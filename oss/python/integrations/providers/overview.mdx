LangChain 提供了一个庞大的生态系统，拥有 1000 多个集成，涵盖聊天与嵌入模型、工具与工具包、文档加载器、向量存储等。

**提供商**是托管 AI 模型并通过 API 公开它们的公司或平台（例如 OpenAI、Anthropic、Google）。许多提供商都有一个专门的 `langchain-<provider>` 包，该包实现了 LangChain 的一个或多个标准接口——聊天模型、嵌入模型、向量存储等——让您无论底层提供商如何都能获得一致的 API。安装包、选择模型名称，即可在不更改代码的情况下更换提供商。
