>[Read the Docs](https://readthedocs.org/) 是一个开源的免费软件文档托管平台，用于托管使用 `Sphinx` 文档生成器编写的文档。

本文档介绍如何从 `Read-The-Docs` 构建生成的 HTML 中加载内容。

有关实际应用示例，请参阅[此处](https://github.com/langchain-ai/chat-langchain)。

本示例假设 HTML 文件已被抓取到某个文件夹中。可以通过取消注释并运行以下命令来完成抓取：

```python
pip install -qU  beautifulsoup4
```

```python
#!wget -r -A.html -P rtdocs https://python.langchain.com/en/latest/
```

```python
from langchain_community.document_loaders import ReadTheDocsLoader
```

```python
loader = ReadTheDocsLoader("rtdocs")
```

```python
docs = loader.load()
```
