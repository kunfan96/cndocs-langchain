[Nimble](https://www.nimbleway.com/) 通过搜索 API 提供实时网页数据访问，该 API 会浏览实时网络，而非依赖预构建的索引。与传统搜索 API 不同，Nimble 使用无头浏览器实时导航网站——处理 JavaScript 渲染、动态内容、分页以及复杂的多步骤导航流程。

**关键技术能力：**

- **实时网页浏览**：直接访问当前网页内容，而非缓存索引
- **JavaScript 渲染**：支持现代单页应用（SPA）、懒加载和客户端渲染
- **灵活模式**：快速模式用于获取 SERP 数据，深度模式用于完整内容提取
- **AI 摘要**：在原始搜索结果旁提供可选的 LLM 生成答案
- **生产就绪**：专为大规模扩展设计，支持异步操作并提供全面的错误处理

如需更多信息，请访问 [nimbleway.com](https://www.nimbleway.com/) 或查阅 [Nimble 官方文档](https://docs.nimbleway.com/)。

## 安装与配置

Nimble 集成位于其独立的合作伙伴包中。您可以使用以下命令进行安装：

<CodeGroup>
```bash pip
pip install -U langchain-nimble
```
```bash uv
uv add langchain-nimble
```
</CodeGroup>

要使用该包，您需要将 `NIMBLE_API_KEY` 环境变量设置为您在 [Nimble](https://www.nimbleway.com/) 注册时获取的 API 密钥。
