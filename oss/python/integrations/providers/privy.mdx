[Privy](https://privy.io) 是专为 AI Agent 构建的强大钱包基础设施，为规模化设计。

为 Agent 提供工具以：

- 自动创建和管理钱包
- 使用各种数字资产（包括稳定币）进行支付
- 签名消息和交易
- 查询钱包余额和地址

### 工作原理
Privy 提供去除区块链交互复杂性的钱包基础设施：

1. 为您的 Agent 提供强大的模块化钱包
2. 使用交易策略保护您的 Agent 资产
3. 快速进行支付
