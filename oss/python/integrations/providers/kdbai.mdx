>[KDB.AI](https://kdb.ai) 是一款强大的基于知识的向量数据库和搜索引擎，允许您利用实时数据构建可扩展、可靠的 AI 应用，提供高级搜索、推荐和个性化功能。


## 安装与设置

安装 Python SDK：

<CodeGroup>
```bash pip
pip install kdbai-client
```

```bash uv
uv add kdbai-client
```
</CodeGroup>


## 向量存储

KDB.AI 索引有一个封装器，可将其作为向量存储使用，支持语义搜索或示例选择。

```python
from langchain_community.vectorstores import KDBAI
```
