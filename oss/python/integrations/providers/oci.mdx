与 [Oracle 云基础设施](https://www.oracle.com/artificial-intelligence/) 相关的 `LangChain` 集成。

## OCI 生成式 AI
> Oracle 云基础设施（OCI）[生成式 AI](https://docs.oracle.com/en-us/iaas/Content/generative-ai/home.htm) 是一项完全托管的服务，提供一套覆盖广泛用例的最先进、可定制的大型语言模型（LLMs），并可通过单一 API 访问。
> 使用 OCI 生成式 AI 服务，您可以访问即用型预训练模型，或根据自己的数据在专用 AI 集群上创建和托管微调后的自定义模型。

要使用此功能，您应该安装最新的 `oci` Python SDK 和 langchain_community 包。

<CodeGroup>
```bash pip
pip install -U oci langchain-community
```

```bash uv
uv add oci langchain-community
```
</CodeGroup>

参见 [聊天](/oss/python/integrations/llms/oci_generative_ai)、[补全](/oss/python/integrations/chat/oci_generative_ai) 和 [嵌入](/oss/python/integrations/text_embedding/oci_generative_ai) 使用示例。

```python
from langchain_community.chat_models import ChatOCIGenAI

from langchain_community.llms import OCIGenAI

from langchain_community.embeddings import OCIGenAIEmbeddings
```

## OCI 数据科学模型部署端点

> [OCI 数据科学](https://docs.oracle.com/en-us/iaas/data-science/using/home.htm) 是一个
> 面向数据科学团队的完全托管的无服务器平台。使用 OCI 数据科学
> 平台，您可以构建、训练和管理机器学习模型，然后使用
> [OCI 数据科学模型部署服务](https://docs.oracle.com/en-us/iaas/data-science/using/model-dep-about.htm)
> 将其部署为 OCI 模型部署端点。

要使用此功能，您应该安装最新的 `oracle-ads` Python SDK。

<CodeGroup>
```bash pip
pip install -U oracle-ads
```

```bash uv
uv add oracle-ads
```
</CodeGroup>

参见 [聊天](/oss/python/integrations/chat/oci_data_science) 和 [补全](/oss/python/integrations/llms/oci_model_deployment_endpoint) 使用示例。


```python
from langchain_community.chat_models import ChatOCIModelDeployment

from langchain_community.llms import OCIModelDeploymentLLM
```
