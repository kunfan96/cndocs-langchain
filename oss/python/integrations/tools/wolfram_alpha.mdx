本 notebook 介绍如何使用 Wolfram Alpha 组件。

首先，您需要注册 Wolfram Alpha 开发者账户并获取 APP ID：

1. 前往 Wolfram Alpha，[在此](https://developer.wolframalpha.com/)注册开发者账户
2. 创建一个应用并获取 APP ID
3. pip install wolframalpha

然后需要设置以下环境变量：

1. 将 APP ID 保存到 WOLFRAM_ALPHA_APPID 环境变量中

```python
pip install wolframalpha
```

```python
import os

os.environ["WOLFRAM_ALPHA_APPID"] = ""
```

```python
from langchain_community.utilities.wolfram_alpha import WolframAlphaAPIWrapper
```

```python
wolfram = WolframAlphaAPIWrapper()
```
