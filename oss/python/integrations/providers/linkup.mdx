> [Linkup](https://www.linkup.so/) 提供了一项 API，用于将 LLM 连接到互联网以及 Linkup 高级合作伙伴来源。

## 安装与设置

要使用 Linkup 提供商，您首先需要有效的 API 密钥，您可以通过[在 Linkup 网站注册](https://app.linkup.so/sign-up)来获取。
您还需要 `langchain-linkup` 包，可以使用 pip 进行安装：

<CodeGroup>
```bash pip
pip install langchain-linkup
```

```bash uv
uv add langchain-linkup
```
</CodeGroup>

## 检索器

请参阅[使用示例](/oss/python/integrations/retrievers/linkup_search)。

```python
from langchain_linkup import LinkupSearchRetriever

retriever = LinkupSearchRetriever(
    depth="deep",  # “standard”或“deep”
    linkup_api_key=None,  # API 密钥可在此处传入，或设置为 LINKUP_API_KEY 环境变量
)
```

## 工具

请参阅[使用示例](/oss/python/integrations/tools/linkup_search)。

```python
from langchain_linkup import LinkupSearchTool

tool = LinkupSearchTool(
    depth="deep",  # “standard”或“deep”
    output_type="searchResults",  # “searchResults”、“sourcedAnswer”或“structured”
    linkup_api_key=None,  # API 密钥可在此处传入，或设置为 LINKUP_API_KEY 环境变量
)
```
