>[Apache Doris](https://doris.apache.org/) 是一个用于实时分析的现代数据仓库，
能够在大规模实时数据上提供闪电般的分析性能。

>通常 `Apache Doris` 被归类为 OLAP，在 [ClickBench（分析型数据库基准测试）](https://benchmark.clickhouse.com/) 中表现优异。
> 由于其超快的向量化执行引擎，它也可用作高速向量数据库。

## 安装与设置

<CodeGroup>
```bash pip
pip install pymysql
```

```bash uv
uv add pymysql
```
</CodeGroup>
