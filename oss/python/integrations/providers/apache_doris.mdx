>[Apache Doris](https://doris.apache.org/) 是一款面向实时分析的现代数据仓库。
它能够在大规模实时数据上提供闪电般的分析速度。

>通常 `Apache Doris` 被归类为 OLAP，它在 [ClickBench —— 分析型数据库基准测试](https://benchmark.clickhouse.com/) 中展现了卓越的性能。
> 由于它拥有超快的向量化执行引擎，因此也可以用作快速的向量数据库。

## 安装与配置

<CodeGroup>
```bash pip
pip install pymysql
```

```bash uv
uv add pymysql
```
</CodeGroup>
