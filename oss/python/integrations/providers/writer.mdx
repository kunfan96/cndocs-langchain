与 WRITER 相关的所有功能


>本页介绍如何在 LangChain 中使用 [WRITER](https://writer.com/) 生态系统。更多信息请参阅 Writer [文档](https://dev.writer.com/home/introduction)。
>[Palmyra](https://writer.com/blog/palmyra/) 是由 `WRITER` 开发的大型语言模型（LLM）。
>
>[Writer API](https://dev.writer.com/api-guides/introduction) 由一组具有不同功能和价格点的 Palmyra 子模型驱动。

## 安装与设置

通过以下方式安装集成包：
<CodeGroup>
```bash pip
pip install langchain-writer
```

```bash uv
uv add langchain-writer
```
</CodeGroup>

获取 WRITER API 密钥并将其设置为环境变量（`WRITER_API_KEY`）

## 聊天模型

```python
from langchain_writer import ChatWriter
```
查看[详情](/oss/python/integrations/chat/writer)。


## PDF 解析器

```python
from langchain_writer.pdf_parser import PDFParser
```

**弃用通知**：PDF 解析工具已被弃用，将于 **2025 年 12 月 22 日**移除。

**迁移路径**：我们计划为聊天完成引入预建 PDF 解析工具，提供类似功能。该工具将与其他预建工具类似地工作。我们将在该替代方案可用时提供更多详情。
</Warning>
