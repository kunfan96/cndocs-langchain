[Moonshot](https://platform.moonshot.cn/) 是一家面向企业和个人提供 LLM 服务的中国初创公司。

本示例介绍如何使用 LangChain 与 Moonshot 进行交互。

```python
from langchain_community.llms.moonshot import Moonshot
```

```python
import os

# 从以下地址生成您的 API 密钥：https://platform.moonshot.cn/console/api-keys
os.environ["MOONSHOT_API_KEY"] = "MOONSHOT_API_KEY"
```

```python
llm = Moonshot()
# 或使用特定模型
# 可用模型：https://platform.moonshot.cn/docs
# llm = Moonshot(model="moonshot-v1-128k")
```

```python
# 向模型发送提示
llm.invoke("What is the difference between panda and bear?")
```
