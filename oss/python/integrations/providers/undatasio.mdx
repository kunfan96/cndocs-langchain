> [UnDatasIO](https://undatas.io) 的 `undatasio` 包从原始源文档（如 PDF）中提取干净的文本。
> 本页介绍如何在 LangChain 中使用 `undatasio` 生态系统。

## 安装与设置

- 使用 `pip install undatasio` 安装 Python SDK，
  并使用 `pip install langchain-undatasio` 来使用 `UnDatasIOLoader` 并远程针对 UnDatasIO API 进行分区处理。
  您需要一个 API 密钥，可以在 [undatas.io](https://undatas.io) 免费生成。

- 不需要本地系统依赖项；所有处理都在云端运行。

## 数据加载器

`UnDatasIO` 的主要用途是通过**文档加载器**。

### UnDatasIOLoader

有关单文件解析和延迟加载的用法示例，请参见[用法示例](/oss/python/integrations/document_loaders/undatasio)。

```python
from langchain_undatasio import UnDatasIOLoader
```
