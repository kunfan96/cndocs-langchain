本页介绍如何在 LangChain 中使用 [SearchApi](https://www.searchapi.io/) Google Search API。SearchApi 是一个实时 SERP API，便于进行 SERP 抓取。

## 设置

- 访问 [https://www.searchapi.io/](https://www.searchapi.io/) 注册免费账户
- 获取 API 密钥并将其设置为环境变量 (`SEARCHAPI_API_KEY`)

## 包装器

### 实用工具

有一个 SearchApiAPIWrapper 实用工具可以包装此 API。要导入此实用工具：

```python
from langchain_community.utilities import SearchApiAPIWrapper
```

您可以将其作为带有搜索工具的代理的一部分使用：

```python
from langchain_community.utilities import SearchApiAPIWrapper
from langchain_openai import OpenAI
from langchain.agents import create_agent, Tool

import os

os.environ["SEARCHAPI_API_KEY"] = ""
os.environ["OPENAI_API_KEY"] = ""

llm = OpenAI(temperature=0)
search = SearchApiAPIWrapper()

tools = [
    Tool(
        name="Intermediate Answer",
        func=search.run,
        description="useful for when you need to ask with search",
    )
]

agent = create_agent(
    model=llm,
    tools=tools,
)

agent.invoke("Who lived longer: Plato, Socrates, or Aristotle?")
```

#### 输出

```

```

### 工具

您也可以轻松地将此包装器作为工具加载（以与代理一起使用）。
您可以通过以下方式实现：

```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["searchapi"])
```

有关工具的更多信息，请参阅 [工具](/oss/python/integrations/tools)。
