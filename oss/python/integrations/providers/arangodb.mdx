>[ArangoDB](https://github.com/arangodb/arangodb) 是一个可扩展的图数据库系统，
> 能够更快速地从关联数据中挖掘价值。它支持原生图、集成搜索引擎和 JSON，通过单一查询语言实现。ArangoDB 支持本地部署、云端部署——随处可用。

## 安装与设置

安装 [ArangoDB Python 驱动](https://github.com/ArangoDB-Community/python-arango) 包：

<CodeGroup>
```bash pip
pip install python-arango
```

```bash uv
uv add python-arango
```
</CodeGroup>

## 图问答链

将您的 `ArangoDB` 数据库与聊天模型连接，从数据中获取洞察。

查看 notebook 示例[在这里](/oss/python/integrations/graphs/arangodb)。

```python
from arango import ArangoClient

from langchain_community.graphs import ArangoGraph
from langchain_classic.chains import ArangoGraphQAChain
```
