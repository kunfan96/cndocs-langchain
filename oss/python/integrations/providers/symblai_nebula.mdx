本页介绍如何在 LangChain 中使用 [Symbl.ai](https://symbl.ai/) 的 LLM [Nebula](https://symbl.ai/nebula) 生态系统。
内容分为两部分：安装与设置，以及对特定 Nebula 封装器的参考说明。

## 安装与设置

- 获取 [Nebula API 密钥](https://info.symbl.ai/Nebula_Private_Beta.html)，并将其设置为环境变量 `NEBULA_API_KEY`
- 更多详情请参阅 [Nebula 文档](https://docs.symbl.ai/docs/nebula-llm)。

### LLM

存在一个 Nebula LLM 封装器，访问方式：
```python
from langchain_community.llms import Nebula
llm = Nebula()
```
