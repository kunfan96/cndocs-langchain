>[Meta Platforms, Inc.](https://www.facebook.com/)（以 `Meta` 名义开展业务，前身为
> `Facebook, Inc.` 和 `TheFacebook, Inc.`）是一家美国跨国科技
> 企业集团。该公司拥有并运营 `Facebook`、`Instagram`、`Threads`
> 以及 `WhatsApp` 等产品与服务。

## 嵌入模型

### LASER

>[LASER](https://github.com/facebookresearch/LASER) 是由
> `Meta AI Research` 团队开发的 Python 库，用于
> 为
> [截至 2024 年 2 月 25 日的 147 多种语言](https://github.com/facebookresearch/flores/blob/main/flores200/README.md#languages-in-flores-200)
> 创建多语言句子嵌入。

<CodeGroup>
```bash pip
pip install laser_encoders
```

```bash uv
uv add laser_encoders
```
</CodeGroup>

请参阅[使用示例](/oss/python/integrations/embeddings/laser)。

```python
from langchain_community.embeddings.laser import LaserEmbeddings
```

## 向量存储

### Facebook FAISS

>[Facebook AI 相似度搜索 (Faiss)](https://engineering.fb.com/2017/03/29/data-infrastructure/faiss-a-library-for-efficient-similarity-search/)
> 是一个用于高效相似度搜索和密集向量聚类的库。它包含可在任意大小的向量集合中进行搜索的算法，甚至支持可能无法完全装入内存的向量集。它还包含用于评估和参数调优的辅助
> 代码。

[Faiss 文档](https://faiss.ai/)。

我们需要安装 `faiss` Python 包。

<CodeGroup>
```bash pip
pip install faiss-gpu # 适用于支持 CUDA 7.5+ 的 GPU。
```

```bash uv
uv add faiss-gpu # 适用于支持 CUDA 7.5+ 的 GPU。
```
</CodeGroup>

或

<CodeGroup>
```bash pip
pip install faiss-cpu # 用于 CPU 安装
```

```bash uv
uv add faiss-cpu # 用于 CPU 安装
```
</CodeGroup>

请参阅[使用示例](/oss/python/integrations/vectorstores/faiss)。

```python
from langchain_community.vectorstores import FAISS
```
