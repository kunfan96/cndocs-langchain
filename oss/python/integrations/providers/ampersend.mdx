[Ampersend](https://ampersend.ai) 使 LangChain Agent 能够为远程 AI Agent 服务付费并使用这些服务。

为 Agent 提供以下工具：

- 自动为远程 Agent 服务付款
- 通过 A2A 协议与远程 Agent 交互

### 工作原理

Ampersend 通过 x402 协议提供透明的支付处理：

1. 将您的 Agent 连接到带有消费控制的钱包
2. 通过 A2A 协议调用远程 Agent
3. 需要时自动处理付款

## 安装与设置

请查看[工具文档](/oss/python/integrations/tools/ampersend)，了解如何设置和安装 Ampersend。
