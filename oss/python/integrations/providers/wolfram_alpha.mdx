>[WolframAlpha](https://en.wikipedia.org/wiki/WolframAlpha) 是由 `Wolfram Research` 开发的答案引擎。
> 它通过从外部来源数据计算答案来回答事实性查询。

本页介绍如何在 LangChain 中使用 `Wolfram Alpha API`。

## 安装与设置
- 使用以下命令安装依赖：
<CodeGroup>
```bash pip
pip install wolframalpha
```

```bash uv
uv add wolframalpha
```
</CodeGroup>

- 注册一个 [Wolfram Alpha 开发者账户](https://developer.wolframalpha.com/)
- 创建一个应用并获取您的 `APP ID`
- 将您的 APP ID 设置为环境变量 `WOLFRAM_ALPHA_APPID`

## 包装器

### 实用工具

存在一个 WolframAlphaAPIWrapper 实用工具，用于包装此 API。要导入此实用工具：

```python
from langchain_community.utilities.wolfram_alpha import WolframAlphaAPIWrapper
```

有关此包装器的更详细说明，请参见[此笔记本](/oss/python/integrations/tools/wolfram_alpha)。

### 工具

您还可以轻松地将此包装器加载为工具（以与代理一起使用）。
您可以通过以下方式实现：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["wolfram-alpha"])
```

有关工具的更多信息，请参见[工具](/oss/python/integrations/tools)。
