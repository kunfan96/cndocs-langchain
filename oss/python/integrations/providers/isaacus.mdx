[Isaacus](https://isaacus.com/) 是一家基础法律人工智能研究公司，致力于构建面向法律科技生态系统的 AI 模型、应用程序和工具。

Isaacus 的产品线包括 [Kanon 2 Embedder](https://isaacus.com/blog/introducing-kanon-2-embedder)（根据 [大规模法律嵌入基准测试](https://isaacus.com/blog/introducing-mleb) 衡量，为全球最佳的法律嵌入模型），以及 [法律零样本分类](https://docs.isaacus.com/models/introduction#universal-classification) 和 [法律抽取式问答模型](https://docs.isaacus.com/models/introduction#answer-extraction)。

Isaacus 为 LangChain 的嵌入接口提供一流支持，可通过 [`langchain-isaacus`](https://pypi.org/project/langchain-isaacus/) 集成包访问。

## 环境配置
要开始将 Isaacus 模型与 LangChain 配合使用，请访问 [Isaacus 平台](https://platform.isaacus.com/accounts/signup/) 并创建一个新账户。

注册完成后，[添加支付方式](https://platform.isaacus.com/billing/)（从而领取您的 [免费额度](https://docs.isaacus.com/pricing/credits)）并 [生成 API 密钥](https://platform.isaacus.com/users/api-keys/)。

接下来，安装 [`langchain-isaacus`](https://pypi.org/project/langchain-isaacus/) 集成包：
<CodeGroup>
```bash pip
pip install langchain-isaacus
```

```bash uv
uv add langchain-isaacus
```
</CodeGroup>

随后，请将 `ISAACUS_API_KEY` 环境变量设置为您的 Isaacus API 密钥。
<CodeGroup>
```bash bash
export ISAACUS_API_KEY="your_api_key_here"
```
```powershell powershell
$env:ISAACUS_API_KEY="your_api_key_here"
```
</CodeGroup>

## 嵌入
下面的代码片段演示了如何使用 Isaacus 的 Kanon 2 Embedder 模型，通过 LangChain 评估法律查询与法律文档之间的语义相似度。有关如何使用 Isaacus LangChain 集成生成嵌入的详细指南，请参阅 [Isaacus 嵌入指南](/oss/python/integrations/embeddings/isaacus)。

```python
import numpy as np # 注意：您可能需要运行 `pip install numpy`。

from langchain_isaacus import IsaacusEmbeddings

# 为 Kanon 2 Embedder 创建 Isaacus API 客户端。
client = IsaacusEmbeddings(
    "kanon-2-embedder",
    # dimensions=1792, # 您也可以根据需要指定较低的维度。
)

# 嵌入一个示例文档。
document_embedding = client.embed_documents(texts=["These are GitHub's billing policies."])[0]

# 嵌入我们的搜索查询。
relevant_query_embedding = client.embed_query(text="What are GitHub's billing policies?")
irrelevant_query_embedding = client.embed_query(text="What are Microsoft's billing policies?")

# 计算查询与文档之间的相似度。
relevant_similarity = np.dot(relevant_query_embedding, document_embedding)
irrelevant_similarity = np.dot(irrelevant_query_embedding, document_embedding)

# 记录结果。
print(f"Similarity of relevant query to the document: {relevant_similarity * 100:.2f}")
print(f"Similarity of irrelevant query to the document: {irrelevant_similarity * 100:.2f}")
```
