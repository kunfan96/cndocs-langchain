>[Chroma](https://docs.trychroma.com/getting-started) 是一款用于构建基于嵌入的 AI 应用的数据库。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-chroma
```

```bash uv
uv add langchain-chroma
```
</CodeGroup>


## VectorStore

Chroma 向量数据库有一个封装，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

```python
from langchain_chroma import Chroma
```

有关 Chroma 封装的更详细介绍，请参阅[此笔记本](/oss/python/integrations/vectorstores/chroma)
