[ScrapingAnt](https://scrapingant.com/) 是一个具有无头浏览器功能、代理和反机器人绕过的网络抓取 API。它允许将网页数据提取为可访问的 LLM Markdown。

此特定集成仅使用 Markdown 提取功能，但如果您需要 ScrapingAnt 提供的更多功能（但尚未在此集成中实现），请随时[联系我们](mailto:support@scrapingant.com)。

### 集成详情

| 类                                                                                                                                                    | 包                                                                                        | 本地 | 可序列化 | JS 支持 |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------|:-----:|:------------:|:----------:|
| [`ScrapingAntLoader`](https://reference.langchain.com/python/langchain-community/document_loaders/scrapingant/ScrapingAntLoader) | [`langchain_community`](https://reference.langchain.com/python/langchain-community) |   ❌   |      ❌       |     ❌      |

### 加载器功能

|      来源       | 文档延迟加载 | 异步支持 |
|:-----------------:|:---------------------:|:-------------:|
| `ScrapingAntLoader` |           ✅           |       ❌       |

## 设置

使用 pip 安装 ScrapingAnt Python SDK 和所需的 LangChain 包：

```shell
pip install scrapingant-client langchain langchain-community
```

## 实例化

```python
from langchain_community.document_loaders import ScrapingAntLoader

scrapingant_loader = ScrapingAntLoader(
    ["https://scrapingant.com/", "https://example.com/"],  # 要抓取的 URL 列表
    api_key="<YOUR_SCRAPINGANT_TOKEN>",  # 从 https://scrapingant.com/ 获取您的 API 密钥
    continue_on_failure=True,  # 忽略无法处理的网页并记录其异常
)
```

ScrapingAntLoader 还允许提供一个 dict - scraping config 用于自定义抓取请求。由于它基于 [ScrapingAnt Python SDK](https://github.com/ScrapingAnt/scrapingant-client-python)，您可以将任何[通用参数](https://github.com/ScrapingAnt/scrapingant-client-python)传递给 `scrape_config` 参数。

```python
from langchain_community.document_loaders import ScrapingAntLoader

scrapingant_config = {
    "browser": True,  # 启用云浏览器的浏览器渲染
    "proxy_type": "datacenter",  # 选择代理类型（datacenter 或 residential）
    "proxy_country": "us",  # 选择代理位置
}

scrapingant_additional_config_loader = ScrapingAntLoader(
    ["https://scrapingant.com/"],
    api_key="<YOUR_SCRAPINGANT_TOKEN>",  # 从 https://scrapingant.com/ 获取您的 API 密钥
    continue_on_failure=True,  # 忽略无法处理的网页并记录其异常
    scrape_config=scrapingant_config,  # 传递 scrape_config 对象
)
```

## 加载

使用 `load` 方法抓取网页并获取提取的 Markdown 内容。

```python
# 从 URL 加载文档作为 Markdown
documents = scrapingant_loader.load()

print(documents)
```

## 延迟加载

使用 'lazy_load' 方法延迟抓取网页并获取提取的 Markdown 内容。

```python
# 延迟从 URL 加载文档作为 Markdown
lazy_documents = scrapingant_loader.lazy_load()

for document in lazy_documents:
    print(document)
```
