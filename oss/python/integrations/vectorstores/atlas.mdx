>[Atlas](https://docs.nomic.ai/index.html) 是 Nomic 开发的一个平台，用于与小型和互联网规模的非结构化数据集进行交互。它使任何人都能在浏览器中可视化、搜索和共享海量数据集。

您需要安装 `langchain-community`，使用 `pip install -qU langchain-community` 来使用此集成。

此笔记本向您展示如何使用与 `AtlasDB` 向量存储相关的功能。

### 加载包

```python
import time

from langchain_community.document_loaders import TextLoader
from langchain_community.vectorstores import AtlasDB
from langchain_text_splitters import SpacyTextSplitter
```

```python
ATLAS_TEST_API_KEY = "x"
```

### 准备数据

```python
loader = TextLoader("../../how_to/state_of_the_union.txt")
documents = loader.load()
text_splitter = SpacyTextSplitter(separator="|")
texts = []
for doc in text_splitter.split_documents(documents):
    texts.extend(doc.page_content.split("|"))

texts = [e.strip() for e in texts]
```

### 使用 nomic 的 atlas 映射数据

```python
db = AtlasDB.from_texts(
    texts=texts,
    name="test_index_" + str(time.time()),  # 向量存储的唯一名称
    description="test_index",  # 向量存储的描述
    api_key=ATLAS_TEST_API_KEY,
    index_kwargs={"build_topic_model": True},
)
```

```python
db.project.wait_for_project_lock()
```

```python
db.project
```
