> [Zvec](https://zvec.org/en/) 是一个开源的进程内向量数据库——轻量、极速，专为直接嵌入应用而设计。基于 Proxima（阿里巴巴经过实战考验的向量搜索引擎）构建，以最小的配置提供生产级、低延迟、可扩展的相似性搜索。

本文档演示如何在 LangChain 生态系统中利用 Zvec。具体展示如何安装 Zvec，以及如何将其用作 LangChain 中的 VectorStore 插件。

## 安装与设置

安装 Python SDK：

<CodeGroup>
```bash pip
pip install zvec
```

```bash uv
uv add zvec
```
</CodeGroup>


## 嵌入模型

```python
from langchain_community.embeddings import DashScopeEmbeddings
```

查看[使用示例](/oss/python/integrations/vectorstores/zvec)。

## 向量存储

Zvec 集合被封装为 LangChain 中熟悉的 VectorStore，可在各种场景中直接使用，例如语义搜索或示例选择。

你可以通过以下方式导入向量存储：

```python
from langchain_community.vectorstores import Zvec
```

有关 Zvec 封装的详细说明，请参阅[此笔记本](/oss/python/integrations/vectorstores/zvec)
