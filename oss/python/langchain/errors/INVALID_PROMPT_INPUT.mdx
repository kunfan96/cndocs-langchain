title: INVALID_PROMPT_INPUT
---

{/* TODO: fix link when porting page */}
当 [prompt template](https://github.com/langchain-ai/langchain/blob/v0.3/docs/docs/concepts/prompt_templates.mdx) 收到缺失或无效的输入变量时发生。




## 故障排除

要解决此错误，你可以：

1. 检查你的 prompt template 是否正确。使用 f-string 格式时，确保正确转义大括号：
   * 在 f-string 中使用 `{{` 表示单个大括号
   * 在 f-string 中使用 `{{{{` 表示双大括号
2. 使用 `MessagesPlaceholder` 组件时，确认你传递的是消息数组或类似消息的对象。如果使用简写元组，请用大括号包装变量名，如 `["placeholder", "{messages}"]`
3. 通过使用 [LangSmith](/langsmith/home) 或日志记录检查 prompt template 的实际输入来调试，以验证它们是否符合预期
4. 如果从 LangChain [Prompt Hub](https://smith.langchain.com/prompts) 获取 prompts，请使用示例输入隔离并测试 prompt，以确保其按预期运行
