[Modal](https://modal.com) 提供支持 GPU 的无服务器容器基础设施。有关注册、身份验证和平台详情，请参阅 [Modal 文档](https://modal.com/docs)。

## 安装

<CodeGroup>
```bash pip
pip install langchain-modal
```

```bash uv
uv add langchain-modal
```
</CodeGroup>

## 创建沙盒后端

在 Python 中，您使用提供商 SDK 创建沙盒，然后使用 [deepagents 后端](/oss/python/deepagents/backends) 封装它。

```python
import modal

from langchain_modal import ModalSandbox

app = modal.App.lookup("your-app")
modal_sandbox = modal.Sandbox.create(app=app)
backend = ModalSandbox(sandbox=modal_sandbox)

result = backend.execute("echo hello")
print(result.output)
```

## 与 Deep Agents 一起使用

```python
import modal
from langchain_anthropic import ChatAnthropic

from deepagents import create_deep_agent
from langchain_modal import ModalSandbox

app = modal.App.lookup("your-app")
modal_sandbox = modal.Sandbox.create(app=app)
backend = ModalSandbox(sandbox=modal_sandbox)

agent = create_deep_agent(
    model=ChatAnthropic(model="claude-sonnet-4-20250514"),
    system_prompt="You are a coding assistant with sandbox access.",
    backend=backend,
)

result = agent.invoke(
    {"messages": [{"role": "user", "content": "Install numpy and calculate pi"}]}
)
```

## 清理

您负责通过 Modal 管理沙盒生命周期。
完成后，请终止沙盒。
