[`deepagents`](https://www.npmjs.com/package/deepagents) 是一个独立的库，用于构建能够处理复杂、多步骤任务的 agent。它基于 LangGraph 构建，受到 Claude Code、Deep Research 和 Manus 等应用的启发，deep agents 具备规划能力、用于上下文管理的文件系统，以及生成子 agent 的能力。


## 何时使用 deep agents

当你需要能够执行以下操作的 agent 时，请使用 deep agents：
- **处理复杂的多步骤任务**，需要规划和分解
- 通过文件系统工具**管理大量上下文**
- **委托工作**给专门的子 agent 以实现上下文隔离
- 跨对话和线程**持久化记忆**



对于较简单的用例，考虑使用 LangChain 的 [`createAgent`](/oss/javascript/langchain/agents) 或构建自定义的 [LangGraph](/oss/javascript/langgraph/overview) 工作流。


## 核心能力

<Card title="规划和任务分解" icon="timeline">
    Deep agents 包含一个内置的 `write_todos` 工具，使 agent 能够将复杂任务分解为离散步骤，跟踪进度，并在新信息出现时调整计划。
</Card>
<Card title="上下文管理" icon="scissors">
    文件系统工具（`ls`、`read_file`、`write_file`、`edit_file`）允许 agent 将大量上下文卸载到内存中，防止上下文窗口溢出，并能够处理可变长度的工具结果。
</Card>
<Card title="子 agent 生成" icon="people-group">
    内置的 `task` 工具使 agent 能够生成专门的子 agent 以实现上下文隔离。这使主 agent 的上下文保持清晰，同时仍能深入处理特定子任务。
</Card>
<Card title="长期记忆" icon="database">
    使用 LangGraph 的 Store 扩展 agent 的跨线程持久记忆。Agent 可以保存和检索之前对话中的信息。
</Card>

## 与 LangChain 生态系统的关系

Deep agents 构建在以下基础之上：
- [LangGraph](/oss/javascript/langgraph/overview) - 提供底层的图执行和状态管理
- [LangChain](/oss/javascript/langchain/overview) - 工具和模型集成与 deep agents 无缝协作
- [LangSmith](/langsmith/home) - 可观测性、评估和部署

Deep agents 应用程序可以通过 [LangSmith Deployment](/langsmith/deployments) 部署，并使用 [LangSmith Observability](/langsmith/observability) 监控。
