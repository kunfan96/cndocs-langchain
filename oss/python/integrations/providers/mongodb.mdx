>[MongoDB](https://www.mongodb.com/) 是一款非关系型、面向文档的
> 数据库，支持具有动态模式的类 JSON 文档。

**注意：**
- 有关其他 `MongoDB` 集成，请参阅 [MongoDB Atlas 页面](/oss/python/integrations/providers/mongodb_atlas)。

## 安装与配置

安装 Python 包：

<CodeGroup>
```bash pip
pip install langchain-mongodb
```

```bash uv
uv add langchain-mongodb
```
</CodeGroup>
