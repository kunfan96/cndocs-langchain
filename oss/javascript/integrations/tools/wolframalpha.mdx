WolframAlpha 工具将您的代理和链连接到 WolframAlpha 的最先进计算智能引擎。

## 设置

您需要从 [WolframAlpha 门户](https://developer.wolframalpha.com/) 创建一个应用并获取 `appid`。

## 用法

```typescript
import { WolframAlphaTool } from "@langchain/community/tools/wolframalpha";

const tool = new WolframAlphaTool({
  appid: "YOUR_APP_ID",
});

const res = await tool.invoke("What is 2 * 2?");

console.log(res);
```
