>[Read the Docs](https://readthedocs.org/) 是一个开源的免费软件文档托管平台。它使用 `Sphinx` 文档生成器生成文档。

本笔记本介绍如何从作为 `Read-The-Docs` 构建一部分生成的 HTML 中加载内容。

有关此示例，请参阅 [chat-langchain 仓库](https://github.com/langchain-ai/chat-langchain)。

这假设 HTML 已被爬取到一个文件夹中。可以通过取消注释并运行以下命令来完成此操作

```python
pip install -qU  beautifulsoup4
```

```python
#!wget -r -A.html -P rtdocs https://python.langchain.com/en/latest/
```

```python
from langchain_community.document_loaders import ReadTheDocsLoader
```

```python
loader = ReadTheDocsLoader("rtdocs")
```

```python
docs = loader.load()
```
