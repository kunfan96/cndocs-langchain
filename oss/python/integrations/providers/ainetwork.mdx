>[AI Network](https://www.ainetwork.ai/build-on-ain) 是一个专为容纳大规模 AI 模型而设计的 Layer 1 区块链，利用由
> [$AIN token](https://www.ainetwork.ai/token) 驱动的分布式 GPU 网络，为 AI 驱动的 `NFTs`（`AINFTs`）增添价值。


## 安装与设置

您需要安装 `ain-py` Python 包。

<CodeGroup>
```bash pip
pip install ain-py
```

```bash uv
uv add ain-py
```
</CodeGroup>
您需要将环境变量 `AIN_BLOCKCHAIN_ACCOUNT_PRIVATE_KEY` 设置为您的 AIN 区块链账户私钥。
## 工具包

请参阅[使用示例](/oss/python/integrations/tools/ainetwork)。

```python
from langchain_community.agent_toolkits.ainetwork.toolkit import AINetworkToolkit
```
