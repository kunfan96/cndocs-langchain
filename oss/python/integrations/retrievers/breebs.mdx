>[BREEBS](https://www.breebs.com/) 是一个开放的协作知识平台。任何人都可以基于存储在 Google Drive 文件夹中的 PDF 创建 Breeb（知识胶囊）。Breeb 可被任何 LLM/聊天机器人使用，以提升其专业知识、减少幻觉并提供访问来源的能力。在幕后，Breebs 实现了多种检索增强生成（RAG）模型，以在每次迭代中无缝提供有用的上下文。

## 可用 Breebs 列表

要获取 Breebs 的完整列表，包括其密钥（breeb_key）和描述：
[breebs.promptbreeders.com/web/listbreebs](https://breebs.promptbreeders.com/web/listbreebs)。
已有数十个由社区创建的 Breebs 可供免费使用。它们涵盖广泛的专业知识，从有机化学到神话学，以及关于诱惑和去中心化金融的技巧。

## 创建新的 Breeb

要生成新的 Breeb，只需将 PDF 文件整理到公共共享的 Google Drive 文件夹中，然后在 [BREEBS 网站](https://www.breebs.com/)上点击"Create Breeb"按钮启动创建过程。目前最多可包含 120 个文件，总字符数限制为 1500 万。

## 检索器示例

```python
from langchain_community.retrievers import BreebsRetriever
```

```python
breeb_key = "Parivoyage"
retriever = BreebsRetriever(breeb_key)
documents = retriever.invoke(
    "What are some unique, lesser-known spots to explore in Paris?"
)
print(documents)
```
