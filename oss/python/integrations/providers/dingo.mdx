> [DingoDB](https://github.com/dingodb) 是一款分布式多模态向量
> 数据库。它融合了数据湖与向量数据库的特性，
> 支持存储任意类型的数据（键值对、PDF、音频、
> 视频等），且不受数据规模限制。借助 DingoDB，您可以构建
> 属于自己的“向量海洋”（继数据仓库和数据湖之后的下一代数据架构）。这使得您能够
> 通过单一的 SQL 语句，以极低的延迟实时分析结构化与非结构化数据。

## 安装与配置

安装 Python SDK

<CodeGroup>
```bash pip
pip install dingodb
```

```bash uv
uv add dingodb
```
</CodeGroup>

## VectorStore

存在一个针对 DingoDB 索引的封装器，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

要导入此向量存储：

```python
from langchain_community.vectorstores import Dingo
```

有关 DingoDB 封装器的更详细教程，请参阅[此笔记本](/oss/python/integrations/vectorstores/dingo)
