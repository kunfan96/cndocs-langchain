[Gel](https://www.geldata.com/) 是一个基于 PostgreSQL 构建的强大数据平台。

- 采用对象和图形的思维模式，而非表和 JOIN 操作。
- 使用高级 Python SDK、内置 GUI、迁移引擎、认证（Auth）与 AI 层等更多功能。
- 支持在本地、远程或[完全托管的云环境](https://www.geldata.com/cloud)中运行。

## 安装

<CodeGroup>
```bash pip
pip install langchain-gel
```

```bash uv
uv add langchain-gel
```
</CodeGroup>

## 配置

1. 运行 `gel project init`
2. 编辑模式（schema）。要使用 LangChain 向量存储，你需要以下类型：

```gel
using extension pgvector;

module default {
    scalar type EmbeddingVector extending ext::pgvector::vector<1536>;

    type Record {
        required collection: str;
        text: str;
        embedding: EmbeddingVector;
        external_id: str {
            constraint exclusive;
        };
        metadata: json;

        index ext::pgvector::hnsw_cosine(m := 16, ef_construction := 128)
            on (.embedding)
    }
}
```

> 注意：此为最小化配置。你可以随意添加任意数量的类型、属性和链接！
> 阅读[文档](https://docs.geldata.com/learn/schema)以了解更多关于利用 Gel 模式的信息。

3. 运行迁移命令：`gel migration create && gel migrate`。

## 使用方法

```python
from langchain_gel import GelVectorStore

vector_store = GelVectorStore(
    embeddings=embeddings,
)
```

请参阅[GEL 向量存储的完整使用示例](/oss/python/integrations/vectorstores/gel)。
