[ScraperAPI](https://www.scraperapi.com/) 通过其网络爬虫 API，无需担心代理、浏览器或验证码处理，即可从任何公共网站收集数据。[langchain-scraperapi](https://github.com/scraperapi/langchain-scraperapi) 封装了此服务，使 AI 代理能够轻松浏览网络并从中抓取数据。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-scraperapi
```

```bash uv
uv add langchain-scraperapi
```
</CodeGroup>

从 [ScraperAPI](https://www.scraperapi.com/) 获取 API 密钥，并将其设置为环境变量：

```python
import os

os.environ["SCRAPERAPI_API_KEY"] = "your-api-key"
```

## 工具

该包提供 3 个工具，分别用于抓取任何网站、获取结构化的 Google 搜索结果和获取结构化的 Amazon 搜索结果。

有关每个工具的详细文档：
- [ScraperAPITool](/oss/python/integrations/tools/scraperapi) - 浏览并抓取任何网站
- ScraperAPIGoogleSearchTool - 获取结构化的 Google 搜索 SERP 数据
- ScraperAPIAmazonSearchTool - 获取结构化的 Amazon 产品搜索数据

有关更详细的演练，请参阅[官方仓库](https://github.com/scraperapi/langchain-scraperapi)。
