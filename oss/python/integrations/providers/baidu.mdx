本页涵盖所有 LangChain 与百度产品的集成，例如 PaddleOCR。

## PaddleOCR

`langchain-paddleocr` 包实现了 PaddleOCR 功能的集成。[PaddleOCR](https://www.paddleocr.com) 是百度开发的一款强大且轻量的 OCR 工具包，可将图像和 PDF 与 LLM 连接起来。它支持超过 100 种语言，并将文档内容转换为结构化、可供 AI 使用的数据。

## 百度智能云

### 安装与配置

在 [百度智能云文心千帆平台](https://cloud.baidu.com/product/wenxinworkshop) 注册并获取 `Qianfan` 的 `AK` 和 `SK` 密钥。

### 大语言模型

#### 百度千帆

查看 [使用示例](/oss/python/integrations/llms/baidu_qianfan_endpoint)。

```python
from langchain_community.llms import QianfanLLMEndpoint
```

### 聊天模型

#### 千帆聊天端点

查看 [使用示例](/oss/python/integrations/chat/baidu_qianfan_endpoint)。
查看另一个 [使用示例](/oss/python/integrations/chat/ernie)。

```python
from langchain_community.chat_models import QianfanChatEndpoint
```

### 嵌入模型

#### 百度千帆

查看 [使用示例](/oss/python/integrations/embeddings/baidu_qianfan_endpoint)。
查看另一个 [使用示例](/oss/python/integrations/embeddings/ernie)。

```python
from langchain_community.embeddings import QianfanEmbeddingsEndpoint
```

### 文档加载器

#### 百度 BOS 目录加载器

```python
from langchain_community.document_loaders.baiducloud_bos_directory import BaiduBOSDirectoryLoader
```

#### 百度 BOS 文件加载器

```python
from langchain_community.document_loaders.baiducloud_bos_file import BaiduBOSFileLoader
```

### 向量存储

#### 百度智能云 ElasticSearch VectorSearch

查看 [使用示例](/oss/python/integrations/vectorstores/baiducloud_vector_search)。

```python
from langchain_community.vectorstores import BESVectorStore
```

#### 百度 VectorDB

查看 [使用示例](/oss/python/integrations/vectorstores/baiduvectordb)。

```python
from langchain_community.vectorstores import BaiduVectorDB
```
