>[Baichuan Inc.](https://www.baichuan-ai.com/) 是一家处于 AGI 时代的中国初创公司，
> 致力于解决人类的基本需求：效率、健康与幸福。


## 安装与设置

前往 [Baichuan 平台](https://platform.baichuan-ai.com/) 注册并获取 API 密钥。

## LLMs

请参阅 [使用示例](/oss/python/integrations/llms/baichuan)。

```python
from langchain_community.llms import BaichuanLLM
```

## 聊天模型

请参阅 [使用示例](/oss/python/integrations/chat/baichuan)。

```python
from langchain_community.chat_models import ChatBaichuan
```

## 嵌入模型

请参阅 [使用示例](/oss/python/integrations/embeddings/baichuan)。

```python
from langchain_community.embeddings import BaichuanTextEmbeddings
```
