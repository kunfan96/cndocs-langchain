`PromptLayer` 是首个允许您跟踪、管理和共享 GPT 提示词工程的平台。`PromptLayer` 充当您的代码与 `OpenAI` Python 库之间的中间件。

`PromptLayer` 会记录所有您的 `OpenAI API` 请求，使您能够在 `PromptLayer` 仪表板中搜索和探索请求历史。

本示例展示了如何连接到 [PromptLayer](https://www.promptlayer.com) 以开始记录您的 OpenAI 请求。

另一个示例位于 [PromptLayer 提供商集成](/oss/python/integrations/providers/promptlayer)。

## 安装 PromptLayer

要在 OpenAI 中使用 PromptLayer，需要安装 `promptlayer` 包。请使用 pip 安装 `promptlayer`。

```python
pip install -qU  promptlayer
```

## 导入

```python
import os

import promptlayer
from langchain_community.llms import PromptLayerOpenAI
```

## 设置环境变量 API 密钥

您可以在 [www.promptlayer.com](https://www.promptlayer.com) 上通过点击导航栏中的设置齿轮图标来创建 PromptLayer API 密钥。

将其设置为名为 `PROMPTLAYER_API_KEY` 的环境变量。

您还需要一个 OpenAI 密钥，称为 `OPENAI_API_KEY`。

```python
from getpass import getpass

PROMPTLAYER_API_KEY = getpass()
```

## 正常使用 PromptLayerOpenAI LLM

*您可以选择传入 `pl_tags`，以便利用 PromptLayer 的标签功能来跟踪您的请求。*

```python
llm = PromptLayerOpenAI(pl_tags=["langchain"])
llm("I am a cat and I want")
```

**上述请求现在应该会显示在您的 [PromptLayer 仪表板](https://www.promptlayer.com) 中。**

## 使用 PromptLayer 跟踪

如果您希望使用任何 [PromptLayer 跟踪功能](https://magniv.notion.site/Track-4deee1b1f7a34c1680d085f82567dab9)，则在实例化 PromptLayer LLM 时需要传入参数 `return_pl_id` 以获取请求 ID。

```python
llm = PromptLayerOpenAI(return_pl_id=True)
llm_results = llm.generate(["Tell me a joke"])

for res in llm_results.generations:
    pl_request_id = res[0].generation_info["pl_request_id"]
    promptlayer.track.score(request_id=pl_request_id, score=100)
```

使用此功能可让您在 PromptLayer 仪表板中跟踪模型的性能。如果您正在使用提示词模板，还可以将模板附加到请求中。
总体而言，这为您提供了在 PromptLayer 仪表板中跟踪不同模板和模型性能的机会。
