>[Confident AI](https://confident-ai.com) 是 `DeepEval` 的创建者。
>
>[DeepEval](https://github.com/confident-ai/deepeval) 是一个用于 LLM 单元测试的包。
> 借助 `DeepEval`，每个人都可以通过更快的迭代构建稳健的语言模型，
> 结合单元测试和集成测试。`DeepEval 支持迭代过程中的每一步，
> 从合成数据生成到测试均提供全面支持。

## 安装与配置

您需要获取 [DeepEval API 凭据](https://app.confident-ai.com)。

您需要安装 `DeepEval` Python 包：

<CodeGroup>
```bash pip
pip install deepeval
```

```bash uv
uv add deepeval
```
</CodeGroup>

## 回调函数

请参阅[示例](/oss/python/integrations/callbacks/confident)。

```python
from langchain.callbacks.confident_callback import DeepEvalCallbackHandler
```
