SWI-Prolog 提供了一个全面的免费 Prolog 环境。

## 安装与设置

安装 SWI-Prolog 后，使用 pip 安装 lanchain-prolog：

<CodeGroup>
```bash pip
pip install langchain-prolog
```

```bash uv
uv add langchain-prolog
```
</CodeGroup>

## 工具

`PrologTool` 类允许生成使用 Prolog 规则生成答案的 langchain 工具。

```python
from langchain_prolog import PrologConfig, PrologTool
```

请参阅[使用示例](/oss/python/integrations/tools/prolog_tool)。

请参阅同一指南以获取 `PrologRunnable` 的使用示例，该类允许生成使用 Prolog 规则生成答案的 LangChain 可运行对象。
