> [LambdaDB](https://lambdadb.ai/) 是一款面向 RAG 和智能体的无服务器 AI 数据库。基于灵活的文档模型，统一全文、多向量和混合搜索。即时处理无限持久化内存与海量并发——成本仅为原来的十分之一。

## 安装与配置

安装 Python SDK：

<CodeGroup>
    ```bash pip
    pip install langchain-lambdadb
    ```

    ```bash uv
    uv add langchain-lambdadb
    ```
</CodeGroup>

你还需要一个包含以下信息的 LambdaDB 账户：
- 项目 URL
- 项目 API 密钥

前往 [lambdadb.ai](https://lambdadb.ai) 获取你的凭据。

## 向量存储

存在一个针对 `LambdaDB` 集合的封装器，允许你将其用作语义搜索、RAG 及其他 AI 应用的向量存储。

导入该向量存储的方式如下：
```python
from langchain_lambdadb import LambdaDBVectorStore
```

### 快速入门

```python
from langchain_lambdadb import LambdaDBVectorStore
from langchain_openai import OpenAIEmbeddings
from lambdadb import LambdaDB
import os

# 初始化客户端
client = LambdaDB(
    server_url=os.environ["LAMBDADB_SERVER_URL"],
    project_api_key=os.environ["LAMBDADB_API_KEY"]
)

# 使用现有集合创建向量存储
vector_store = LambdaDBVectorStore(
    client=client,
    collection_name="my_collection",  # 必须预先存在
    embedding=OpenAIEmbeddings()
)
```

如需了解 LambdaDB 封装器的更详细教程，请参阅[完整文档](/oss/python/integrations/vectorstores/lambdadb)。
