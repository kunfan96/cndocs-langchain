## 用例

以下是常见用例的教程，按框架组织。

[Deep Agents](/oss/javascript/deepagents/overview) 包含用于管理上下文、虚拟文件系统和其他常见代理需求的内置功能。

  构建一个向 Slack 发送报告的数据分析代理。
</Card>

<Card title="深度研究" icon="search" href="/oss/javascript/deepagents/deep-research" horizontal>
  构建一个多步骤网络研究代理，具有子代理委托和战略反思功能。
</Card>

### LangChain

[LangChain](/oss/javascript/langchain/overview) [代理](/oss/javascript/langchain/agents) 实现使简单用例易于上手。

<Card title="语义搜索" icon="search" href="/oss/javascript/langchain/knowledge-base" horizontal>
  使用 LangChain 组件构建 PDF 上的语义搜索引擎。
</Card>

<Card title="RAG 代理" icon="user-search" href="/oss/javascript/langchain/rag" horizontal>
  创建一个检索增强生成 (RAG) 代理。
</Card>

<Card title="SQL 代理" icon="database" href="/oss/javascript/langchain/sql-agent" horizontal>
  构建一个 SQL 代理，通过人工在环审查与数据库交互。
</Card>

<Card title="语音代理" icon="microphone" href="/oss/javascript/langchain/voice-agent" horizontal>
  构建一个您可以与之对话和倾听的代理。

### LangGraph

LangChain 的 [代理](/oss/javascript/langchain/agents) 实现使用 [LangGraph](/oss/javascript/langgraph/overview) 原语。
如果需要更深层次的定制，可以直接在 LangGraph 中实现代理。

<Card title="自定义 RAG 代理" icon="user-search" href="/oss/javascript/langgraph/agentic-rag" horizontal>
  使用 LangGraph 原语构建 RAG 代理，实现细粒度控制。
</Card>



### 多代理

这些教程演示了 [多代理模式](/oss/javascript/langchain/multi-agent)，将 LangChain 代理与 LangGraph 工作流相结合。
