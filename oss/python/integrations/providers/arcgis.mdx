>[ArcGIS](https://www.esri.com/en-us/arcgis/about-arcgis/overview) 是由 [Esri](https://www.esri.com/) 开发和维护的一系列客户端、服务器和在线地理信息系统软件。
>
>`ArcGISLoader` 使用 `arcgis` 包。
> `arcgis` 是一个用于向量和栅格分析、地理编码、地图制作、路径规划和导航的 Python 库，
> 还可以管理您的 GIS 中的用户、组和信息条目。
>它可以访问来自 `Esri` 和其他权威来源的现成地图和精选地理数据，同时也可使用您自己的数据。

## 安装与设置

我们需要安装 `arcgis` 包。

<CodeGroup>
```bash pip
pip install -U arcgis
```

```bash uv
uv add arcgis
```
</CodeGroup>

## 文档加载器

```python
from langchain_community.document_loaders import ArcGISLoader
```
