---
title: MODEL_NOT_FOUND
---

<Note>
    目前仅用于 `langchainjs`（JavaScript/TypeScript）。
</Note>

你指定的模型名称未被提供商识别。

## 故障排查

要解决此错误：

1. **验证模型标识符**：仔细检查你传入的模型字符串，确保拼写和格式正确
2. **检查代理/包装器配置**：如果你使用代理或其他替代主机配合模型包装器，确认允许使用的模型名称没有被限制或更改

此错误通常源于模型名称字符串中的拼写错误，或代码与提供商 API 之间的代理服务或模型包装器施加的限制。
