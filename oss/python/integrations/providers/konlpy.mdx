>[KoNLPy](https://konlpy.org/) 是一个用于韩语自然语言处理（NLP）
> 的 Python 包。


## 安装与设置

您需要安装 `konlpy` Python 包。

<CodeGroup>
```bash pip
pip install konlpy
```

```bash uv
uv add konlpy
```
</CodeGroup>

from langchain_text_splitters import KonlpyTextSplitter
