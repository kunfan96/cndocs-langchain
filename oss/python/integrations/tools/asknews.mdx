> [AskNews](https://asknews.app) 通过单一自然语言查询，为任何 LLM 注入最新全球新闻（或历史新闻）。具体而言，AskNews 每天对超过 30 万篇文章进行翻译、摘要、实体提取并将其索引到冷热向量数据库中。AskNews 在低延迟端点上提供这些向量数据库。当您查询 AskNews 时，您将获得一个经过提示优化的字符串，其中包含所有最相关的信息增强内容（例如实体、分类、翻译、摘要）。这意味着您无需管理自己的新闻 RAG，也无需担心如何向 LLM 以简洁方式传达新闻信息。
> AskNews 同样致力于透明度，因此我们的覆盖范围会在数百个国家、13 种语言和 5 万个来源中持续监控和多样化。如果您希望追踪我们的来源覆盖情况，可以访问我们的[透明度仪表板](https://asknews.app/en/transparency)。

## 安装配置

该集成位于 `langchain-community` 包中。我们还需要安装 `asknews` 包本身。

```bash
pip install -U langchain-community asknews
```

我们还需要设置 AskNews API 凭证，可在 [AskNews 控制台](https://my.asknews.app)获取。

```python
import getpass
import os

os.environ["ASKNEWS_CLIENT_ID"] = getpass.getpass()
os.environ["ASKNEWS_CLIENT_SECRET"] = getpass.getpass()
```

## 使用方法

以下展示如何单独使用该工具。

```python
from langchain_community.tools.asknews import AskNewsSearch

tool = AskNewsSearch(max_results=2)
tool.invoke({"query": "Effect of fed policy on tech sector"})
```

## 链式调用

以下展示如何将其作为 agent 的一部分使用。我们使用 OpenAI Functions Agent，因此需要安装相应的依赖。我们还将使用 [LangSmith Hub](https://smith.langchain.com/hub) 来拉取提示词，因此需要安装该组件。

```bash
pip install -U langchain-openai langchainhub
```

```python
import getpass
import os

os.environ["OPENAI_API_KEY"] = getpass.getpass()
```

```python
from langchain_classic import hub
from langchain.agents import AgentExecutor, create_openai_functions_agent
from langchain_community.tools.asknews import AskNewsSearch
from langchain_openai import ChatOpenAI

prompt = hub.pull("hwchase17/openai-functions-agent")
llm = ChatOpenAI(temperature=0)
asknews_tool = AskNewsSearch()
tools = [asknews_tool]
agent = create_openai_functions_agent(llm, tools, prompt)
agent_executor = AgentExecutor(agent=agent, tools=tools)
agent_executor.invoke({"input": "How is the tech sector being affected by fed policy?"})
```
