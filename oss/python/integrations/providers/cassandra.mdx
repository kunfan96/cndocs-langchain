> [Apache Cassandra®](https://cassandra.apache.org/) 是一款面向行、高度可扩展且高可用的 NoSQL 数据库。
> 从 5.0 版本起，该数据库内置了[向量搜索功能](https://cassandra.apache.org/doc/trunk/cassandra/vector-search/overview.html)。

本页介绍的集成可与 `Cassandra` 以及其他兼容 CQL 的数据库一起使用，
即使用 `Cassandra Query Language` 协议的数据库。


## 安装与设置

安装以下 Python 包：

<CodeGroup>
```bash pip
pip install "cassio>=0.1.6"
```

```bash uv
uv add "cassio>=0.1.6"
```
</CodeGroup>

## 向量存储

```python
from langchain_community.vectorstores import Cassandra
```

在[示例笔记本](/oss/python/integrations/vectorstores/cassandra)中了解更多。

## LLM 缓存

```python
from langchain.globals import set_llm_cache
from langchain_community.cache import CassandraCache
set_llm_cache(CassandraCache())
```

## 语义 LLM 缓存

```python
from langchain.globals import set_llm_cache
from langchain_community.cache import CassandraSemanticCache
set_llm_cache(CassandraSemanticCache(
    embedding=my_embedding,
    table_name="my_store",
))
```

## 文档加载器

```python
from langchain_community.document_loaders import CassandraLoader
```

在[示例笔记本](/oss/python/integrations/document_loaders/cassandra)中了解更多。

## 工具包

`Cassandra Database toolkit` 可帮助 AI 工程师高效地将智能体与 Cassandra 数据集成。

```python
from langchain_community.agent_toolkits.cassandra_database.toolkit import (
    CassandraDatabaseToolkit,
)
```

在[示例笔记本](/oss/python/integrations/tools/cassandra_database)中了解更多。


Cassandra Database 独立工具：

### 获取 Schema

用于获取 Apache Cassandra 数据库中 keyspace 的 schema。

```python
from langchain_community.tools import GetSchemaCassandraDatabaseTool
```

### 获取表数据

用于从 Apache Cassandra 数据库中的表获取数据。

```python
from langchain_community.tools import GetTableDataCassandraDatabaseTool
```

### 查询

使用提供的 CQL 查询 Apache Cassandra 数据库。

```python
from langchain_community.tools import QueryCassandraDatabaseTool
```
