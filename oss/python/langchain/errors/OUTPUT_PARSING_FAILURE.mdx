---
title: OUTPUT_PARSING_FAILURE
---

[output parser](https://reference.langchain.com/python/langchain_core/output_parsers/) 无法按预期处理模型输出。

<Note>
    一些预构建的结构（如旧版 LangChain agents 和 chains）可能在内部使用 output parsers，因此即使你没有明显地实例化和使用 output parser，你也可能会看到此错误。
</Note>

## 故障排除

- 如果可能，考虑使用工具调用或其他结构化输出技术，而不是使用 output parser，以可靠地输出可解析的值。
- 向你的 prompt 添加更精确的格式化说明。
- 如果你使用的是较小或能力较弱的模型，请尝试使用能力更强的模型。
