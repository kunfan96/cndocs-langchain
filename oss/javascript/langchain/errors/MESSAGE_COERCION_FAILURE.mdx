## 故障排除

要解决此错误：

1. **确保格式正确**：聊天模型的所有输入必须是 LangChain 消息类数组或受支持的类消息格式
2. 验证您的消息是否未发生意外的字符串化或转换
3. 检查错误的堆栈跟踪，并添加日志语句以在消息对象传递给模型之前检查它们
