[ERNIE-Bot](https://cloud.baidu.com/doc/WENXINWORKSHOP/s/jlil56u11) 是由百度开发的大语言模型，涵盖了大量中文数据。
本笔记本介绍了如何使用 ErnieBot 聊天模型。

**已弃用警告**

我们建议用户从 `langchain_community.chat_models.ErnieBotChat` 切换到 `langchain_community.chat_models.QianfanChatEndpoint`。

有关 `QianfanChatEndpoint` 的文档可以在 [Ernie 文档](/oss/python/integrations/chat/baidu_qianfan_endpoint/) 中找到。

我们推荐用户使用 `QianfanChatEndpoint` 的 4 个原因如下：

1. `QianfanChatEndpoint` 支持 Qianfan 平台上的更多大模型。
2. `QianfanChatEndpoint` 支持流式模式。
3. `QianfanChatEndpoint` 支持函数调用功能。
4. `ErnieBotChat` 已不再维护并已被弃用。

迁移时的几点建议：

- 将 `ernie_client_id` 更改为 `qianfan_ak`，也将 `ernie_client_secret` 更改为 `qianfan_sk`。
- 安装 `qianfan` 包。例如：`pip install qianfan`
- 将 `ErnieBotChat` 更改为 `QianfanChatEndpoint`。

```python
from langchain_community.chat_models.baidu_qianfan_endpoint import QianfanChatEndpoint

chat = QianfanChatEndpoint(
    qianfan_ak="你的 qianfan ak",
    qianfan_sk="你的 qianfan sk",
)
```

## 使用方法

```python
from langchain_community.chat_models import ErnieBotChat
from langchain.messages import HumanMessage

chat = ErnieBotChat(
    ernie_client_id="YOUR_CLIENT_ID", ernie_client_secret="YOUR_CLIENT_SECRET"
)
```

或者您可以在环境变量中设置 `client_id` 和 `client_secret`

```bash
export ERNIE_CLIENT_ID=YOUR_CLIENT_ID
export ERNIE_CLIENT_SECRET=YOUR_CLIENT_SECRET
```

```python
chat([HumanMessage(content="hello there, who are you?")])
```
