>[DuckDuckGo Search](https://github.com/deedy5/duckduckgo_search) 是一个可以使用
> `DuckDuckGo.com` 搜索引擎搜索词语、文档、图像、视频、新闻、地图和文本翻译的包。
> 它还可以将文件和图像下载到本地硬盘。

## 安装与设置

您需要安装 Python 包：

<CodeGroup>
```bash pip
pip install duckduckgo-search
```

```bash uv
uv add duckduckgo-search
```
</CodeGroup>

## 工具

查看[使用示例](/oss/python/integrations/tools/ddg)。

有两个可用工具：

```python
from langchain_community.tools import DuckDuckGoSearchRun
from langchain_community.tools import DuckDuckGoSearchResults
```
