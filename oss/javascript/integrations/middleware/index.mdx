浏览不同提供商的可用中间件，或为生态系统贡献您自己的中间件。了解中间件如何工作，请参阅[中间件概述](/oss/javascript/langchain/middleware/overview)；了解如何在 Deep Agents 中使用中间件，请参阅[Deep Agents 文档](/oss/javascript/deepagents/customization#middleware)。

## 分享您的中间件

中间件支持上下文工程、工具定制和运行时安全控制。它是 LangChain 中一个有用的扩展点，我们喜欢突出社区使用它构建的内容：

    <Card title="添加官方集成" icon="package" href="/oss/javascript/contributing/implement-langchain#middleware">
        遵循贡献指南来构建和发布中间件包。
    </Card>
    <Card title="分享社区中间件" icon="users" href="https://github.com/langchain-ai/docs">
        向文档仓库提交 PR，将您的中间件添加到下方表格中。
    </Card>
</CardGroup>
