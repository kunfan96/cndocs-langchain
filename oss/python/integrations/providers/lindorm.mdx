Lindorm 是阿里云的云原生多模态数据库，支持对宽表、时序、文本、对象、流和空间数据等多种类型数据的统一访问和集成处理。它兼容 SQL、HBase/Cassandra/S3、TSDB、HDFS、Solr 和 Kafka 等多种标准接口，并与第三方生态工具无缝集成。这使其适用于日志、监控、计费、广告、社交网络、出行和风控等场景。Lindorm 也是支持阿里核心业务的数据库之一。

要使用 Lindorm 的 AI 和向量能力，您应[获取服务](https://help.aliyun.com/document_detail/174640.html?spm=a2c4g.11186623.help-menu-172543.d_0_1_0.4c6367558DN8Uq)并安装 `langchain-lindorm-integration` 包。

```python
!pip install -U langchain-lindorm-integration
```

## 嵌入

要使用 Lindorm AI 服务中部署的嵌入模型，请导入 LindormAIEmbeddings。

```python
from langchain_lindorm_integration import LindormAIEmbeddings
```

## 重排序

Lindorm AI 服务还支持重排序。

```python
from langchain_lindorm_integration.reranker import LindormAIRerank
```

## 向量存储

Lindorm 还支持向量存储。

```python
from langchain_lindorm_integration import LindormVectorStore
```

## ByteStore

使用 Lindorm 的 ByteStore

```python
from langchain_lindorm_integration import LindormByteStore
```
