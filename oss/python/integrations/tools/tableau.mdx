本指南提供了快速入门 [Tableau](https://help.tableau.com/current/api/vizql-data-service/en-us/index.html) 的概述。

### 概述

Tableau 的 VizQL 数据服务（简称 VDS）为开发人员提供了对其 Tableau 发布数据源的程序化访问，允许他们为任何自定义工作负载或应用程序（包括 AI 代理）扩展其业务语义。simple_datasource_qa 工具将 VDS 添加到 LangChain 框架中。本笔记本向您展示如何使用它来构建基于企业语义模型回答分析问题的代理。

关注 [tableau-langchain](https://github.com/Tab-SE/tableau_langchain) 项目，获取更多即将推出的工具！

#### 设置

确保您正在运行并有权访问：

1. Python 3.12.2 或更高版本
2. 一个 Tableau Cloud 或 Server 环境，至少有一个已发布的数据源

### 凭据

您可以显式声明环境变量，如本文档中的几个示例所示。但是，如果未提供这些参数，simple_datasource_qa 工具将尝试从环境变量中自动读取它们。

对于您选择查询的数据源，请确保已在 Tableau 中更新 VizqlDataApiAccess 权限，以允许 VDS API 通过 REST 访问该数据源。更多信息，请参见 [Tableau 权限和功能](https://help.tableau.com/current/server/en-us/permissions_capabilities.htm#data-sources)。

```python
# langchain package imports
from langchain_openai import ChatOpenAI

# langchain_tableau and langgraph imports
from langchain_tableau.tools.simple_datasource_qa import initialize_simple_datasource_qa
from langchain.agents import create_agent
```

## 身份验证变量

您可以显式声明环境变量，如本食谱中的几个示例所示。但是，如果未提供这些参数，simple_datasource_qa 工具将尝试从环境变量中自动读取它们。

对于您选择的数据源，请确保已在 Tableau 中更新 VizqlDataApiAccess 权限，以允许 VDS API 通过 REST 访问该数据源。更多信息，请参见 [Tableau 权限和功能](https://help.tableau.com/current/server/en-us/permissions_capabilities.htm#data-sources
)。

```python
import os

from dotenv import load_dotenv

load_dotenv()

tableau_server = "https://stage-dataplane2.tableau.sfdc-shbmgi.svc.sfdcfc.net/"  # 替换为您的 Tableau 服务器名称
tableau_site = "vizqldataservicestage02"  # 替换为您的 Tableau 站点
tableau_jwt_client_id = os.getenv(
    "TABLEAU_JWT_CLIENT_ID"
)  # JWT 客户端 ID（通过 Tableau 管理员界面获取）
tableau_jwt_secret_id = os.getenv(
    "TABLEAU_JWT_SECRET_ID"
)  # JWT 密钥 ID（通过 Tableau 管理员界面获取）
tableau_jwt_secret = os.getenv(
    "TABLEAU_JWT_SECRET"
)  # JWT 密钥 ID（通过 Tableau 管理员界面获取）
tableau_api_version = "3.21"  # 当前 Tableau REST API 版本
tableau_user = "joe.constantino@salesforce.com"  # 输入查询目标 Tableau 数据源的用户名

# 对于本食谱，我们连接到每个 Tableau 服务器默认附带的 Superstore 数据集
datasource_luid = (
    "0965e61b-a072-43cf-994c-8c6cf526940d"  # 此工具的目标数据源
)
model_provider = "openai"  # 您为代理使用的模型提供者的名称
# 添加变量以控制代理和工具的 LLM 模型
os.environ["OPENAI_API_KEY"]  # 将您的模型 API 密钥设置为环境变量
tooling_llm_model = "gpt-4.1-mini"
```

## 实例化

initialize_simple_datasource_qa 初始化名为 [simple_datasource_qa](https://github.com/Tab-SE/tableau_langchain/blob/3ff9047414479cd55d797c18a78f834d57860761/pip_package/langchain_tableau/tools/simple_datasource_qa.py#L101) 的 LangGraph 工具，该工具可用于对 Tableau 数据源进行分析问答。

此初始化函数：

1. 使用 Tableau 的连接应用框架进行基于 JWT 的身份验证来认证到 Tableau。所有必需的变量必须在运行时定义或作为环境变量定义。
2. 异步查询 datasource_luid 变量中指定的目标数据源的字段元数据。
3. 基于目标数据源的元数据，将自然语言问题转换为执行 VDS 查询数据源请求所需的 JSON 格式查询负载。
4. 执行到 VDS 的 POST 请求。
5. 格式化结果并以结构化响应返回。

```python
# 初始化 simple_datasource_qa 以通过 VDS 查询 Tableau 数据源
analyze_datasource = initialize_simple_datasource_qa(
    domain=tableau_server,
    site=tableau_site,
    jwt_client_id=tableau_jwt_client_id,
    jwt_secret_id=tableau_jwt_secret_id,
    jwt_secret=tableau_jwt_secret,
    tableau_api_version=tableau_api_version,
    tableau_user=tableau_user,
    datasource_luid=datasource_luid,
    tooling_llm_model=tooling_llm_model,
    model_provider=model_provider,
)

# 加载代理要使用的工具列表。在本例中，我们将只加载数据源问答工具。
tools = [analyze_datasource]
```

## 调用 - LangGraph 示例

首先，我们将初始化我们选择的 LLM。然后，我们使用 langgraph 代理构造函数类定义一个代理，并使用与目标数据源相关的查询调用它。

```python
from IPython.display import Markdown, display

model = ChatOpenAI(model="gpt-4.1", temperature=0)

tableauAgent = create_agent(model, tools)

# 运行代理
messages = tableauAgent.invoke(
    {
        "messages": [
            (
                "human",
                "what's going on with table sales?",
            )
        ]
    }
)
messages
# display(Markdown(messages['messages'][3].content)) #display a nicely formatted answer for successful generations
```
