> [LocalAI](https://localai.io/) 是一款免费、开源的 OpenAI 替代方案。
> `LocalAI` 作为一个无缝替换的 REST API，兼容用于本地推理的 OpenAI API 规范。它允许您使用消费级硬件在本地或私有化部署环境中运行 LLMs、生成图像、音频（以及更多功能），并支持多种模型家族与架构。

<Info>
`langchain-localai` 是 LocalAI 的第三方集成包。它为在 LangChain 中使用 LocalAI 服务提供了一种简便的方式。
源代码可在 [GitHub](https://github.com/mkhludnev/langchain-localai) 上获取。
</Info>

## 安装与配置

<CodeGroup>
```bash pip
pip install langchain-localai
```

```bash uv
uv add langchain-localai
```
</CodeGroup>
