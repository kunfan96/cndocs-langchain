`GooseAI` 是一种通过 API 交付的完全托管型 NLP 即服务（NLP-as-a-Service）。GooseAI 提供对[这些模型](https://goose.ai/docs/models)的访问权限。

本笔记本将介绍如何在 [GooseAI](https://goose.ai/) 中使用 LangChain。

## 安装 openai

使用 GooseAI API 需要 `openai` 包。请使用 `pip install openai` 安装 `openai`。

```python
pip install -qU  langchain-openai
```

## 导入模块

```python
import os

from langchain_classic.chains import LLMChain
from langchain_community.llms import GooseAI
from langchain_core.prompts import PromptTemplate
```

## 设置环境变量 API Key

请确保从 GooseAI 获取您的 API 密钥。您将获得 10 美元的免费额度用于测试不同的模型。

```python
from getpass import getpass

GOOSEAI_API_KEY = getpass()
```

```python
os.environ["GOOSEAI_API_KEY"] = GOOSEAI_API_KEY
```

## 创建 GooseAI 实例

您可以指定不同的参数，例如模型名称、最大生成 token 数、温度等。

```python
llm = GooseAI()
```

## 创建提示词模板

我们将为问答场景创建一个提示词模板。

```python
template = """Question: {question}

Answer: Let's think step by step."""

prompt = PromptTemplate.from_template(template)
```

## 初始化 LLMChain

```python
llm_chain = LLMChain(prompt=prompt, llm=llm)
```

## 运行 LLMChain

提供一个问题并运行 LLMChain。

```python
question = "What NFL team won the Super Bowl in the year Justin Beiber was born?"

llm_chain.run(question)
```
