集成包是用户可以安装并在其项目中使用的 Python 包。它们实现了一个或多个遵循 LangChain 接口标准的组件。

LangChain 组件是 [`langchain-core`](https://github.com/langchain-ai/langchain/tree/master/libs/core) 中基类的子类。示例包括 [聊天模型](/oss/javascript/integrations/chat)、[工具](/oss/javascript/integrations/tools)、[检索器](/oss/javascript/integrations/retrievers) 等。

您的集成包通常将实现这些组件中至少一个的子类。展开下面的选项卡以查看每个组件的详细信息。

<Tabs>
    <Tab title="聊天模型">
        聊天模型是 [`BaseChatModel`](https://reference.langchain.com/javascript/langchain-core/language_models/chat_models/BaseChatModel) 类的子类。它们实现了生成聊天完成、处理消息格式和管理模型参数的方法。

        <Warning>
            聊天模型集成指南目前正在完善中（WIP）。与此同时，请阅读 [聊天模型概念指南](/oss/javascript/langchain/models) 以了解 LangChain 聊天模型如何工作。
        </Warning>
    </Tab>
    <Tab title="工具">
        工具主要以 2 种方式使用：

        1. 定义一个“输入模式”或“参数模式”，与文本请求一起传递给聊天模型的工具调用功能，以便聊天模型可以生成“工具调用”或调用工具的参数。
        2. 获取上面生成的“工具调用”，执行某些操作并返回一个响应，该响应可以作为 ToolMessage 传回聊天模型。

        Tools 类必须继承自 [`BaseTool`](https://reference.langchain.com/javascript/classes/_langchain_core.tools.StructuredTool.html) 基类。此接口有 3 个属性和 2 个方法应在子类中实现。

        <Warning>
            工具集成指南目前正在完善中（WIP）。与此同时，请阅读 [工具概念指南](/oss/javascript/langchain/tools) 以了解 LangChain 工具如何工作。
        </Warning>
    </Tab>
    <Tab title="检索器">
        检索器用于根据查询从 API、数据库或其他来源检索文档。Retriever 类必须继承自 BaseRetriever 基类。

        <Warning>
            检索器集成指南目前正在完善中（WIP）。与此同时，请阅读 [检索器概念指南](/oss/javascript/integrations/retrievers) 以了解 LangChain 检索器如何工作。
        </Warning>
    </Tab>
    <Tab title="向量存储">
        所有向量存储必须继承自 [`VectorStore`](https://reference.langchain.com/javascript/langchain-core/vectorstores/VectorStore) 基类。此接口由用于在向量存储中写入、删除和搜索文档的方法组成。

        请参阅 [向量存储集成指南](/oss/javascript/integrations/vectorstores) 以了解实现向量存储集成的详细信息。

        <Warning>
            向量存储集成指南目前正在完善中（WIP）。与此同时，请阅读 [向量存储概念指南](/oss/javascript/integrations/vectorstores) 以了解 LangChain 向量存储如何工作。
        </Warning>
    </Tab>
    <Tab title="嵌入">
        嵌入模型是 [`Embeddings`](https://reference.langchain.com/javascript/langchain-core/embeddings/Embeddings) 类的子类。

        <Warning>
            嵌入模型集成指南目前正在完善中（WIP）。与此同时，请阅读 [嵌入模型概念指南](/oss/javascript/integrations/text_embedding) 以了解 LangChain 嵌入模型如何工作。
        </Warning>
    </Tab>
    <Tab title="沙盒">
        沙盒集成使 [Deep Agents](/oss/javascript/deepagents/overview) 能够在隔离环境中运行代码。

        **参考实现：** 请参阅 [Daytona 合作伙伴集成](https://github.com/langchain-ai/deepagents/tree/main/libs/partners/daytona) 以了解结构和模式。
    </Tab>
</Tabs>
