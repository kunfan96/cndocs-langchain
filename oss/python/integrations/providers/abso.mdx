[Abso](https://abso.ai/#router) 是一款开源的 LLM 代理，可根据提示词复杂度在快慢模型之间自动路由请求。它采用多种启发式方法来选择正确的模型。其速度极快且延迟极低。

## 安装与配置

```bash
pip install langchain-abso
```
