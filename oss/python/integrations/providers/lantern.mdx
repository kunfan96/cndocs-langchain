本页介绍如何在 LangChain 中使用 [Lantern](https://github.com/lanterndata/lantern)。
内容分为两部分：设置，以及特定 Lantern 包装器的参考文档。

## 设置
1. 第一步是创建一个已安装 `lantern` 扩展的数据库。

    请按照 [Lantern 安装指南](https://github.com/lanterndata/lantern#-quick-install) 中的步骤安装数据库和扩展。使用 Docker 镜像是最简单的入门方式。

## 包装器

### VectorStore

提供了一个针对 Postgres 向量数据库的包装器，允许您将其用作向量存储，无论用于语义搜索还是示例选择。

要导入此向量存储：
```python
from langchain_community.vectorstores import Lantern
```

有关 Lantern 包装器的更详细逐步指南，请参阅[此笔记本](/oss/python/integrations/vectorstores/lantern)。
