>[阿里云 AnalyticDB for PostgreSQL](https://www.alibabacloud.com/help/en/analyticdb-for-postgresql/latest/product-introduction-overview)
> 是[阿里云](https://www.alibabacloud.com/)推出的一种大规模并行处理（MPP）数据仓库服务，
> 专为在线分析大量数据而设计。

>`AnalyticDB for PostgreSQL` 基于开源 `Greenplum Database` 项目开发，
> 并由`阿里云`进行了深度扩展。AnalyticDB for PostgreSQL 兼容 ANSI SQL 2003 语法以及 PostgreSQL 和
> Oracle 数据库生态系统，同时支持行存储和列存储。AnalyticDB for PostgreSQL 能以高性能离线处理 PB 级数据，
> 并支持高并发场景。

本页介绍如何在 LangChain 中使用 AnalyticDB 生态系统。

## 安装与设置

您需要安装 `sqlalchemy` Python 包。

<CodeGroup>
```bash pip
pip install sqlalchemy
```

```bash uv
uv add sqlalchemy
```
</CodeGroup>

## 向量存储

查看[使用示例](/oss/python/integrations/vectorstores/analyticdb)。

```python
from langchain_community.vectorstores import AnalyticDB
```
