>[Context](https://context.ai/) 为 LLM 驱动的产品和功能提供用户分析。

## 安装与设置

我们需要安装 `context-python` Python 包：

<CodeGroup>
```bash pip
pip install context-python
```

```bash uv
uv add context-python
```
</CodeGroup>


## 回调

查看[使用示例](/oss/python/integrations/callbacks/context)。

```python
from langchain.callbacks import ContextCallbackHandler
```
