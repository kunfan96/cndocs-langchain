`arXiv Retriever` 允许用户查询 arXiv 数据库以获取学术文章。它支持全文档检索（PDF 解析）和基于摘要的检索。

有关所有 `ArxivRetriever` 功能和配置的详细文档，请前往 [API 参考](https://reference.langchain.com/javascript/langchain-community/retrievers/arxiv/ArxivRetriever)

## 功能

- 查询灵活性：使用自然语言查询或特定的 arXiv ID 进行搜索。
- 全文档检索：可选择获取和解析 PDF。
- 摘要作为文档：检索摘要以获得更快的结果。
- 可自定义选项：配置最大结果数和输出格式。

## 集成详情

| 检索器            | 来源                         | 包                                                                          |
| ---------------- | ---------------------------- | ---------------------------------------------------------------------------- |
| `ArxivRetriever` | 来自 arXiv 的学术文章         | [`@langchain/community`](https://www.npmjs.com/package/@langchain/community) |

## 设置

确保安装了以下依赖项：

- `pdf-parse` 用于解析 PDF
- `fast-xml-parser` 用于解析来自 arXiv API 的 XML 响应

```
npm install pdf-parse fast-xml-parser
```

## 实例化

```typescript
const retriever = new ArxivRetriever({
  getFullDocuments: false, // 设置为 true 以获取全文档（PDF）
  maxSearchResults: 5, // 要检索的最大结果数
});
```

## 用法

使用 `invoke` 方法搜索 arXiv 以查找相关文章。您可以使用自然语言查询或特定的 arXiv ID。

```typescript
const query = "quantum computing";

const documents = await retriever.invoke(query);
documents.forEach((doc) => {
  console.log("Title:", doc.metadata.title);
  console.log("Content:", doc.pageContent); // 解析的 PDF 内容
});
```

## 在链中使用

与其他检索器一样，`ArxivRetriever` 可以通过链集成到 LLM 应用程序中。以下是检索器在链中使用的示例：

```typescript
import { ChatOpenAI } from "@langchain/openai";
import { ChatPromptTemplate } from "@langchain/core/prompts";
import {
  RunnablePassthrough,
  RunnableSequence,
} from "@langchain/core/runnables";
import { StringOutputParser } from "@langchain/core/output_parsers";
import type { Document } from "@langchain/core/documents";

const llm = new ChatOpenAI({
  model: "gpt-4.1-mini",
  temperature: 0,
});

const prompt = ChatPromptTemplate.fromTemplate(`
Answer the question based only on the context provided.

Context: {context}

Question: {question}`);

const formatDocs = (docs: Document[]) => {
  return docs.map((doc) => doc.pageContent).join("\n\n");
};

const ragChain = RunnableSequence.from([
  {
    context: retriever.pipe(formatDocs),
    question: new RunnablePassthrough(),
  },
  prompt,
  llm,
  new StringOutputParser(),
]);

await ragChain.invoke("What are the latest advances in quantum computing?");
```
