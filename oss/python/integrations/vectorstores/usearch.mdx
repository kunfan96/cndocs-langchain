>[USearch](https://unum-cloud.github.io/usearch/) 是一个更小、更快的单文件向量搜索引擎

>USearch 的基础功能与 FAISS 完全相同，如果你曾研究过近似最近邻搜索，其接口应该会让你感到熟悉。FAISS 是高性能向量搜索引擎领域广泛认可的标准。USearch 和 FAISS 都采用相同的 HNSW 算法，但两者在设计理念上存在显著差异。USearch 在不牺牲性能的前提下，体积紧凑、兼容性广，主要专注于用户自定义指标并减少依赖项。

我们希望使用 [`OpenAIEmbeddings`](https://reference.langchain.com/python/langchain-openai/embeddings/base/OpenAIEmbeddings)，因此需要获取 OpenAI API Key。

```python
import getpass
import os

if "OPENAI_API_KEY" not in os.environ:
    os.environ["OPENAI_API_KEY"] = getpass.getpass("OpenAI API Key:")
```

```python
from langchain_community.document_loaders import TextLoader
from langchain_community.vectorstores import USearch
from langchain_openai import OpenAIEmbeddings
from langchain_text_splitters import CharacterTextSplitter
```

```python
from langchain_community.document_loaders import TextLoader

loader = TextLoader("../../../extras/modules/state_of_the_union.txt")
documents = loader.load()
text_splitter = CharacterTextSplitter(chunk_size=1000, chunk_overlap=0)
docs = text_splitter.split_documents(documents)

embeddings = OpenAIEmbeddings()
```

```python
db = USearch.from_documents(docs, embeddings)

query = "What did the president say about Ketanji Brown Jackson"
docs = db.similarity_search(query)
```

```python
print(docs[0].page_content)
```

## 带分数的相似度搜索

`similarity_search_with_score` 方法不仅可以返回文档，还可以返回查询与文档之间的距离分数。返回的距离分数为 L2 距离，因此分数越低越好。

```python
docs_and_scores = db.similarity_search_with_score(query)
```

```python
docs_and_scores[0]
```
