> [ClickHouse](https://clickhouse.com/) 是一款快速且资源高效的开源数据库，用于实时应用与分析，具备完整的 SQL 支持以及广泛的功能，可协助用户编写分析查询。
> 它包含数据结构与距离搜索函数（如 `L2Distance`）以及
> [近似最近邻搜索索引](https://clickhouse.com/docs/en/engines/table-engines/mergetree-family/annindexes)
> 这使得 ClickHouse 能够作为高性能且可扩展的向量数据库，通过 SQL 存储和搜索向量。


## 安装与配置

我们需要安装 `clickhouse-connect` Python 包。

<CodeGroup>
```bash pip
pip install clickhouse-connect
```

```bash uv
uv add clickhouse-connect
```
</CodeGroup>
