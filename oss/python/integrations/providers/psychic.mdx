>[Psychic](https://www.psychic.dev/) 是一个通过 OAuth 与 `Notion`、`Zendesk`、
> `Confluence` 和 `Google Drive` 等 SaaS 工具集成的平台，并将来自这些应用程序的文档同步到您的 SQL 或向量数据库。您可以把它看作是非结构化数据的 Plaid。

## 安装与设置

<CodeGroup>
```bash pip
pip install psychicapi
```

```bash uv
uv add psychicapi
```
</CodeGroup>

Psychic 设置简单——您导入 `react` 库并使用从 [Psychic 仪表板](https://dashboard.psychic.dev/) 获取的 `Sidekick API` 密钥配置它。连接应用程序后，您可以从仪表板查看这些连接并使用服务器端库检索数据。

1. 在 [仪表板](https://dashboard.psychic.dev/) 创建账户。
2. 使用 [react 库](https://docs.psychic.dev/sidekick-link) 将 Psychic link 模态框添加到您的前端 React 应用。您将使用它来连接 SaaS 应用。
3. 创建连接后，您可以按照 [示例 notebook](/oss/python/integrations/document_loaders/psychic) 使用 `PsychicLoader`


## 相比其他文档加载器的优势

1. **通用 API：** 不需要为每个 SaaS 应用构建 OAuth 流程和学习 API，您只需集成一次 Psychic 并利用我们的通用 API 检索数据。
2. **数据同步：** 客户 SaaS 应用中的数据可能很快就会过时。使用 Psychic，您可以配置 Webhook，以每天或实时保持文档更新。
3. **简化的 OAuth：** Psychic 端到端处理 OAuth，这样您就不必花时间为每个集成创建 OAuth 客户端、保持访问令牌最新并处理 OAuth 重定向逻辑。
