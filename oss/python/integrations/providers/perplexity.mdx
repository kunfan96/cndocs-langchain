>[Perplexity](https://www.perplexity.ai/pro) 是最强大的网络搜索方式，
> 提供无限次 Pro 搜索、升级的 AI 模型、无限文件上传、
> 图像生成和 API 积分。
>
> 您可以查看 [可用模型列表](https://docs.perplexity.ai/docs/model-cards)。

## 安装与设置

安装 Perplexity x LangChain 集成包：

<CodeGroup>
```bash pip
pip install langchain-perplexity
```

```bash uv
uv add langchain-perplexity
```
</CodeGroup>

从 [此处](https://docs.perplexity.ai/docs/getting-started) 获取您的 API 密钥。
