>[Confident AI](https://confident-ai.com) 是 `DeepEval` 的创建者。
>
>[DeepEval](https://github.com/confident-ai/deepeval) 是一个用于对 LLM 进行单元测试的包。
> 使用 `DeepEval`，每个人都可以通过单元测试和集成测试更快地迭代，从而构建健壮的语言模型。`DeepEval` 为从合成数据创建到测试的每个迭代步骤提供支持。

## 安装与设置

您需要获取 [DeepEval API 凭据](https://app.confident-ai.com)。

您需要安装 `DeepEval` Python 包：

<CodeGroup>
```bash pip
pip install deepeval
```

```bash uv
uv add deepeval
```
</CodeGroup>

## 回调

查看[示例](/oss/python/integrations/callbacks/confident)。

```python
from langchain.callbacks.confident_callback import DeepEvalCallbackHandler
```
