## 概述

本笔记本概述了如何直接在 LangChain 中使用 ADS4GPTs 工具和工具包。在您的 LangGraph 应用程序中，您很可能会使用我们预构建的 LangGraph 代理。

## 设置

### 安装 ADS4GPTs 包

使用 pip 安装 ADS4GPTs 包。

```python
# 安装 ADS4GPTs 包
# 使用 pip 安装 ADS4GPTs 包
!pip install ads4gpts-langchain
```

设置 API 认证的环境变量（[获取 API 密钥](https://www.ads4gpts.com)）。

```python
# 设置环境变量
# 提示用户安全地输入其 ADS4GPTs API 密钥
if not os.environ.get("ADS4GPTS_API_KEY"):
    os.environ["ADS4GPTS_API_KEY"] = getpass("输入您的 ADS4GPTs API 密钥: ")
```

## 实例化

导入必要的库，包括 ADS4GPTs 工具和工具包。

初始化 ADS4GPTs 工具，例如 Ads4gptsInlineSponsoredResponseTool。我们将使用一个工具进行演示，因为该过程对于其他工具是相同的。

```python
# 导入所需库

import os
from getpass import getpass

from ads4gpts_langchain import Ads4gptsInlineSponsoredResponseTool, Ads4gptsToolkit
```

```python
# 初始化 ADS4GPTs 工具
# 初始化 Ads4gptsInlineSponsoredResponseTool
inline_sponsored_response_tool = Ads4gptsInlineSponsoredResponseTool(
    ads4gpts_api_key=os.environ["ADS4GPTS_API_KEY"],
)
```

### 工具包实例化

使用所需参数初始化 Ads4gptsToolkit。

```python
# 工具包初始化
# 使用所需参数初始化 Ads4gptsToolkit
toolkit = Ads4gptsToolkit(
    ads4gpts_api_key=os.environ["ADS4GPTS_API_KEY"],
)

# 从工具包中检索工具
tools = toolkit.get_tools()

# 打印已初始化的工具
for tool in tools:
    print(f"已初始化工具: {tool.__class__.__name__}")
```

```text
已初始化工具: Ads4gptsInlineSponsoredResponseTool
已初始化工具: Ads4gptsSuggestedPromptTool
```

## 调用

使用示例输入运行 ADS4GPTs 工具并显示结果。

```python
# 运行 ADS4GPTs 工具
# 工具的示例输入数据
sample_input = {
    "id": "test_id",
    "user_gender": "female",
    "user_age": "25-34",
    "user_persona": "test_persona",
    "ad_recommendation": "test_recommendation",
    "undesired_ads": "test_undesired_ads",
    "context": "test_context",
    "num_ads": 1,
    "style": "neutral",
}

# 运行 Ads4gptsInlineSponsoredResponseTool
inline_sponsored_response_result = inline_sponsored_response_tool._run(
    **sample_input, ad_format="INLINE_SPONSORED_RESPONSE"
)
print("内联赞助响应结果:", inline_sponsored_response_result)
```


### 异步运行 ADS4GPTs 工具

使用示例输入异步运行 ADS4GPTs 工具并显示结果。

```python
import asyncio


# 定义一个异步函数来异步运行工具
async def run_ads4gpts_tools_async():
    # 异步运行 Ads4gptsInlineSponsoredResponseTool
    inline_sponsored_response_result = await inline_sponsored_response_tool._arun(
        **sample_input, ad_format="INLINE_SPONSORED_RESPONSE"
    )
    print("异步内联赞助响应结果:", inline_sponsored_response_result)
```

### 工具包调用

使用 Ads4gptsToolkit 获取并运行工具。

```python
# 工具的示例输入数据
sample_input = {
    "id": "test_id",
    "user_gender": "female",
    "user_age": "25-34",
    "user_persona": "test_persona",
    "ad_recommendation": "test_recommendation",
    "undesired_ads": "test_undesired_ads",
    "context": "test_context",
    "num_ads": 1,
    "style": "neutral",
}

# 运行一个工具并打印结果
tool = tools[0]
result = tool._run(**sample_input)
print(f"来自 {tool.__class__.__name__} 的结果:", result)


# 定义一个异步函数来异步运行工具
async def run_toolkit_tools_async():
    result = await tool._arun(**sample_input)
    print(f"来自 {tool.__class__.__name__} 的异步结果:", result)


# 执行异步函数
await run_toolkit_tools_async()
```

```python
    os.environ["OPENAI_API_KEY"] = getpass("输入您的 OPENAI_API_KEY API 密钥: ")
```

```python
import os

from langchain_openai import ChatOpenAI

openai_model = ChatOpenAI(model="gpt-4.1", openai_api_key=os.environ["OPENAI_API_KEY"])
model = openai_model.bind_tools(tools)
model_response = model.invoke(
```
