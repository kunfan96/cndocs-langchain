>[Konko](https://www.konko.ai/) API 是一个完全托管的 Web API，旨在帮助应用程序开发者：

1. **选择**适合其应用程序的开源或专有 LLM
2. **构建**应用程序更快，通过集成领先的应用程序框架和完全托管的 API
3. **微调**较小的开源 LLM，以极低的成本实现行业领先的性能
4. **部署生产级 API**，利用符合 SOC 2 标准的多云基础设施，无需进行基础设施设置或管理，即可满足安全、隐私、吞吐量和延迟的 SLA 要求

本示例介绍如何使用 LangChain 与 \`Konko\` 补全 [模型](https://docs.konko.ai/docs/list-of-models#konko-hosted-models-for-completion) 进行交互

要运行此笔记本，您需要 Konko API 密钥。登录我们的 Web 应用以 [创建 API 密钥](https://platform.konko.ai/settings/api-keys) 来访问模型

#### 设置环境变量

1. 您可以为以下项设置环境变量
    1. KONKO_API_KEY（必需）
    2. OPENAI_API_KEY（可选）
2. 在当前 shell 会话中，使用 export 命令：

```shell
export KONKO_API_KEY={your_KONKO_API_KEY_here}
export OPENAI_API_KEY={your_OPENAI_API_KEY_here} # 可选
```

## 调用模型

在 [Konko 概览页面](https://docs.konko.ai/docs/list-of-models) 上查找模型

另一种查看 Konko 实例上运行的模型列表的方法是通过此 [端点](https://docs.konko.ai/reference/get-models)。

从这里开始，我们可以初始化我们的模型：

```python
from langchain_community.llms import Konko

llm = Konko(model="mistralai/mistral-7b-v0.1", temperature=0.1, max_tokens=128)

input_ = """You are a helpful assistant. Explain Big Bang Theory briefly."""
print(llm.invoke(input_))
```
