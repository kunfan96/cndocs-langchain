本页介绍如何在 LangChain 中使用 Teradata 向量存储。
内容分为两部分：安装与设置，以及对特定 Teradata 封装器的参考说明。

## 安装
- 使用 `pip install langchain-teradata` 安装 Python 包

## 设置
第一步是创建到 Teradata Vantage 系统的连接。

    您需要 Teradata 凭证，包括主机名、用户名、密码，以及可选的云部署 API 令牌。有关详细设置说明，请参阅 [Teradata 向量存储用户指南](https://docs.teradata.com/r/Enterprise_IntelliFlex_VMware/Teradata-Vector-Store-User-Guide/Introduction-to-the-Enterprise-Vector-Store-User-Guide) 和 [Vantage Cloud Lake 入门指南](https://docs.teradata.com/r/Lake-Getting-Started-with-VantageCloud-Lake/)

## 封装器

### 向量存储

存在一个围绕 Teradata 向量数据库的封装器，允许您将其用作向量存储，无论是用于相似性搜索还是 RAG 管道。

导入此向量存储：
```python
from langchain_teradata import TeradataVectorStore
```

### 使用

有关 Teradata VectorStore 的更详细演练，请参阅 [langchain-teradata 用户指南](https://docs.teradata.com/r/Enterprise_IntelliFlex_VMware/Teradata-Package-for-LangChain-User-Guide) 和 [langchain-teradata 工作流](https://github.com/Teradata/langchain-teradata)
