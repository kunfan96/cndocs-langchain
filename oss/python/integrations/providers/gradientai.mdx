## 环境配置

langchain-gradient 使用 DigitalOcean Gradient 平台。

在 DigitalOcean 上创建账户，从 Gradient 平台获取 `DIGITALOCEAN_INFERENCE_KEY` API 密钥，并安装 `langchain-gradient` 集成包。

### 凭据

前往 [DigitalOcean Gradient](https://www.digitalocean.com/products/gradient)

1. 注册/登录 DigitalOcean Cloud Console
2. 进入 Gradient 平台并导航至 Serverless Inference。
3. 点击 Create model access key，输入名称并创建密钥。

完成上述操作后，设置 `DIGITALOCEAN_INFERENCE_KEY` 环境变量：

```python
import os
os.environ["DIGITALOCEAN_INFERENCE_KEY"] = "your-api-key"
```

### 安装

LangChain Gradient 集成位于 `langchain-gradient` 包中：

<CodeGroup>
```bash pip
pip install -qU langchain-gradient
```

```bash uv
uv add langchain-gradient
```
</CodeGroup>

## 实例化

```python
from langchain_gradient import ChatGradient

llm = ChatGradient(
    model="llama3.3-70b-instruct",
    api_key=os.environ.get("DIGITALOCEAN_INFERENCE_KEY")
)
```

## 调用

```python
messages = [
    (
        "system",
        "你是一位富有创意的故事讲述者。请以引人入胜且充满想象力的方式续写你收到的任何故事提示。",
    ),
    ("human", "很久以前，在一个神秘森林边缘的村庄里，一个叫米拉的小女孩发现了一块发光的石头……"),
]
ai_msg = llm.invoke(messages)
ai_msg
print(ai_msg.content)
```

## 链式调用

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate(
    [
        (
            "system",
            "你是一位知识渊博的助手。请仔细阅读提供的上下文并回答用户的问题。如果答案在上下文中，请引用相关句子。如果没有，请回复“未在上下文中找到”。",
        ),
        ("human", "上下文：{context}\n问题：{question}"),
    ]
)

chain = prompt | llm
chain.invoke(
    {
        "context": (
            "埃菲尔铁塔位于巴黎，于 1889 年完工。它由古斯塔夫·埃菲尔的工程公司设计。该塔是世界上最著名的建筑之一。自由女神像是法国赠送给美国的礼物。"
        ),
        "question": "谁设计了埃菲尔铁塔，它何时完工？"
    }
)
```
