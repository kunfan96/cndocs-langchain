本页介绍如何在 LangChain 中使用 [Epsilla](https://github.com/epsilla-cloud/vectordb)。
内容分为两部分：安装与设置，以及对特定 Epsilla 封装器的参考。

## 安装与设置

- 使用 `pip/pip3 install pyepsilla` 安装 Python SDK

## 封装器

### 向量存储

存在一个围绕 Epsilla 向量数据库的封装器，可以将其用作向量存储，
无论是用于语义搜索还是示例选择。

导入此向量存储：

```python
from langchain_community.vectorstores import Epsilla
```

关于 Epsilla 封装器的更详细介绍，请参阅 [此 notebook](/oss/python/integrations/vectorstores/epsilla)
