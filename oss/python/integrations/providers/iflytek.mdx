>[iFlytek](https://www.iflytek.com) 是一家成立于 1999 年的中国信息技术公司。
> 它开发语音识别软件以及覆盖教育、通信、音乐、智能玩具等行业的语音互联网/移动产品。


## 安装与配置

- 从 [iFlyTek SparkLLM API 控制台](https://console.xfyun.cn/services/bm3) 获取 `SparkLLM` 的 `app_id`、`api_key` 和 `api_secret`（更多信息，请参阅 [iFlyTek SparkLLM 介绍](https://xinghuo.xfyun.cn/sparkapi)）。
- 安装 Python 包（不适用于嵌入模型）：

<CodeGroup>
```bash pip
pip install websocket-client
```

```bash uv
uv add websocket-client
```
</CodeGroup>

## LLMs

请参阅 [使用示例](/oss/python/integrations/llms/sparkllm)。

```python
from langchain_community.llms import SparkLLM
```
