>[Context](https://context.ai/) 为基于 LLM 的产品和功能提供用户分析服务。

## 安装与配置

我们需要安装 `context-python` Python 包：

<CodeGroup>
```bash pip
pip install context-python
```

```bash uv
uv add context-python
```
</CodeGroup>


## 回调

请参阅[使用示例](/oss/python/integrations/callbacks/context)。

```python
from langchain.callbacks import ContextCallbackHandler
```
