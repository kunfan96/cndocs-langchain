>[USearch](https://unum-cloud.github.io/usearch/) 是一个更小、更快的单文件向量搜索引擎。

>`USearch` 的基本功能与 `FAISS` 相同，如果您曾经研究过近似最近邻搜索，其界面应该看起来很熟悉。
>`USearch` 和 `FAISS` 都采用 `HNSW` 算法，但它们在设计原则上存在显著差异。
>`USearch` 紧凑且与 FAISS 广泛兼容，同时不牺牲性能，主要关注用户定义的指标和更少的依赖项。
>
## 安装和设置

我们需要安装 `usearch` Python 包。

<CodeGroup>
```bash pip
pip install usearch
```

```bash uv
uv add usearch
```
</CodeGroup>
