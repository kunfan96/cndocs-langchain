>[LLMonitor](https://llmonitor.com?utm_source=langchain&utm_medium=py&utm_campaign=docs) 是一个开源的可观测性平台，提供成本与使用量分析、用户追踪、追踪以及评估工具。

## 安装与设置

在 [llmonitor.com](https://llmonitor.com?utm_source=langchain&utm_medium=py&utm_campaign=docs) 上创建账户，然后复制新应用的 `tracking id`。

获取后，通过运行以下命令将其设置为环境变量：

```bash
export LLMONITOR_APP_ID="..."
```


## 回调

请参阅[使用示例](/oss/python/integrations/callbacks/llmonitor)。

```python
from langchain.callbacks import LLMonitorCallbackHandler
```
