> [Embedchain](https://github.com/embedchain/embedchain) 是一个用于创建数据管道的 RAG 框架。
> 它可以加载、索引、检索和同步所有数据。
>
>它以[开源包](https://github.com/embedchain/embedchain)和
> [托管平台解决方案](https://app.embedchain.ai/)两种形式提供。

## 安装与设置

使用 pip 安装包：

<CodeGroup>
```bash pip
pip install embedchain
```

```bash uv
uv add embedchain
```
</CodeGroup>
