## 故障排除

要解决此错误：

1. **确保格式正确**：所有传递给 chat models 的输入必须是 LangChain 消息类的数组或支持的类似消息的格式
2. 验证你的消息没有发生意外的字符串化或转换
3. 检查错误的堆栈跟踪，并添加日志语句来检查消息对象在传递给模型之前的状态
