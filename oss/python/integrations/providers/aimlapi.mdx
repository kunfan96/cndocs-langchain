>[AI/ML API](https://aimlapi.com/app/?utm_source=langchain&utm_medium=github&utm_campaign=integration) 提供单一 API，用于访问 300 多种托管的基础模型（包括 DeepSeek、Gemini、GPT 等），并具备企业级的高可用性与吞吐量。

## 安装与配置

- 安装 AI/ML API 集成包。

  ```bash
  pip install langchain-aimlapi
  ```

- 在 [aimlapi.com](https://aimlapi.com/app/?utm_source=langchain&utm_medium=github&utm_campaign=integration) 创建账户并生成 API 密钥。
- 通过设置 `AIMLAPI_API_KEY` 环境变量完成身份验证。

```python
import os

os.environ["AIMLAPI_API_KEY"] = "aimlapi_..."
```
