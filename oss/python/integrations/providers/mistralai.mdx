>[Mistral AI](https://docs.mistral.ai/api/) 是一个为其强大的开源模型提供托管服务的平台。


## 安装与设置

与 API 通信需要有效的 [API 密钥](https://console.mistral.ai/users/api-keys/)。

还需要安装 `langchain-mistralai` 包：

<CodeGroup>
```bash pip
pip install langchain-mistralai
```

```bash uv
uv add langchain-mistralai
```
</CodeGroup>

## 对话模型

### ChatMistralAI

查看[使用示例](/oss/python/integrations/chat/mistralai)。

```python
from langchain_mistralai.chat_models import ChatMistralAI
```

## 嵌入模型

### MistralAIEmbeddings

查看[使用示例](/oss/python/integrations/text_embedding/mistralai)。

```python
from langchain_mistralai import MistralAIEmbeddings
```
