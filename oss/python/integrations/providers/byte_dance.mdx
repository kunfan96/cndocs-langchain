## 文档加载器

>[飞书](https://www.larksuite.com/) 是一款企业协作平台
> 由 \`字节跳动\` 开发。

### 用于文档的飞书套件

请参阅[使用示例](/oss/python/integrations/document_loaders/larksuite/#load-from-document)。

```python
from langchain_community.document_loaders.larksuite import LarkSuiteDocLoader
```

### 用于 Wiki 的飞书套件

请参阅[使用示例](/oss/python/integrations/document_loaders/larksuite/#load-from-wiki)。

```python
from langchain_community.document_loaders.larksuite import LarkSuiteWikiLoader
```
