>[Fiddler](https://www.fiddler.ai/) 提供统一平台，用于在企业规模上监控、解释、分析
> 和改进 ML 部署。

## 安装与设置

使用 [Fiddler](https://demo.fiddler.ai) 设置你的模型：

* 用于连接 Fiddler 的 URL
* 你的组织 ID
* 你的授权令牌

安装 Python 包：

<CodeGroup>
```bash pip
pip install fiddler-client
```

```bash uv
uv add fiddler-client
```
</CodeGroup>

## 回调

```python
from langchain_community.callbacks.fiddler_callback import FiddlerCallbackHandler
```

查看[示例](/oss/python/integrations/callbacks/fiddler)。
