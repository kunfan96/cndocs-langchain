> Smabbler 的图驱动平台通过将数据转化为结构化知识基础来提升 AI 开发效率。

## Galaxia

> Galaxia 知识库是一个用于 RAG 的集成知识库和检索机制。与标准解决方案不同，它基于使用符号 NLP 和知识表示方案构建的知识图谱。提供的文本会被分析并转换为包含文本、语言和语义信息的图谱。这种丰富的结构允许基于语义信息而非向量相似度/距离进行检索。

使用 Galaxia 实现 RAG，需要先将文件上传到 [Galaxia](https://beta.cloud.smabbler.com/home)，在那里分析它们，然后构建模型（知识图谱）。模型构建完成后，你可以使用 `GalaxiaRetriever` 连接到 API 并开始检索。

更多信息：[文档](https://smabbler.gitbook.io/smabbler)

## 安装
<CodeGroup>
```bash pip
pip install langchain-galaxia-retriever
```

```bash uv
uv add langchain-galaxia-retriever
```
</CodeGroup>

## 使用

```
from langchain_galaxia_retriever.retriever import GalaxiaRetriever

gr = GalaxiaRetriever(
    api_url="beta.api.smabbler.com",
    api_key="<key>",
    knowledge_base_id="<knowledge_base_id>",
    n_retries=10,
    wait_time=5,
)

result = gr.invoke('<test question>')
print(result)
