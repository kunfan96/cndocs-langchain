>[Blackboard Learn](https://en.wikipedia.org/wiki/Blackboard_Learn)（原 `Blackboard Learning Management System`）
>是由 Blackboard Inc. 开发的基于网络的虚拟学习环境和教学管理系统。
>该软件具备课程管理功能、可定制的开放架构以及可扩展的设计，支持与
>学生信息系统和身份验证协议集成。它可以安装在本地服务器上，
>由 `Blackboard ASP Solutions` 托管，或作为软件即服务在 Amazon Web Services 上提供。
>其主要目的包括为传统面对面授课的课程增加在线元素，以及开发几乎或完全不需要面对面会议的纯在线课程。

## 安装与设置

无需进行任何特殊设置。
