>[Steam (Wikipedia)](https://en.wikipedia.org/wiki/Steam_(service)) 是由 `Valve Corporation` 开发的视频游戏数字发行服务和商店。它为 Valve 的游戏提供自动更新，并扩展到发行第三方游戏。`Steam` 提供各种功能，如带有 Valve Anti-Cheat 措施的游戏服务器匹配、社交网络和游戏流媒体服务。

>[Steam](https://store.steampowered.com/about/) 是玩游戏、讨论游戏和创作游戏的终极目的地。

Steam 工具包包含两个工具：

- `Game Details`
- `Recommended Games`

本指南提供了使用 Steam API 与 LangChain 的逐步说明，以根据您当前的 Steam 游戏库存检索 Steam 游戏推荐，或收集您提供的某些 Steam 游戏的信息。

## 设置

我们需要安装两个 Python 库。

## 导入

```python
pip install -qU python-steam-api python-decouple steamspypi
```

## 分配环境变量

要使用此工具包，请准备好您的 OpenAI API 密钥、Steam API 密钥（来自 [Steam 开发者 API 密钥页面](https://steamcommunity.com/dev/apikey)）并保留您自己的 SteamID。收到 Steam API 密钥后，您可以将其作为环境变量输入下方。
工具包将读取 "STEAM_KEY" API 密钥作为环境变量进行身份验证，因此请在此处设置。您还需要设置您的 "OPENAI_API_KEY" 和 "STEAM_ID"。

```python
import os

os.environ["STEAM_KEY"] = ""
os.environ["STEAM_ID"] = ""
os.environ["OPENAI_API_KEY"] = ""
```

## 初始化

初始化 LLM、SteamWebAPIWrapper、SteamToolkit，最重要的是 LangChain 代理来处理您的查询！

## 示例

```python
from langchain_community.agent_toolkits.steam.toolkit import SteamToolkit
from langchain_community.utilities.steam import SteamWebAPIWrapper

steam = SteamWebAPIWrapper()
tools = [steam.run]
```

```python
from langchain.agents import create_agent

agent = create_agent("gpt-4.1-mini", tools)
```

```python
events = agent.stream(
    {"messages": [("user", "can you give the information about the game Terraria?")]},
    stream_mode="values",
)
for event in events:
    event["messages"][-1].pretty_print()
```
