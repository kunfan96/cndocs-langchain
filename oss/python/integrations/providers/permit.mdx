[Permit.io](https://permit.io/) 提供细粒度访问控制和策略执行。通过 LangChain，您可以集成 Permit 检查，以确保只有授权用户才能在您的 LLM 应用程序中访问或检索数据。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-permit
pip install permit
```

```bash uv
uv add langchain-permit
uv add permit
```
</CodeGroup>

为您的 Permit PDP 和凭证设置环境变量：

```python
export PERMIT_API_KEY="your_permit_api_key"
export PERMIT_PDP_URL="http://localhost:7766"   # 或您的真实 PDP 端点
```

确保您的 PDP 正在运行并已配置。有关策略设置，请参阅
[Permit 文档](https://docs.permit.io/sdk/python/quickstart-python/#2-setup-your-pdp-policy-decision-point-container)。
