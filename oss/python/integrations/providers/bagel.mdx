> [Bagel](https://www.bagel.net/)（`AI 开放向量数据库`）类似于 AI 数据领域的 GitHub。
它是一个协作平台，用户可以在其中创建、
共享和管理向量数据集。它支持独立开发者的私有项目、
企业的内部协作以及数据 DAO 的公开贡献。

## 安装与配置

<CodeGroup>
```bash pip
pip install bagelML
```

```bash uv
uv add bagelML
```
</CodeGroup>
