>[OpenWeatherMap](https://openweathermap.org/) 是一个开源的天气服务提供商。

## 安装与设置

<CodeGroup>
```bash pip
pip install pyowm
```

```bash uv
uv add pyowm
```
</CodeGroup>

我们必须设置 `OpenWeatherMap API token`。

## 文档加载器

请参阅[使用示例](/oss/python/integrations/document_loaders/weather)。

