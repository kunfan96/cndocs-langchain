>[Stochastic 加速平台](https://docs.stochastic.ai/docs/introduction/) 旨在简化深度学习模型的生命周期。涵盖从模型的上传与版本管理，到训练、压缩与加速，直至投入生产环境的完整流程。

本示例介绍如何使用 LangChain 与 `StochasticAI` 模型进行交互。

请从您的 [Stochastic AI 工作区个人资料设置](https://app.stochastic.ai/workspace/profile/settings?tab=profile) 中获取 `API_KEY` 和 `API_URL`。

```python
YOUR_API_URL = getpass()
```

```text
 ········
```

```python
from langchain_classic.chains import LLMChain
from langchain_community.llms import StochasticAI
from langchain_core.prompts import PromptTemplate
```

```python
template = """Question: {question}

Answer: Let's think step by step."""

prompt = PromptTemplate.from_template(template)
```

```python
llm = StochasticAI(api_url=YOUR_API_URL)
```

```python
llm_chain = LLMChain(prompt=prompt, llm=llm)
```

```python
question = "What NFL team won the Super Bowl in the year Justin Beiber was born?"

llm_chain.run(question)
```

```text
"\n\nStep 1: In 1999, the St. Louis Rams won the Super Bowl.\n\nStep 2: In 1999, Beiber was born.\n\nStep 3: The Rams were in Los Angeles at the time.\n\nStep 4: So they didn't play in the Super Bowl that year.\n"
```
