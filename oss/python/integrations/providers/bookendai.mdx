LangChain 实现了与 [bookend.ai](https://bookend.ai/) 提供的嵌入模型的集成。


## 安装与配置


你需要在 [bookend.ai](https://bookend.ai/) 网站上注册并获取 `API_KEY`。

## 嵌入模型

请参阅[使用示例](/oss/python/integrations/embeddings/bookend)。

```python
from langchain_community.embeddings import BookendEmbeddings
```
