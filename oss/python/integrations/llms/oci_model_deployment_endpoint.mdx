> 有关最新更新、示例和实验性功能，请参阅 [ADS LangChain 集成](https://accelerated-data-science.readthedocs.io/en/latest/user_guide/large_language_model/langchain_models.html)。

本笔记本将介绍如何使用托管在 [OCI Data Science Model Deployment](https://docs.oracle.com/en-us/iaas/data-science/using/model-dep-about.htm) 上的 LLM。

为了进行身份验证，使用 [oracle-ads](https://accelerated-data-science.readthedocs.io/en/latest/user_guide/cli/authentication.html) 库来自动加载调用端点所需的凭据。

## 前置条件

### 部署模型

您可以使用 OCI Data Science Model Deployment 上的 [AI Quick Actions](https://docs.oracle.com/en-us/iaas/data-science/using/ai-quick-actions.htm) 轻松部署、微调并评估基础模型。如需更多部署示例，请访问 [Oracle GitHub 示例仓库](https://github.com/oracle-samples/oci-data-science-ai-samples/blob/main/ai-quick-actions/llama3-with-smc.md)。

### 策略

请确保已配置访问 OCI Data Science Model Deployment 端点所需的 [策略](https://docs.oracle.com/en-us/iaas/data-science/using/model-dep-policies-auth.htm#model_dep_policies_auth__predict-endpoint)。

## 设置

部署模型后，您需要设置以下调用所需参数：

- **`endpoint`**：来自已部署模型的模型 HTTP 端点，例如 `https://modeldeployment.<region>.oci.customer-oci.com/<md_ocid>/predict`。

### 身份验证

您可以通过 ads 或环境变量设置身份验证。当您在 OCI Data Science Notebook Session 中工作时，可以利用资源主体（resource principal）访问其他 OCI 资源。有关更多选项，请查看 [Accelerated Data Science 身份验证选项](https://accelerated-data-science.readthedocs.io/en/latest/user_guide/cli/authentication.html)。

## 示例

```python
import ads
from langchain_community.llms import OCIModelDeploymentLLM

# 通过 ads 设置身份验证
# 若您在已配置基于资源主体身份验证的
# OCI 服务中运行，请使用资源主体
ads.set_auth("resource_principal")

# 创建 OCI 模型部署端点的实例
# 将端点 URI 和模型名称替换为您自己的
# 使用通用类作为入口点，您将能够
# 在实例化期间通过 model_kwargs 传递模型参数。
llm = OCIModelDeploymentLLM(
    endpoint="https://modeldeployment.<region>.oci.customer-oci.com/<md_ocid>/predict",
    model="odsc-llm",
)

# 运行 LLM
llm.invoke("谁是美国第一任总统？")
```

```python
import ads
from langchain_community.llms import OCIModelDeploymentVLLM

# 通过 ads 设置身份验证
# 若您在已配置基于资源主体身份验证的
# OCI 服务中运行，请使用资源主体
ads.set_auth("resource_principal")

# 创建 OCI 模型部署端点的实例
# 将端点 URI 和模型名称替换为您自己的
# 使用特定于框架的类作为入口点，您将能够
# 在构造函数中传递模型参数。
llm = OCIModelDeploymentVLLM(
    endpoint="https://modeldeployment.<region>.oci.customer-oci.com/<md_ocid>/predict",
)

# 运行 LLM
llm.invoke("谁是美国第一任总统？")
```

```python
import os

from langchain_community.llms import OCIModelDeploymentTGI

# 通过环境变量设置身份验证
# 当您在本地工作站或不支持
# 资源主体的平台上工作时，请使用 API Key 设置。
os.environ["OCI_IAM_TYPE"] = "api_key"
os.environ["OCI_CONFIG_PROFILE"] = "default"
os.environ["OCI_CONFIG_LOCATION"] = "~/.oci"

# 通过环境变量设置端点
# 将端点 URI 替换为您自己的
os.environ["OCI_LLM_ENDPOINT"] = (
    "https://modeldeployment.<region>.oci.customer-oci.com/<md_ocid>/predict"
)

# 创建 OCI 模型部署端点的实例
# 使用特定于框架的类作为入口点，您将能够
# 在构造函数中传递模型参数。
llm = OCIModelDeploymentTGI()

# 运行 LLM
llm.invoke("谁是美国第一任总统？")
```

### 异步调用

```python
await llm.ainvoke("给我讲个笑话。")
```

### 流式调用

```python
for chunk in llm.stream("给我讲个笑话。"):
    print(chunk, end="", flush=True)
```
