[Oxylabs](https://oxylabs.io/) 是一家市场领先的网络情报采集平台，遵循最高的商业、道德与合规标准，助力全球企业挖掘数据驱动的洞察。

[langchain-oxylabs](https://pypi.org/project/langchain-oxylabs/) 实现了使 LLM 能够与 Oxylabs Web Scraper API 交互的工具。


## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-oxylabs
```

```bash uv
uv add langchain-oxylabs
```
</CodeGroup>

## 工具

有关可用工具的详细信息，请参阅 [Oxylabs 工具文档](/oss/python/integrations/tools/oxylabs/)。
