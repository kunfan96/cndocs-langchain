>[USearch](https://unum-cloud.github.io/usearch/) 是一个更小更快的单文件向量搜索引擎。

>`USearch` 的基础功能与 `FAISS` 相同，如果你曾研究过近似最近邻搜索，其接口应该很熟悉。
> `USearch` 和 `FAISS` 都采用 `HNSW` 算法，但设计理念有所不同。`USearch` 体积紧凑，与 FAISS 广泛兼容且不牺牲性能，主要关注用户自定义度量和更少的依赖。
>
## 安装与设置

我们需要安装 `usearch` Python 包。

<CodeGroup>
```bash pip
pip install usearch
```

```bash uv
uv add usearch
```
</CodeGroup>
