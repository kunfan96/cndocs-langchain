>[Rockset](https://rockset.com/product/) 是一个实时分析数据库服务，用于以低延迟、高并发的方式大规模提供分析查询。它为结构化和半结构化数据构建了 Converged Index™，并为向量嵌入提供了高效的存储。其对无模式数据运行 SQL 的支持使其成为运行带元数据过滤器的向量搜索的完美选择。

## 安装与设置

确保您拥有 Rockset 账户，并前往 Web 控制台获取 API 密钥。详细信息可在[网站](https://rockset.com/docs/rest-api/)上找到。

<CodeGroup>
```bash pip
pip install rockset
```

```bash uv
uv add rockset
```
</CodeGroup>
