>[OctoAI](https://docs.octoai.cloud/docs) 提供对高效算力的便捷访问，
> 并允许用户将其选择的 AI 模型集成到应用程序中。
> `OctoAI` 计算服务可帮助您轻松运行、调优和扩展 AI 应用。


## 安装与配置

- 安装 `openai` Python 包：
  ```bash
  pip install openai
  ```
- 在 `OctoAI` 注册，并从 [您的 OctoAI 账户页面](https://octoai.cloud/settings) 获取 API Token。


## 聊天模型

查看 [使用示例](/oss/python/integrations/chat/octoai)。

```python
from langchain_community.chat_models import ChatOctoAI
```
## LLMs

查看 [使用示例](/oss/python/integrations/llms/octoai)。

```python
from langchain_community.llms.octoai_endpoint import OctoAIEndpoint
```
## Embedding 模型

```python
from langchain_community.embeddings.octoai_embeddings import OctoAIEmbeddings
```
