[Scrapeless](https://scrapeless.com) 提供灵活且功能丰富的数据采集服务，支持广泛的参数自定义和多种格式导出。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-scrapeless
```

```bash uv
uv add langchain-scrapeless
```
</CodeGroup>

您需要设置 Scrapeless API 密钥：

```python
import os
os.environ["SCRAPELESS_API_KEY"] = "your-api-key"
```

## 工具

Scrapeless 集成提供以下几种工具：

- [ScrapelessDeepSerpGoogleSearchTool](/oss/python/integrations/tools/scrapeless_scraping_api) - 支持全面提取所有结果类型的 Google SERP 数据。
- [ScrapelessDeepSerpGoogleTrendsTool](/oss/python/integrations/tools/scrapeless_scraping_api) - 检索 Google 关键词趋势数据，包括时间段内的热度、地区兴趣及相关搜索。
- [ScrapelessUniversalScrapingTool](/oss/python/integrations/tools/scrapeless_universal_scraping) - 访问和提取通常会屏蔽爬虫的 JS 渲染网站数据。
- [ScrapelessCrawlerCrawlTool](/oss/python/integrations/tools/scrapeless_crawl) - 爬取网站及其链接页面以提取全面数据。
- [ScrapelessCrawlerScrapeTool](/oss/python/integrations/tools/scrapeless_crawl) - 从单个网页提取信息。
