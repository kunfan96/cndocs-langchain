[AgentQL](https://www.agentql.com/) 提供网页交互功能，并可使用 [AgentQL 查询](https://docs.agentql.com/agentql-query) 或自然语言提示词从任意网页中提取结构化数据。AgentQL 可跨多种语言和网页使用，且不会因时间推移或页面变更而失效。

## 安装与配置

安装集成包：

<CodeGroup>
```bash pip
pip install langchain-agentql
```

```bash uv
uv add langchain-agentql
```
</CodeGroup>

## API 密钥

从我们的 [开发者门户](https://dev.agentql.com/) 获取 API 密钥，并将其添加到环境变量中：
```
export AGENTQL_API_KEY="your-api-key-here"
```

## DocumentLoader
AgentQL 的文档加载器可使用 AgentQL 查询从任意网页中提取结构化数据。

```python
from langchain_agentql.document_loaders import AgentQLLoader
```
请参阅我们的[文档加载器文档及使用示例](/oss/python/integrations/document_loaders/agentql)。

## 工具与工具包
AgentQL 工具可提供网页交互功能，并可使用 AgentQL 查询或自然语言提示词从任意网页中提取结构化数据。

```python
from langchain_agentql.tools import ExtractWebDataTool, ExtractWebDataBrowserTool, GetWebElementBrowserTool
from langchain_agentql import AgentQLBrowserToolkit
```
请参阅我们的[工具文档及使用示例](/oss/python/integrations/tools/agentql)。
