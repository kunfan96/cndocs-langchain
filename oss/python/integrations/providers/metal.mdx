本页介绍如何在 LangChain 中使用 [Metal](https://getmetal.io)。

## 什么是 Metal？

Metal 是一个面向生产环境构建的托管检索与记忆平台。您可以轻松将数据索引到 `Metal` 中，并对其进行语义搜索和检索。

## 快速开始

通过 [创建 Metal 账户](https://app.getmetal.io/signup) 开始使用。

随后，您可以轻松利用 `MetalRetriever` 类来开始检索您的数据，用于语义搜索、提示上下文等。该类接收一个 `Metal` 实例以及一个要传递给 Metal API 的参数字典。

```python
from langchain_classic.retrievers import MetalRetriever
from metal_sdk.metal import Metal


metal = Metal("API_KEY", "CLIENT_ID", "INDEX_ID");
retriever = MetalRetriever(metal, params={"limit": 2})

docs = retriever.invoke("search term")
```
