---
title: MODEL_AUTHENTICATION
---

<Note>
    目前仅用于 `langchainjs`（JavaScript/TypeScript）。
</Note>

你的模型提供商拒绝了你对其服务的访问。

此错误通常发生在认证凭据或 API 密钥存在问题时。

## 故障排查

* 确认你的 API 密钥或认证凭据准确有效。
* 如果使用基于环境变量的认证，请验证：
    - 变量名拼写正确
    - 变量已被赋值
    - 第三方包（如 `dotenv`）没有干扰加载过程
* 如果使用代理或非标准端点，确保你的自定义提供商不期望使用其他认证方案。
* 通过显式传递凭据来绕过环境变量问题：

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(api_key="YOUR_KEY_HERE")
```
