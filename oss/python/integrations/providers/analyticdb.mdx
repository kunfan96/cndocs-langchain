>[AnalyticDB for PostgreSQL](https://www.alibabacloud.com/help/en/analyticdb-for-postgresql/latest/product-introduction-overview)
> 是一项来自 [Alibaba Cloud](https://www.alibabacloud.com/) 的大规模并行处理（MPP）数据仓库服务，
> 专为在线分析海量数据而设计。

>`AnalyticDB for PostgreSQL` 基于开源的 `Greenplum Database`
> 项目开发，并由 `Alibaba Cloud` 进行了深度扩展增强。AnalyticDB
> for PostgreSQL 兼容 ANSI SQL 2003 语法以及 PostgreSQL 和
> Oracle 数据库生态系统。AnalyticDB for PostgreSQL 同时支持行存储和
> 列存储。AnalyticDB for PostgreSQL 能够以高性能水平离线处理 PB 级数据，并支持高并发。

本页介绍如何在 LangChain 中使用 AnalyticDB 生态系统。

## 安装与配置

您需要安装 `sqlalchemy` Python 包。

<CodeGroup>
```bash pip
pip install sqlalchemy
```

```bash uv
uv add sqlalchemy
```
</CodeGroup>
