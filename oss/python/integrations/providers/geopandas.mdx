>[GeoPandas](https://geopandas.org/) 是一个开源项目，旨在使 Python 中的地理空间数据处理更加简便。`GeoPandas` 扩展了 `pandas` 使用的数据类型，以允许对几何类型进行空间操作。
> 几何操作由 `shapely` 执行。

## 安装与设置

我们需要安装几个 Python 包。

<CodeGroup>
```bash pip
pip install -U sodapy pandas geopandas
```

```bash uv
uv add sodapy pandas geopandas
```
</CodeGroup>
