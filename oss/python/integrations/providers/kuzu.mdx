> [Kùzu](https://kuzudb.com/) 是一款可嵌入、可扩展且速度极快的图数据库。
> 它采用宽松的 MIT 许可证授权，您可以在 [GitHub](https://github.com/kuzudb/kuzu) 上查看其源代码。

> Kùzu 的关键特性：
>- 性能与可扩展性：为图数据实现了现代、最先进的连接算法。
>- 易用性：极易设置和上手，因为无需服务器（嵌入式架构）。
>- 互操作性：可方便地从外部列式格式、CSV、JSON 和关系数据库中扫描和复制数据。
>- 结构化属性图模型：实现了属性图模型，并增加了结构支持。
>- Cypher 支持：允许使用声明式查询语言 Cypher 方便地查询图数据。

> 请访问他们的[文档](https://docs.kuzudb.com/)以开始使用 Kùzu。


## 安装与配置

按以下方式安装 Python SDK：

<CodeGroup>
```bash pip
pip install -U langchain-kuzu
```

```bash uv
uv add langchain-kuzu
```
</CodeGroup>
