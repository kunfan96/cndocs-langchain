**Dappier：以动态、实时数据模型赋能 AI**

Dappier 提供了一个前沿的平台，让开发者能够立即访问涵盖新闻、娱乐、金融、市场数据、天气等领域的广泛实时数据模型。借助我们的预训练数据模型，您可以加速您的 AI 应用程序的发展，确保它们提供精准且最新的响应，并减少不准确性。

Dappier 数据模型帮助您构建下一代 LLM 应用程序，通过与全球领先品牌的可信内容进行交互，增强任何 GPT App 或 AI 工作流。通过简单的 API 获取可操作的专有数据，释放您的创造力。使用来自值得信赖来源的专有数据来增强您的 AI 是确保事实准确、最新响应并减少幻觉的最佳方式，无论问题是什么。

由开发者为开发者
Dappier 旨在简化从数据整合到变现的旅程，提供清晰明了的路径来部署和从中获利。体验新互联网的 monetization 基础设施，请访问 **[dappier.com/](https://dappier.com/)**。

本示例介绍了如何使用 LangChain 与 Dappier AI 模型进行交互

-----------------------------------------------------------------------------------

要使用我们的 Dappier AI 数据模型，您需要一个 API 密钥。请访问 Dappier 平台 ([platform.dappier.com/](https://platform.dappier.com/)) 登录并在个人资料中创建 API 密钥。

您可以参阅更多关于 API 的详细信息：[docs.dappier.com/introduction](https://docs.dappier.com/introduction)

要与我们的 Dappier 聊天模型进行交互，您可以在初始化类时直接通过名为 dappier_api_key 的参数传递密钥，或者将其设置为环境变量。

```bash
export DAPPIER_API_KEY="..."
```

```python
from langchain_community.chat_models.dappier import ChatDappierAI
from langchain.messages import HumanMessage
```

```python
chat = ChatDappierAI(
    dappier_endpoint="https://api.dappier.com/app/datamodelconversation",
    dappier_model="dm_01hpsxyfm2fwdt2zet9cg6fdxt",
    dappier_api_key="...",
)
```

```python
messages = [HumanMessage(content="Who won the super bowl in 2024?")]
chat.invoke(messages)
```

```text
AIMessage(content='Hey there! The Kansas City Chiefs won Super Bowl LVIII in 2024. They beat the San Francisco 49ers in overtime with a final score of 25-22. It was quite the game! 🏈')
```

await chat.ainvoke(messages)
