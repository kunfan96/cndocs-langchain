>[CTranslate2](https://opennmt.net/CTranslate2/quickstart.html) 是一个用于 Transformer 模型高效推理的 C++ 和 Python 库。
>
>该项目实现了一个自定义运行时，应用了权重量化、层融合、批量重排序等多种性能优化技术，以加速 CPU 和 GPU 上 Transformer 模型的推理并降低内存使用。
>
>功能完整列表和支持的模型可在[项目仓库](https://opennmt.net/CTranslate2/guides/transformers.html)中找到。首先，请查阅官方[快速入门指南](https://opennmt.net/CTranslate2/quickstart.html)。


## 安装与设置

安装 Python 包：

<CodeGroup>
```bash pip
pip install ctranslate2
```

```bash uv
uv add ctranslate2
```
</CodeGroup>


## LLM

查看[使用示例](/oss/python/integrations/llms/ctranslate2)。

```python
from langchain_community.llms import CTranslate2
```
