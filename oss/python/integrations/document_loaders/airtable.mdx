```python
pip install -qU  pyairtable
```

```python
from langchain_community.document_loaders import AirtableLoader
```

* 按照 [Airtable API 密钥文档](https://support.airtable.com/docs/creating-and-using-api-keys-and-access-tokens) 获取您的 API 密钥。
* 按照 [Airtable Web API 介绍](https://airtable.com/developers/web/api/introduction) 获取您的 Base ID。
* 按照 [查找 Airtable Base ID 和 Table ID 指南](https://www.highviewapps.com/kb/where-can-i-find-the-airtable-base-id-and-table-id/#:~:text=Both%20the%20Airtable%20Base%20ID,URL%20that%20begins%20with%20tbl) 从表 URL 获取您的 Table ID。

```python
api_key = "xxx"
base_id = "xxx"
table_id = "xxx"
view = "xxx"  # 可选
```

```python
loader = AirtableLoader(api_key, table_id, base_id, view=view)
docs = loader.load()
```

将每行表数据作为 `dict` 返回。

```python
len(docs)
```

```text
3
```

```python
eval(docs[0].page_content)
```
