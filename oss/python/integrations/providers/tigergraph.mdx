>[TigerGraph](https://www.tigergraph.com/tigergraph-db/) 是一个原生分布式且高性能的图数据库。
> 以顶点和边的图格式存储数据，能够建立丰富的关联关系，
> 非常适合为 LLM 响应提供基础。

## 安装与设置

请按照[如何连接到 `TigerGraph` 数据库](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection)的说明进行操作。

安装 Python SDK：

<CodeGroup>
```bash pip
pip install pyTigerGraph
```

```bash uv
uv add pyTigerGraph
```
</CodeGroup>

## 图存储

### TigerGraph

请参阅[使用示例](/oss/python/integrations/graphs/tigergraph)。

```python
from langchain_community.graphs import TigerGraph
```
