> [Zep](https://www.getzep.com) 是 AI 助手应用的长期记忆服务。
> 借助 Zep，您可以为 AI 助手提供回忆过去对话的能力，无论对话多么久远，
> 同时还能减少幻觉、延迟和成本。

> 对 Zep Cloud 感兴趣？请参阅 [Zep Cloud 安装指南](https://help.getzep.com/sdks)

**注意：** `ZepVectorStore` 与 `Documents` 配合使用，旨在用作 `Retriever`。
它提供了与 Zep 的 `ZepMemory` 类分开的功能，后者设计用于持久化、丰富和搜索用户的聊天历史。

## 为什么选择 zep 的 VectorStore？

Zep 会使用 Zep 服务器本地的低延迟模型自动嵌入添加到 Zep 向量存储中的文档。
Zep TS/JS 客户端可在非 Node 边缘环境中使用。这两者结合 Zep 的聊天记忆功能，
使 Zep 成为构建对话式 LLM 应用的理想选择，其中延迟和性能至关重要。

### 支持的搜索类型

Zep 支持相似性搜索和最大边际相关性 (MMR) 搜索。MMR 搜索特别适用于
检索增强生成应用，因为它会重新排序结果以确保返回文档的多样性。

## 安装

请按照 [Zep 开源快速入门指南](https://help.getzep.com/quickstart) 安装并开始使用 Zep。

## 用法

您需要 Zep API URL 和可选的 API 密钥才能使用 Zep VectorStore。有关更多信息，请参阅 [Zep 文档](https://help.getzep.com/welcome)。

在下面的示例中，我们使用 Zep 的自动嵌入功能，该功能使用
低延迟嵌入模型在 Zep 服务器上自动嵌入文档。由于 LangChain 需要传入 [`Embeddings`](https://reference.langchain.com/javascript/langchain-core/embeddings/Embeddings) 实例，我们传入 `FakeEmbeddings`。

**注意：** 如果您传入除 `FakeEmbeddings` 之外的 [`Embeddings`](https://reference.langchain.com/javascript/langchain-core/embeddings/Embeddings) 实例，该类将用于嵌入文档。
您还必须将文档集合设置为 `isAutoEmbedded === false`。请参阅下面的 `OpenAIEmbeddings` 示例。

### 示例：从文档创建 ZepVectorStore 并进行查询

<Tip>
请参阅[此部分了解安装 LangChain 包的一般说明](/oss/javascript/langchain/install)。
</Tip>

```bash npm
npm install @langchain/openai @langchain/community @langchain/core
```

```typescript
import { ZepVectorStore } from "@langchain/community/vectorstores/zep";
import { FakeEmbeddings } from "@langchain/core/utils/testing";
import { TextLoader } from "@langchain/classic/document_loaders/fs/text";
import { randomUUID } from "crypto";

const loader = new TextLoader("src/document_loaders/example_data/example.txt");
const docs = await loader.load();
export const run = async () => {
  const collectionName = `collection${randomUUID().split("-")[0]}`;

  const zepConfig = {
    apiUrl: "http://localhost:8000", // 这应该是您的 Zep 实现的 URL
    collectionName,
    embeddingDimensions: 1536, // 这必须与您使用的嵌入宽度匹配
    isAutoEmbedded: true, // 如果为 true，向量存储将在添加文档时自动嵌入文档
  };

  const embeddings = new FakeEmbeddings();

  const vectorStore = await ZepVectorStore.fromDocuments(
    docs,
    embeddings,
    zepConfig
  );

  // 等待文档被嵌入
  // eslint-disable-next-line no-constant-condition
  while (true) {
    const c = await vectorStore.client.document.getCollection(collectionName);
    console.log(
      `嵌入状态: ${c.document_embedded_count}/${c.document_count} 个文档已嵌入`
    );
    // eslint-disable-next-line no-promise-executor-return
    await new Promise((resolve) => setTimeout(resolve, 1000));
    if (c.status === "ready") {
      break;
    }
  }

  const results = await vectorStore.similaritySearchWithScore("bar", 3);

  console.log("相似性结果:");
  console.log(JSON.stringify(results));

  const results2 = await vectorStore.maxMarginalRelevanceSearch("bar", {
    k: 3,
  });

  console.log("MMR 结果:");
  console.log(JSON.stringify(results2));
};
```

### 示例：使用元数据过滤器查询 ZepVectorStore

```typescript
import { FakeEmbeddings } from "@langchain/core/utils/testing";
import { randomUUID } from "crypto";
import { Document } from "@langchain/core/documents";

const docs = [
  new Document({

    pageContent:

    pageContent:

  }),
  new Document({
    metadata: { album: "Physical Graffiti", year: 1975 },
    pageContent:
      "Kashmir，来自 Physical Graffiti，展示了 Led Zeppelin 独特的摇滚与世界音乐融合。",
  }),
  new Document({
    metadata: { album: "Houses of the Holy", year: 1973 },
    pageContent:
      "The Rain Song 是来自 Houses of the Holy 的一首优美、忧郁的作品。",
  }),
  new Document({
    metadata: { band: "Black Sabbath", album: "Paranoid", year: 1970 },
    pageContent:
      "Paranoid 是 Black Sabbath 的第二张录音室专辑，包含一些他们最著名的歌曲。",
  }),
  new Document({
    metadata: {
      band: "Iron Maiden",
      album: "The Number of the Beast",
      year: 1982,
    },
    pageContent:
      "The Number of the Beast 通常被认为是 Iron Maiden 最好的专辑。",
  }),
  new Document({
    metadata: { band: "Metallica", album: "Master of Puppets", year: 1986 },
    pageContent:
      "Master of Puppets 被广泛认为是 Metallica 最杰出的作品。",
  }),
  new Document({
    metadata: { band: "Megadeth", album: "Rust in Peace", year: 1990 },
    pageContent:
      "Rust in Peace 是 Megadeth 的第四张录音室专辑，具有复杂的吉他演奏。",
  }),
];

export const run = async () => {
  const collectionName = `collection${randomUUID().split("-")[0]}`;

  const zepConfig = {
    apiUrl: "http://localhost:8000", // 这应该是您的 Zep 实现的 URL
    collectionName,
    embeddingDimensions: 1536, // 这必须与您使用的嵌入宽度匹配
    isAutoEmbedded: true, // 如果为 true，向量存储将在添加文档时自动嵌入文档
  };

  const embeddings = new FakeEmbeddings();

  const vectorStore = await ZepVectorStore.fromDocuments(
    docs,
    embeddings,
    zepConfig
  );

  // 等待文档被嵌入
  // eslint-disable-next-line no-constant-condition
  while (true) {
    const c = await vectorStore.client.document.getCollection(collectionName);
    console.log(
      `嵌入状态: ${c.document_embedded_count}/${c.document_count} 个文档已嵌入`
    );
    // eslint-disable-next-line no-promise-executor-return
    await new Promise((resolve) => setTimeout(resolve, 1000));
    if (c.status === "ready") {
      break;
    }
  }

  vectorStore
    .similaritySearchWithScore("sad music", 3, {
      where: { jsonpath: "$[*] ? (@.year == 1973)" }, // 我们应该看到一个结果：The Rain Song
    })
    .then((results) => {
      console.log(`\n\n相似性结果:\n${JSON.stringify(results)}`);
    })
    .catch((e) => {
      if (e.name === "NotFoundError") {
        console.log("未找到结果");
      } else {
        throw e;
      }
    });

  // 我们这里没有进行过滤，而是演示 MMR 的工作原理。
  // 我们也可以像上面的相似性搜索一样，为 MMR 搜索添加过滤器。
  vectorStore
    .maxMarginalRelevanceSearch("sad music", {
      k: 3,
    })
    .then((results) => {
      console.log(`\n\nMMR 结果:\n${JSON.stringify(results)}`);
    })
    .catch((e) => {
      if (e.name === "NotFoundError") {
        console.log("未找到结果");
      } else {
        throw e;
      }
    });
};
```
