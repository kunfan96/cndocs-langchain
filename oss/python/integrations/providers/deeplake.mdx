[Deeplake](https://www.deeplake.ai/) 是一款专为 AI 和深度学习应用优化的数据库。


## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-deeplake
```

```bash uv
uv add langchain-deeplake
```
</CodeGroup>
