>[Iugu](https://www.iugu.com/) 是一家巴西服务及软件即服务（SaaS）公司，为电商网站和移动应用提供支付处理软件及应用程序编程接口。

本文档介绍如何将 `Iugu REST API` 中的数据加载为可被 LangChain 处理的格式，并提供向量化使用示例。

```python
from langchain.indexes import VectorstoreIndexCreator
from langchain_community.document_loaders import IuguLoader
```

Iugu API 需要访问令牌，可在 Iugu 控制面板中找到。

此文档加载器还需要一个 `resource` 选项，用于指定要加载的数据类型。

可用资源如下：

`Documentation` [文档](https://dev.iugu.com/reference/metadados)

```python
iugu_loader = IuguLoader("charges")
```

```python
# Create a vectorstore retriever from the loader
# see https://python.langchain.com/en/latest/modules/data_connection/getting_started.html for more details

index = VectorstoreIndexCreator().from_loaders([iugu_loader])
iugu_doc_retriever = index.vectorstore.as_retriever()
```
