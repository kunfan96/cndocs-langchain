与 YDB 相关的所有功能。

> [YDB](https://ydb.tech/) 是一个多功能开源分布式 SQL 数据库，将高可用性和可扩展性与强一致性和 ACID 事务相结合。它可以同时处理事务型（OLTP）、分析型（OLAP）和流式工作负载。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-ydb
```

```bash uv
uv add langchain-ydb
```
</CodeGroup>

## 向量存储

导入 YDB 向量存储：

```python
from langchain_ydb.vectorstores import YDB
```

有关 YDB 向量存储的更详细说明，请参阅[此笔记本](/oss/python/integrations/vectorstores/ydb)。
