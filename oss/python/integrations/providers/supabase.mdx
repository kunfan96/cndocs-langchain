>[Supabase](https://supabase.com/docs) 是一个开源的 `Firebase` 替代品。
> `Supabase` 构建在 `PostgreSQL` 之上，后者提供强大的 `SQL` 查询能力，并支持与现有工具和框架的简单接口。

>[PostgreSQL](https://en.wikipedia.org/wiki/PostgreSQL) 也称为 `Postgres`，
> 是一个免费且开源的关系型数据库管理系统 (RDBMS)，
> 强调可扩展性和 `SQL` 合规性。

## 安装和设置

我们需要安装 `supabase` Python 包。

<CodeGroup>
```bash pip
pip install supabase
```

```bash uv
uv add supabase
```
</CodeGroup>
