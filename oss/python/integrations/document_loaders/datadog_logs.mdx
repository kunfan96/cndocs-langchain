>[Datadog](https://www.datadoghq.com/) 是一个面向云规模应用程序的监控与分析平台。

此加载器使用 `datadog_api_client` Python 包从 Datadog 中获取应用程序日志。您需要使用 `Datadog API key` 和 `APP key` 初始化加载器，并传入查询条件以提取所需的日志。

```python
from langchain_community.document_loaders import DatadogLogsLoader
```

```python
pip install -qU  datadog-api-client
```

```python
DD_API_KEY = "..."
DD_APP_KEY = "..."
```

```python
query = "service:agent status:error"

loader = DatadogLogsLoader(
    query=query,
    api_key=DD_API_KEY,
    app_key=DD_APP_KEY,
    from_time=1688732708951,  # Optional, timestamp in milliseconds
    to_time=1688736308951,  # Optional, timestamp in milliseconds
    limit=100,  # Optional, default is 100
)
```

```python
documents = loader.load()
documents
```
