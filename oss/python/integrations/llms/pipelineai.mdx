>[PipelineAI](https://pipeline.ai) 允许您在云端大规模运行 ML 模型，并提供对[多种 LLM 模型](https://pipeline.ai)的 API 访问。

本 notebook 介绍如何将 LangChain 与 [PipelineAI](https://docs.pipeline.ai/docs) 配合使用。

## PipelineAI 示例

[此示例展示了 PipelineAI 与 LangChain 的集成方式](https://docs.pipeline.ai/docs/langchain)，由 PipelineAI 创建。

## 设置

使用 `PipelineAI` API（即 `Pipeline Cloud`）需要 `pipeline-ai` 库。使用 `pip install pipeline-ai` 安装。

```python
# Install the package
pip install -qU  pipeline-ai
```

## 示例

### 导入

```python
import os

from langchain_community.llms import PipelineAI
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import PromptTemplate
```

### 设置环境 API 密钥

请从 PipelineAI 获取您的 API 密钥。查阅[云端快速入门指南](https://docs.pipeline.ai/docs/cloud-quickstart)。您将获得 30 天免费试用，包含 10 小时的无服务器 GPU 计算，用于测试不同模型。

```python
os.environ["PIPELINE_API_KEY"] = "YOUR_API_KEY_HERE"
```

## 创建 PipelineAI 实例

实例化 PipelineAI 时，需要指定要使用的 pipeline 的 id 或 tag，例如 `pipeline_key = "public/gpt-j:base"`。您还可以选择传入其他特定于 pipeline 的关键字参数：

```python
llm = PipelineAI(pipeline_key="YOUR_PIPELINE_KEY", pipeline_kwargs={...})
```

### 创建提示词模板

我们将创建一个问答提示词模板。

```python
template = """Question: {question}

Answer: Let's think step by step."""

prompt = PromptTemplate.from_template(template)
```

### 初始化 LLMChain

```python
llm_chain = prompt | llm | StrOutputParser()
```

### 运行 LLMChain

提供问题并运行 LLMChain。

```python
question = "What NFL team won the Super Bowl in the year Justin Beiber was born?"

llm_chain.invoke(question)
```
