## 安装

要访问 GreenNode 模型，您需要创建一个 GreenNode 账户、获取 API 密钥并安装 `langchain-greennode` 集成包。

### 凭据

前往 [此页面](https://aiplatform.console.greennode.ai/api-keys) 注册 GreenNode AI 平台并生成 API 密钥。完成这些步骤后，请设置 GREENNODE_API_KEY 环境变量：

```python
import getpass
import os

if not os.getenv("GREENNODE_API_KEY"):
    os.environ["GREENNODE_API_KEY"] = getpass.getpass("请输入您的 GreenNode API 密钥: ")
```

如果您希望在模型调用时自动进行跟踪，还可以设置 [LangSmith](/langsmith/home) 的 API 密钥：

```python
os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("请输入您的 LangSmith API 密钥: ")
```

### 安装

LangChain GreenNode 集成位于 `langchain-greennode` 包中：

```python
pip install -qU langchain-greennode
```

## 实例化

现在我们可以实例化我们的模型对象并生成聊天完成结果：

```python
from langchain_greennode import ChatGreenNode

# 初始化聊天模型
llm = ChatGreenNode(
    # api_key="YOUR_API_KEY",  # 可以直接传递 API 密钥
    model="deepseek-ai/DeepSeek-R1-Distill-Qwen-32B",  # 选择可用的模型之一
    temperature=0.6,
    top_p=0.95,
)
```

### 聊天消息

您可以使用不同的消息类型来结构化与模型的对话：

```python
from langchain.messages import AIMessage, HumanMessage, SystemMessage

messages = [
    SystemMessage(content="您是一个具有科学专长的帮助型 AI 助手。"),
    HumanMessage(content="黑洞是什么？"),
    AIMessage(
        content="黑洞是时空中的区域，在那里引力如此之强，以至于连光都无法逃脱。",
    ),
    HumanMessage(content="它们是如何形成的？"),
]

response = llm.invoke(messages)
print(response.content[:100])
```

```text
Black holes are formed through several processes, depending on their type. The most common way bla
```

## 链接

您可以在 LangChain 链和代理中使用 `ChatGreenNode`：

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate(
    [
        (
            "system",
            "您是一个能够将 {input_language} 翻译成 {output_language} 的帮助型助手。",
        ),
        ("human", "{input}"),
    ]
)

chain = prompt | llm
chain.invoke(
    {
        "input_language": "英语",
        "output_language": "德语",
        "input": "我爱编程。",
    }
)
```
