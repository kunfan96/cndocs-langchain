>[FireCrawl](https://firecrawl.dev/?ref=langchain) 可抓取并转换任意网站为可直接供大语言模型使用的数据。
> 它会抓取所有可访问的子页面，并返回干净的 Markdown
> 以及各页面的元数据。无需站点地图。


## 安装与配置

安装 Python SDK：

<CodeGroup>
```bash pip
pip install firecrawl-py==0.0.20
```

```bash uv
uv add "firecrawl-py==0.0.20"
```
</CodeGroup>

## 文档加载器

请参阅[使用示例](/oss/python/integrations/document_loaders/firecrawl)。

```python
from langchain_community.document_loaders import FireCrawlLoader
```
