>[AI21 Labs](https://www.ai21.com/about) 是一家专注于自然语言处理（NLP）的公司，其开发的 AI 系统能够理解和生成自然语言。

本页介绍如何在 `LangChain` 中使用 `AI21` 生态系统。

## 安装与配置

- 获取 AI21 API 密钥，并将其设置为环境变量（`AI21_API_KEY`）
- 安装 Python 包：

<CodeGroup>
```bash pip
pip install langchain-ai21
```

```bash uv
uv add langchain-ai21
```
</CodeGroup>

## 聊天模型

### AI21 聊天

请参阅[使用示例](/oss/python/integrations/chat/ai21)。

```python
from langchain_ai21 import ChatAI21
```

## 已弃用的功能

:::caution 以下功能已被弃用。
:::

### AI21 LLM

```python
from langchain_ai21 import AI21LLM
```

### AI21 上下文回答

```python
from langchain_ai21 import AI21ContextualAnswers
```

## 文本分割器

### AI21 语义文本分割器

```python
from langchain_ai21 import AI21SemanticTextSplitter
```
