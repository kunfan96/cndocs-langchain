>[Konko](https://www.konko.ai/) API 是一个完全托管的 Web API，旨在帮助应用程序开发者：

1. **选择**适合其应用的开源或专有 LLM
2. 通过与主流应用框架的集成及完全托管的 API **更快地构建**应用
3. **微调**较小的开源 LLM，以极低的成本实现行业领先的性能
4. 在无需基础设施搭建或管理的情况下，使用 Konko AI 符合 SOC 2 标准的多云基础设施**部署生产级 API**，满足安全性、隐私性、吞吐量和延迟的 SLA

本示例介绍如何使用 LangChain 与 `Konko` 的 ChatCompletion [模型](https://docs.konko.ai/docs/list-of-models#konko-hosted-models-for-chatcompletion)进行交互

要运行本 notebook，您需要 Konko API 密钥。登录我们的 Web 应用以[创建 API 密钥](https://platform.konko.ai/settings/api-keys)来访问模型

```python
from langchain_community.chat_models import ChatKonko
from langchain.messages import HumanMessage, SystemMessage
```

#### 设置环境变量

1. 您可以设置以下环境变量：
   1. KONKO_API_KEY（必填）
   2. OPENAI_API_KEY（可选）
2. 在当前 shell 会话中，使用 export 命令：

```shell
export KONKO_API_KEY={your_KONKO_API_KEY_here}
export OPENAI_API_KEY={your_OPENAI_API_KEY_here} #Optional
```

## 调用模型

在 [Konko 概览页面](https://docs.konko.ai/docs/list-of-models)查找模型

另一种查找在 Konko 实例上运行的模型列表的方式是通过此[接口](https://docs.konko.ai/reference/get-models)。

从这里，我们可以初始化我们的模型：

```python
chat = ChatKonko(max_tokens=400, model="meta-llama/llama-2-13b-chat")
```

```python
messages = [
    SystemMessage(content="You are a helpful assistant."),
    HumanMessage(content="Explain Big Bang Theory briefly"),
]
chat(messages)
```
