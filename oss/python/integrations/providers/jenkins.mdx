[Jenkins](https://www.jenkins.io/) 是一个开源自动化平台，可帮助软件团队简化开发工作流程。
它在 DevOps 社区中被广泛采用，用于通过 CI/CD 流水线自动化应用程序的构建、测试和部署。


## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-jenkins
```

```bash uv
uv add langchain-jenkins
```
</CodeGroup>
