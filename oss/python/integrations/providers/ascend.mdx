>[Ascend](https://https://www.hiascend.com/) 是由华为提供的自然处理单元（Natural Process Unit）

本页介绍如何在 LangChain 中使用 Ascend NPU。

### 安装

请使用以下命令安装 torch-npu：

<CodeGroup>
```bash pip
pip install torch-npu
```

```bash uv
uv add torch-npu
```
</CodeGroup>

请按照如下指定的安装说明进行操作：
* 按照 [CANN 安装文档](https://www.hiascend.com/document/detail/zh/canncommercial/700/quickstart/quickstart/quickstart_18_0002.html) 中的说明安装 CANN。

### 嵌入模型

请参阅[使用示例](/oss/python/integrations/embeddings/ascend)。

```python
from langchain_community.embeddings import AscendEmbeddings
```
