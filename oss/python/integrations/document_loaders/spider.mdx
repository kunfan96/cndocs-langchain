[Spider](https://spider.cloud/) 是[速度最快](https://github.com/spider-rs/spider/blob/main/benches/BENCHMARKS.md)且最经济实惠的爬虫与抓取工具，可返回适合 LLM 使用的数据。

## 安装

```python
pip install spider-client
```

## 使用方法

使用 Spider 需要从 [spider.cloud](https://spider.cloud/) 获取 API 密钥。

```python
from langchain_community.document_loaders import SpiderLoader

loader = SpiderLoader(
    api_key="YOUR_API_KEY",
    url="https://spider.cloud",
    mode="scrape",  # 如果未提供 API 密钥，则从环境变量 SPIDER_API_KEY 中读取
)

data = loader.load()
print(data)
```

## 模式

- `scrape`：默认模式，抓取单个 URL
- `crawl`：爬取所提供域名 URL 的所有子页面

## 爬虫选项

`params` 参数是一个字典，可传入加载器中。请参阅 [Spider 文档](https://spider.cloud/docs/api) 查看所有可用参数。
