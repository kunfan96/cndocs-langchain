**Scrapeless** 提供灵活且功能丰富的数据采集服务，具备广泛的参数自定义和多格式导出支持。这些能力使 LangChain 能够更有效地集成和利用外部数据。核心功能模块包括：

**DeepSerp**

- **Google 搜索**：支持全面提取 Google SERP 数据，涵盖所有结果类型。
  - 支持选择本地化 Google 域名（例如 `google.com`、`google.ad`）以获取特定地区的搜索结果。
  - 支持分页以检索第一页之后的结果。
  - 支持搜索结果过滤开关，以控制是否排除重复或相似内容。
- **Google 趋势**：从 Google 检索关键词趋势数据，包括随时间变化的受欢迎程度、区域兴趣和相关搜索。
  - 支持多关键词比较。
  - 支持多种数据类型：`interest_over_time`、`interest_by_region`、`related_queries` 和 `related_topics`。
  - 允许按特定 Google 属性（网络、YouTube、新闻、购物）过滤，以进行特定来源的趋势分析。

**通用爬取**

- 专为现代、JavaScript 密集型网站设计，支持动态内容提取。
  - 全球高级代理支持，用于绕过地理限制并提高可靠性。

**爬虫**

- **爬取**：递归爬取网站及其链接页面，以提取全站内容。
  - 支持可配置的爬取深度和范围 URL 定位。
- **抓取**：从单个网页高精度提取内容。
  - 支持“仅提取主要内容”以排除广告、页脚和其他非必要元素。
  - 允许批量抓取多个独立 URL。

## 实例化

这里我们展示如何实例化 Scrapeless 通用爬取工具的实例。此工具允许您使用无头浏览器（具备 JavaScript 渲染功能）、可自定义的输出类型和地理特定代理支持来抓取任何网站。

该工具在实例化时接受以下参数：

- `url`（必需，字符串）：要抓取的网站 URL。
- `headless`（可选，布尔值）：是否使用无头浏览器。默认为 True。
- `js_render`（可选，布尔值）：是否启用 JavaScript 渲染。默认为 True。
- `js_wait_until`（可选，字符串）：定义何时认为 JavaScript 渲染的页面已准备好。默认为 `'domcontentloaded'`。选项包括：
  - `load`：等待页面完全加载。
  - `domcontentloaded`：等待 DOM 完全加载。
  - `networkidle0`：等待网络空闲。
  - `networkidle2`：等待网络空闲 2 秒。
- `outputs`（可选，字符串）：要从页面提取的特定数据类型。选项包括：
  - `phone_numbers`
  - `headings`
  - `images`
  - `audios`
  - `videos`
  - `links`
  - `menus`
  - `hashtags`
  - `emails`
  - `metadata`
  - `tables`
  - `favicon`
- `response_type`（可选，字符串）：定义响应的格式。默认为 `'html'`。选项包括：
  - `html`：返回页面的原始 HTML。
  - `plaintext`：返回纯文本内容。
  - `markdown`：返回页面的 Markdown 版本。
  - `png`：返回 PNG 截图。
  - `jpeg`：返回 JPEG 截图。
- `response_image_full_page`（可选，布尔值）：使用截图输出（png 或 jpeg）时，是否捕获并返回全页图像。默认为 False。
- `selector`（可选，字符串）：用于限定页面内抓取范围的特定 CSS 选择器。默认为 `None`。
- `proxy_country`（可选，字符串）：用于地理特定代理访问的两字母国家代码（例如 `'us'`、`'gb'`、`'de'`、`'jp'`）。默认为 `'ANY'`。

# 基本用法

print(result)

```python

tool = ScrapelessUniversalScrapingTool()

result = tool.invoke({"url": "https://exmaple.com", "response_type": "markdown"})
print(result)
```

### 在代理中使用

```python
from langchain_openai import ChatOpenAI
from langchain_scrapeless import ScrapelessUniversalScrapingTool
from langchain.agents import create_agent


model = ChatOpenAI()

tool = ScrapelessUniversalScrapingTool()

# 将工具与代理一起使用
tools = [tool]
agent = create_agent(model, tools)

for chunk in agent.stream(
    {
        "messages": [
            (
                "human",
                "使用 scrapeless 爬取工具获取 https://www.scrapeless.com/en 并提取 h1 标签。",
            )
        ]
    },
    stream_mode="values",
):
    chunk["messages"][-1].pretty_print()
```
