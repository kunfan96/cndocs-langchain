[Ionic](https://www.ioniccommerce.com/) 是一个即插即用的 AI 助手电子商务市场。通过在您的代理中包含 [Ionic 工具](https://github.com/ioniccommerce/ionic_langchain)，您可以轻松地为用户提供在代理内直接购物和交易的能力，并且您将获得交易分成。

这是一个基本的 Jupyter Notebook，演示了如何将 Ionic 工具集成到您的代理中。有关如何使用 Ionic 设置代理的更多信息，请参阅 Ionic [文档](https://docs.ioniccommerce.com/introduction)。

此 Jupyter Notebook 演示了如何将 Ionic 工具与代理一起使用。

**注意：ionic-langchain 包由 Ionic Commerce 团队维护，而非 LangChain 维护者。**

## 设置代理

```python
from ionic_langchain.tool import Ionic, IonicTool
from langchain_classic import hub
from langchain.agents import AgentExecutor, Tool, create_agent
from langchain_openai import OpenAI

# 基于 ReAct 代理
# https://python.langchain.com/docs/modules/agents/agent_types/react
# 参见论文 "ReAct: Synergizing Reasoning and Acting in Language Models" (https://arxiv.org/abs/2210.03629)
# 如需其他代理类型帮助，请联系 support@ionicapi.com。

open_ai_key = "YOUR KEY HERE"
model = "gpt-3.5-turbo-instruct"
temperature = 0.6

llm = OpenAI(openai_api_key=open_ai_key, model_name=model, temperature=temperature)


ionic_tool = IonicTool().tool()


# 该工具自带提示，
# 但您也可以通过 description 属性直接更新它：

ionic_tool.description = str(
    """
Ionic 是一个电子商务购物工具。助手使用 Ionic Commerce 购物工具从数千家在线零售商中查找、发现和比较产品。当用户正在寻找产品推荐或试图查找特定产品时，助手应使用该工具。

用户可以指定他们希望查看结果的数量、最低价格和最高价格。
Ionic 工具输入是一个逗号分隔的值字符串：
  - 查询字符串（必需，不能包含逗号）
  - 结果数量（默认为 4，不超过 10）
  - 最低价格（以美分为单位，$5 变为 500）
  - 最高价格（以美分为单位）
例如，如果寻找 5 到 10 美元之间的咖啡豆，工具输入将是 `coffee beans, 5, 500, 1000`。

将它们作为 Markdown 格式的列表返回，每个推荐来自工具结果，确保包含完整的 PDP URL。例如：

1. 产品 1：[价格] -- 链接
2. 产品 2：[价格] -- 链接
3. 产品 3：[价格] -- 链接
4. 产品 4：[价格] -- 链接
"""
)

tools = [ionic_tool]

# create_agent 的默认提示
prompt = hub.pull("hwchase17/react")

agent = create_agent(
    llm,
    tools,
    prompt=prompt,
)

agent_executor = AgentExecutor(
    agent=agent, tools=tools, handle_parsing_errors=True, verbose=True, max_iterations=5
)
```

## 运行

```python
input = (
    "我正在寻找一个新的 4k 显示器，你能为我找到一些低于 1000 美元的选择吗"
)
agent_executor.invoke({"input": input})
```
