>[01.AI](https://www.lingyiwanwu.com/en)，由李开复博士创立，是一家处于 AI 2.0 前沿的全球公司。他们提供尖端的大语言模型，包括 Yi 系列，参数范围从 6B 到数千亿。01.AI 还提供多模态模型、开放 API 平台以及开源选项，如 Yi-34B/9B/6B 和 Yi-VL。

## 安装与设置

从 [中国站点](https://platform.lingyiwanwu.com/apikeys) 或 [全球站点](https://platform.01.ai/apikeys) 注册并获取 API 密钥。

## LLMs

查看[使用示例](/oss/python/integrations/llms/yi)。

```python
from langchain_community.llms import YiLLM
```

## 聊天模型

查看[使用示例](/oss/python/integrations/chat/yi)。

```python
from langchain_community.chat_models import ChatYi
```
