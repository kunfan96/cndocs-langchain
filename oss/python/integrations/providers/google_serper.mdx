本页介绍如何在 LangChain 中使用 [Serper](https://serper.dev) Google 搜索 API。Serper 是一款低成本的 Google 搜索 API，可用于添加来自 Google 搜索的答案框、知识图谱和自然搜索结果数据。

内容分为两部分：设置步骤，以及针对特定 Google Serper 包装器的参考文档。

## 设置

- 访问 [serper.dev](https://serper.dev) 注册免费账户
- 获取 API 密钥并将其设置为环境变量（`SERPER_API_KEY`）

## 包装器

### 工具类

存在一个名为 `GoogleSerperAPIWrapper` 的工具类，用于封装该 API。要导入此工具类：

```python
from langchain_community.utilities import GoogleSerperAPIWrapper
```

您可以将其作为包含搜索工具的代理的一部分来使用：

```python
from langchain_community.utilities import GoogleSerperAPIWrapper
from langchain_openai import OpenAI
from langchain.agents import create_agent, Tool

import os

os.environ["SERPER_API_KEY"] = ""
os.environ["OPENAI_API_KEY"] = ""

llm = OpenAI(temperature=0)
search = GoogleSerperAPIWrapper()

tools = [
    Tool(
        name="Intermediate Answer",
        func=search.run,
        description="useful for when you need to ask with search",
    )
]

agent = create_agent(
    model=llm,
    tools=tools,
)

agent.invoke("What is the hometown of the reigning men's U.S. Open champion?")
```


#### 输出
```
正在进入新的 AgentExecutor 链...
 是的。
追问：现任美国网球公开赛男子单打冠军是谁？
中间答案：现任冠军卡洛斯·阿尔卡拉斯，2022年男子单打冠军。
追问：卡洛斯·阿尔卡拉斯来自哪里？
中间答案：西班牙埃尔帕尔马尔
因此最终答案是：西班牙埃尔帕尔马尔

> 链执行完成。

'西班牙埃尔帕尔马尔'
```

如需了解此包装器的更详细操作指南，请参阅 [Google Serper 集成](/oss/python/integrations/tools/google_serper)。

### 工具

您还可以轻松将此包装器加载为 Tool（以便与 Agent 配合使用）。
您可以通过以下方式实现：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["google-serper"])
```

有关工具的更多信息，请参阅[本文档概述](/oss/python/integrations/tools)。
