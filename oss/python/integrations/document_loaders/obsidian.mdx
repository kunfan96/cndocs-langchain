>[Obsidian](https://obsidian.md/) 是一个强大且可扩展的知识库，
基于您本地的纯文本文件夹运行。

本 notebook 介绍如何从 `Obsidian` 数据库中加载文档。

由于 `Obsidian` 本质上只是存储在磁盘上的一个 Markdown 文件文件夹，加载器只需提供该目录的路径。

`Obsidian` 文件有时还包含[元数据](https://help.obsidian.md/Editing+and+formatting/Metadata)，即文件顶部的 YAML 块。这些值将被添加到文档的元数据中。（`ObsidianLoader` 也可以传入 `collect_metadata=False` 参数来禁用此行为。）

```python
from langchain_community.document_loaders import ObsidianLoader
```

```python
loader = ObsidianLoader("<path-to-obsidian>")
```

```python
docs = loader.load()
```
