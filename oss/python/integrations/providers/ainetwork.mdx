>[AI Network](https://www.ainetwork.ai/build-on-ain) 是一个专为承载大规模 AI 模型而设计的第一层区块链，
> 利用由 [$AIN 代币](https://www.ainetwork.ai/token)驱动的去中心化 GPU 网络，丰富了 AI 驱动的 `NFTs`（`AINFTs`）生态。


## 安装与设置

您需要安装 `ain-py` Python 包。

<CodeGroup>
```bash pip
pip install ain-py
```

```bash uv
uv add ain-py
```
</CodeGroup>
您需要将 `AIN_BLOCKCHAIN_ACCOUNT_PRIVATE_KEY` 环境变量设置为您的 AIN 区块链账户私钥。

## 工具包

查看[使用示例](/oss/python/integrations/tools/ainetwork)。

```python
from langchain_community.agent_toolkits.ainetwork.toolkit import AINetworkToolkit
```
