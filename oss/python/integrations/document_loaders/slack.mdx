>[Slack](https://slack.com/) 是一款即时通讯程序。

本笔记本介绍如何从 `Slack` 导出的 Zip 文件加载文档。

要获取此 `Slack` 导出，请遵循以下说明：

## 🧑 摄取您自己的数据集的说明

导出您的 Slack 数据。您可以通过访问您的工作区管理页面并点击 导入/导出选项（\{your_slack_domain\}.slack.com/services/export）来完成此操作。然后，选择正确的日期范围并点击 `开始导出`。Slack 将在导出准备就绪时向您发送电子邮件和直接消息。

下载将在您的下载文件夹（或根据您的操作系统配置，下载文件所在的位置）中生成一个 `.zip` 文件。

复制 `.zip` 文件的路径，并将其分配给下面的 `LOCAL_ZIPFILE`。

```python
from langchain_community.document_loaders import SlackDirectoryLoader
```

```python
# 可选设置您的 Slack URL。这将为您提供文档源中的正确 URL。
SLACK_WORKSPACE_URL = "https://xxx.slack.com"
LOCAL_ZIPFILE = ""  # 在此处粘贴您的 Slack zip 文件的本地路径。

loader = SlackDirectoryLoader(LOCAL_ZIPFILE, SLACK_WORKSPACE_URL)
```

```python
docs = loader.load()
docs
```
