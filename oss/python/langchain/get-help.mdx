---
title: 获取帮助
---

与 LangChain 社区联系，访问学习资源，并获得构建信心所需的支持。

## 学习资源

通过我们全面的学习材料开始您的旅程或加深您的知识。

- **[Chat LangChain](https://chat.langchain.com/)**: 实时文档支持，随时提问
- **[API 参考](https://reference.langchain.com/python/)**: LangChain 所有包的完整文档

## 社区支持

通过我们活跃的社区渠道，从其他开发者和 LangChain 团队获得帮助。

- **[社区论坛](https://forum.langchain.com/)**: 提问、分享解决方案并讨论最佳实践
- **[社区 Slack](https://www.langchain.com/join-community)**: 与其他开发者联系并快速获得帮助

## 专业支持

对于企业需求和关键应用，访问专用支持渠道。

- **[支持门户](https://support.langchain.com/)**: 提交工单并跟踪支持请求
- **[LangSmith 状态](https://status.smith.langchain.com/)**: LangSmith 服务和 API 的实时状态

## 贡献

帮助我们改进 LangChain，让每个人都受益。无论是修复错误、添加功能还是改进文档，我们欢迎您的贡献。

- **[贡献指南](/oss/python/contributing/overview)**: 有关贡献 LangChain 的所有信息

## 保持联系

关注我们以获取最新更新、公告和社区亮点。

- **[X (Twitter)](https://x.com/langchain)**: 每日更新和社区亮点
- **[LinkedIn](https://www.linkedin.com/company/langchain/)**: 专业网络和公司更新
