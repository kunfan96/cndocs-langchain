> Smabbler 的图驱动平台通过将数据转化为结构化的知识基础，从而加速 AI 开发。

## Galaxia

> Galaxia 知识库是专为 RAG 设计的集成式知识库与检索机制。与标准方案不同，它基于使用符号自然语言处理（NLP）和知识表示解决方案构建的知识图谱。提供的文本会被分析并转换为包含文本、语言和语义信息的图谱。这种丰富的结构支持基于语义信息的检索，而非基于向量相似度/距离。

使用 Galaxia 实现 RAG 需要先将文件上传至 [Galaxia](https://beta.cloud.smabbler.com/home)，在其平台上进行分析，然后构建模型（知识图谱）。模型构建完成后，您可以使用 `GalaxiaRetriever` 连接 API 并开始检索。

更多信息：[文档](https://smabbler.gitbook.io/smabbler)

## 安装
<CodeGroup>
```bash pip
pip install langchain-galaxia-retriever
```

```bash uv
uv add langchain-galaxia-retriever
```
</CodeGroup>

## 用法

```python
from langchain_galaxia_retriever.retriever import GalaxiaRetriever

gr = GalaxiaRetriever(
    api_url="beta.api.smabbler.com",
    api_key="<key>",
    knowledge_base_id="<knowledge_base_id>",
    n_retries=10,
    wait_time=5,
)

result = gr.invoke('<test question>')
print(result)
