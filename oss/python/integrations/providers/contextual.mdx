Contextual AI 提供最先进的 RAG 组件，专为准确且可靠的企业级 AI 应用而设计。我们的 LangChain 集成为我们的专用模型暴露了独立的 API 端点：

- Grounded Language Model (GLM)：全球最具依据的语言模型，旨在通过优先忠于检索到的知识来最小化幻觉。GLM 提供卓越的事实准确性并附带内联引用，使其成为对可靠性至关重要的企业级 RAG 和智能体应用的理想选择。

- 指令遵循重排序器（Instruction-Following Reranker）：首款遵循自定义指令的智能重排序器，能够根据时效性、来源或文档类型等特定标准对文档进行优先级排序。在行业基准测试中超越竞争对手，我们的重排序器解决了企业知识库中的冲突信息挑战。

由 RAG 技术发明者创立，Contextual AI 的专用组件帮助创新团队加速开发可直接投入生产环境的 RAG 智能体，以提供卓越准确性的响应。

## Grounded 语言模型（GLM）

Grounded 语言模型（GLM）专为最小化企业级 RAG 和智能体应用中的幻觉而设计。GLM 提供：

- 在 FACTS 基准测试中表现强劲，事实准确率达到 88%（[查看基准测试结果](https://venturebeat.com/ai/contextual-ais-new-ai-model-crushes-gpt-4o-in-accuracy-heres-why-it-matters/)）
- 响应严格基于提供的知识来源，并附带内联引用（[阅读产品详情](https://contextual.ai/blog/introducing-grounded-language-model/)）
- 精确的来源引用直接集成在生成的响应中
- 优先采用检索到的上下文而非参数化知识（[查看技术概述](https://contextual.ai/blog/platform-benchmarks-2025/)）
- 在信息不可用时明确承认不确定性

GLM 可作为 RAG 流水线中通用 LLM 的直接替换方案，显著提升关键任务企业级应用的可靠性。

## 指令遵循重排序器

全球首款指令遵循重排序器以前所未有的控制力和准确性彻底革新了文档排名。核心功能包括：

- 使用自然语言指令根据时效性、来源、元数据等对文档进行优先级排序（[查看工作原理](https://contextual.ai/blog/introducing-instruction-following-reranker/)）
- 在 BEIR 基准测试中表现优异，得分为 61.2，大幅领先竞争对手（[查看基准数据](https://contextual.ai/blog/platform-benchmarks-2025/)）
- 智能解决来自多个知识来源的冲突信息
- 无缝集成，作为现有重排序器的直接替换方案
- 通过自然语言命令动态控制文档排名

该重排序器擅长处理可能包含矛盾信息的企业知识库，允许您精确指定在不同场景下哪些来源应享有更高优先级。

## 将 Contextual AI 与 LangChain 配合使用

详细信息请参阅 [Contextual 聊天集成文档](/oss/python/integrations/chat/contextual)。

此集成允许您将 Contextual AI 的 GLM 和指令遵循重排序器轻松融入您的 LangChain 工作流。GLM 确保您的应用提供严格有依据的响应，而重排序器则通过智能优先排序最相关文档来显著提升检索质量。

无论您是为受监管行业还是注重安全的环境构建应用，Contextual AI 都能为您的企业用例提供所需的准确性、控制力和可靠性。

立即开始免费试用，体验面向企业 AI 应用的最具依据的语言模型和指令遵循重排序器。

### Grounded 语言模型

```python
# 集成 Grounded 语言模型
import getpass
import os

from langchain_contextual import ChatContextual

# 设置凭据
if not os.getenv("CONTEXTUAL_AI_API_KEY"):
    os.environ["CONTEXTUAL_AI_API_KEY"] = getpass.getpass(
        "输入您的 Contextual API 密钥: "
    )

# 初始化 Contextual LLM
llm = ChatContextual(
    model="v1",
    api_key="",
)
# 包含系统提示词（可选）
system_prompt = "You are a helpful assistant that uses all of the provided knowledge to answer the user's query to the best of your ability."

# 在此处以字符串数组形式提供来自知识库的知识
knowledge = [
    "There are 2 types of dogs in the world: good dogs and best dogs.",
    "There are 2 types of cats in the world: good cats and best cats.",
]

# 创建您的消息
messages = [
    ("human", "What type of cats are there in the world and what are the types?"),
]

# 通过提供知识字符串调用 GLM，可选系统提示词
# 如果希望关闭 GLM 的评论功能，请将 True 传递给 avoid_commentary 参数
ai_msg = llm.invoke(
    messages, knowledge=knowledge, system_prompt=system_prompt, avoid_commentary=True
)

print(ai_msg.content)
```

```text
根据可用信息，世界上有两种猫：

1. 好猫
2. 最佳猫
```

### 指令遵循重排序器

```python
import getpass
import os

from langchain_contextual import ContextualRerank

if not os.getenv("CONTEXTUAL_AI_API_KEY"):
    os.environ["CONTEXTUAL_AI_API_KEY"] = getpass.getpass(
        "Enter your Contextual API key: "
    )


api_key = ""
model = "ctxl-rerank-en-v1-instruct"

compressor = ContextualRerank(
    model=model,
    api_key=api_key,
)

from langchain_core.documents import Document

query = "What is the current enterprise pricing for the RTX 5090 GPU for bulk orders?"
instruction = "Prioritize internal sales documents over market analysis reports. More recent documents should be weighted higher. Enterprise portal content supersedes distributor communications."

document_contents = [
    "Following detailed cost analysis and market research, we have implemented the following changes: AI training clusters will see a 15% uplift in raw compute performance, enterprise support packages are being restructured, and bulk procurement programs (100+ units) for the RTX 5090 Enterprise series will operate on a $2,899 baseline.",
    "Enterprise pricing for the RTX 5090 GPU bulk orders (100+ units) is currently set at $3,100-$3,300 per unit. This pricing for RTX 5090 enterprise bulk orders has been confirmed across all major distribution channels.",
    "RTX 5090 Enterprise GPU requires 450W TDP and 20% cooling overhead.",
]

metadata = [
    {
        "Date": "January 15, 2025",
        "Source": "NVIDIA Enterprise Sales Portal",
        "Classification": "Internal Use Only",
    },
    {"Date": "11/30/2023", "Source": "TechAnalytics Research Group"},
    {
        "Date": "January 25, 2025",
        "Source": "NVIDIA Enterprise Sales Portal",
        "Classification": "Internal Use Only",
    },
]

documents = [
    Document(page_content=content, metadata=metadata[i])
    for i, content in enumerate(document_contents)
]
reranked_documents = compressor.compress_documents(
    query=query,
    instruction=instruction,
    documents=documents,
)
```
