>[Aleph Alpha](https://docs.aleph-alpha.com/) 成立于 2019 年，致力于研究和构建强 AI 时代的基础技术。团队由国际科学家、工程师和创新者组成，负责研究、开发和部署大型语言和多模态模型等变革性 AI，并运营着欧洲速度最快的商业 AI 集群。

>[Luminous 系列](https://docs.aleph-alpha.com/docs/introduction/luminous/)是一组大型语言模型家族。

## 安装与设置

<CodeGroup>
```bash pip
pip install aleph-alpha-client
```

```bash uv
uv add aleph-alpha-client
```
</CodeGroup>

您需要创建一个新的令牌。请参阅[说明](https://docs.aleph-alpha.com/docs/account/#create-a-new-token)。

```python
from getpass import getpass

ALEPH_ALPHA_API_KEY = getpass()
```

查看[使用示例](/oss/python/integrations/llms/aleph_alpha)。
