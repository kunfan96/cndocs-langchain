在 Cerebras，我们研发了全球最大、最快的 AI 处理器——晶圆级引擎-3（WSE-3）。由 WSE-3 驱动的 Cerebras CS-3 系统代表了一类全新的 AI 超级计算机，以其无与伦比的性能和可扩展性，为生成式 AI 的训练与推理树立了新标准。

将 Cerebras 作为您的推理服务提供商，您可以：
- 实现前所未有的 AI 推理工作负载速度
- 以高吞吐量进行商业化构建
- 借助我们无缝的集群技术轻松扩展 AI 工作负载

我们的 CS-3 系统可以快速且轻松地组建集群，打造全球最大的 AI 超级计算机，从而简化大型模型的部署与运行。领先的企业、研究机构以及政府机构已在采用 Cerebras 解决方案来开发专有模型并训练流行的开源模型。

想体验 Cerebras 的强大实力吗？请访问我们的[官网](https://cerebras.ai)获取更多资源，并探索通过 Cerebras Cloud 或本地部署访问我们技术的方案！

如需了解更多关于 Cerebras Cloud 的信息，请访问 [cloud.cerebras.ai](https://cloud.cerebras.ai/)。我们的 API 参考文档位于 [inference-docs.cerebras.ai](https://inference-docs.cerebras.ai/)。

## 安装与配置
安装集成包：

<CodeGroup>
```bash pip
pip install langchain-cerebras
```

```bash uv
uv add langchain-cerebras
```
</CodeGroup>

## API 密钥
从 [cloud.cerebras.ai](https://cloud.cerebras.ai/) 获取 API 密钥，并将其添加到环境变量中：
```
export CEREBRAS_API_KEY="your-api-key-here"
```
