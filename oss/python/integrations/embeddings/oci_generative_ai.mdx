本文档将帮助您开始使用 OCI 生成式 AI [嵌入模型](/oss/python/integrations/embeddings)。Oracle Cloud Infrastructure (OCI) 生成式 AI 提供最先进的文本和图像嵌入模型，支持语义搜索、RAG、聚类和跨模态应用。

有关详细文档，请参阅 [OCI 生成式 AI 文档](https://docs.oracle.com/en-us/iaas/Content/generative-ai/home.htm) 和 [API 参考](https://docs.oracle.com/en-us/iaas/api/#/en/generative-ai/20231130/)。

## 实例化

```python
from langchain_oci import OCIGenAIEmbeddings

embeddings = OCIGenAIEmbeddings(
    model_id="cohere.embed-english-v3.0",
    service_endpoint="https://inference.generativeai.us-chicago-1.oci.oraclecloud.com",
    compartment_id="ocid1.compartment.oc1..your-compartment-id",
)
```

## 使用方法

构建针对技术文档的语义搜索：

```python
# 索引代码文档
docs = [
    "authenticate() 验证 JWT 令牌并返回用户对象",
    "authorize() 检查用户对资源访问的权限",
    "audit_log() 记录用户操作以进行合规性跟踪"
]
doc_vectors = embeddings.embed_documents(docs)

# 使用自然语言查询进行搜索
query = "如何验证用户身份？"
query_vector = embeddings.embed_query(query)

# 查找最相关的文档
import numpy as np
similarities = [
    np.dot(query_vector, doc_vec) /
    (np.linalg.norm(query_vector) * np.linalg.norm(doc_vec))
    for doc_vec in doc_vectors
]
best_match = docs[np.argmax(similarities)]
# 返回："authenticate() 验证 JWT 令牌..."
```

**应用场景：** 代码搜索、文档问答、日志分析、重复项检测

## 图像嵌入

使用多模态嵌入通过文本查询搜索视觉资产：

```python
import numpy as np
from langchain_oci import OCIGenAIEmbeddings

embeddings = OCIGenAIEmbeddings(
    model_id="cohere.embed-v4.0",
    service_endpoint="https://inference.generativeai.us-chicago-1.oci.oraclecloud.com",
    compartment_id="ocid1.compartment.oc1..your-compartment-id",
)

# 索引架构图
diagrams = ["microservices.png", "database_schema.png", "network.png"]
image_vectors = embeddings.embed_image_batch(diagrams)

# 使用文本查询进行搜索
query_vector = embeddings.embed_query("数据库关系")

# 查找最佳匹配
similarities = [np.dot(query_vector, v) / (np.linalg.norm(query_vector) * np.linalg.norm(v))
                for v in image_vectors]
print(diagrams[np.argmax(similarities)])  # "database_schema.png"
```

**应用场景：** 技术图表搜索、资产管理、可视化文档检索

## 可用模型

| 模型 | 维度 | 类型 |
|-------|------------|------|
| `cohere.embed-english-v3.0` | 1024 | 仅文本 |
| `cohere.embed-multilingual-v3.0` | 1024 | 仅文本 |
| `cohere.embed-v4.0` | 256-1536 | 文本 + 图像 |

查看所有模型，请参阅 [OCI 模型目录](https://docs.oracle.com/en-us/iaas/Content/generative-ai/pretrained-models.htm)。

## RAG 示例

```python
from langchain_community.vectorstores import FAISS

# 创建向量存储
vectorstore = FAISS.from_documents(documents, embeddings)

# 搜索
results = vectorstore.similarity_search("your query", k=3)
```

## 异步操作

用于生产环境的异步操作：

```python
query_vector = await embeddings.aembed_query("What is AI?")
doc_vectors = await embeddings.aembed_documents(["Doc 1", "Doc 2"])
```
