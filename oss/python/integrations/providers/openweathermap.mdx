>[OpenWeatherMap](https://openweathermap.org/api/) 提供特定位置的所有关键天气数据：
>- 当前天气
>- 未来1小时的分钟级预报
>- 未来48小时的逐小时预报
>- 未来8天的每日预报
>- 国家级天气预警
>- 回溯40年以上的历史天气数据

本页介绍如何在 LangChain 中使用 OpenWeatherMap API。

## 安装与配置

- 使用以下命令安装依赖项：

<CodeGroup>
```bash pip
pip install pyowm
```

```bash uv
uv add pyowm
```
</CodeGroup>

- 访问 OpenWeatherMap 并[注册账户以获取您的 API 密钥](https://openweathermap.org/api/)
- 将您的 API 密钥设置为环境变量 `OPENWEATHERMAP_API_KEY`

## 封装器

### 工具类

存在一个名为 OpenWeatherMapAPIWrapper 的工具类，用于封装该 API。要导入此工具类：

```python
from langchain_community.utilities.openweathermap import OpenWeatherMapAPIWrapper
```

有关此封装器的更详细教程，请参阅[此笔记本](/oss/python/integrations/tools/openweathermap)。

### 工具

您还可以轻松将此封装器加载为 Tool（以便与 Agent 配合使用）。
您可以通过以下方式实现：

```python
import os
from langchain_community.utilities import OpenWeatherMapAPIWrapper

os.environ["OPENWEATHERMAP_API_KEY"] = ""
weather = OpenWeatherMapAPIWrapper()
tools = [weather.run]
```

有关工具的更多信息，请参阅[工具](/oss/python/integrations/tools)。
