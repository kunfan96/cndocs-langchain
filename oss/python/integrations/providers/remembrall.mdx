>[Remembrall](https://remembrall.dev/) 是一个为语言模型提供
> 长期记忆、检索增强生成和完整可观测性的平台。

## 安装与设置

要开始，请 [使用 GitHub 在 Remembrall 平台登录](https://remembrall.dev/login)
并从 [设置页面](https://remembrall.dev/dashboard/settings) 复制您的 API 密钥。
