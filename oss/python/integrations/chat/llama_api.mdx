这个笔记本展示了如何使用 LangChain 与 [LlamaAPI](https://llama-api.com/)（Llama2 的托管版本，支持函数调用功能）进行集成。

```python
pip install -qU llamaapi
```

```python
from llamaapi import LlamaAPI

# 将 'Your_API_Token' 替换为您的实际 API 令牌
llama = LlamaAPI("Your_API_Token")
```

```python
from langchain_experimental.llms import ChatLlamaAPI
```

model = ChatLlamaAPI(client=llama)

```python
from langchain_classic.chains import create_tagging_chain

schema = {
    "properties": {
        "sentiment": {
            "type": "string",
            "description": "passage 中遇到的情感",
        },
        "aggressiveness": {
            "type": "integer",
            "description": "0-10 分数，表示 passage 的激进程度",
        },
        "language": {"type": "string", "description": "passage 的语言"},
    }
}

chain = create_tagging_chain(schema, model)
```

```python
chain.run("give me your money")
```
