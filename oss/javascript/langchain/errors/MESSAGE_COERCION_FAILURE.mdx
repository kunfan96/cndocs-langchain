## 故障排除

要解决此错误：

1. **确格式正确**: 聊天模型的所有输入必须是 LangChain 消息类的数组或受支持的类似消息的格式
2. 验证您的消息没有发生意外的字符串化或转换
3. 检查错误的堆栈跟踪并添加日志语句，以便在将消息对象传递给模型之前检查它们
