>[Stack Exchange](https://en.wikipedia.org/wiki/Stack_Exchange) 是一个涵盖各领域主题的问答（Q&A）网站网络，每个站点覆盖特定主题，问题、答案和用户均受声誉奖励机制约束。

本页介绍如何在 LangChain 中使用 `Stack Exchange API`。

## 安装与设置
- 安装依赖：
<CodeGroup>
```bash pip
pip install stackapi
```

```bash uv
uv add stackapi
```
</CodeGroup>

## 封装器

### 工具类

存在一个 StackExchangeAPIWrapper 工具类封装此 API。导入方式：

```python
from langchain_community.utilities import StackExchangeAPIWrapper
```

有关此封装器的更详细演练，请参阅[此 notebook](/oss/python/integrations/tools/stackexchange)。

### 工具

您也可以轻松地将此封装器加载为工具（与代理一起使用）。
操作方式：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["stackexchange"])
```

有关工具的更多信息，请参阅[此页面](/oss/python/integrations/tools)。
