[Yuan2.0](https://github.com/IEIT-Yuan/Yuan-2.0) 是由 IEIT System 开发的一代基础大语言模型。我们已发布全部三款模型：Yuan 2.0-102B、Yuan 2.0-51B 和 Yuan 2.0-2B。同时，我们为其他开发者提供了用于预训练、微调及推理服务的相关脚本。Yuan2.0 基于 Yuan1.0，利用更广泛的高质量预训练数据和指令微调数据集，以增强模型在语义、数学、推理、代码、知识等方面的理解能力。

本示例介绍如何使用 LangChain 与 `Yuan2.0`（2B/51B/102B）推理服务进行交互以实现文本生成。

Yuan2.0 搭建了推理服务，用户只需请求推理 API 即可获取结果，具体介绍见 [Yuan2.0 Inference-Server](https://github.com/IEIT-Yuan/Yuan-2.0/blob/main/docs/inference_server.md)。

```python
from langchain_classic.chains import LLMChain
from langchain_community.llms.yuan2 import Yuan2
```

```python
# 本地部署的 Yuan2.0 推理服务器的默认 infer_api
infer_api = "http://127.0.0.1:8000/yuan"

# 代理环境中的直接访问端点
# import os
# os.environ["no_proxy"]="localhost,127.0.0.1,::1"

yuan_llm = Yuan2(
    infer_api=infer_api,
    max_tokens=2048,
    temp=1.0,
    top_p=0.9,
    use_history=False,
)

# 仅当您希望 Yuan2.0 跟踪对话历史时开启 use_history
# 并将累积的上下文发送给后端模型 API，使其具备状态。默认情况下它是无状态的。
# llm.use_history = True
```

```python
question = "请介绍一下中国。"
```

```python
print(yuan_llm.invoke(question))
```
