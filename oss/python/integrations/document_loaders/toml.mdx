>[TOML](https://en.wikipedia.org/wiki/TOML) 是一种用于配置文件的文件格式。它旨在易于读写，并旨在明确地映射到字典。其规范是开源的。`TOML` 已在多种编程语言中实现。名称 `TOML` 是 "Tom's Obvious, Minimal Language" 的首字母缩写，指的是其创建者 Tom Preston-Werner。

如果需要加载 `Toml` 文件，请使用 `TomlLoader`。

```python
from langchain_community.document_loaders import TomlLoader
```

```python
loader = TomlLoader("example_data/fake_rule.toml")
```

```python
rule = loader.load()
```
