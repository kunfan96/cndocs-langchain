| :--- | :--- | :---: | :---: |

### 工具功能

- 按主题、作者或查询检索学术出版物。
- 获取元数据，如标题、作者和出版年份。
- 高级搜索过滤器，包括引用次数和期刊名称。

## 设置

集成位于 `@langchain/community` 包中。

```bash
npm install @langchain/community
```

### 凭证

确保您拥有访问 Google 学术的适当 API 密钥。在环境变量中设置它：

```typescript
SERPAPI_API_KEY="your-serp-api-key"
```

设置 [LangSmith](https://smith.langchain.com/) 以获得最佳可观测性也很有帮助：

```typescript
process.env.LANGSMITH_TRACING="true"
process.env.LANGSMITH_API_KEY="your-langchain-api-key"
```

## 实例化

您可以像这样导入并实例化 `SERPGoogleScholarAPITool` 工具：

```python
import { SERPGoogleScholarAPITool } from "@langchain/community/tools/google_scholar";

const tool = new SERPGoogleScholarAPITool({
  apiKey: process.env.SERPAPI_API_KEY,
});
```

## 调用

### 直接使用参数调用

您可以使用查询参数直接调用工具：

```python
const results = await tool.invoke({
  query: "neural networks",
  maxResults: 5,
});

console.log(results);
```

### 使用 ToolCall 调用

我们也可以使用模型生成的 `ToolCall` 调用工具：

```python
const modelGeneratedToolCall = {
  args: { query: "machine learning" },
  id: "1",
  name: tool.name,
  type: "tool_call",
};
await tool.invoke(modelGeneratedToolCall);
```
