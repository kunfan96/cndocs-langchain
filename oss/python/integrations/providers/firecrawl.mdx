>[FireCrawl](https://firecrawl.dev/?ref=langchain) 可以爬取任何网站并将其转换为 LLM 就绪的数据。
> 它可以爬取所有可访问的子页面，并为每个页面提供干净的 Markdown
> 和元数据。无需站点地图。

## 安装与设置

安装 Python SDK：

<CodeGroup>
```bash pip
pip install firecrawl-py==0.0.20
```

```bash uv
uv add "firecrawl-py==0.0.20"
```
</CodeGroup>

## 文档加载器

查看[使用示例](/oss/python/integrations/document_loaders/firecrawl)。

```python
from langchain_community.document_loaders import FireCrawlLoader
```
