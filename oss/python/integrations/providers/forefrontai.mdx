> [Forefront AI](https://forefront.ai/) 是一个允许你对开源文本生成模型进行微调与推理的平台


## 安装与设置

获取 `ForefrontAI` API 密钥，
请访问[此页面](https://accounts.forefront.ai/sign-in?redirect_url=https%3A%2F%2Fforefront.ai%2Fapp%2Fapi-keys)。
 并将其设置为环境变量（`FOREFRONTAI_API_KEY`）。

## LLM

请参阅[使用示例](/oss/python/integrations/llms/forefrontai)。

```python
from langchain_community.llms import ForefrontAI
```
