>[阿里云机器学习平台（PAI）](https://www.alibabacloud.com/help/en/pai) 是一款面向企业和开发者打造的机器学习或深度学习工程平台。它提供易于使用、高性价比、高性能且易于扩展的插件，可应用于多种行业场景。凭借内置的 140 多种优化算法，`Machine Learning Platform for AI` 提供涵盖数据标注（`PAI-iTAG`）、模型构建（`PAI-Designer` 和 `PAI-DSW`）、模型训练（`PAI-DLC`）、编译优化以及推理部署（`PAI-EAS`）的全流程 AI 工程能力。`PAI-EAS` 支持 CPU 和 GPU 等多种硬件资源，具备高吞吐和低延迟的特性。它允许您通过几次点击即可部署大规模复杂模型，并实时执行弹性扩缩容操作。此外，它还提供了全面的运维与监控系统。

```python
##安装用于该集成的 langchain 包
pip install -qU langchain-community
```

```python
from langchain_classic.chains import LLMChain
from langchain_community.llms.pai_eas_endpoint import PaiEasEndpoint
from langchain_core.prompts import PromptTemplate

template = """Question: {question}

Answer: Let's think step by step."""

prompt = PromptTemplate.from_template(template)
```

想要使用 EAS LLM 的用户必须先配置 EAS 服务。当 EAS 服务启动后，可以获取 `EAS_SERVICE_URL` 和 `EAS_SERVICE_TOKEN`。更多详细信息请参阅 [www.alibabacloud.com/help/en/pai/user-guide/service-deployment/](https://www.alibabacloud.com/help/en/pai/user-guide/service-deployment/)。

```python
import os

os.environ["EAS_SERVICE_URL"] = "Your_EAS_Service_URL"
os.environ["EAS_SERVICE_TOKEN"] = "Your_EAS_Service_Token"
llm = PaiEasEndpoint(
    eas_service_url=os.environ["EAS_SERVICE_URL"],
    eas_service_token=os.environ["EAS_SERVICE_TOKEN"],
)
```

```python
llm_chain = prompt | llm

question = "What NFL team won the Super Bowl in the year Justin Beiber was born?"
llm_chain.invoke({"question": question})
```
