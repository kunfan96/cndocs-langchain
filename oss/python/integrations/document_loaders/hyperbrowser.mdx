[Hyperbrowser](https://hyperbrowser.ai) 是一个用于运行和扩展无头浏览器的平台。它可以让您大规模启动和管理浏览器会话，并为各种网页抓取需求提供易用的解决方案，例如抓取单个页面或爬取整个网站。

主要特性：

- 即时扩展性 - 几秒内启动数百个浏览器会话，无需基础设施烦恼
- 简单集成 - 与 Puppeteer 和 Playwright 等流行工具无缝协作
- 强大的 API - 易于使用的 API，可抓取/爬取任何网站，以及更多功能
- 绕过反爬虫措施 - 内置隐身模式、广告拦截、自动 CAPTCHA 解决和代理轮换

本指南提供了使用 Hyperbrowser [文档加载器](https://python.langchain.com/docs/concepts/#document-loaders)快速入门的概述。

如需了解更多关于 Hyperbrowser 的信息，请访问 [Hyperbrowser 官网](https://hyperbrowser.ai)，或查阅 [Hyperbrowser 文档](https://docs.hyperbrowser.ai)。

## 设置

要访问 Hyperbrowser 文档加载器，您需要安装 `langchain-hyperbrowser` 集成包，并创建一个 Hyperbrowser 账户并获取 API 密钥。

### 凭据

前往 [Hyperbrowser](https://app.hyperbrowser.ai/) 注册并生成 API 密钥。完成后，设置 HYPERBROWSER_API_KEY 环境变量：

### 安装

安装 **langchain-hyperbrowser**。

```python
pip install -qU langchain-hyperbrowser
```

## 初始化

现在可以实例化模型对象并加载文档：

```python
from langchain_hyperbrowser import HyperbrowserLoader

loader = HyperbrowserLoader(
    urls="https://example.com",
    api_key="YOUR_API_KEY",
)
```

docs = loader.load()

## 懒加载

```python
page = []
for doc in loader.lazy_load():
    page.append(doc)
    if len(page) >= 10:
        # do some paged operation, e.g.
        # index.upsert(page)

        page = []
```

## 高级用法

您可以指定加载器执行的操作。默认操作为 `scrape`（抓取）。对于 `scrape`，可以提供单个 URL 或 URL 列表。对于 `crawl`（爬取），只能提供单个 URL，该操作会爬取指定页面及其子页面，并为每个页面返回一个文档。

```python
loader = HyperbrowserLoader(
    urls="https://hyperbrowser.ai", api_key="YOUR_API_KEY", operation="crawl"
)
```

也可以在 `params` 参数中提供加载器的可选参数。有关支持参数的更多信息，请访问 [docs.hyperbrowser.ai/reference/sdks/python/scrape#start-scrape-job-and-wait](https://docs.hyperbrowser.ai/reference/sdks/python/scrape#start-scrape-job-and-wait) 或 [docs.hyperbrowser.ai/reference/sdks/python/crawl#start-crawl-job-and-wait](https://docs.hyperbrowser.ai/reference/sdks/python/crawl#start-crawl-job-and-wait)。

```python
loader = HyperbrowserLoader(
    urls="https://example.com",
    api_key="YOUR_API_KEY",
    operation="scrape",
    params={"scrape_options": {"include_tags": ["h1", "h2", "p"]}},
)
```
