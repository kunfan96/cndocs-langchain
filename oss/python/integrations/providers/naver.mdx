与 `Naver` 相关的所有功能，包括 HyperCLOVA X 模型，特别是通过 `Naver Cloud` [CLOVA Studio](https://clovastudio.ncloud.com/) 访问的模型。

> [Naver](https://navercorp.com/) 是一家拥有尖端技术和多元化业务组合的全球科技公司，业务涵盖搜索、电商、金融科技、内容、云和 AI。

> [Naver Cloud](https://www.navercloudcorp.com/lang/en/) 是 Naver 的云计算部门，是领先的云服务提供商，通过其 [Naver Cloud Platform（NCP）](https://www.ncloud.com/)向企业提供全面的云服务。

更详细的说明（包括韩语版本）请参阅 [NCP 用户指南](https://guide.ncloud-docs.com/docs/clovastudio-overview)。

## 安装与设置

- 通过[申请](https://api.ncloud-docs.com/docs/ai-naver-clovastudio-summary#API%ED%82%A4)获取 CLOVA Studio API 密钥，并将其设置为环境变量（`CLOVASTUDIO_API_KEY`）。


Naver 集成分布在两个包中：

- `langchain-naver`：专用的 Naver 集成包。
- `langchain-naver-community`：社区维护的包，未经 Naver 或 LangChain 官方维护。

<CodeGroup>
```bash pip
pip install -U langchain-naver
# pip install -U langchain-naver-community // 安装以使用 Naver Search 工具
```

```bash uv
uv add langchain-naver
# uv add langchain-naver-community // 安装以使用 Naver Search 工具
```
</CodeGroup>

> **（注意）** 通过 `langchain-community`（第三方集成合集）使用 Naver 集成已过时。
> - **请改用 `langchain-naver`，新功能应仅通过该包实现**。
> - 如果您使用的是 `langchain-community`（已过时）且拥有旧版 API 密钥（不以 `nv-*` 前缀开头），则应将其设置为 `NCP_CLOVASTUDIO_API_KEY`，并可能还需要通过[创建应用](https://guide.ncloud-docs.com/docs/en/clovastudio-playground01#create-test-app)获取额外的 API 网关密钥，并将其设置为 `NCP_APIGW_API_KEY`。

## 工具

### Naver 搜索

Naver Search 集成允许您的 LangChain 应用从 Naver 搜索引擎检索信息。这对于韩语查询和获取关于韩国话题的最新信息特别有用。

要使用 Naver Search 工具，您需要：

1. 登录 [Naver 开发者门户](https://developers.naver.com/main/)
2. 创建新应用并启用 Search API
3. 从"应用列表"部分获取 **NAVER_CLIENT_ID** 和 **NAVER_CLIENT_SECRET**
4. 在应用中将这些设置为环境变量

```python
from langchain_naver_community.tool import NaverSearchResults
from langchain_naver_community.utils import NaverSearchAPIWrapper

# Set up the search wrapper
search = NaverSearchAPIWrapper()

# Create a tool
tool = NaverSearchResults(api_wrapper=search)
```

更多详情请参阅[使用示例](/oss/python/integrations/tools/naver_search)。

### 专项搜索工具

该包还提供针对不同内容类型的专项搜索工具：

```python
from langchain_naver_community.tool import NaverNewsSearch  # 用于新闻文章
from langchain_naver_community.tool import NaverBlogSearch  # 用于博客文章
from langchain_naver_community.tool import NaverImageSearch  # 用于图片
```

这些工具均可在 LangChain Agent 中使用，提供更有针对性的搜索能力。
