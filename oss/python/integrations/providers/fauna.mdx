>[Fauna](https://fauna.com/) 是一个分布式文档关系型数据库
> 它将文档的灵活性与关系型数据库的强大功能相结合，
> 是一款符合 ACID 规范、可跨区域、云或全球范围扩展的数据库。


## 安装与设置

我们需要获取密钥。
请参阅详细的[指南](https://docs.fauna.com/fauna/current/learn/security_model/)。

我们需要安装 `fauna` 包。

<CodeGroup>
```bash pip
pip install -U fauna
```

```bash uv
uv add fauna
```
</CodeGroup>

## 文档加载器

请参阅[使用示例](/oss/python/integrations/document_loaders/fauna)。

```python
from langchain_community.document_loaders.fauna import FaunaLoader
```
