>[DuckDuckGo Search](https://github.com/deedy5/ddgs) 是一个软件包，
> 使用 `DuckDuckGo.com` 搜索引擎搜索单词、文档、图片、视频、新闻、地图和文本
> 翻译。它会将文件和
> 图片下载到本地硬盘。

## 安装与设置

您需要安装一个 Python 软件包：

<CodeGroup>
```bash pip
pip install ddgs
```

```bash uv
uv add ddgs
```
</CodeGroup>

## 工具

请参阅[使用示例](/oss/python/integrations/tools/ddg)。

有两种工具可用：

```python
from langchain_community.tools import DuckDuckGoSearchRun
from langchain_community.tools import DuckDuckGoSearchResults
```
