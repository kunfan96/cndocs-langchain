>[DuckDB](https://duckdb.org/) 是一款进程内 SQL OLAP 数据库管理系统。

## 安装与设置

首先，您需要安装 `duckdb` Python 包。

<CodeGroup>
```bash pip
pip install duckdb
```

```bash uv
uv add duckdb
```
</CodeGroup>
