**Scrapeless** 提供灵活且功能丰富的数据采集服务，支持广泛的参数自定义和多种格式导出。这些能力使 LangChain 能够更有效地集成和利用外部数据。核心功能模块包括：

**DeepSerp**

- **Google 搜索**：支持跨所有结果类型全面提取 Google SERP 数据。
  - 支持选择本地化 Google 域名（例如 `google.com`、`google.ad`），以检索特定地区的搜索结果。
  - 支持分页以获取第一页以外的结果。
  - 支持搜索结果过滤开关，用于控制是否排除重复或相似内容。
- **Google Trends**：从 Google 获取关键词趋势数据，包括随时间变化的热度、地区兴趣以及相关搜索。
  - 支持多关键词对比。
  - 支持多种数据类型：`interest_over_time`、`interest_by_region`、`related_queries` 和 `related_topics`。
  - 允许按特定 Google 属性（网页、YouTube、新闻、购物）进行过滤，以进行特定来源的趋势分析。

**通用抓取**

- 专为现代 JavaScript 密集型网站设计，支持动态内容提取。
  - 全球高级代理支持，可绕过地理限制并提高可靠性。

**爬虫**

- **Crawl（爬取）**：递归爬取网站及其链接页面，提取全站内容。
  - 支持可配置的爬取深度和范围化 URL 定向。
- **Scrape（抓取）**：高精度地从单个网页提取内容。
  - 支持"仅主要内容"提取，排除广告、页脚和其他非核心元素。
  - 支持批量抓取多个独立 URL。

## 实例化

以下展示如何实例化 Scrapeless 通用抓取工具。该工具允许您使用无头浏览器抓取任意网站，支持 JavaScript 渲染、可自定义的输出类型以及地理特定代理。

实例化时，工具接受以下参数：

- `url`（必填，str）：要抓取的网站 URL。
- `headless`（可选，bool）：是否使用无头浏览器。默认为 True。
- `js_render`（可选，bool）：是否启用 JavaScript 渲染。默认为 True。
- `js_wait_until`（可选，str）：定义 JavaScript 渲染页面就绪的时机。默认为 `'domcontentloaded'`。选项包括：
  - `load`：等待页面完全加载。
  - `domcontentloaded`：等待 DOM 完全加载。
  - `networkidle0`：等待网络空闲。
  - `networkidle2`：等待网络空闲 2 秒。
- `outputs`（可选，str）：从页面提取的特定数据类型。选项包括：
  - `phone_numbers`
  - `headings`
  - `images`
  - `audios`
  - `videos`
  - `links`
  - `menus`
  - `hashtags`
  - `emails`
  - `metadata`
  - `tables`
  - `favicon`
- `response_type`（可选，str）：定义响应的格式。默认为 `'html'`。选项包括：
  - `html`：返回页面的原始 HTML。
  - `plaintext`：返回纯文本内容。
  - `markdown`：返回页面的 Markdown 版本。
  - `png`：返回 PNG 截图。
  - `jpeg`：返回 JPEG 截图。
- `response_image_full_page`（可选，bool）：使用截图输出（png 或 jpeg）时是否捕获并返回整页图片。默认为 False。
- `selector`（可选，str）：用于将抓取范围限定在页面某部分的 CSS 选择器。默认为 `None`。
- `proxy_country`（可选，str）：用于地理特定代理访问的两字母国家代码（例如 `'us'`、`'gb'`、`'de'`、`'jp'`）。默认为 `'ANY'`。

# Basic usage

print(result)

```python

tool = ScrapelessUniversalScrapingTool()

result = tool.invoke({"url": "https://exmaple.com", "response_type": "markdown"})
print(result)
```

### 在 Agent 中使用

```python
from langchain_openai import ChatOpenAI
from langchain_scrapeless import ScrapelessUniversalScrapingTool
from langchain.agents import create_agent


model = ChatOpenAI()

tool = ScrapelessUniversalScrapingTool()

# Use the tool with an agent
tools = [tool]
agent = create_agent(model, tools)

for chunk in agent.stream(
    {
        "messages": [
            (
                "human",
                "Use the scrapeless scraping tool to fetch https://www.scrapeless.com/en and extract the h1 tag.",
            )
        ]
    },
    stream_mode="values",
):
    chunk["messages"][-1].pretty_print()
```
