## 智能体框架（如 LangChain）

智能体框架提供抽象，使使用 LLM 构建时更容易入门。

[LangChain](/oss/javascript/langchain/overview) 是一个智能体框架，提供结构化内容块、智能体循环和中间件等抽象。

LangChain 的抽象设计旨在易于上手，同时仍为高级用例提供所需的灵活性。

虽然 LangChain 构建在 [LangGraph](/oss/javascript/langgraph/overview) 之上，但您无需了解 LangGraph 即可使用 LangChain。

其他智能体框架示例包括 [Vercel 的 AI SDK](https://ai-sdk.dev/docs/introduction)、[CrewAI](https://www.crewai.com/)、[OpenAI Agents SDK](https://openai.github.io/openai-agents-python/)、[Google ADK](https://google.github.io/adk-docs/)、[LlamaIndex](https://www.llamaindex.ai/) 等。

### 何时使用 LangChain

在以下情况下使用 LangChain：

- 您想快速构建智能体和自主应用程序。
- 您需要模型、工具和智能体循环的标准抽象。
- 您想要一个易于使用但仍提供灵活性的框架。
- 您正在构建简单的智能体应用程序，无需复杂的编排需求。

## 智能体运行时（如 LangGraph）

智能体运行时提供在生产环境中运行智能体的工具。
支持的工具可能包括：

- **持久执行**：智能体在故障后持续存在，并可以长时间运行，从上次中断处恢复。
- **流式处理**：支持流式工作流和响应。
- **人机回环**：通过检查和修改智能体状态来纳入人工监督。
- **持久化**：用于状态管理的线程级和跨线程持久化。
- **低级控制**：直接控制智能体编排，无需高级抽象。

[LangGraph](/oss/javascript/langgraph/overview) 是一个低级编排框架和运行时，用于构建、管理和部署长时间运行、有状态的智能体。

智能体框架通常更高级，并在智能体运行时上运行。
例如，LangChain 1.0 构建在 LangGraph 之上。

其他智能体运行时示例包括 [Temporal](https://temporal.io/)、[Inngest](https://www.inngest.com/) 和其他持久执行引擎。

### 何时使用 LangGraph

在以下情况下使用 LangGraph：

- 您需要对智能体编排进行细粒度的低级控制。
- 您需要为长时间运行、有状态的智能体提供持久执行。
- 您正在构建结合确定性和智能体步骤的复杂工作流。
- 您需要用于智能体部署的生产就绪基础设施。

## 智能体工具集（如 Deep Agents SDK）

智能体工具集是自以为是的、开箱即用的框架，具有内置工具和功能，用于构建复杂、长时间运行的智能体。
支持的工具可能包括：

- **规划能力**：使用待办事项列表跟踪多个任务。
- **任务委派**：委派工作并通过子智能体保持上下文清洁。
- **文件系统**：对不同可插拔存储后端上的文件进行读写访问。
- **令牌管理**：对话历史记录摘要和大型工具结果驱逐。

[Deep Agents SDK](/oss/javascript/deepagents/overview) 构建在 LangGraph 之上，并添加了规划能力、用于上下文管理的文件系统、生成子智能体等功能。
Deep Agents 专为需要规划和分解的复杂多步骤任务而设计。

示例任务包括处理搜索结果、脚本和状态中的其他工件。

其他智能体工具集示例包括 [Claude Agent SDK](https://platform.claude.com/docs/en/agent-sdk/overview)、[Manus](https://manus.im/) 和其他编码 CLI。

### 何时使用 Deep Agents SDK

在以下情况下使用 [Deep Agents SDK](/oss/javascript/deepagents/overview)：

- 您正在构建需要长时间运行的智能体。
- 您正在构建需要处理复杂多步骤任务的智能体。
- 您想使用预定义工具，例如文件系统操作、bash 执行和自动化上下文工程。
- 您想使用预定义提示和子智能体。

## 功能比较

虽然您可以使用 LangChain、LangGraph 和 Deep Agents 完成类似的任务，但集成它们的级别不同：

| 功能 | LangChain | LangGraph   | Deep Agents |
| ------- | --------- | ----------- | ----------- |
| 短期记忆 | [短期记忆](/oss/javascript/langchain/short-term-memory) | [短期记忆](/oss/javascript/langgraph/add-memory#add-short-term-memory) | [`StateBackend`](/oss/javascript/deepagents/backends#statebackend-ephemeral) |
| 长期记忆 | [长期记忆](/oss/javascript/langchain/long-term-memory) | [长期记忆](/oss/javascript/langgraph/add-memory#add-long-term-memory) | [长期记忆](/oss/javascript/deepagents/memory) |
| 技能  | [多智能体技能](/oss/javascript/langchain/multi-agent/skills) | - | [技能](/oss/javascript/deepagents/skills) |
| 子智能体 | [多智能体子智能体](/oss/javascript/langchain/multi-agent/subagents) | [子图](/oss/javascript/langgraph/use-subgraphs) | [子智能体](/oss/javascript/deepagents/subagents) |
| 人机回环 | [人机回环中间件](/oss/javascript/langchain/human-in-the-loop) | [中断](/oss/javascript/langgraph/interrupts) | [`interrupt_on` 参数](/oss/javascript/deepagents/harness#human-in-the-loop) |
| 流式处理 | [智能体流式处理](/oss/javascript/langchain/streaming/overview) | [流式处理](/oss/javascript/langgraph/streaming) | [流式处理](/oss/javascript/deepagents/streaming/overview) |
