本页介绍如何在 LangChain 中使用 [Epsilla](https://github.com/epsilla-cloud/vectordb)。
本文档分为两部分：安装与设置，以及针对特定 Epsilla 包装器的参考文档。

## 安装与设置

- 使用 `pip/pip3 install pyepsilla` 安装 Python SDK

## 包装器

### VectorStore

提供了一个封装 Epsilla 向量数据库的包装器，使您能够将其作为向量存储（VectorStore）使用，无论是用于语义搜索还是示例选择。

要导入此向量存储：

```python
from langchain_community.vectorstores import Epsilla
```

如需了解 Epsilla 包装器的更详细操作指南，请参阅[此笔记本](/oss/python/integrations/vectorstores/epsilla)
