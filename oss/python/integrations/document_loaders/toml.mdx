>[TOML](https://en.wikipedia.org/wiki/TOML) 是一种配置文件格式，设计目标是易于读写，并能明确无歧义地映射到字典结构。其规范是开源的。`TOML` 已在多种编程语言中实现。`TOML` 这个名称是"Tom's Obvious, Minimal Language"的首字母缩写，以其创始人 Tom Preston-Werner 命名。

如果需要加载 `Toml` 文件，请使用 `TomlLoader`。

```python
from langchain_community.document_loaders import TomlLoader
```

```python
loader = TomlLoader("example_data/fake_rule.toml")
```

```python
rule = loader.load()
```
