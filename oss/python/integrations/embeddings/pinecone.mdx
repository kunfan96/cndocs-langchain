可以通过 `PineconeEmbeddings` 访问 Pinecone 的推理 API。通过 Pinecone 服务提供文本嵌入。我们首先安装必要的依赖库：

```python
!pip install -qU "langchain-pinecone>=0.2.0"
```

接下来，我们[注册/登录 Pinecone](https://app.pinecone.io) 以获取 API 密钥：

```python
import os
from getpass import getpass

os.environ["PINECONE_API_KEY"] = os.getenv("PINECONE_API_KEY") or getpass(
    "请输入您的 Pinecone API 密钥： "
)
```

查阅文档以了解可用的[模型](https://docs.pinecone.io/models/overview)。现在，我们按如下方式初始化嵌入模型：

```python
from langchain_pinecone import PineconeEmbeddings

embeddings = PineconeEmbeddings(model="multilingual-e5-large")
```

从这里开始，我们可以同步或异步创建嵌入，让我们从同步开始！我们使用 `embed_query` 将单个文本嵌入为查询嵌入（即我们在 RAG 中用于搜索的内容）：

```python
docs = [
    "苹果是一种广受欢迎的水果，以其甜美和清脆的口感而闻名。",
    "科技公司苹果公司以其 iPhone 等创新产品而闻名。",
    "许多人喜欢将苹果作为健康零食食用。",
    "苹果公司凭借其时尚的设计和用户友好的界面彻底改变了科技行业。",
    "正如谚语所说，一天一苹果，医生远离我。",
]
```

```python
doc_embeds = embeddings.embed_documents(docs)
doc_embeds
```

```python
query = "告诉我关于被称为苹果的科技公司的信息"
query_embed = embeddings.embed_query(query)
query_embed
```
