>[Dedoc](https://dedoc.readthedocs.io) 是一个[开源](https://github.com/ispras/dedoc)库/服务，能够从各种格式的文件中提取文本、表格、附件和文档结构（例如标题、列表项等）。

`Dedoc` 支持 `DOCX`、`XLSX`、`PPTX`、`EML`、`HTML`、`PDF`、图像等更多格式。您可以在 [Dedoc 支持的格式文档](https://dedoc.readthedocs.io/en/latest/#id1) 中找到完整的受支持格式列表。

## 安装与设置

### Dedoc 库

您可以使用 `pip` 安装 `Dedoc`。在这种情况下，您需要安装依赖项。更多信息，请参阅 [Dedoc 安装指南](https://dedoc.readthedocs.io/en/latest/getting_started/installation.html)。

<CodeGroup>

```bash pip
pip install dedoc
```

```bash uv
uv add dedoc
```

</CodeGroup>

### Dedoc API

如果您打算使用 `Dedoc` API，则无需安装 `dedoc` 库。在这种情况下，您应该运行 `Dedoc` 服务，例如 `Docker` 容器（详情请参阅[该文档](https://dedoc.readthedocs.io/en/latest/getting_started/installation.html#install-and-run-dedoc-using-docker)）：

```bash
docker pull dedocproject/dedoc
docker run -p 1231:1231
```

## 文档加载器

* 用于处理任何格式的文件（由 `Dedoc` 支持），您可以使用 `DedocFileLoader`：

    ```python
    from langchain_community.document_loaders import DedocFileLoader
    ```

* 用于处理 PDF 文件（带或不带文本层），您可以使用 `DedocPDFLoader`：

    ```python
    from langchain_community.document_loaders import DedocPDFLoader
    ```

* 用于在不安装库的情况下处理任何格式的文件，您可以将 `Dedoc API` 与 `DedocAPIFileLoader` 配合使用：

    ```python
    from langchain_community.document_loaders import DedocAPIFileLoader
    ```

有关更多详细信息，请参阅[使用示例](/oss/python/integrations/document_loaders/dedoc)。
