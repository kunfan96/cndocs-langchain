MHTML 既用于电子邮件，也用于存档的网页。MHTML，有时称为 MHT，代表 MIME HTML，是一个包含整个网页存档的单个文件。当将网页保存为 MHTML 格式时，此文件扩展名将包含 HTML 代码、图像、音频文件、Flash 动画等。

```python
from langchain_community.document_loaders import MHTMLLoader
```

```python
# 为 MHTML 文件创建一个新的加载器对象
loader = MHTMLLoader(
    file_path="../../../../../../tests/integration_tests/examples/example.mht"
)

# 从文件加载文档
documents = loader.load()

# 打印文档以查看结果
for doc in documents:
    print(doc)
```
