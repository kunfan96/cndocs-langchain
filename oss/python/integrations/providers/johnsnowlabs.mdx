通过开源的 `johnsnowlabs` 库，您可以访问 [johnsnowlabs](https://www.johnsnowlabs.com/) 的企业级 NLP 库生态系统，其中包含 200 多种语言的 21,000 多个企业级 NLP 模型。如需查看所有 24,000 多个模型，请参阅 [John Snow Labs 模型中心](https://nlp.johnsnowlabs.com/models)

## 安装与配置


<CodeGroup>
```bash pip
pip install johnsnowlabs
```

```bash uv
uv add johnsnowlabs
```
</CodeGroup>

要[安装企业功能](https://nlp.johnsnowlabs.com/docs/en/jsl/install_licensed_quick)，请运行：
```python
# 更多详情请参阅 https://nlp.johnsnowlabs.com/docs/en/jsl/install_licensed_quick
nlp.install()
```


您可以使用基于 `gpu`、`cpu`、`apple_silicon` 或 `aarch` 的优化二进制文件来嵌入查询和文档。
默认情况下使用 CPU 二进制文件。
会话启动后，若要切换 GPU 或 CPU，必须重启笔记本，否则更改将不会生效。

## 使用 CPU 嵌入查询：
```python
document = "foo bar"
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert')
output = embedding.embed_query(document)
```


## 使用 GPU 嵌入查询：


```python
document = "foo bar"
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','gpu')
output = embedding.embed_query(document)
```




## 使用 Apple Silicon（M1、M2 等）嵌入查询：

```python
documents = ["foo bar", 'bar foo']
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','apple_silicon')
output = embedding.embed_query(document)
```



## 使用 AARCH 嵌入查询：

```python
documents = ["foo bar", 'bar foo']
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','aarch')
output = embedding.embed_query(document)
```






## 使用 CPU 嵌入文档：
```python
documents = ["foo bar", 'bar foo']
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','gpu')
output = embedding.embed_documents(documents)
```



## 使用 GPU 嵌入文档：

```python
documents = ["foo bar", 'bar foo']
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','gpu')
output = embedding.embed_documents(documents)
```





## 使用 Apple Silicon（M1、M2 等）嵌入文档：

```python
```python
documents = ["foo bar", 'bar foo']
embedding = JohnSnowLabsEmbeddings('embed_sentence.bert','apple_silicon')
output = embedding.embed_documents(documents)
```



## 使用 AARCH 嵌入文档：

```python
```python
documents = ["foo bar", 'bar foo']

output = embedding.embed_documents(documents)

底层通过 [nlp.load](https://nlp.johnsnowlabs.com/docs/en/jsl/load_api) 加载模型，并通过 [nlp.start()](https://nlp.johnsnowlabs.com/docs/en/jsl/start-a-sparksession) 启动 Spark 会话。
