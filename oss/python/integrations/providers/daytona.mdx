[Daytona](https://daytona.io) 提供支持多语言的快速启动沙箱环境。有关注册、身份验证和平台详情，请参阅 [Daytona 文档](https://www.daytona.io/docs)。

## 安装

<CodeGroup>
```bash pip
pip install langchain-daytona
```

```bash uv
uv add langchain-daytona
```
</CodeGroup>

## 创建沙箱后端

在 Python 中，您使用提供商 SDK 创建沙箱，然后用 [deepagents 后端](/oss/python/deepagents/backends)将其封装。

```python
from daytona import Daytona

from langchain_daytona import DaytonaSandbox

sandbox = Daytona().create()
backend = DaytonaSandbox(sandbox=sandbox)

result = backend.execute("echo hello")
print(result.output)
```

## 与 deepagents 一起使用

```python
from daytona import Daytona
from langchain_anthropic import ChatAnthropic

from deepagents import create_deep_agent
from langchain_daytona import DaytonaSandbox

sandbox = Daytona().create()
backend = DaytonaSandbox(sandbox=sandbox)

agent = create_deep_agent(
    model=ChatAnthropic(model="claude-sonnet-4-20250514"),
    system_prompt="You are a coding assistant with sandbox access.",
    backend=backend,
)

result = agent.invoke(
    {
        "messages": [
            {"role": "user", "content": "Create a hello world Python script and run it"}
        ]
    }
)
```

## 清理

您需要通过 Daytona SDK 管理沙箱生命周期。
完成后，请停止或销毁沙箱。

另请参阅：[沙箱](/oss/python/deepagents/sandboxes)。
