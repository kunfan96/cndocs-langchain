>[Git](https://en.wikipedia.org/wiki/Git) 是一个分布式版本控制系统，可以跟踪任何计算机文件集合中的变更，通常用于协调程序员在软件开发过程中协作开发源代码。

## 安装与设置

首先，你需要安装 `GitPython` Python 包。

<CodeGroup>
```bash pip
pip install GitPython
```

```bash uv
uv add GitPython
```
</CodeGroup>

## 文档加载器

查看[使用示例](/oss/python/integrations/document_loaders/git)。

```python
from langchain_community.document_loaders import GitLoader
```
