>[Vespa](https://vespa.ai/) 是一个功能完整的搜索引擎和向量数据库。
> 它在同一查询中支持向量搜索（ANN）、词法搜索以及结构化数据搜索。

## 安装与设置


<CodeGroup>
```bash pip
pip install pyvespa
```

```bash uv
uv add pyvespa
```
</CodeGroup>



## 检索器

查看[使用示例](/oss/python/integrations/retrievers/vespa)。
