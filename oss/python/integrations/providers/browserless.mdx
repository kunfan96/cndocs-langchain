>[Browserless](https://www.browserless.io/docs/start) 是一种允许您在云端运行无头 Chrome 实例的服务。
> 它是在规模化运行基于浏览器的自动化的绝佳方式，无需担心管理自己的基础设施。

## 安装与设置

我们需要[在此处](https://www.browserless.io/pricing/)获取 API 密钥。
