> [Forefront AI](https://forefront.ai/) 是一个平台，让你能够
> 对开源文本生成模型进行微调和推理。

## 安装与设置

访问[此页面](https://accounts.forefront.ai/sign-in?redirect_url=https%3A%2F%2Fforefront.ai%2Fapp%2Fapi-keys)获取 `ForefrontAI` API 密钥，
并将其设置为环境变量（`FOREFRONTAI_API_KEY`）。

## LLMs

查看[使用示例](/oss/python/integrations/llms/forefrontai)。

```python
from langchain_community.llms import ForefrontAI
```
