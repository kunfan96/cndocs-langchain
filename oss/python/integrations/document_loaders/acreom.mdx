---

[acreom](https://acreom.com) 是一个以开发者为中心的知识库，支持在本地 Markdown 文件上运行任务。

以下是如何将本地 acreom vault 加载到 LangChain 的示例。由于 acreom 中的本地 vault 是一个包含纯文本 .md 文件的文件夹，加载器需要指向该目录的路径。

Vault 文件可能包含一些存储为 YAML 头部的元数据。如果将 `collect_metadata` 设置为 true，这些值将被添加到文档的元数据中。

```python
from langchain_community.document_loaders import AcreomLoader
```

```python
loader = AcreomLoader("<path-to-acreom-vault>", collect_metadata=False)
```

```python
docs = loader.load()
```
