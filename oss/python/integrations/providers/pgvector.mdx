本页介绍如何在 LangChain 中使用 Postgres [PGVector](https://github.com/pgvector/pgvector) 生态系统。
内容分为两部分：安装与设置，以及特定 PGVector 包装器的参考。

## 安装
- 使用 `pip install pgvector` 安装 Python 包

## 设置
1. 第一步是创建一个安装了 `pgvector` 扩展的数据库。

    按照 [PGVector 安装步骤](https://github.com/pgvector/pgvector#installation) 中的说明安装数据库和扩展。Docker 镜像是最简单的入门方式。

## 包装器

### VectorStore

Postgres 向量数据库周围存在一个包装器，允许您将其用作向量存储，
无论是用于语义搜索还是示例选择。

要导入此向量存储：
```python
from langchain_community.vectorstores.pgvector import PGVector
```
