## 概述

`LocalFileStore` 是 `ByteStore` 的持久化实现，将所有内容存储在你指定的文件夹中。适合在单台机器上使用，且允许添加或删除文件的场景。

### 安装

LangChain `LocalFileStore` 集成位于 `langchain` 包中：

```python
pip install -qU langchain-classic
```

## 实例化

现在可以实例化字节存储：

```python
from pathlib import Path

from langchain_classic.storage import LocalFileStore

root_path = Path.cwd() / "data"  # can also be a path set by a string

kv_store = LocalFileStore(root_path)
```

## 使用方法

可以使用 `mset` 方法在键下设置数据：

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[b'value1', b'value2']
```

可以在 `data` 文件夹中查看创建的文件：

```python
!ls {root_path}
```

```text
key1 key2
```

可以使用 `mdelete` 方法删除数据：

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[None, None]
```
