[@decodo/langchain-ts](https://www.npmjs.com/package/@decodo/langchain-ts) 包使开发者能够在 LangChain 应用程序中使用 Decodo 的 Web Scraper API。

Web Scraper API 的特点：

- 易于访问网络数据。简化从网站和在线源检索信息的过程。
- 地理灵活性。无论区域限制如何，均可访问内容。
- 可靠的抓取。采用先进技术以避免检测和封锁。

## 功能

@decodo/langchain-ts 插件的功能包括：

- **Web 抓取**：抓取任何 URL 并检索 Markdown 内容
- **Google 搜索**：搜索 Google 并检索结构化结果
- **Amazon 搜索**：搜索 Amazon 并检索结构化产品数据
- **地理位置选择**，用于位置敏感内容
- **JavaScript 渲染**，适用于需要真实浏览器的目标
- **Markdown 输出**，用于与 LLM 进行高效的令牌连接

## 安装

```bash
npm install @decodo/langchain-ts
```

## 快速开始

要使用此项目中的工具，您需要 [Decodo 高级 Web 抓取 API](https://help.decodo.com/docs/web-scraping-api-core-and-advanced-plans) 订阅。可在 [仪表板](https://dashboard.decodo.com/) 上获取免费试用。

## 示例

每个工具的一组代理示例。

### 通用工具

```typescript
import dotenv from "dotenv";
import { ChatOpenAI } from "@langchain/openai";
import { createAgent } from "@langchain/agents";
import { DecodoUniversalTool } from "@decodo/langchain-ts";

dotenv.config();

const main = async () => {
  const username = process.env.SCRAPER_API_USERNAME!;
  const password = process.env.SCRAPER_API_PASSWORD!;

  const decodoUniversalTool = new DecodoUniversalTool({ username, password });

  const model = new ChatOpenAI({
    model: "gpt-4.1-mini",
  });

  const agent = createAgent({
    llm: model,
    tools: [decodoUniversalTool],
  });

  const result = await agent.invoke({
    messages: [
      {
        role: "user",
        content:
          "scrape the wikipedia NBA 2025 season page and tell me who won in 2025?",
      },
    ],
  });

  console.log(result.messages[result.messages.length - 1].content);
};

if (require.main === module) {
  main();
}
```

更多示例，请参见 GitHub 上的 [@decodo/langchain-ts](https://github.com/Decodo/decodo-langchain-ts)。

## 配置

所有工具都接受一个 `DecodoConfig` 对象：

```typescript
type DecodoConfig = {
  username: string; // 您的 Web Advanced 产品用户名
  password: string; // 您的 Web Advanced 产品密码
};
```
