>[百川智能](https://www.baichuan-ai.com/) 是 AGI 时代的一家中国创业公司，
> 致力于满足人类的基本需求：效率、健康与幸福。


## 安装与设置

[在此](https://platform.baichuan-ai.com/)注册并获取 API 密钥。

## LLM

查看[使用示例](/oss/python/integrations/llms/baichuan)。

```python
from langchain_community.llms import BaichuanLLM
```

## 聊天模型

查看[使用示例](/oss/python/integrations/chat/baichuan)。

```python
from langchain_community.chat_models import ChatBaichuan
```

## 嵌入模型

查看[使用示例](/oss/python/integrations/text_embedding/baichuan)。

```python
from langchain_community.embeddings import BaichuanTextEmbeddings
```
