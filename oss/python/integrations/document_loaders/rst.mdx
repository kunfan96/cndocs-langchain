>[reStructured Text（RST）](https://en.wikipedia.org/wiki/ReStructuredText) 是一种文本数据文件格式，主要用于 Python 编程语言社区的技术文档编写。

## `UnstructuredRSTLoader`

您可以通过以下工作流程，使用 `UnstructuredRSTLoader` 从 RST 文件加载数据。

```python
from langchain_community.document_loaders import UnstructuredRSTLoader

loader = UnstructuredRSTLoader(file_path="./example_data/README.rst", mode="elements")
docs = loader.load()

print(docs[0])
```
