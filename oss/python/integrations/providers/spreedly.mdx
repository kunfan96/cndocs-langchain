>[Spreedly](https://docs.spreedly.com/) 是一项服务，允许您安全地存储信用卡并使用它们与任意数量的支付网关和第三方 API 进行交易。它通过同时提供卡令牌化/保险库服务以及网关和接收器集成服务来实现这一点。由 Spreedly 令牌化的支付方法存储在 `Spreedly` 中，允许您独立存储一张卡，然后根据您的业务需求将其传递到不同的端点。
