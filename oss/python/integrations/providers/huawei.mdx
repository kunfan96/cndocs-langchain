>[华为技术有限公司](https://www.huawei.com/) 是一家中国跨国
> 数字通信技术公司。
>
>[华为云](https://www.huaweicloud.com/intl/en-us/product/) 提供全面的
> 全球云计算服务套件。


## 安装与配置

要访问 `华为云`，您需要一个访问令牌。

您还需要安装一个 Python 库：

<CodeGroup>
```bash pip
pip install -U esdk-obs-python
```

```bash uv
uv add esdk-obs-python
```
</CodeGroup>


## 文档加载器

### 华为 OBS 目录

请参阅[使用示例](/oss/python/integrations/document_loaders/huawei_obs_directory)。

```python
from langchain_community.document_loaders import OBSDirectoryLoader
```

### 华为 OBS 文件

请参阅[使用示例](/oss/python/integrations/document_loaders/huawei_obs_file)。

```python
from langchain_community.document_loaders.obs_file import OBSFileLoader
```
