>[ElevenLabs](https://elevenlabs.io/about) 是一家语音 AI 研究与部署公司，
> 致力于让内容以任何语言和声音实现全球无障碍访问。
>
>`ElevenLabs` 打造最逼真、多功能且具备上下文感知能力的
> AI 音频，支持以数百种全新及现有声音在 29 种语言中生成语音。

## 安装与设置

首先，您需要注册一个 ElevenLabs 账户。您可以参考
[此处说明](https://docs.elevenlabs.io/welcome/introduction)。

安装 Python 包：

<CodeGroup>
```bash pip
pip install elevenlabs
```

```bash uv
uv add elevenlabs
```
</CodeGroup>

## 工具

请参阅[使用示例](/oss/python/integrations/tools/eleven_labs_tts)。

```python
from langchain_community.tools import ElevenLabsText2SpeechTool
```
