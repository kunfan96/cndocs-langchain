本示例介绍如何在 LangChain 中使用 AI21SemanticTextSplitter。

## 安装

```python
pip install langchain-ai21
```

## 环境设置

我们需要获取 AI21 API 密钥并设置 AI21_API_KEY 环境变量：

```python
import os
from getpass import getpass

if "AI21_API_KEY" not in os.environ:
    os.environ["AI21_API_KEY"] = getpass()
```

## 使用示例

### 按语义含义拆分文本

此示例展示如何使用 AI21SemanticTextSplitter 根据语义含义将文本拆分为块。

```python
from langchain_ai21 import AI21SemanticTextSplitter

TEXT = (
    "我们都经历过阅读冗长、乏味且无聊的文本——财务报告、"
    "法律文件或条款与条件（不过，说实话，谁真的会去读那些条款和条件呢？）。\n"
    "想象一家雇佣数十万员工的公司。在当今信息"
    "过载的时代，近30%的工作日都花在处理文档上。这并不奇怪，"
    "因为其中一些文档是故意冗长且复杂的（你知道吗，"
    "阅读你所有的隐私政策几乎需要四分之一年的时间？）。除了"
    "效率低下，员工可能干脆避免阅读某些文档（例如，只有16%的"
    "员工在签署前完全阅读他们的雇佣合同！）。\n这就是人工智能驱动的摘要"
    "工具可以发挥作用的地方：与其阅读整个文档，这既乏味又耗时，"
    "用户可以（理想情况下）快速从文本中提取相关信息。借助大型语言模型，"
    "这些工具的开发比以往任何时候都更容易，你可以为用户提供"
    "专门根据其偏好定制的摘要。\n大型语言模型自然遵循输入"
    "（提示）中的模式，并提供遵循相同模式的连贯完成。为此，我们希望"
    '在输入中为它们提供几个示例（"少样本提示"），以便它们能够遵循。'
    "为你的问题创建正确提示的过程称为提示工程，"
    "你可以在这里阅读更多相关信息。"
)

semantic_text_splitter = AI21SemanticTextSplitter()
chunks = semantic_text_splitter.split_text(TEXT)

print(f"文本已被拆分为 {len(chunks)} 个块。")
for chunk in chunks:
    print(chunk)
    print("====")
```
