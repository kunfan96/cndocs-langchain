>[Figma](https://www.figma.com/) 是一款用于界面设计的协作式 Web 应用程序。

## 安装与设置

Figma API 需要 `access token`、`node_ids` 和 `file key`。

`file key` 可从 URL 中提取。 https://www.figma.com/file/\{filekey\}/sampleFilename

`Node IDs` 同样可在 URL 中找到。点击任意元素并查找 `'?node-id=\{node_id\}'` 参数。

`Access token` [说明文档](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens)。
