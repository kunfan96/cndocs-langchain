[Bright Data](https://brightdata.com) 是一个网络数据平台，提供用于网页抓取、搜索引擎结果页面（SERP）收集以及访问地理限制内容的工具。

Bright Data 允许开发者从网站提取结构化数据、执行搜索引擎查询，并访问可能原本被屏蔽或受地理限制的内容。该平台旨在帮助克服常见的网页抓取挑战，包括反机器人系统、验证码（CAPTCHA）和 IP 封禁。

## 安装与配置

<CodeGroup>
```bash pip
pip install langchain-brightdata
```

```bash uv
uv add langchain-brightdata
```
</CodeGroup>

您需要设置 Bright Data API 密钥：

在 [Bright Data](https://brightdata.com/?utm_source=tech-partner&utm_medium=link&utm_campaign=langchain&hs_signup=1) 注册，并从您的账户设置中获取 API 密钥。在下面的示例中，将 `"your-api-key"` 替换为您的实际 API 密钥：

```python
import os
os.environ["BRIGHT_DATA_API_KEY"] = "your-api-key"
```

或者，您可以在初始化工具时直接传递它：

```python
from langchain_brightdata import BrightDataSERP

tool = BrightDataSERP(bright_data_api_key="your-api-key")
```

## 工具

Bright Data 集成提供了以下多种工具：

- [BrightDataSERP](/oss/python/integrations/tools/brightdata_serp) - 支持地理定向和自定义区域支持的搜索引擎结果收集
- [BrightDataUnlocker](/oss/python/integrations/tools/brightdata_unlocker) - 访问任何可能受地理限制或受机器人保护的公共网站
- [BrightDataWebScraperAPI](/oss/python/integrations/tools/brightdata-webscraperapi) - 从 44 个热门域名（包括 Amazon、LinkedIn、Instagram、TikTok 等）中提取结构化数据
