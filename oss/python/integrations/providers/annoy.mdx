> [Annoy](https://github.com/spotify/annoy)（`Approximate Nearest Neighbors Oh Yeah`）
> 是一个带有 Python 绑定的 C++ 库，用于在空间中搜索与给定查询点相近的点。它还会创建大型只读的基于文件的数据
> 结构，这些结构被映射到内存中，以便多个进程可以共享同一份数据。

## 安装与设置

<CodeGroup>
```bash pip
pip install annoy
```

```bash uv
uv add annoy
```
</CodeGroup>
