Dell 是一家全球性科技公司，提供包括 AI 解决方案在内的各种硬件、软件和服务。其 AI 产品组合包括专为 AI 工作负载设计的基础设施，以及针对 AI 数据管理优化的 Dell PowerScale 存储系统。

## PowerScale

Dell [PowerScale](https://www.dell.com/en-us/shop/powerscale-family/sf/powerscale) 是一款企业级扩展存储系统，搭载业界领先的 OneFS 文件系统，可本地部署或在云端运行。

### 安装与设置

<CodeGroup>
```bash pip
pip install powerscale-rag-connector
```

```bash uv
uv add powerscale-rag-connector
```
</CodeGroup>
