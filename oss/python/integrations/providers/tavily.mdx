[Tavily](https://tavily.com) 是一款专为 AI 代理设计的搜索引擎。
Tavily 同时提供搜索和抽取 API，AI 开发者可以轻松将其应用程序与实时在线信息集成。Tavily 的核心使命是从可信来源提供真实可靠的信息，提升 AI 生成内容和推理的准确性与可靠性。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-tavily
```

```bash uv
uv add langchain-tavily
```
</CodeGroup>
