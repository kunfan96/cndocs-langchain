---
title: GRAPH_RECURSION_LIMIT
---

您的 LangGraph [`StateGraph`](https://langchain-ai.github.io/langgraph/reference/graphs/#langgraph.graph.state.StateGraph) 在触及停止条件之前已达到最大步骤数。
这通常是由类似以下示例的代码引起的无限循环所导致的：

```python
class State(TypedDict):
    some_key: str

builder = StateGraph(State)
builder.add_node("a", ...)
builder.add_node("b", ...)
builder.add_edge("a", "b")
builder.add_edge("b", "a")
...

graph = builder.compile()
```




然而，复杂的图也可能自然地触及默认限制。

## 故障排除

* 如果您的图不需要经历很多次迭代，则很可能存在循环。请检查您的逻辑中是否有无限循环。

* 如果您的图比较复杂，可以在调用图时向 `config` 对象传入更高的 `recursion_limit` 值，如下所示：

```python
graph.invoke({...}, {"recursion_limit": 100})
```
