所有与 YDB 相关的功能。

> [YDB](https://ydb.tech/) 是一个多功能的开源分布式 SQL 数据库，它结合了
> 高可用性和可扩展性，以及强一致性和 ACID 事务。
> 它同时支持事务性 (OLTP)、分析性 (OLAP) 和流式工作负载。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-ydb
```

```bash uv
uv add langchain-ydb
```
</CodeGroup>

## 向量存储

要导入 YDB 向量存储：

```python
from langchain_ydb.vectorstores import YDB
```

有关 YDB 向量存储的更详细演练，请参见[此笔记本](/oss/python/integrations/vectorstores/ydb)。
