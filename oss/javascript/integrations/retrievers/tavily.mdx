[Tavily 的搜索 API](https://tavily.com) 是一个专为 AI 代理（LLM）构建的搜索引擎，能够快速提供实时、准确且基于事实的结果。

## 概述

这将帮助您开始使用 `TavilySearchAPIRetriever` [检索](/oss/javascript/langchain/retrieval)。有关所有 `TavilySearchAPIRetriever` 功能和配置的详细文档，请前往 [API 参考](https://reference.langchain.com/javascript/langchain-community/retrievers/tavily_search_api/TavilySearchAPIRetriever)。

### 集成详情

| 检索器 | 来源 | 包 |
| :--- | :--- | :---: |
[`TavilySearchAPIRetriever`](https://reference.langchain.com/javascript/langchain-community/retrievers/tavily_search_api/TavilySearchAPIRetriever) | 网络上的信息。 | [`@langchain/community`](https://npmjs.com/@langchain/community/) |

## 设置

您需要使用您的 Tavily API 密钥填充 `TAVILY_API_KEY` 环境变量，或将其作为 `apiKey` 传递给构造函数。通过[在其网站上注册](https://tavily.com/)获取密钥。

如果您想从单个查询获取自动跟踪，还可以通过取消注释以下内容来设置您的 [LangSmith](/langsmith/home) API 密钥：

```typescript
// process.env.LANGSMITH_API_KEY = "<YOUR API KEY HERE>";
// process.env.LANGSMITH_TRACING = "true";
```

### 安装

此检索器位于 `@langchain/community` 包中：

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core
```
```bash yarn
yarn add @langchain/community @langchain/core
```
```bash pnpm
pnpm add @langchain/community @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化我们的检索器：

```typescript
import { TavilySearchAPIRetriever } from "@langchain/community/retrievers/tavily_search_api";

const retriever = new TavilySearchAPIRetriever({
  k: 3,
});
```

有关允许参数的完整列表，请参阅[官方文档](https://docs.tavily.com/docs/tavily-api/rest_api#parameters)。您可以通过 `kwargs` 对象将任何参数传递给 SDK。

## 用法

```typescript
const query = "what is the current weather in SF?";

await retriever.invoke(query);
```
