>[SparkLLM](https://xinghuo.xfyun.cn/spark) 是科大讯飞自主研发的大规模认知模型。
它通过学习大量文本、代码和图像，具备跨领域知识和语言理解能力。
它能够理解并基于自然对话执行任务。

## 聊天模型

参见[使用示例](/oss/python/integrations/chat/sparkllm)。

```python
from langchain_community.chat_models import ChatSparkLLM
```

## LLM

参见[使用示例](/oss/python/integrations/llms/sparkllm)。

```python
from langchain_community.llms import SparkLLM
```

## 嵌入模型

参见[使用示例](/oss/python/integrations/embeddings/sparkllm)

```python
from langchain_community.embeddings import SparkLLMTextEmbeddings
```
