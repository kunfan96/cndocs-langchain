`Cerebrium` 是 AWS SageMaker 的替代方案。它还通过 API 提供对[多种大语言模型](https://docs.cerebrium.ai/cerebrium/prebuilt-models/deployment)的访问权限。

本笔记本将介绍如何将 LangChain 与 [CerebriumAI](https://docs.cerebrium.ai/introduction) 配合使用。

## 安装 cerebrium

使用 `CerebriumAI` API 需要安装 `cerebrium` 包。请使用 `pip3 install cerebrium` 安装 `cerebrium`。

```python
# 安装包
!pip3 install cerebrium
```

## 导入模块

```python
import os

from langchain_classic.chains import LLMChain
from langchain_community.llms import CerebriumAI
from langchain_core.prompts import PromptTemplate
```

## 设置环境变量 API Key

请确保从 [CerebriumAI 控制台](https://dashboard.cerebrium.ai/login) 获取您的 API Key。您将获得 1 小时的免费无服务器 GPU 算力，用于测试不同的模型。

```python
os.environ["CEREBRIUMAI_API_KEY"] = "YOUR_KEY_HERE"
```

## 创建 CerebriumAI 实例

您可以指定不同的参数，例如模型端点 URL、最大长度、温度等。您必须提供一个端点 URL。

```python
llm = CerebriumAI(endpoint_url="YOUR ENDPOINT URL HERE")
```

## 创建 Prompt 模板

我们将为问答（Question and Answer）创建一个 prompt 模板。

```python
template = """Question: {question}

Answer: Let's think step by step."""

prompt = PromptTemplate.from_template(template)
```

## 初始化 LLMChain

```python
llm_chain = LLMChain(prompt=prompt, llm=llm)
```

## 运行 LLMChain

输入一个问题并运行 LLMChain。

```python
question = "What NFL team won the Super Bowl in the year Justin Beiber was born?"

llm_chain.run(question)
```
