Eden AI 正通过汇集最佳的 AI 提供商，革新 AI 场景，让用户解锁无限可能并挖掘人工智能的真正潜力。借助一个全面的一站式平台，它允许用户快速将 AI 功能部署到生产环境中，通过单一 API 访问全部 AI 能力。（网站: [edenai.co/](https://edenai.co/)）

本示例介绍了如何使用 LangChain 与 Eden AI 模型进行交互。

-----------------------------------------------------------------------------------

`EdenAI` 不仅限于简单的模型调用。它还提供了高级功能，包括：

- **多个提供商**：访问由不同提供商提供的多种语言模型，为您提供选择最适合您应用场景的模型的自由。

- **故障转移机制**：设置故障转移机制以确保即使主提供商不可用时也能无缝运行，您可以轻松切换到替代提供商。

- **使用量跟踪**：按项目和 API 密钥跟踪使用统计信息。此功能允许您有效监控和管理资源消耗。

- **监控与可观测性**：`EdenAI` 提供平台上的全面监控和可观测性工具。监测语言模型的性能、分析使用模式并获得有价值的见解以优化您的应用程序。

访问 EDENAI 的 API 需要一个 API 密钥，

您可以通过在 [app.edenai.run/user/register](https://app.edenai.run/user/register) 创建帐户并在 [app.edenai.run/admin/iam/api-keys](https://app.edenai.run/admin/iam/api-keys) 处获取。

一旦我们有了密钥，我们将通过运行以下命令将其设置为环境变量：

```bash
export EDENAI_API_KEY="..."
```

您可以在 API 参考中找到更多详细信息：[docs.edenai.co/reference](https://docs.edenai.co/reference)

如果您不希望设置环境变量，也可以直接在初始化 EdenAI 会话模型类时通过 `edenai_api_key` 命名参数传递密钥。

```python
from langchain_community.chat_models.edenai import ChatEdenAI
from langchain.messages import HumanMessage
```

```python
chat = ChatEdenAI(
    edenai_api_key="...", provider="openai", temperature=0.2, max_tokens=250
)
```

```python
messages = [HumanMessage(content="Hello !")]
chat.invoke(messages)
```

```text
AIMessage(content='Hello! How can I assist you today?')
```

```python
await chat.ainvoke(messages)
```

```text
AIMessage(content='Hello! How can I assist you today?')
```

## 流式传输和批量处理

`ChatEdenAI` 支持流式传输和批量处理。以下是一个示例。

```python
for chunk in chat.stream(messages):
    print(chunk.content, end="", flush=True)
```

```text
Hello! How can I assist you today?
```

```python
chat.batch([messages])
```

```text
[AIMessage(content='Hello! How can I assist you today?')]
```

## 故障转移机制

在 Eden AI 中，您可以设置故障转移机制以确保即使主提供商不可用时也能无缝运行。您可以通过轻松切换到替代提供商来实现这一点。

```python
chat = ChatEdenAI(
    edenai_api_key="...",
    provider="openai",
    temperature=0.2,
    max_tokens=250,
    fallback_providers="google",
)
```

在这个示例中，如果 OpenAI 遇到任何问题，您可以使用 Google 作为备用提供商。

有关更多关于 Eden AI 的信息和详细内容，请参阅此链接：[docs.edenai.co/docs/additional-parameters](https://docs.edenai.co/docs/additional-parameters)

## 工具

### bind_tools()

使用 `ChatEdenAI.bind_tools`，我们可以轻松地将 Pydantic 类、字典模式、LangChain 工具或甚至函数作为工具传递给模型。

```python
from pydantic import BaseModel, Field

llm = ChatEdenAI(provider="openai", temperature=0.2, max_tokens=500)


class GetWeather(BaseModel):
    """获取给定位置的当前天气"""

    location: str = Field(description="城市和州，例如：旧金山，加利福尼亚")


llm_with_tools = llm.bind_tools([GetWeather])
```

```python
ai_msg = llm_with_tools.invoke(
    "what is the weather like in San Francisco",
)
ai_msg
```

### with_structured_output()

`BaseChatModel.with_structured_output` 接口使从聊天模型中获取结构化输出变得容易。您可以使用 `ChatEdenAI.with_structured_output`（它在幕后使用工具调用），以更可靠地返回特定格式的输出：

```python
structured_llm = llm.with_structured_output(GetWeather)
structured_llm.invoke(
    "what is the weather like in San Francisco",
)
```

```text
GetWeather(location='San Francisco')
```

### 传递工具结果给模型

以下是如何使用工具的一个完整示例。将工具输出传递给模型，从模型中获取结果。

```python
from langchain.messages import HumanMessage, ToolMessage
from langchain.tools import tool


@tool
def add(a: int, b: int) -> int:
    """添加 `a` 和 `b`。

    参数：
        a: 第一个整数
        b: 第二个整数
    """
    return a + b


llm = ChatEdenAI(
    provider="openai",
    max_tokens=1000,
    temperature=0.2,
)

llm_with_tools = llm.bind_tools([add], tool_choice="required")

query = "What is 11 + 11?"

messages = [HumanMessage(query)]
ai_msg = llm_with_tools.invoke(messages)
messages.append(ai_msg)

tool_call = ai_msg.tool_calls[0]
tool_output = add.invoke(tool_call["args"])

# 将工具结果添加到模型
messages.append(ToolMessage(tool_output, tool_call_id=tool_call["id"]))

llm_with_tools.invoke(messages).content
```

```text
'11 + 11 = 22'
```
