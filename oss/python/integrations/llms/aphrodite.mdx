[Aphrodite](https://github.com/PygmalionAI/aphrodite-engine) 是一个开源的大规模推理引擎，旨在为 [PygmalionAI](https://pygmalion.chat) 网站上的数千名用户提供支持。

* 采用 vLLM 的注意力机制，实现高吞吐量与低延迟
* 支持多种最先进的（SOTA）采样方法
* 使用 Exllamav2 GPTQ 内核，在较小批次大小下获得更高的吞吐量

本笔记本将介绍如何在 LangChain 和 Aphrodite 中使用大语言模型。

使用前，请确保已安装 `aphrodite-engine` Python 包。

```python
## 安装用于此集成的所需 langchain 包
pip install -qU langchain-community
```

```python
pip install -qU  aphrodite-engine==0.4.2
# pip list | grep aphrodite
```

```python
from langchain_community.llms import Aphrodite

llm = Aphrodite(
    model="PygmalionAI/pygmalion-2-7b",
    trust_remote_code=True,  # Hugging Face 模型必需
    max_tokens=128,
    temperature=1.2,
    min_p=0.05,
    mirostat_mode=0,  # 更改为 2 以启用 mirostat
    mirostat_tau=5.0,
    mirostat_eta=0.1,
)

print(
    llm.invoke(
        '<|system|>Enter RP mode. You are Ayumu "Osaka" Kasuga.<|user|>Hey Osaka. Tell me about yourself.<|model|>'
    )
)
```

## 将模型集成到 LLMChain 中

```python
from langchain_classic.chains import LLMChain
from langchain_core.prompts import PromptTemplate

template = """问题: {question}

回答: 让我们一步步思考。"""
prompt = PromptTemplate.from_template(template)

llm_chain = LLMChain(prompt=prompt, llm=llm)

question = "第一款宝可梦游戏发布时，美国总统是谁？"

print(llm_chain.run(question))
```

## 分布式推理

Aphrodite 支持分布式张量并行推理与服务。

要使用 LLM 类运行多 GPU 推理，请将 `tensor_parallel_size` 参数设置为您想要使用的 GPU 数量。例如，要在 4 个 GPU 上运行推理：

```python
from langchain_community.llms import Aphrodite

llm = Aphrodite(
        model="PygmalionAI/mythalion-13b",
        tensor_parallel_size=4,
        trust_remote_code=True,  # Hugging Face 模型必需
)

llm("AI 的未来是什么？")
```
