>[Chaindesk](https://chaindesk.ai) 是一个[开源](https://github.com/gmpetrov/databerry)的文档检索平台，可帮助将个人数据与大型语言模型连接起来。


## 安装与设置

我们需要注册 Chaindesk，创建数据存储，添加一些数据，并获取数据存储 API 端点 URL。
我们需要 [API 密钥](https://docs.chaindesk.ai/api-reference/authentication)。

## 检索器

查看[使用示例](/oss/python/integrations/retrievers/chaindesk)。

```python
from langchain_classic.retrievers import ChaindeskRetriever
```
