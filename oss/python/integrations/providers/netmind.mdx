[Netmind AI](https://www.netmind.ai/) 更快、更智能、更具成本效益地构建 AI。
利用我们的全球 GPU 网络进行训练、微调、推理运行与规模扩展——您的全能 AI 引擎。

本示例介绍如何使用 LangChain 与 Netmind AI 模型进行交互。

## 安装与设置

```bash
pip install langchain-netmind
```

获取一个 Netmind API 密钥，并将其设置为环境变量（`NETMIND_API_KEY`）。
访问 [www.netmind.ai/](https://www.netmind.ai/) 注册 Netmind 并生成 API 密钥。
