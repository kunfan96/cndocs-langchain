# Moonshot

>[Moonshot AI](https://platform.moonshot.cn/) 是一家中国人工智能公司，提供大语言模型服务，包括 Kimi 系列模型。

本页介绍如何在 LangChain 中使用 Moonshot。

## 安装与配置

安装 Moonshot 集成包：

```bash
pip install langchain-moonshot
```

从 [Moonshot 控制台](https://platform.moonshot.cn/console/api-keys) 获取 Moonshot API 密钥，并将其设置为环境变量：

```bash
export MOONSHOT_API_KEY="your-api-key-here"
```

## 聊天模型

查看[详细使用示例](/oss/python/integrations/chat/moonshot)。

```python
from langchain_moonshot import ChatMoonshot
```

## LLM

查看[详细使用示例](/oss/python/integrations/llms/moonshot)。

```python
from langchain_community.llms.moonshot import Moonshot
```
