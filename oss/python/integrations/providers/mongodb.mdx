>[MongoDB](https://www.mongodb.com/) 是一款 NoSQL 文档型数据库，支持具有动态模式的类 JSON 文档。

**注意：**
- 其他 `MongoDB` 集成请参阅 [MongoDB Atlas 页面](/oss/python/integrations/providers/mongodb_atlas)。

## 安装与设置

安装 Python 包：

<CodeGroup>
```bash pip
pip install langchain-mongodb
```

```bash uv
uv add langchain-mongodb
```
</CodeGroup>
