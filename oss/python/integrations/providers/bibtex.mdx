>[BibTeX](https://www.ctan.org/pkg/bibtex) 是一种文件格式和参考文献管理系统，通常与 `LaTeX` 排版软件配合使用。它用于组织和存储学术与研究文档的文献信息。

## 安装与设置

我们需要安装 `bibtexparser` 和 `pymupdf` 包。

<CodeGroup>
```bash pip
pip install bibtexparser pymupdf
```

```bash uv
uv add bibtexparser pymupdf
```
</CodeGroup>


## 文档加载器

请参阅[使用示例](/oss/python/integrations/document_loaders/bibtex)。

```python
from langchain_community.document_loaders import BibtexLoader
```
