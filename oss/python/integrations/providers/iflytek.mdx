>[科大讯飞](https://www.iflytek.com) 是一家成立于 1999 年的中国信息技术公司。
> 它创建语音识别软件和
> 基于语音的互联网/移动产品，涵盖教育、通信、
> 音乐、智能玩具等行业。

## 安装与设置

- 从 [iFlyTek SparkLLM API 控制台](https://console.xfyun.cn/services/bm3) 获取 `SparkLLM` 的 app_id、api_key 和 api_secret（更多信息请参阅 [iFlyTek SparkLLM 简介](https://xinghuo.xfyun.cn/sparkapi)）。
- 安装 Python 包（嵌入模型不需要）：

<CodeGroup>
```bash pip
pip install websocket-client
```

```bash uv
uv add websocket-client
```
</CodeGroup>

## LLMs

查看[使用示例](/oss/python/integrations/llms/sparkllm)。

```python
from langchain_community.llms import SparkLLM
```

## 聊天模型

查看[使用示例](/oss/python/integrations/chat/sparkllm)。

```python
from langchain_community.chat_models import ChatSparkLLM
```

```python
from langchain_community.embeddings import SparkLLMTextEmbeddings
```
