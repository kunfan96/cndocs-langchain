---
title: MODEL_AUTHENTICATION
---

<Note>
    目前仅在 `langchainjs` (JavaScript/TypeScript) 中使用。
</Note>

你的模型提供商拒绝你访问其服务。

此错误通常在你的身份验证凭据或 API 密钥存在问题时发生。

## 故障排除

* 确认你的 API 密钥或身份验证凭据准确有效。
* 如果使用基于环境变量的身份验证，请验证：
    - 变量名拼写正确
    - 变量包含已分配的值
    - 第三方包如 `dotenv` 没有干扰加载
* 如果使用代理或非标准端点，请确保你的自定义提供商不需要替代的身份验证方案。
* 通过显式传递凭据来绕过环境变量问题：

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(api_key="YOUR_KEY_HERE")
```
