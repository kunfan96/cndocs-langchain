本页介绍如何在 LangChain 中使用 [cloro](https://cloro.dev/) 监控 API。

## 安装与设置
- 通过 `pip install -U langchain-cloro` 安装依赖项
- 从 [cloro 控制台](https://dashboard.cloro.dev/)获取 cloro API 密钥，并将其设置为环境变量（`CLORO_API_KEY`）

## 工具

cloro 提供多种工具用于抓取 AI 平台并提取结构化数据。使用这些工具：

```python
from langchain_cloro import CloroGoogleSearch, CloroChatGPT, CloroGemini, CloroPerplexity, CloroGrok, CloroCopilot
```

详细文档和示例请参阅：
- [Google 搜索抓取器](/oss/python/integrations/tools/cloro#google-search)
- [ChatGPT 抓取器](/oss/python/integrations/tools/cloro#chatgpt-monitoring)
- [Gemini 抓取器](/oss/python/integrations/tools/cloro#gemini-monitoring)
- [Perplexity 抓取器](/oss/python/integrations/tools/cloro#perplexity-monitoring)
- [Grok 抓取器](/oss/python/integrations/tools/cloro#grok-monitoring)
- [Copilot 抓取器](/oss/python/integrations/tools/cloro#copilot-monitoring)

更多信息，请访问 [cloro 文档](https://docs.cloro.dev)。
