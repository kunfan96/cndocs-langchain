>[DingoDB](https://github.com/dingodb) 是一个分布式多模态向量数据库。它融合了数据湖和向量数据库的特点，
> 可存储任何类型的数据（键值对、PDF、音频、视频等），不受大小限制。利用 DingoDB，您可以构建自己的 Vector Ocean（继数据仓库和数据湖之后的下一代数据架构），使您能够通过单一 SQL 以极低的实时延迟分析结构化和非结构化数据。

## 安装与设置

安装 Python SDK

<CodeGroup>
```bash pip
pip install dingodb
```

```bash uv
uv add dingodb
```
</CodeGroup>

## VectorStore

有一个 DingoDB 索引的封装，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

要导入此向量存储：

```python
from langchain_community.vectorstores import Dingo
```

有关 DingoDB 封装的更详细介绍，请参阅[此笔记本](/oss/python/integrations/vectorstores/dingo)
