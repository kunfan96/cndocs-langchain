>[E2B](https://e2b.dev/) 提供开源的安全沙箱
> 用于执行 AI 生成的代码。更多详情请访问 [e2b GitHub 仓库](https://github.com/e2b-dev)。

## 安装与配置

您需要安装一个 Python 包：

<CodeGroup>
```bash pip
pip install e2b_code_interpreter
```

```bash uv
uv add e2b_code_interpreter
```
</CodeGroup>

## 工具

```python
from langchain_community.tools import E2BDataAnalysisTool
```
