## 用例

以下是按框架组织的常见用例教程。

### 深度 Agents

[深度 agents](/oss/javascript/deepagents/overview) 包含用于管理上下文、虚拟文件系统和其他常见 agent 需求的内置功能。

<Card title="数据分析" icon="chart-pie" href="/oss/javascript/deepagents/data-analysis" horizontal>
  构建一个向 Slack 发送报告的数据分析 agent。
</Card>

### LangChain

[LangChain](/oss/javascript/langchain/overview) [agent](/oss/javascript/langchain/agents) 实现使得简单的用例易于上手。

<Card title="语义搜索" icon="search" href="/oss/javascript/langchain/knowledge-base" horizontal>
  使用 LangChain 组件构建一个基于 PDF 的语义搜索引擎。
</Card>

<Card title="RAG Agent" icon="user-search" href="/oss/javascript/langchain/rag" horizontal>
  创建一个检索增强生成 (RAG) agent。
</Card>

<Card title="SQL Agent" icon="database" href="/oss/javascript/langchain/sql-agent" horizontal>
  构建一个 SQL agent，通过人机回环审查与数据库交互。
</Card>

<Card title="语音 Agent" icon="microphone" href="/oss/javascript/langchain/voice-agent" horizontal>
  构建一个您可以对其说话并聆听的 agent。
</Card>

### LangGraph

LangChain 的 [agent](/oss/javascript/langchain/agents) 实现使用了 [LangGraph](/oss/javascript/langgraph/overview) 原语。
如果需要更深度的定制，可以直接在 LangGraph 中实现 agents。

<Card title="自定义 RAG Agent" icon="user-search" href="/oss/javascript/langgraph/agentic-rag" horizontal>
  使用 LangGraph 原语构建一个 RAG agent 以实现细粒度控制。
</Card>



### 多 Agent

这些教程演示了 [多 agent 模式](/oss/javascript/langchain/multi-agent)，将 LangChain agents 与 LangGraph 工作流结合在一起。
