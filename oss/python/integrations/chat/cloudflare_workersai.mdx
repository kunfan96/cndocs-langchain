## 配置

要访问 CloudflareWorkersAI 模型，您需要创建一个 CloudflareWorkersAI 账户、获取 API 密钥，并安装 `langchain-cloudflare` 整合包。

### 凭据

请前往 [www.cloudflare.com/developer-platform/products/workers-ai/](https://www.cloudflare.com/developer-platform/products/workers-ai/) 注册 CloudflareWorkersAI 并生成一个 API 密钥。完成此操作后，请设置 CF_AI_API_KEY 环境变量和 CF_ACCOUNT_ID 环境变量：

```python
import getpass
import os

if not os.getenv("CF_AI_API_KEY"):
    os.environ["CF_AI_API_KEY"] = getpass.getpass(
        "请输入您的 CloudflareWorkersAI API 密钥: "
    )

if not os.getenv("CF_ACCOUNT_ID"):
    os.environ["CF_ACCOUNT_ID"] = getpass.getpass(
        "请输入您的 CloudflareWorkersAI 账户 ID: "
    )
```

如果您希望自动跟踪模型调用，还可以通过取消注释以下内容设置 [LangSmith](/langsmith/home) API 密钥：

```python
os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("请输入您的 LangSmith API 密钥: ")
```

### 安装

LangChain CloudflareWorkersAI 整合位于 `langchain-cloudflare` 包中：

```python
pip install -qU langchain-cloudflare
```

## 实例化

现在我们可以实例化我们的模型对象并生成聊天完成结果：

- 更新模型实例化参数。

```python
from langchain_cloudflare.chat_models import ChatCloudflareWorkersAI

llm = ChatCloudflareWorkersAI(
    model="@cf/meta/llama-3.3-70b-instruct-fp8-fast",
    temperature=0,
    max_tokens=1024,
    # 其他参数...
)
```

## 调用

```python
messages = [
    (
        "system",
        "您是一个帮助助手，可以将英语翻译成法语。请翻译用户句子。",
    ),
    ("human", "我爱编程。"),
]
ai_msg = llm.invoke(messages)
```

## 结构化输出

```python
json_schema = {
    "title": "joke",
    "description": "向用户讲述的笑话。",
    "type": "object",
    "properties": {
        "setup": {
            "type": "string",
            "description": "笑话的设定。",
        },
        "punchline": {
            "type": "string",
            "description": "笑话的笑点。",
        },
        "rating": {
            "type": "integer",
            "description": "从 1 到 10 的评分，表示笑话有多好笑。",
            "default": None,
        },
    },
    "required": ["setup", "punchline"],
}
structured_llm = llm.with_structured_output(json_schema)

structured_llm.invoke("给我讲个关于猫的笑话")
```

```text
{'setup': '为什么这只猫加入了一个乐队？',
 'punchline': '因为它想成为打击乐手（purr-cussionist）',
 'rating': 8}
```

## 绑定工具

```python
from typing import List

from langchain.tools import tool


@tool
def validate_user(user_id: int, addresses: List[str]) -> bool:
    """使用历史地址验证用户。

    参数:
        user_id (int): 用户 ID。
        addresses (List[str]): 字符串列表形式的历史住址。
    """
    return True


llm_with_tools = llm.bind_tools([validate_user])

result = llm_with_tools.invoke(
    "你能验证一下 123 号用户吗？他们之前住在波士顿 MA 的 123 假设街和休斯顿 TX 的 234 装扮大道。"
)
result.tool_calls
```

```text
[{'name': 'validate_user',
  'args': {'user_id': '123',
   'addresses': '["123 Fake St in Boston MA", "234 Pretend Boulevard in Houston TX"]'},
  'id': '31ec7d6a-9ce5-471b-be64-8ea0492d1387',
  'type': 'tool_call'}]
```
