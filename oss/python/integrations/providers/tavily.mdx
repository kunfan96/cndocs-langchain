[Tavily](https://tavily.com) 是一个搜索引擎，专为 AI 代理设计。
Tavily 提供搜索和提取 API，AI 开发者可以轻松地将其应用程序与实时在线信息集成。Tavily 的主要使命是提供来自可信来源的事实和可靠信息，增强 AI 生成内容和推理的准确性与可靠性。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-tavily
```

```bash uv
uv add langchain-tavily
```
</CodeGroup>

## 工具

有关可用工具的详细信息，请参见 [tavily_search](/oss/python/integrations/tools/tavily_search) 和 [tavily_extract](/oss/python/integrations/tools/tavily_extract)。
