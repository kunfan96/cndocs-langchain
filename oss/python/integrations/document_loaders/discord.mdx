>[Discord](https://discord.com/) 是一个 VoIP 和即时消息社交平台。用户可以通过私聊或在称为"服务器"的社区中进行语音通话、视频通话、文字消息、媒体和文件交流。服务器是一组持久化聊天室和语音频道的集合，可通过邀请链接访问。

请按照以下步骤下载您的 `Discord` 数据：

1. 进入您的**用户设置**
2. 然后进入**隐私与安全**
3. 前往**请求我的所有数据**，点击**请求数据**按钮

您可能需要最多 30 天才能收到数据。您将在注册 Discord 时使用的邮箱地址收到一封邮件，该邮件中会有一个下载按钮，可用于下载您的个人 Discord 数据。

```python
import os

import pandas as pd
```

```python
path = input('Please enter the path to the contents of the Discord "messages" folder: ')
li = []
for f in os.listdir(path):
    expected_csv_path = os.path.join(path, f, "messages.csv")
    csv_exists = os.path.isfile(expected_csv_path)
    if csv_exists:
        df = pd.read_csv(expected_csv_path, index_col=None, header=0)
        li.append(df)

df = pd.concat(li, axis=0, ignore_index=True, sort=False)
```

```python
from langchain_community.document_loaders.discord import DiscordChatLoader
```

```python
loader = DiscordChatLoader(df, user_id_col="ID")
print(loader.load())
```
