>[Datadog](https://www.datadoghq.com/) 是一个面向云规模应用的监控和分析平台。

## 安装与设置

<CodeGroup>
```bash pip
pip install datadog_api_client
```

```bash uv
uv add datadog_api_client
```
</CodeGroup>

我们必须使用 Datadog API 密钥和 APP 密钥初始化加载器，并设置查询以提取所需日志。

## 文档加载器

查看[使用示例](/oss/python/integrations/document_loaders/datadog_logs)。
