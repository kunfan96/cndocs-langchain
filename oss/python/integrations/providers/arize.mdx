[Arize](https://arize.com) 是一个 AI 可观测性与 LLM 评估平台，为 LangChain 应用提供支持，提供输入、嵌入、检索、函数和输出消息的详细追踪。


## 安装与设置

首先，您需要安装 `arize` Python 包。

<CodeGroup>
```bash pip
pip install arize
```

```bash uv
uv add arize
```
</CodeGroup>

其次，您需要设置您的 [Arize 账户](https://app.arize.com/auth/join)
并获取您的 `API_KEY` 或 `SPACE_KEY`。


## 回调处理器

```python
from langchain_community.callbacks import ArizeCallbackHandler
```
