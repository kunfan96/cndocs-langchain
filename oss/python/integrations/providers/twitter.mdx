>[Twitter](https://twitter.com/) 是一个在线社交媒体和社交网络服务。

## 安装与设置

<CodeGroup>
```bash pip
pip install tweepy
```

```bash uv
uv add tweepy
```
</CodeGroup>

我们必须使用 `Twitter API` 令牌初始化加载器，并且需要设置 Twitter `username`。

## 文档加载器

参见[使用示例](/oss/python/integrations/document_loaders/twitter)。

```python
from langchain_community.document_loaders import TwitterTweetLoader
```
