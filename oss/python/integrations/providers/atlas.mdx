>[Nomic Atlas](https://docs.nomic.ai/index.html) 是一个用于同时处理
> 小规模和互联网规模非结构化数据集的平台。


## 安装与设置

- 使用 `pip install nomic` 安装 Python 包
- `Nomic` 也包含在 langchain 的 poetry 扩展中，可通过 `poetry install -E all` 安装
