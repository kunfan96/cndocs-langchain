>[Brave Search](https://en.wikipedia.org/wiki/Brave_Search) 是由 Brave Software 开发的搜索引擎。
> - `Brave Search` 使用自己的网页索引。截至 2022 年 5 月，其索引已覆盖超过 100 亿个页面，
> 92% 的搜索结果无需依赖任何第三方，其余结果从 Bing API（服务端）或 Google（客户端选择加入）获取。
> 根据 Brave 的说法，其索引"有意保持小于 Google 或 Bing"，以避免垃圾信息和低质量内容，
> 但缺点是"Brave Search 在处理长尾查询时尚不如 Google。"
>- `Brave Search Premium`：截至 2023 年 4 月，Brave Search 是无广告网站，但将逐步切换为包含广告和高级会员无广告体验的新模式。
> 默认情况下不会收集用户数据（包括 IP 地址）。高级账户用户可以选择加入数据收集。


## 安装与设置

要访问 Brave Search API，您需要[创建账户并获取 API 密钥](https://api.search.brave.com/app/dashboard)。
