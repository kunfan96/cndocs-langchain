>[ArangoDB](https://github.com/arangodb/arangodb) 是一个可扩展的图数据库系统，旨在
> 更快地从关联数据中挖掘价值。通过单一查询语言提供原生图支持、集成搜索引擎和 JSON 支持。ArangoDB 可在本地部署、云端 – 任何地方运行。

## 安装与设置

使用以下命令安装 [ArangoDB Python 驱动程序](https://github.com/ArangoDB-Community/python-arango) 包：

<CodeGroup>
```bash pip
pip install python-arango
```

```bash uv
uv add python-arango
```
</CodeGroup>

## 图问答链

将您的 `ArangoDB` 数据库与聊天模型连接，以获取数据洞察。

请参阅 [ArangoDB 笔记本示例](/oss/python/integrations/graphs/arangodb)。

```python
from arango import ArangoClient

from langchain_community.graphs import ArangoGraph
from langchain_classic.chains import ArangoGraphQAChain
```
