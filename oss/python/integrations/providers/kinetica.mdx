[Kinetica](https://www.kinetica.com/) 是一款专为支持时序与空间数据的分析及生成式 AI 而实时构建的数据库。

## 聊天模型

Kinetica LLM 包装器使用 [Kinetica SqlAssist
LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/) 将自然语言转换为
SQL，以简化数据检索过程。

请参阅 [Kinetica 语言转 SQL 聊天模型](/oss/python/integrations/chat/kinetica) 以获取使用方法。


```python
from langchain_kinetica import ChatKinetica
```

## 向量存储

Kinetica 向量存储包装器利用了 Kinetica 对 [向量
相似度搜索](https://docs.kinetica.com/7.2/vector_search/) 的原生支持。

请参阅 [Kinetica 向量存储 API](/oss/python/integrations/vectorstores/kinetica) 以获取使用方法。


```python
from langchain_kinetica import KineticaVectorstore
```

## 文档加载器

Kinetica 文档加载器可用于从
[Kinetica](https://www.kinetica.com/) 数据库加载 LangChain [文档](https://reference.langchain.com/python/langchain-core/documents/base/Document)。

请参阅 [Kinetica 文档加载器](/oss/python/integrations/document_loaders/kinetica) 以获取使用方法。


```python
from langchain_kinetica import KineticaLoader
```

## 检索器

给定非结构化查询时，Kinetica 检索器可返回相关文档。

请参阅 [基于 Kinetica 向量存储的检索器](/oss/python/integrations/retrievers/kinetica) 以获取使用方法。
