LangChain 中的 [MCP Toolbox](https://googleapis.github.io/genai-toolbox/getting-started/introduction/) 允许您为代理配备一套工具。当代理收到查询时，它可以智能地选择并使用 MCP Toolbox 提供的最合适工具来满足请求。

## 什么是 MCP Toolbox？

MCP Toolbox 本质上是您工具的容器。将其视为代理的多功能工具设备；它可以持有您创建的任何工具。代理随后根据用户输入决定使用哪个具体工具。

当您有一个需要执行各种需要不同能力任务的代理时，这特别有用。

## 安装

要开始使用，您需要安装必要的包：

<CodeGroup>
```bash pip
pip install toolbox-langchain
```

```bash uv
uv add toolbox-langchain
```
</CodeGroup>
