LangChain.js 支持调用 [YandexGPT](https://cloud.yandex.com/en/services/yandexgpt) LLM。

## 设置

首先，您应 [创建服务账户](https://cloud.yandex.com/en/docs/iam/operations/sa/create)，并赋予其 `ai.languageModels.user` 角色。

接下来，您有两种认证选项：

- [IAM 令牌](https://cloud.yandex.com/en/docs/iam/operations/iam-token/create-for-sa)。
  您可以在构造函数参数 `iam_token` 中指定令牌，或在环境变量 `YC_IAM_TOKEN` 中指定。
- [API 密钥](https://cloud.yandex.com/en/docs/iam/operations/api-key/create)
  您可以在构造函数参数 `api_key` 中指定密钥，或在环境变量 `YC_API_KEY` 中指定。

## 用法

<Tip>
请参阅 [此部分获取安装 LangChain 包的一般说明](/oss/javascript/langchain/install)。
</Tip>

```bash npm
npm install @langchain/yandex @langchain/core
```

```typescript
import { YandexGPT } from "@langchain/yandex/llms";

const model = new YandexGPT();
const res = await model.invoke(['将 "I love programming" 翻译成法语。']);
console.log({ res });
```
