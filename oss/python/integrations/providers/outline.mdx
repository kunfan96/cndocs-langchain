> [Outline](https://www.getoutline.com/) 是一个开源的协作知识库平台，专为团队信息共享而设计。

## 设置

首先需要为您的 Outline 实例 [创建一个 API 密钥](https://www.getoutline.com/developers#section/Authentication)。然后设置以下环境变量：

```python
import os

os.environ["OUTLINE_API_KEY"] = "xxx"
os.environ["OUTLINE_INSTANCE_URL"] = "https://app.getoutline.com"
```

## 检索器

参见 [使用示例](/oss/python/integrations/retrievers/outline)。

from langchain_classic.retrievers import OutlineRetriever
