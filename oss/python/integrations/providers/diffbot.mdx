> [Diffbot](https://docs.diffbot.com/docs) 是一套基于机器学习的产品套件，可轻松实现网页数据的结构化与集成。

## 安装与配置

[获取免费的 Diffbot API 令牌](https://app.diffbot.com/get-started/)，并[按照以下说明](https://docs.diffbot.com/reference/authentication)对请求进行身份验证。

## 文档加载器

Diffbot 的 [Extract API](https://docs.diffbot.com/reference/extract-introduction) 是一项用于对网页数据进行结构化与标准化的服务。

与传统网络爬虫工具不同，`Diffbot Extract` 无需任何规则即可读取页面内容。它使用计算机视觉模型将页面分类为 20 种可能类型之一，然后将原始 HTML 标记转换为 JSON。生成的结构化 JSON 遵循一致的[基于类型的本体](https://docs.diffbot.com/docs/ontology)，这使得使用相同的模式从多个不同的 Web 源提取数据变得非常容易。

请参阅[使用示例](/oss/python/integrations/document_loaders/diffbot)。

```python
from langchain_community.document_loaders import DiffbotLoader
```

## 图谱

Diffbot 的 [自然语言处理 API](https://www.diffbot.com/products/natural-language/) 支持从未结构化文本数据中提取实体、关系和语义信息。

请参阅[使用示例](/oss/python/integrations/graphs/diffbot)。

```python
from langchain_experimental.graph_transformers.diffbot import DiffbotGraphTransformer
```
