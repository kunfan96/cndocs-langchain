>[Stack Exchange](https://en.wikipedia.org/wiki/Stack_Exchange) 是一个由问答（Q&A）网站组成的网络，涵盖多个领域的主题，每个网站专注于特定主题，其中问题、答案和用户都遵循声誉奖励流程。

本页介绍如何在 LangChain 中使用 `Stack Exchange API`。

## 安装与设置
- 使用以下命令安装依赖：
<CodeGroup>
```bash pip
pip install stackapi
```

```bash uv
uv add stackapi
```
</CodeGroup>

## 包装器

### 实用工具

存在一个 StackExchangeAPIWrapper 实用工具，用于包装此 API。要导入此实用工具：

```python
from langchain_community.utilities import StackExchangeAPIWrapper
```

有关此包装器的更详细说明，请参阅[此笔记本](/oss/python/integrations/tools/stackexchange)。

### 工具

您还可以轻松地将此包装器加载为工具（与代理一起使用）。
您可以通过以下方式实现：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["stackexchange"])
```

有关工具的更多信息，请参阅[工具](/oss/python/integrations/tools)。
