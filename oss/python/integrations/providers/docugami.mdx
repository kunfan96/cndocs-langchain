>[Docugami](https://docugami.com) 将业务文档转换为文档 XML 知识图谱，生成代表整个文档的 XML 语义树森林。这是一种丰富的表示形式，以 XML 树的形式包含文档中各个分块的语义和结构特征。

## 安装与设置


<CodeGroup>
```bash pip
pip install dgml-utils
pip install docugami-langchain
```

```bash uv
uv add dgml-utils
uv add docugami-langchain
```
</CodeGroup>

## 文档加载器

请参阅 [使用示例](/oss/python/integrations/document_loaders/docugami)。

```python
from docugami_langchain.document_loaders import DocugamiLoader
```
