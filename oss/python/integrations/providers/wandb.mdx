>[Weights & Biases](https://wandb.ai/) 是 AI 开发者平台的提供商，用于训练和
> 微调 AI 模型以及开发 AI 应用程序。

`Weights & Biases` 产品可用于在训练期间记录指标和工件，并跟踪代码的执行。

在 LangChain 中使用 `Weights & Biases` 产品主要有几种方式：
- 使用 `wandb_tracing_enabled`
- 使用 `Weave` 轻量级工具包
- 使用 `WandbCallbackHandler`（已弃用）


## wandb_tracing_enabled

请参阅[使用示例](/oss/python/integrations/providers/wandb_tracing)。

请参阅 [W&B 文档](https://docs.wandb.ai/guides/integrations/langchain)。

```python
from langchain_community.callbacks import wandb_tracing_enabled
```

## Weave

请参阅 [W&B 文档](https://weave-docs.wandb.ai/guides/integrations/langchain)。


## WandbCallbackHandler

**注意：** `WandbCallbackHandler` 正在被弃用，推荐使用 `wandb_tracing_enabled`。

请参阅[使用示例](/oss/python/integrations/providers/wandb_tracking)。

请参阅 [W&B 文档](https://docs.wandb.ai/guides/integrations/langchain)。

```python
from langchain_community.callbacks import WandbCallbackHandler
```
