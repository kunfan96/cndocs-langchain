>[WhatsApp](https://www.whatsapp.com/)（也称为 `WhatsApp Messenger`）是一款免费、跨平台、集中式即时通讯（IM）和 VoIP 服务应用。它允许用户发送文字和语音消息、拨打语音和视频通话，以及共享图片、文档、位置信息和其他内容。

本笔记本介绍如何将 `WhatsApp 聊天记录` 中的数据加载为 LangChain 可用的格式。

```python
from langchain_community.document_loaders import WhatsAppChatLoader
```

```python
loader = WhatsAppChatLoader("example_data/whatsapp_chat.txt")
```

```python
loader.load()
```
