>[Joplin](https://joplinapp.org/) 是一款开源的笔记应用。它捕获您的想法
> 并允许您从任何设备安全地访问它们。


## 安装与设置

`Joplin API` 需要访问令牌。
您可以在 [Joplin REST API 文档](https://joplinapp.org/api/references/rest_api/) 中找到安装说明。


## 文档加载器

请参阅[使用示例](/oss/python/integrations/document_loaders/joplin)。

```python
from langchain_community.document_loaders import JoplinLoader
```
