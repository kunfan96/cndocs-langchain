>[Nomic Atlas](https://docs.nomic.ai/index.html) 是一个用于与小规模和互联网规模非结构化数据集交互的平台。


## 安装与设置

- 使用 `pip install nomic` 安装 Python 包
- `Nomic` 也包含在 langchain 的 poetry extras 中，使用 `poetry install -E all` 安装


## 向量存储

查看[使用示例](/oss/python/integrations/vectorstores/atlas)。

```python
from langchain_community.vectorstores import AtlasDB
```
