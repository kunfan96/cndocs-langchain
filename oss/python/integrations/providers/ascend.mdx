>[Ascend](https://https://www.hiascend.com/) 是华为提供的神经网络处理单元（NPU）

本页介绍如何在 LangChain 中使用 Ascend NPU。

### 安装

使用以下命令安装 torch-npu：

<CodeGroup>
```bash pip
pip install torch-npu
```

```bash uv
uv add torch-npu
```
</CodeGroup>

请按照以下安装说明操作：
* 按照[此处](https://www.hiascend.com/document/detail/zh/canncommercial/700/quickstart/quickstart/quickstart_18_0002.html)所示安装 CANN。

### 嵌入模型

查看[使用示例](/oss/python/integrations/text_embedding/ascend)。

```python
from langchain_community.embeddings import AscendEmbeddings
```
