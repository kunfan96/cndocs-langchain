[Zotero](https://www.zotero.org/) 是一个开源的参考文献管理系统，用于管理书目数据和相关研究资料。你可以通过 [API](https://www.zotero.org/support/dev/web_api/v3/start) 连接到你的个人库以及共享群组库。此检索器实现使用 [PyZotero](https://github.com/urschrei/pyzotero) 访问库。

## 安装

```bash
pip install pyzotero
```

## 检索器

查看[使用示例](/oss/python/integrations/retrievers/zotero)。

```python
from langchain_zotero_retriever.retrievers import ZoteroRetriever
```
