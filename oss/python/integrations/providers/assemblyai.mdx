>[AssemblyAI](https://www.assemblyai.com/) 构建用于语音转文字、说话人分离、语音摘要等任务的 `语音 AI` 模型。
> `AssemblyAI` 的语音 AI 模型包括用于语音数据（如通话、虚拟会议和播客）的高精度语音转文字、
> 说话人检测、情感分析、章节检测、PII 脱敏等功能。



## 安装与设置

获取您的 [API 密钥](https://www.assemblyai.com/dashboard/signup)。

安装 `assemblyai` 包。

<CodeGroup>
```bash pip
pip install -U assemblyai
```

```bash uv
uv add assemblyai
```
</CodeGroup>

## 文档加载器

### AssemblyAI 音频转录

`AssemblyAIAudioTranscriptLoader` 使用 `AssemblyAI API` 转录音频文件，
并将转录文本加载到文档中。

查看[使用示例](/oss/python/integrations/document_loaders/assemblyai)。

```python
from langchain_community.document_loaders import AssemblyAIAudioTranscriptLoader
```

### 按 ID 加载 AssemblyAI 音频

`AssemblyAIAudioLoaderById` 使用 AssemblyAI API 获取已有的转录记录，
并根据指定格式将转录文本加载到一个或多个文档中。

```python
from langchain_community.document_loaders import AssemblyAIAudioLoaderById
```
