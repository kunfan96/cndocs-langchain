## 安装与配置

可通过 pip 安装 GreenNode 集成：

```python
pip install -qU langchain-greennode
```

### API 密钥

要使用 GreenNode Serverless AI，您需要一个 API 密钥，您可以从 [GreenNode Serverless AI](https://aiplatform.console.greennode.ai/api-keys) 获取。API 密钥可以作为初始化参数 `api_key` 传递，也可以设置为环境变量 `GREENNODE_API_KEY`。

```python
import getpass
import os

if not os.getenv("GREENNODE_API_KEY"):
    os.environ["GREENNODE_API_KEY"] = getpass.getpass("请输入您的 GreenNode API 密钥：")
```

## 聊天模型

```python
from langchain_greennode import ChatGreenNode

chat = ChatGreenNode(
    model="deepseek-ai/DeepSeek-R1-Distill-Qwen-32B",  # 从可用模型中选择
    temperature=0.6,
    top_p=0.95,
)
```

使用 GreenNode [聊天模型](https://python.langchain.com/docs/integrations/chat/greennode/)

## 嵌入模型

```python
from langchain_greennode import GreenNodeEmbeddings

# 初始化嵌入模型
embeddings = GreenNodeEmbeddings(
    model="BAAI/bge-m3"  # 从可用模型中选择
)
```

使用 GreenNode [嵌入模型](https://python.langchain.com/docs/integrations/embeddings/greennode)

## 重排序

```python
from langchain_greennode import GreenNodeRerank

# 初始化重排序器
rerank = GreenNodeRerank(
    model="BAAI/bge-reranker-v2-m3",  # 从可用模型中选择
    top_n=-1,
)
```

使用 GreenNode [重排序模型](https://python.langchain.com/docs/integrations/retrievers/greennode-reranker)
