我们将加载 LLMRails Embeddings 类。

使用 LLMRails 嵌入时，需通过参数传入 API 密钥，或在环境变量中配置 `LLM_RAILS_API_KEY`。
获取 API 密钥需在 [console.llmrails.com/signup](https://console.llmrails.com/signup) 完成注册，随后访问 [console.llmrails.com/api-keys](https://console.llmrails.com/api-keys)，在平台创建密钥后将其复制。

```python
from langchain_community.embeddings import LLMRailsEmbeddings
```

```python
embeddings = LLMRailsEmbeddings(model="embedding-english-v1")  # or embedding-multi-v1
```

```python
text = "This is a test document."
```

生成嵌入向量时，可传入单个文本进行查询，也可传入文本列表。

```python
query_result = embeddings.embed_query(text)
query_result[:5]
```

```text
[-0.09996652603149414,
 0.015568195842206478,
 0.17670190334320068,
 0.16521021723747253,
 0.21193109452724457]
```

```python
doc_result = embeddings.embed_documents([text])
doc_result[0][:5]
```

```text

```
