>[scikit-learn](https://scikit-learn.org/stable/) 是一个开源机器学习算法集合，
> 其中包含 [k 近邻](https://scikit-learn.org/stable/modules/generated/sklearn.neighbors.NearestNeighbors.html) 的部分实现。`SKLearnVectorStore` 封装了此实现，并增加了以 JSON、BSON（二进制 JSON）或 Apache Parquet 格式持久化向量存储的功能。

## 安装与设置

- 使用 `pip install scikit-learn` 安装 Python 包


## 向量存储

`SKLearnVectorStore` 为 scikit-learn 包中的最近邻实现提供了简单封装，允许您将其用作向量存储。

导入此向量存储：

```python
from langchain_community.vectorstores import SKLearnVectorStore
```

有关 SKLearnVectorStore 封装器的更详细演练，请参阅[此 notebook](/oss/python/integrations/vectorstores/sklearn)。


## 检索器

`支持向量机（SVM）` 是用于分类、回归和异常检测的监督学习方法。

查看[使用示例](/oss/python/integrations/retrievers/svm)。

```python
from langchain_community.retrievers import SVMRetriever
```
