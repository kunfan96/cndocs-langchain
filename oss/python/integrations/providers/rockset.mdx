>[Rockset](https://rockset.com/product/) 是一款实时分析数据库服务，可在大规模场景下提供低延迟、高并发的分析查询。它在结构化和半结构化数据上构建 Converged Index™，并为向量嵌入提供高效存储。其对无模式数据运行 SQL 的支持，使其成为带元数据过滤的向量搜索的理想选择。

## 安装与设置

确保您有 Rockset 账户，并前往 Web 控制台获取 API 密钥。详细信息请参见[官网](https://rockset.com/docs/rest-api/)。

<CodeGroup>
```bash pip
pip install rockset
```

```bash uv
uv add rockset
```
</CodeGroup>
