[Dappier](https://dappier.com) 将任何大语言模型（LLM）或您的智能体 AI 连接到来自可信来源的实时、已获授权、专有的数据，使您的 AI 成为任何领域的专家。我们的专业模型包括实时网络搜索、新闻、体育、金融股票市场数据、加密货币数据以及来自优质出版商的独家内容。请访问 [marketplace.dappier.com](https://marketplace.dappier.com) 探索我们市场中广泛的数据模型。

[Dappier](https://dappier.com) 提供丰富、可直接用于提示词且上下文相关的数据字符串，专为与 LangChain 无缝集成而优化。无论您是在构建对话式 AI、推荐引擎还是智能搜索，Dappier 的 LLM 无关 RAG 模型都能确保您的 AI 能够访问经过验证的最新数据——无需承担构建和管理自有检索流程的复杂性。

## 安装与配置

安装 `langchain-dappier` 并设置环境变量 `DAPPIER_API_KEY`。

<CodeGroup>
```bash pip
pip install -U langchain-dappier
export DAPPIER_API_KEY="your-api-key"
```

```bash uv
uv add langchain-dappier
export DAPPIER_API_KEY="your-api-key"
```
</CodeGroup>

我们还需要设置 Dappier API 凭据，可在 [Dappier 网站](https://platform.dappier.com/profile/api-keys) 生成。

前往 [Dappier 市场](https://platform.dappier.com/marketplace) 即可查看支持的数据模型。
