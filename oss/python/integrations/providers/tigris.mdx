> [Tigris](https://tigrisdata.com) 是一个开源无服务器 NoSQL 数据库和搜索平台，旨在简化高性能向量搜索应用的构建。
> `Tigris` 消除了管理、运营和同步多个工具的基础设施复杂性，让您可以专注于构建出色的应用程序。

## 安装与设置


<CodeGroup>
```bash pip
pip install tigrisdb openapi-schema-pydantic
```

```bash uv
uv add tigrisdb openapi-schema-pydantic
```
</CodeGroup>
