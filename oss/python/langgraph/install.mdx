安装 LangGraph 基础包：

<CodeGroup>
```bash pip
pip install -U langgraph
```

```bash uv
uv add langgraph
```
</CodeGroup>




使用 LangGraph 通常需要访问 LLM 并定义工具。
您可以根据自己的需要选择方式。

一种常见的方式（文档中我们将使用这种方式）是使用 [LangChain](/oss/python/langchain/overview)。

安装 LangChain：

<CodeGroup>
```bash pip
pip install -U langchain
# Requires Python 3.10+
```

```bash uv
uv add langchain
# Requires Python 3.10+
```
</CodeGroup>




要使用特定 LLM 提供商的包，您需要单独安装它们。

请参阅 [integrations](/oss/python/integrations/providers/overview) 页面，获取各提供商的安装说明。
