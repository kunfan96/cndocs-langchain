[Kinetica](https://www.kinetica.com/) 是一款专为时序和空间数据上的分析与生成式 AI 而构建的实时数据库。

## 对话模型

Kinetica LLM 封装器使用 [Kinetica SqlAssist LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/) 将自然语言转换为 SQL，从而简化数据检索过程。

使用方法请参阅 [Kinetica 自然语言转 SQL 对话模型](/oss/python/integrations/chat/kinetica)。


```python
from langchain_kinetica import ChatKinetica
```

## 向量存储

Kinetica 向量存储封装器利用 Kinetica 对[向量相似度搜索](https://docs.kinetica.com/7.2/vector_search/)的原生支持。

使用方法请参阅 [Kinetica 向量存储 API](/oss/python/integrations/vectorstores/kinetica)。


```python
from langchain_kinetica import KineticaVectorstore
```

## 文档加载器

Kinetica 文档加载器可用于从 [Kinetica](https://www.kinetica.com/) 数据库加载 LangChain [文档](https://python.langchain.com/api_reference/core/documents/langchain_core.documents.base.Document.html)。

使用方法请参阅 [Kinetica 文档加载器](/oss/python/integrations/document_loaders/kinetica)。


```python
from langchain_kinetica import KineticaLoader
```

## 检索器

Kinetica 检索器可根据非结构化查询返回文档。

使用方法请参阅 [Kinetica 基于向量存储的检索器](/oss/python/integrations/retrievers/kinetica)。
