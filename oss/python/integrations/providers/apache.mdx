>[Apache 软件基金会（维基百科）](https://en.wikipedia.org/wiki/The_Apache_Software_Foundation)
> 是一个去中心化的开源开发者社区。他们开发的软件遵循 Apache 许可证的条款进行分发，这是一种针对自由和开源软件（FOSS）的宽松型开源许可证。Apache 项目
> 以协作式、基于共识的开发流程以及开放且务实的软件许可证为特征，这意味着它
> 允许免费获取软件的开发者在非免费条款下重新分发该软件。每个项目均由一支自主组建的
> 技术专家团队管理，这些专家也是该项目的活跃贡献者。

## Apache AGE

>[Apache AGE](https://age.apache.org/) 是一个提供图数据库功能的 `PostgreSQL` 扩展。`AGE` 是 `A Graph Extension` 的缩写，其灵感来源于 Bitnine 对 `PostgreSQL 10` 的分叉版本 `AgensGraph`，后者是一种多模态数据库。该项目的目标是创建一种单一
> 存储方案，能够同时处理关系型和图模型数据，以便用户可以使用标准的 ANSI SQL 结合 `openCypher` 这一图查询语言。
> `Apache AGE` 存储的数据元素包括节点、连接它们的边以及节点和边的属性。

See more about [integrating with Apache AGE](/oss/python/integrations/graphs/apache_age).

## Apache cassandra

>[Apache Cassandra](https://cassandra.apache.org/) 是一款 NoSQL、面向行、高可扩展且高可用的数据库。从 5.0 版本起，
> 该数据库内置了向量搜索功能。

See more about [integrating with Apache Cassandra](/oss/python/integrations/providers/cassandra/).

## Apache doris

>[Apache Doris](https://doris.apache.org/) 是一款面向实时分析的现代数据仓库。它能够对大规模实时数据进行闪电般的快速分析。
>
>通常 `Apache Doris` 被归类为 OLAP，它在 ClickBench——一个分析型数据库基准测试中展现了卓越的性能。由于它拥有
> 超快的向量化执行引擎，因此也可用作快速的向量数据库。

See more about [integrating with Apache Doris](/oss/python/integrations/providers/apache_doris/).

## Apache kafka

>[Apache Kafka](https://github.com/apache/kafka) 是一个分布式消息系统，用于发布和订阅记录流。


## Apache spark

>[Apache Spark](https://spark.apache.org/) 是一款用于大规模数据处理的统一分析引擎。它提供了 Scala、Java、
> Python 和 R 的高级 API，以及一个优化的引擎，支持用于数据分析的通用计算图。它还支持一套丰富的高级
> 工具，包括用于 SQL 和数据框的 `Spark SQL`、用于 pandas 工作负载的 `pandas API on Spark`、用于机器学习的 `MLlib`、
> 用于图处理的 `GraphX`，以及用于流处理的 `Structured Streaming`。

See more about [integrating with Apache Spark](/oss/python/integrations/providers/spark).
