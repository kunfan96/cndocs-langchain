>[Golden](https://golden.com) 提供一套自然语言 API，用于使用 Golden 知识图谱进行查询和丰富。例如，`OpenAI 的产品`、`获得 A 轮融资的生成式 AI 公司` 和 `投资的说唱歌手` 等查询可用于检索相关实体的结构化数据。
>
>`golden-query` langchain 工具是对 [Golden Query API](https://docs.golden.com/reference/query-api) 的封装器，可以以编程方式访问这些结果。
>更多信息请参阅 [Golden Query API 文档](https://docs.golden.com/reference/query-api)。

## 安装与设置
- 访问 [Golden API 文档](https://docs.golden.com/) 了解 Golden API 的概述。
- 从 [Golden API 设置](https://golden.com/settings/api) 页面获取你的 API 密钥。
- 将你的 API 密钥保存到 GOLDEN_API_KEY 环境变量中

## 封装器

### 工具函数

有一个 GoldenQueryAPIWrapper 工具函数封装了此 API。要导入此工具函数：

```python
from langchain_community.utilities.golden_query import GoldenQueryAPIWrapper
```

有关此封装器的更详细说明，请参阅 [此 notebook](/oss/python/integrations/tools/golden_query)。

### 工具

你也可以轻松地将此封装器作为工具加载（与代理一起使用）。
你可以这样做：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["golden-query"])
```

有关工具的更多信息，请参阅[此页面](/oss/python/integrations/tools)。
