>[Zilliz Cloud](https://zilliz.com/doc/quick_start) 是基于 `LF AI Milvus®` 的全托管云服务，


## 安装与设置

安装 Python SDK：
<CodeGroup>
```bash pip
pip install pymilvus
```

```bash uv
uv add pymilvus
```
</CodeGroup>

## 向量存储

Zilliz 索引有对应的封装，允许你将其用作向量存储，用于语义搜索或示例选择。

```python
from langchain_community.vectorstores import Milvus
```

有关 Milvus 封装的更详细说明，请参阅[此笔记本](/oss/python/integrations/vectorstores/zilliz)
