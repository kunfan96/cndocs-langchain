>[Argilla](https://argilla.io/) 是一个面向大语言模型的开源数据策展平台。
> 借助 `Argilla`，每个人都可以通过结合人类和机器反馈来加速数据策展，从而构建稳健的语言模型。`Argilla` 支持 MLOps 周期中的每一步，从数据标注到模型监控。

## 安装与设置

获取您的 [API 密钥](https://platform.openai.com/account/api-keys)。

安装 Python 包：

<CodeGroup>
```bash pip
pip install argilla
```

```bash uv
uv add argilla
```
</CodeGroup>

## 回调函数


```python
from langchain.callbacks import ArgillaCallbackHandler
```

请参阅[示例](/oss/python/integrations/callbacks/argilla)。
