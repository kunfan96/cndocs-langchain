>[GeoPandas](https://geopandas.org/) 是一个开源项目，旨在简化
> Python 中地理空间数据的处理工作。`GeoPandas` 扩展了 `pandas` 使用的数据类型，以支持对几何类型执行空间操作。
> 几何操作由 `shapely` 执行。


## 安装与设置

我们需要安装几个 Python 包。

<CodeGroup>
```bash pip
pip install -U sodapy pandas geopandas
```

```bash uv
uv add sodapy pandas geopandas
```
</CodeGroup>
