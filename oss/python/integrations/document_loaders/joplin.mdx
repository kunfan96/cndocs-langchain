>[Joplin](https://joplinapp.org/) 是一款开源笔记应用，可随时随地安全地记录和访问您的想法。

本文档介绍如何从 `Joplin` 数据库加载文档。

`Joplin` 提供了用于访问本地数据库的 [REST API](https://joplinapp.org/api/references/rest_api/)。此加载器使用该 API 检索数据库中的所有笔记及其元数据，需要按以下步骤从应用中获取访问令牌：

1. 打开 `Joplin` 应用。加载文档期间应用必须保持运行状态。
2. 进入设置/选项，选择"Web Clipper"。
3. 确保 Web Clipper 服务已启用。
4. 在"高级选项"下复制授权令牌。

您可以在初始化加载器时直接传入访问令牌，也可以将其存储在环境变量 JOPLIN_ACCESS_TOKEN 中。

另一种方式是将 `Joplin` 的笔记数据库导出为 Markdown 文件（可选包含 Front Matter 元数据），再使用 Markdown 加载器（如 ObsidianLoader）加载。

```python
from langchain_community.document_loaders import JoplinLoader
```

```python
loader = JoplinLoader(access_token="<access-token>")
```

```python
docs = loader.load()
```
