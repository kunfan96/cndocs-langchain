[Jenkins](https://www.jenkins.io/) 是一个开源自动化平台，使软件团队能够优化其开发工作流。它在 DevOps 社区中被广泛采用，作为通过 CI/CD 流水线自动化应用程序构建、测试和部署的工具。


## 安装与配置

<CodeGroup>
```bash pip
pip install langchain-jenkins
```

```bash uv
uv add langchain-jenkins
```
</CodeGroup>
