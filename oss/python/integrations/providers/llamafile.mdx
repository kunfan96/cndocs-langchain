>[llamafile](https://github.com/Mozilla-Ocho/llamafile) 允许你通过单个文件分发和运行 LLMs。
> 
>`llamafile` 使开源 LLMs 对开发者和最终用户都更加易于获取。`llamafile` 通过将 [llama.cpp](https://github.com/ggerganov/llama.cpp) 与 [Cosmopolitan Libc](https://github.com/jart/cosmopolitan) 结合到一个框架中来实现这一目标，该框架将所有 LLMs 的复杂性简化为一个单文件可执行程序（称为“llamafile”），可在大多数计算机上本地运行，且无需安装。


## 安装与设置

请参阅 [安装说明](https://github.com/Mozilla-Ocho/llamafile?tab=readme-ov-file#quickstart)。

## LLMs

请参阅 [使用示例](/oss/python/integrations/llms/llamafile)。

```python
from langchain_community.llms.llamafile import Llamafile
```

## 嵌入模型


```python
from langchain_community.embeddings import LlamafileEmbeddings
```
