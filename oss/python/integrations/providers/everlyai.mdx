> [Everly AI](https://everlyai.xyz/) 允许你在云端大规模运行 ML 模型。
> 它还提供对[多个 LLM 模型](https://everlyai.xyz/)的 API 访问。

## 安装与设置

要使用 `Everly AI`，你需要一个 API 密钥。访问
[Everly AI](https://everlyai.xyz/) 在你的个人资料中创建 API 密钥。

## 聊天模型

查看[使用示例](/oss/python/integrations/chat/everlyai)。
