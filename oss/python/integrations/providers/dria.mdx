>[Dria](https://dria.co/) 是一个公共 RAG 模型中心，供开发者贡献和使用共享的嵌入湖。

有关 LangChain 与 Dria 集成的更多详情，请参阅[此页面](https://dria.co/docs/integrations/langchain)。

## 安装与设置

您需要安装 Python 包：

<CodeGroup>
```bash pip
pip install dria
```

```bash uv
uv add dria
```
</CodeGroup>

您需要从 Dria 获取 API 密钥。您可以通过在 [Dria](https://dria.co/) 注册来获取。
