>[Slack](https://slack.com/) 是一款即时通讯程序。

本笔记本介绍如何从 `Slack` 导出生成的压缩文件中加载文档。

按照以下说明操作，可将导出数据导入系统：

## 🧑 导入自有数据集的操作说明

导出您的 Slack 数据。您可以前往工作区管理页面，点击导入/导出选项（`{your_slack_domain}.slack.com/services/export`）。然后选择合适的日期范围，点击 `开始导出`。导出完成后，Slack 将通过邮件和私信通知您。

下载完成后，您的下载文件夹（或根据操作系统配置所指定的下载位置）中将出现一个 `.zip` 文件。

复制该 `.zip` 文件的路径，并将其赋值给下方的 `LOCAL_ZIPFILE`。

```python
from langchain_community.document_loaders import SlackDirectoryLoader
```

```python
# 可选：设置您的 Slack URL，这将为文档来源提供正确的 URL。
SLACK_WORKSPACE_URL = "https://xxx.slack.com"
LOCAL_ZIPFILE = ""  # 在此粘贴本地 Slack zip 文件路径。

loader = SlackDirectoryLoader(LOCAL_ZIPFILE, SLACK_WORKSPACE_URL)
```

```python
docs = loader.load()
docs
```
