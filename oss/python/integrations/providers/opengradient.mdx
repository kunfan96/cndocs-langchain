[OpenGradient](https://www.opengradient.ai/) 是一个去中心化的 AI 计算网络，提供全球可访问、无需许可且可验证的机器学习模型推理服务。

OpenGradient langchain 包目前提供了一个工具包，允许开发者为 OpenGradient 网络上的模型构建自定义的机器学习推理工具。这以前一直是个挑战，因为大型模型参数具有污染上下文窗口的特性——想象一下，你需要向你的智能体提供一个 200x200 的浮点数数组！

该工具包通过将所有的数据处理逻辑封装在工具定义本身来解决这个问题。这种方法保持了智能体上下文窗口的整洁，同时赋予开发者完全的自由度，为其机器学习模型实现自定义的数据处理和实时数据检索。

## 安装与设置

请确保您拥有 OpenGradient API 密钥，以便访问 OpenGradient 网络。如果您已经拥有 API 密钥，只需设置环境变量：

```python
!export OPENGRADIENT_PRIVATE_KEY="your-api-key"
```

如果您需要设置新的 API 密钥，请下载 opengradient SDK 并按照说明初始化新配置。

```python
!pip install opengradient
!opengradient config init
```

设置好 API 密钥后，安装 langchain-opengradient 包。

```python
pip install -U langchain-opengradient
```

## OpenGradient 工具包

OpenGradientToolkit 赋能开发者基于部署在 OpenGradient 去中心化网络上的 [机器学习模型](https://hub.opengradient.ai/models) 和 [工作流](https://docs.opengradient.ai/developers/sdk/ml_workflows.html) 创建专用工具。此集成使 LangChain 智能体能够访问强大的机器学习能力，同时保持高效的上下文使用。

### 核心优势

* 🔄 实时数据集成 - 在您的工具中处理实时数据流

* 🎯 动态处理 - 适应特定智能体输入的自定义数据管道

* 🧠 上下文效率 - 处理复杂的机器学习操作而不会淹没您的上下文窗口

* 🔌 无缝部署 - 轻松集成已部署在 OpenGradient 网络上的模型

* 🔧 完全定制 - 通过 [OpenGradient SDK](https://docs.opengradient.ai/developers/sdk/model_management.html) 创建并部署您自己的特定模型，然后从中构建自定义工具

* 🔐 可验证推理 - 所有推理均在去中心化的 OpenGradient 网络上运行，允许用户选择多种安全选项（如 ZKML 和 TEE），以实现无需信任的可验证模型执行

如需详细示例和实施指南，请参阅我们的[综合教程](/oss/python/integrations/tools/opengradient_toolkit.ipynb)。
