[rank_bm25](https://github.com/dorianbrown/rank_bm25) 是一个开源的算法集合，
旨在查询文档并返回最相关的结果，通常用于创建搜索引擎。

查看其 [项目页面](https://github.com/dorianbrown/rank_bm25) 了解可用算法。


## 安装与设置

首先，您需要安装 `rank_bm25` Python 包。

<CodeGroup>
```bash pip
pip install rank_bm25
```

```bash uv
uv add rank_bm25
```
</CodeGroup>


## 检索器

参见 [使用示例](/oss/python/integrations/retrievers/bm25)。

from langchain_community.retrievers import BM25Retriever
