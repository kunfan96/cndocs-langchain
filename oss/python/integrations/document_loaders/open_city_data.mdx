[Socrata](https://dev.socrata.com/foundry/data.sfgov.org/vw6y-z8j6) 提供了一个用于访问城市开放数据的 API。

对于如 [旧金山犯罪数据](https://data.sfgov.org/Public-Safety/Police-Department-Incident-Reports-Historical-2003/tmnf-yvry) 之类的数据集，请查看右上角的 `API` 标签页。

该标签页会提供 `数据集标识符`。

使用数据集标识符可以获取指定城市 ID（`data.sfgov.org`）的特定表格——

例如，`vw6y-z8j6` 对应 [旧金山 311 数据](https://dev.socrata.com/foundry/data.sfgov.org/vw6y-z8j6)。

例如，`tmnf-yvry` 对应 [旧金山警察数据](https://dev.socrata.com/foundry/data.sfgov.org/tmnf-yvry)。

```python
pip install -qU  sodapy
```

```python
from langchain_community.document_loaders import OpenCityDataLoader
```

```python
dataset = "vw6y-z8j6"  # 311 data
dataset = "tmnf-yvry"  # crime data
loader = OpenCityDataLoader(city_id="data.sfgov.org", dataset_id=dataset, limit=2000)
```

```python
docs = loader.load()
```
