>[GooseAI](https://goose.ai) 让 NLP 服务的部署变得更容易和更便捷。
> `GooseAI` 是一个通过 API 提供的完全托管的推理服务。
> 与其他知名 API 功能相当，`GooseAI` 只需更改代码中的两行，就能以业界最佳经济性
> 提供开源语言模型服务。

## 安装与设置

- 使用 `pip install openai` 安装 Python SDK
- 从[此链接](https://goose.ai/)获取你的 GooseAI API 密钥。
- 设置环境变量（`GOOSEAI_API_KEY`）。

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```

## LLMs

查看[使用示例](/oss/python/integrations/llms/gooseai)。

```python
from langchain_community.llms import GooseAI
```
