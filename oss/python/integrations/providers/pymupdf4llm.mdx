[PyMuPDF4LLM](https://pymupdf.readthedocs.io/en/latest/pymupdf4llm) 旨在使提取 PDF 内容为 Markdown 格式变得更加容易，这是 LLM 和 RAG 应用程序所需的。

[langchain-pymupdf4llm](https://github.com/lakinduboteju/langchain-pymupdf4llm) 将 PyMuPDF4LLM 集成到 LangChain 中，作为一个 Document Loader。

```python
pip install -qU langchain-pymupdf4llm
```

```python
from langchain_pymupdf4llm import PyMuPDF4LLMLoader, PyMuPDF4LLMParser
```
