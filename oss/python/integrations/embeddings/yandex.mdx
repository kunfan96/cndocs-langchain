本笔记本将介绍如何在 LangChain 中使用 [YandexGPT](https://cloud.yandex.com/en/services/yandexgpt) 嵌入模型。

使用前，请确保已安装 `yandexcloud` Python 包。

```python
pip install -qU  yandexcloud
```

首先，您需要[创建服务账号](https://cloud.yandex.com/en/docs/iam/operations/sa/create)，并分配 `ai.languageModels.user` 角色。

接下来，您有两种认证选项：

- [IAM 令牌](https://cloud.yandex.com/en/docs/iam/operations/iam-token/create-for-sa)。
    您可以在构造函数参数 `iam_token` 中指定令牌，或在环境变量 `YC_IAM_TOKEN` 中指定。
- [API 密钥](https://cloud.yandex.com/en/docs/iam/operations/api-key/create)
    您可以在构造函数参数 `api_key` 中指定密钥，或在环境变量 `YC_API_KEY` 中指定。

要指定模型，您可以使用 `model_uri` 参数，更多详情请参阅[官方文档](https://cloud.yandex.com/en/docs/yandexgpt/concepts/models#yandexgpt-embeddings)。

默认情况下，将使用 `folder_id` 参数或 `YC_FOLDER_ID` 环境变量中指定的文件夹下的最新 `text-search-query` 版本。

```python
from langchain_community.embeddings.yandex import YandexGPTEmbeddings
```

```python
embeddings = YandexGPTEmbeddings()
```

```python
text = "This is a test document."
```

```python
query_result = embeddings.embed_query(text)
```

```python
doc_result = embeddings.embed_documents([text])
```

```python
query_result[:5]
```

