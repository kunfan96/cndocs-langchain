[Nia](https://trynia.ai) 是一个搜索和索引 API，它持续从文档、研究论文、数据集、代码库等中提供上下文——因此代理程序永远不会依赖过时的数据。可扩展、成本降低 5 倍且可靠。

## 概述

### 集成详情

| 类 | 包 | [PY 支持](/oss/javascript/integrations/tools/nia) | 版本 |
| :--- | :--- | :---: | :---: |
| [`NiaToolkit`](https://github.com/nozomio-labs/nia-langchain) | [`@nozomioai/langchain-nia`](https://www.npmjs.com/package/@nozomioai/langchain-nia) | ✅ | ![NPM - Version](https://img.shields.io/npm/v/@nozomioai/langchain-nia?style=flat-square&label=%20&) |

## 设置

集成位于 `@nozomioai/langchain-nia` 包中：

<CodeGroup>
```bash npm
npm install @nozomioai/langchain-nia @langchain/core
```
```bash yarn
yarn add @nozomioai/langchain-nia @langchain/core
```
```bash pnpm
pnpm add @nozomioai/langchain-nia @langchain/core
```
</CodeGroup>

### 凭证

在 [trynia.ai](https://trynia.ai) 注册以获取 API 密钥。

```typescript
process.env.NIA_API_KEY = "nk_...";
```

设置 [LangSmith](https://smith.langchain.com/) 以获得最佳可观测性也很有帮助（但不是必需的）：

```typescript
process.env.LANGSMITH_TRACING = "true";
process.env.LANGSMITH_API_KEY = "your-api-key";
```

## 实例化

### 使用工具包

```typescript
import { NiaToolkit } from "@nozomioai/langchain-nia";

const toolkit = new NiaToolkit({
  includeSearch: true,        // 5 个搜索工具
  includeSources: true,       // 7 个源管理工具
  includeGithub: true,        // 4 个 GitHub 工具
  includeContexts: true,      // 2 个上下文/内存工具
  includeDependencies: true,  // 2 个依赖工具
});

const tools = toolkit.getTools(); // 20 个工具
```

### 使用单个工具

```typescript
import { NiaSearch } from "@nozomioai/langchain-nia";

const tool = new NiaSearch();
```

## 调用

### 跨索引源搜索

```typescript
import { NiaSearch } from "@nozomioai/langchain-nia";

const tool = new NiaSearch();
const result = await tool.invoke({ query: "how to use React hooks" });
```

### 搜索网络

```typescript
import { NiaWebSearch } from "@nozomioai/langchain-nia";

const tool = new NiaWebSearch();
const result = await tool.invoke({ query: "latest Node.js release", numResults: 5 });
```

### 在代理中使用

```typescript
import { NiaToolkit } from "@nozomioai/langchain-nia";
import { ChatOpenAI } from "@langchain/openai";

const toolkit = new NiaToolkit({ includeSearch: true });
const tools = toolkit.getTools();

const llm = new ChatOpenAI({ model: "gpt-4o" });
const llmWithTools = llm.bindTools(tools);

const result = await llmWithTools.invoke("Search for React hooks best practices");
```

## 可用工具

| 工具 | 描述 |
|------|-------------|
| `NiaSearch` | 对索引的仓库、文档和数据源进行语义搜索 |
| `NiaWebSearch` | 带有类别过滤和日期范围的网络搜索 |
| `NiaDeepResearch` | 多步骤综合研究 |
| `NiaUniversalSearch` | 同时搜索所有源 |
| `NiaAdvisor` | 根据索引的文档分析代码 |
| `NiaIndex` | 索引新源（仓库、文档、论文、数据集） |
| `NiaSourceList` | 列出索引的源并进行过滤 |
| `NiaSourceSubscribe` | 订阅预索引的公共源 |
| `NiaSourceSync` | 重新同步源以拉取最新更改 |
| `NiaRead` | 从索引的源读取文件/页面 |
| `NiaGrep` | 在索引的源中进行正则表达式搜索 |
| `NiaExplore` | 浏览索引源的文件树 |
| `NiaGitHubSearch` | 在 GitHub 仓库中搜索代码 |
| `NiaGitHubRead` | 从 GitHub 仓库读取文件 |
| `NiaGitHubGlob` | 查找匹配 glob 模式的文件 |
| `NiaGitHubTree` | 浏览仓库文件树 |
| `NiaContextSave` | 保存上下文以供跨代理共享 |
| `NiaContextSearch` | 对保存的上下文进行语义搜索 |
| `NiaDependencySubscribe` | 自动订阅项目依赖的文档 |
| `NiaDependencyAnalyze` | 从清单预览可索引的依赖项 |
