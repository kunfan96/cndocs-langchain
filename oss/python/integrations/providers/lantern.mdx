本页介绍如何在 LangChain 中使用 [Lantern](https://github.com/lanterndata/lantern)。
内容分两个部分：设置，以及对特定 Lantern 封装器的说明。

## 设置
1. 第一步是创建安装了 `lantern` 扩展的数据库。

    按照 [Lantern 安装指南](https://github.com/lanterndata/lantern#-quick-install)安装数据库和扩展。Docker 镜像是最简单的入门方式。

## 封装器

### 向量存储

Postgres 向量数据库有一个封装器，可将其作为向量存储使用，支持语义搜索或示例选择。

导入该向量存储：
```python
from langchain_community.vectorstores import Lantern
```
