> [ClickHouse](https://clickhouse.com/) 是一款快速、资源高效的开源数据库，适用于实时应用和分析，支持完整 SQL 并提供大量函数以帮助用户编写分析查询。它具有数据结构和距离搜索函数（如 `L2Distance`）以及[近似最近邻搜索索引](https://clickhouse.com/docs/en/engines/table-engines/mergetree-family/annindexes)，使 ClickHouse 能够作为高性能、可扩展的向量数据库，用于通过 SQL 存储和搜索向量。


## 安装与设置

我们需要安装 `clickhouse-connect` Python 包。

<CodeGroup>
```bash pip
pip install clickhouse-connect
```

```bash uv
uv add clickhouse-connect
```
</CodeGroup>
