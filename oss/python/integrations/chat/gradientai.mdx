## 配置

langchain-gradient 使用 DigitalOcean Gradient 平台。

在 DigitalOcean 创建账户，从 Gradient 平台上获取 `DIGITALOCEAN_INFERENCE_KEY` API 密钥，并安装 `langchain-gradient` 集成包。

### 资源凭证

前往 [DigitalOcean 登录](https://cloud.digitalocean.com/login)

1. 登录 DigitalOcean 云控制台
2. 进入 Gradient 平台，导航到无服务器推理。
3. 点击创建模型访问密钥，输入名称并生成密钥。

完成上述操作后，请设置 `DIGITALOCEAN_INFERENCE_KEY` 环境变量：

```python
import getpass
import os

if not os.getenv("DIGITALOCEAN_INFERENCE_KEY"):
    os.environ["DIGITALOCEAN_INFERENCE_KEY"] = getpass.getpass(
        "请输入您的 DIGITALOCEAN_INFERENCE_KEY API 密钥: "
    )
```

如果你想获取模型调用的自动跟踪，还可以通过取消注释下方代码设置 [LangSmith](/langsmith/home) API 密钥：

```python
os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("请输入您的 LangSmith API 密钥: ")
```

### 安装

DigitalOcean Gradient 集成位于 `langchain-gradient` 包中：

```python
pip install -qU langchain-gradient
```

## 实例化

现在我们可以实例化我们的模型对象并生成聊天完成内容：

```python
from langchain_gradient import ChatGradient

llm = ChatGradient(
    model="llama3.3-70b-instruct",
    # 其他参数...
)
```

## 链接

我们可以通过提示模板链接我们的模型：

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate(
    [
        (
            "system",
            '你是一个博学的助手。仔细阅读提供的上下文并回答用户的问题。如果答案在上下文中存在，请引用相关的句子。否则，回复“未在上下文中找到。”',
        ),
        ("human", "Context: {context}\nQuestion: {question}"),
    ]
)

chain = prompt | llm
chain.invoke(
    {
        "context": (
            "埃菲尔铁塔位于巴黎，并于1889年完工。它是由古斯塔夫·埃菲尔的工程公司设计的。这座塔是世界上最著名的建筑之一。自由女神像是法国赠送给美国的一份礼物。"
        ),
        "question": "谁设计了埃菲尔铁塔，它是什么时候完工的？",
    }
)
```
