本文介绍如何在 LangChain 中使用 [YandexGPT](https://cloud.yandex.com/en/services/yandexgpt) 嵌入模型。

使用前，你需要安装 `yandexcloud` Python 包。

```python
pip install -qU  yandexcloud
```

首先，你需要[创建一个服务账号](https://cloud.yandex.com/en/docs/iam/operations/sa/create)，并赋予其 `ai.languageModels.user` 角色。

接下来，你有两种身份验证方式可供选择：

- [IAM 令牌](https://cloud.yandex.com/en/docs/iam/operations/iam-token/create-for-sa)。
    你可以在构造参数 `iam_token` 或环境变量 `YC_IAM_TOKEN` 中指定令牌。
- [API 密钥](https://cloud.yandex.com/en/docs/iam/operations/api-key/create)
    你可以在构造参数 `api_key` 或环境变量 `YC_API_KEY` 中指定密钥。

要指定模型，可以使用 `model_uri` 参数，详情请参阅[文档](https://cloud.yandex.com/en/docs/yandexgpt/concepts/models#yandexgpt-embeddings)。

默认情况下，将使用由 `folder_id` 参数或 `YC_FOLDER_ID` 环境变量所指定文件夹中 `text-search-query` 的最新版本。

```python
from langchain_community.embeddings.yandex import YandexGPTEmbeddings
```

```python
embeddings = YandexGPTEmbeddings()
```

```python
text = "This is a test document."
```

```python
query_result = embeddings.embed_query(text)
```

```python
doc_result = embeddings.embed_documents([text])
```

```python
query_result[:5]
```
