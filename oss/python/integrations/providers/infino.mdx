>[Infino](https://github.com/infinohq/infino) 是一个开源的可观测性平台，它将指标和应用程序日志存储在一起。

`Infino` 的主要功能包括：
- **指标跟踪**：捕获 LLM 模型处理请求所花费的时间、错误、Token 数量以及针对该特定 LLM 的成本估算。
- **数据跟踪**：记录并存储每次 LangChain 交互的 Prompt、请求和响应数据。
- **图表可视化**：生成随时间变化的基础图表，展示请求持续时间、错误发生次数、Token 数量和成本等指标。

## 安装与设置

首先，你需要按如下方式安装 `infinopy` Python 包：

<CodeGroup>
```bash pip
pip install infinopy
```

```bash uv
uv add infinopy
```
</CodeGroup>

如果你已经运行了 `Infino Server`，即可直接开始；若未运行，请按照以下步骤启动它：

- 确保已安装 Docker
- 在终端中运行以下命令：
    ```
    docker run --rm --detach --name infino-example -p 3000:3000 infinohq/infino:latest
    ```
