---
title: 模型认证
---

<Note>
    目前仅在 `langchainjs`（JavaScript/TypeScript）中使用。
</Note>

您的模型提供商正在拒绝您访问其服务。

此错误通常发生在您的认证凭据或 API 密钥存在问题时。

## 故障排除

* 确认您的 API 密钥或认证凭据准确且有效。
* 如果使用基于环境的认证，请验证：
    - 变量名称拼写正确
    - 变量包含已分配的值
    - 第三方包（如 `dotenv`）未干扰加载过程
* 如果使用代理或非标准端点，请确保您的自定义提供商不期望替代的认证方案。
* 通过显式传递凭据来绕过环境变量问题：

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(api_key="YOUR_KEY_HERE")
```
