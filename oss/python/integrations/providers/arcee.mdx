>[Arcee](https://www.arcee.ai/about/about-us) 致力于开发并推进
> 我们定义的 SLMs——即小型、专用、安全且可扩展的语言模型。
> 通过提供 SLM 适配系统及无缝、安全的集成体验，
> `Arcee` 使企业能够充分发挥领域自适应语言模型的全部潜力，
> 从而驱动运营层面的变革性
> 创新。


## 安装与设置

获取您的 `Arcee API` 密钥。


## LLMs

请参阅[使用示例](/oss/python/integrations/llms/arcee)。

```python
from langchain_community.llms import Arcee
```

## 检索器

请参阅[使用示例](/oss/python/integrations/retrievers/arcee)。

```python
from langchain_community.retrievers import ArceeRetriever
```
