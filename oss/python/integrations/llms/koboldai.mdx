[KoboldAI](https://github.com/KoboldAI/KoboldAI-Client) 是一个“支持多种本地和远程 AI 模型的基于浏览器的 AI 辅助写作前端……”。它提供公开和本地 API，可在 LangChain 中使用。

本示例介绍如何将该 API 与 LangChain 配合使用。

在浏览器中访问文档时，只需在你的端点地址末尾添加 `/api` 即可（例如 [127.0.0.1/:5000/api](http://127.0.0.1/:5000/api)）。

```python
from langchain_community.llms import KoboldApiLLM
```

将下方显示的端点替换为通过 `--api` 或 `--public-api` 启动 WebUI 后输出中显示的端点。

可选地，你可以传入 `temperature` 或 `max_length` 等参数。

```python
llm = KoboldApiLLM(endpoint="http://192.168.1.144:5000", max_length=80)
```

```python
response = llm.invoke(
    "### Instruction:\nWhat is the first book of the bible?\n### Response:"
)
```
