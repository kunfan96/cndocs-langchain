[Browserbase](https://browserbase.com) 是一个用于可靠运行、管理和监控无头浏览器的开发者平台。

通过以下方式提升您的 AI 数据检索能力：
- [Serverless Infrastructure](https://docs.browserbase.com/under-the-hood) 提供可靠的浏览器，用于从复杂 UI 中提取数据
- [Stealth Mode](https://docs.browserbase.com/features/stealth-mode) 内置指纹伪装策略和自动验证码解决功能
- [Session Debugger](https://docs.browserbase.com/features/sessions) 支持通过网络时间线和日志检查您的浏览器会话
- [Live Debug](https://docs.browserbase.com/guides/session-debug-connection/browser-remote-control) 快速调试您的自动化流程

## 安装与配置

- 从 [browserbase.com](https://browserbase.com) 获取 API Key 和项目 ID（Project ID），并将其设置为环境变量（`BROWSERBASE_API_KEY`、`BROWSERBASE_PROJECT_ID`）。
- 安装 [Browserbase SDK](http://github.com/browserbase/python-sdk)：

```python
pip install browserbase
```

## 文档加载器

请参阅[使用示例](/oss/python/integrations/document_loaders/browserbase)。

```python
from langchain_community.document_loaders import BrowserbaseLoader
```
