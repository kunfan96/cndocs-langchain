## 概述

`RedisStore` 是 `ByteStore` 的一个实现，将所有内容存储在你的 Redis 实例中。

## 配置

要创建 Redis 字节存储，需要先搭建一个 Redis 实例。可以在本地搭建或使用云服务提供商——详见我们的 [Redis 指南](/oss/python/integrations/providers/redis)。

### 安装

LangChain `RedisStore` 集成位于 `langchain-community` 包中：

```python
pip install -qU langchain-community redis
```

## 实例化

现在可以实例化字节存储：

```python
from langchain_community.storage import RedisStore

kv_store = RedisStore(redis_url="redis://localhost:6379")
```

## 使用方法

可以使用 `mset` 方法在键下设置数据：

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[b'value1', b'value2']
```

可以使用 `mdelete` 方法删除数据：

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[None, None]
```
