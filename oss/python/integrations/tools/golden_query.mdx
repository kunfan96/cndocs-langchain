>[Golden](https://golden.com) 提供了一组用于查询和信息增强的自然语言 API，基于 Golden 知识图谱运作。例如，可使用以下查询来检索相关实体的结构化数据：`Products from OpenAI`、`Generative ai companies with series a funding`、`rappers who invest` 等。
>
>`golden-query` LangChain 工具是 [Golden Query API](https://docs.golden.com/reference/query-api) 的封装，支持以编程方式访问查询结果。
>详情请参阅 [Golden Query API 文档](https://docs.golden.com/reference/query-api)。

本笔记本介绍如何使用 `golden-query` 工具。

- 访问 [Golden API 文档](https://docs.golden.com/) 了解 Golden API 概览。
- 在 [Golden API 设置](https://golden.com/settings/api)页面获取您的 API 密钥。
- 将您的 API 密钥保存到环境变量 GOLDEN_API_KEY 中。

```python
pip install -qU langchain-community
```

```python
import os

os.environ["GOLDEN_API_KEY"] = ""
```

```python
from langchain_community.utilities.golden_query import GoldenQueryAPIWrapper
```

```python
golden_query = GoldenQueryAPIWrapper()
```

```python
import json

json.loads(golden_query.run("companies in nanotech"))
```
