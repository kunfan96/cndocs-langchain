> [IEIT Systems](https://en.ieisystem.com/) 是一家成立于 1999 年的中国信息技术公司，
> 提供 IT 基础设施产品、解决方案与服务，以及在云计算、
> 大数据和人工智能领域的创新 IT 产品与解决方案。


## LLMs

请参阅 [使用示例](/oss/python/integrations/llms/yuan2)。

```python
from langchain_community.llms.yuan2 import Yuan2
```

## Chat models

请参阅 [安装说明](/oss/python/integrations/chat/yuan2/#setting-up-your-api-server)。

Yuan2.0 提供了兼容 OpenAI 的 API，ChatYuan2 通过使用 `OpenAI client` 集成到 langchain 中。
因此，请确保已安装 `openai` 包。

<CodeGroup>
```bash pip
pip install openai
```

```bash uv
uv add openai
```
</CodeGroup>

请参阅 [使用示例](/oss/python/integrations/chat/yuan2)。

```python
from langchain_community.chat_models import ChatYuan2
```
