本页介绍如何使用 [Yeager.ai](https://yeager.ai) 生成 LangChain 工具和 agent。

## 什么是 yeager.ai？
Yeager.ai 是一个旨在简化 AI agent 和工具创建过程的生态系统。

它提供 yAgents，这是一个无代码的 LangChain Agent 构建器，使用户能够轻松构建、测试和部署 AI 解决方案。利用 LangChain 框架，yAgents 可以与各种语言模型和资源无缝集成，适合开发者、研究人员和 AI 爱好者在多种应用场景中使用。

## yAgents
低代码生成式 agent，旨在帮助你轻松构建、原型化和部署 LangChain 工具。

### 如何使用？
<CodeGroup>
```bash pip
pip install yeagerai-agent
yeagerai-agent
```

```bash uv
uv add yeagerai-agent
yeagerai-agent
```
</CodeGroup>
访问 http://127.0.0.1:7860

这将安装必要的依赖项并在你的系统上设置 yAgents。首次运行后，yAgents 将创建一个 .env 文件，你可以在其中输入你的 OpenAI API 密钥。你也可以直接在 Gradio 界面的"Settings"选项卡中进行设置。

`OPENAI_API_KEY=<your_openai_api_key_here>`

我们推荐使用 GPT-4。不过，如果问题被充分分解，该工具也可以与 GPT-3 一起使用。

### 使用 yAgents 创建和执行工具
yAgents 使创建和执行 AI 驱动的工具变得简单。以下是该过程的简要概述：
1. 创建工具：要创建工具，向 yAgents 提供自然语言提示。提示应清楚描述工具的目的和功能。例如：
`create a tool that returns the n-th prime number`

2. 将工具加载到工具箱：要将工具加载到 yAgents 中，只需向 yAgents 提供相应的命令。例如：
`load the tool that you just created it into your toolkit`

3. 执行工具：要运行工具或 agent，只需向 yAgents 提供包含工具名称和任何所需参数的命令。例如：
`generate the 50th prime number`

你可以在[这里](https://www.youtube.com/watch?v=KA5hCM3RaWE)观看其工作原理的视频。

随着你对 yAgents 越来越熟悉，你可以创建更高级的工具和 agent 来自动化工作并提高工作效率。

更多信息，请参阅 [yAgents 的 GitHub](https://github.com/yeagerai/yeagerai-agent) 或我们的[文档](https://yeagerai.gitbook.io/docs/general/welcome-to-yeager.ai)
