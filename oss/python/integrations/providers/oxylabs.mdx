[Oxylabs](https://oxylabs.io/) 是一个市场领先的网络智能收集平台，以最高的商业、道德和合规标准驱动，帮助全球企业解锁数据驱动的洞察。

[langchain-oxylabs](https://pypi.org/project/langchain-oxylabs/) 实现了使 LLMs 能够与 Oxylabs Web Scraper API 交互的工具。


## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-oxylabs
```

```bash uv
uv add langchain-oxylabs
```
</CodeGroup>
