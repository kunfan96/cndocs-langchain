>[Robocorp](https://robocorp.com/) 帮助构建和操作 Python 工作器，这些工作器可以在任何地方、任何规模下无缝运行。

## 安装与设置

您需要安装 `langchain-robocorp` Python 包：

<CodeGroup>
```bash pip
pip install langchain-robocorp
```

```bash uv
uv add langchain-robocorp
```
</CodeGroup>

您需要一个正在运行的 `Action Server` 实例，以便从您的代理应用程序进行通信。
请参阅 [Robocorp 快速入门](https://github.com/robocorp/robocorp#quickstart) 了解如何设置 Action Server 并创建您的 Actions。

您可以使用 Action Server 的 `new` 命令引导一个新项目。

```bash
action-server new
cd ./your-project-name
action-server start
```

## 工具

```python
from langchain_robocorp.toolkits import ActionServerRequestTool
```
