[xAI](https://console.x.ai) 提供了一个 API 用于与 Grok 模型交互。此示例介绍了如何使用 LangChain 与 xAI 模型交互。

## 安装

```python
pip install -U langchain-xai
```

## 环境

要使用 xAI，您需要[创建一个 API 密钥](https://console.x.ai/)。API 密钥可以作为初始化参数 `xai_api_key` 传入，也可以设置为环境变量 `XAI_API_KEY`。

## 示例

有关详细信息和支持的功能，请参阅 [ChatXAI 文档](/oss/python/integrations/chat/xai)。

```python
# 使用 xAI 查询聊天模型

from langchain_xai import ChatXAI

chat = ChatXAI(
    # xai_api_key="YOUR_API_KEY",
    model="grok-4",
)

# 从模型流式传输响应
for m in chat.stream("Tell me fun things to do in NYC"):
    print(m.content, end="", flush=True)

# 如果您不想进行流式传输，可以使用 invoke 方法
# chat.invoke("Tell me fun things to do in NYC")
```
