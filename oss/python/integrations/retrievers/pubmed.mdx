>[PubMed®](https://pubmed.ncbi.nlm.nih.gov/) 由 `美国国家生物技术信息中心、国家医学图书馆` 提供，包含超过 3500 万条生物医学文献引用，来源包括 `MEDLINE`、生命科学期刊和在线书籍。引用可能包含指向 `PubMed Central` 和出版商网站全文内容的链接。

本笔记本介绍如何将 `PubMed` 用作检索器。

```python
from langchain_community.retrievers import PubMedRetriever
```

```python
retriever = PubMedRetriever()
```

```python
retriever.invoke("chatgpt")
```
