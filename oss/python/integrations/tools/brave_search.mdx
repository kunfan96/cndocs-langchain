本笔记本介绍如何使用 Brave 搜索工具。
请访问 [Brave 网站](https://brave.com/search/api/) 注册免费账户并获取 API 密钥。

```python
pip install -qU langchain-community
```

```python
from langchain_community.tools import BraveSearch
```

```python
api_key = "API KEY"
```

```python
tool = BraveSearch.from_api_key(api_key=api_key, search_kwargs={"count": 3})

# 或者，如果您想从环境变量 BRAVE_SEARCH_API_KEY 获取 API 密钥，并留空 search_kwargs
# tool = BraveSearch()

# 或者，如果您只想提供 API 密钥，并留空 search_kwargs
# tool = BraveSearch.from_api_key(api_key=api_key)

# 或者，如果您只想提供 search_kwargs 并从 BRAVE_SEARCH_API_KEY 环境变量读取 API 密钥
# tool = BraveSearch.from_search_kwargs(search_kwargs={"count": 3})
```

```python
tool.run("obama middle name")
```
