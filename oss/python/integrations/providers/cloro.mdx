本文档介绍了如何在 LangChain 中使用 [cloro](https://cloro.dev/) 监控 API。

## 安装与配置
- 使用 `pip install -U langchain-cloro` 安装依赖项
- 从 [cloro 仪表板](https://dashboard.cloro.dev/) 获取 cloro API 密钥，并将其设置为环境变量（`CLORO_API_KEY`）

## 工具

cloro 提供了多种用于抓取 AI 平台并提取结构化数据的工具。要使用这些工具：

```python
from langchain_cloro import CloroGoogleSearch, CloroChatGPT, CloroGemini, CloroPerplexity, CloroGrok, CloroCopilot
```

有关详细文档和示例，请参阅：
- [Google 搜索爬虫](/oss/python/integrations/tools/cloro#google-search-scraper)
- [ChatGPT 爬虫](/oss/python/integrations/tools/cloro#chatgpt-scraper)
- [Gemini 爬虫](/oss/python/integrations/tools/cloro#gemini-scraper)
- [Perplexity 爬虫](/oss/python/integrations/tools/cloro#perplexity-scraper)
- [Grok 爬虫](/oss/python/integrations/tools/cloro#grok-scraper)
- [Copilot 爬虫](/oss/python/integrations/tools/cloro#copilot-scraper)

如需更多信息，请访问 [cloro 文档](https://docs.cloro.dev)。
