`Hugging Face Spaces` 上有数以千计的 `Gradio` 应用。这个库让您的 LLM 能够轻松使用这些应用 🦾

具体来说，`gradio-tools` 是一个 Python 库，可将 `Gradio` 应用转换为工具，供基于大型语言模型（LLM）的智能体使用以完成任务。例如，LLM 可以使用 `Gradio` 工具对在线找到的语音录音进行转录，然后为您生成摘要；或使用另一个 `Gradio` 工具对 Google Drive 上的文档进行 OCR，并回答相关问题。

如果您想使用某个非预置工具的 Space，创建自定义工具非常简单，请参阅 gradio-tools 文档中的相关章节了解详情。欢迎所有贡献！

```python
pip install -qU  gradio_tools langchain-community
```

## 在智能体中使用

```python
from gradio_tools.tools import (
    ImageCaptioningTool,
    StableDiffusionPromptGeneratorTool,
    StableDiffusionTool,
    TextToVideoTool,
)
from langchain.agents import create_agent
from langchain.memory import ConversationBufferMemory
from langchain_openai import OpenAI

llm = OpenAI(temperature=0)
memory = ConversationBufferMemory(memory_key="chat_history")
tools = [
    StableDiffusionTool().langchain,
    ImageCaptioningTool().langchain,
    StableDiffusionPromptGeneratorTool().langchain,
    TextToVideoTool().langchain,
]

agent = create_agent(
    model=llm,
    tools=tools,
    memory=memory,
    verbose=True,
)

output = agent.invoke(
    {
        "input": (
            "Please create a photo of a dog riding a skateboard "
            "but improve my prompt prior to using an image generator."
            "Please caption the generated image and create a video for it using the improved prompt."
        )
    }
)
```
