>[Zilliz Cloud](https://zilliz.com/doc/quick_start) 是 `LF AI Milvus®` 的完全托管云服务，

## 安装与设置

安装 Python SDK：
<CodeGroup>
```bash pip
pip install pymilvus
```

```bash uv
uv add pymilvus
```
</CodeGroup>

## 向量存储

Zilliz 索引的包装器允许您将其用作向量存储，
无论是用于语义搜索还是示例选择。

```python
from langchain_community.vectorstores import Milvus
```

有关 Miluvs 包装器的更详细演练，请参见[此笔记本](/oss/python/integrations/vectorstores/zilliz)
