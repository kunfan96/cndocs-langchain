[Box](https://box.com) 是智能内容云（Intelligent Content Cloud），一个单一平台，使组织能够促进协作、管理完整的内容生命周期、保护关键内容，并利用企业级 AI 改造业务流程。Box 成立于 2005 年，为包括阿斯利康（AstraZeneca）、JLL、摩根士丹利（Morgan Stanley）和 Nationwide 在内的全球领先组织简化了工作流程。

在本软件包中，我们提供了多种将 Box 内容集成到 AI 工作流中的方法。

### 安装与配置

<CodeGroup>
```bash pip
pip install -U langchain-box
```

```bash uv
uv add langchain-box
```
</CodeGroup>

# langchain-box

本软件包包含 LangChain 与 Box 的集成。有关 Box 的更多信息，请参阅我们的[开发者文档](https://developer.box.com)。

## 前置条件

要与 Box 集成，您需要准备以下内容：

* 一个 Box 实例——如果您还不是 Box 客户，请注册一个[免费开发者账号](https://account.box.com/signup/n/developer#ty9l3)。
* 一个 Box 应用——了解如何[创建应用](https://developer.box.com/guides/getting-started/first-application/)的更多详情
* 您的应用在 Box 实例中获得批准——这由您的管理员完成。
好消息是，如果您使用的是免费开发者账号，您自己就是管理员。
[授权您的应用](https://developer.box.com/guides/authorization/custom-app-approval/#manual-approval)

## 身份验证

`box-langchain` 软件包在身份验证方面提供了一定的灵活性。最基本的身份验证方法是使用开发者令牌。您可以在[Box 开发者控制台](https://account.box.com/developers/console)的配置页面上找到它。该令牌的有效期故意设置得很短（1 小时），仅用于开发环境。使用该令牌时，您可以将其作为环境变量 `BOX_DEVELOPER_TOKEN` 添加到环境中，也可以直接传递给加载器，或者使用 `BoxAuth` 身份验证辅助类。

我们将在下一节介绍如何直接将其传递给加载器。

### BoxAuth 辅助类

`BoxAuth` 支持以下身份验证方法：

* 令牌（Token）——可以是开发者令牌，也可以是任何通过 Box SDK 生成的令牌
* 使用服务账号的 JWT
* 使用指定用户的 JWT
* 使用服务账号的 CCG
* 使用指定用户的 CCG

<Note>
**如果使用 JWT 身份验证，在生成公钥/私钥对后，您需要从 Box**

开发者控制台下载配置文件。将此文件放置在应用程序目录结构的某个位置。在使用
`BoxAuth` 辅助类时，您将需要用到此文件的路径。
</Note>

如需更多信息，请了解如何
[设置 Box 应用](https://developer.box.com/guides/getting-started/first-application/)，
并查看
[Box 身份验证指南](https://developer.box.com/guides/authentication/select/)
以了解更多关于我们不同身份验证选项的信息。

示例：

**令牌（Token）**

```python
from langchain_box.document_loaders import BoxLoader
from langchain_box.utilities import BoxAuth, BoxAuthType

auth = BoxAuth(
    auth_type=BoxAuthType.TOKEN,
    box_developer_token=box_developer_token
)

loader = BoxLoader(
    box_auth=auth,
    ...
)
```

**使用服务账号的 JWT**

```python
from langchain_box.document_loaders import BoxLoader
from langchain_box.utilities import BoxAuth, BoxAuthType

auth = BoxAuth(
    auth_type=BoxAuthType.JWT,
    box_jwt_path=box_jwt_path
)

loader = BoxLoader(
    box_auth=auth,
    ...
```

**使用指定用户的 JWT**

```python
from langchain_box.document_loaders import BoxLoader
from langchain_box.utilities import BoxAuth, BoxAuthType

auth = BoxAuth(
    auth_type=BoxAuthType.JWT,
    box_jwt_path=box_jwt_path,
    box_user_id=box_user_id
)

loader = BoxLoader(
    box_auth=auth,
    ...
```

**使用服务账号的 CCG**

```python
from langchain_box.document_loaders import BoxLoader
from langchain_box.utilities import BoxAuth, BoxAuthType

auth = BoxAuth(
    auth_type=BoxAuthType.CCG,
    box_client_id=box_client_id,
    box_client_secret=box_client_secret,
    box_enterprise_id=box_enterprise_id
)

loader = BoxLoader(
    box_auth=auth,
    ...
```

**使用指定用户的 CCG**

```python
from langchain_box.document_loaders import BoxLoader
from langchain_box.utilities import BoxAuth, BoxAuthType

auth = BoxAuth(
    auth_type=BoxAuthType.CCG,
    box_client_id=box_client_id,
    box_client_secret=box_client_secret,
    box_user_id=box_user_id
)

loader = BoxLoader(
    box_auth=auth,
    ...
```

如果您希望使用带有 authorization_code 流程的 OAuth2，请使用您已获取的令牌配合 `BoxAuthType.TOKEN`。
