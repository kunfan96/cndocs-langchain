ModelScope（[首页](https://www.modelscope.cn/) | [GitHub](https://github.com/modelscope/modelscope)）基于“模型即服务”（MaaS）的理念构建。它旨在汇聚来自人工智能社区的最先进机器学习模型，并简化在实际应用中利用 AI 模型的过程。该仓库中开源的核心 ModelScope 库提供了接口和实现，使开发者能够执行模型推理、训练和评估。这将帮助您使用 LangChain 快速上手 ModelScope 补全模型（LLM）。

## 环境配置

要访问 ModelScope 模型，您需要创建一个 ModelScope 账号，获取一个 SDK 令牌，并安装 `langchain-modelscope-integration` 集成包。

### 凭据

前往 [ModelScope](https://modelscope.cn/) 注册账号并生成 [SDK 令牌](https://modelscope.cn/my/myaccesstoken)。完成此操作后，请设置 `MODELSCOPE_SDK_TOKEN` 环境变量：

```python
import getpass
import os

if not os.getenv("MODELSCOPE_SDK_TOKEN"):
    os.environ["MODELSCOPE_SDK_TOKEN"] = getpass.getpass(
        "请输入您的 ModelScope SDK 令牌： "
    )
```

### 安装

LangChain 的 ModelScope 集成位于 `langchain-modelscope-integration` 包中：

```python
pip install -qU langchain-modelscope-integration
```

## 实例化

现在我们可以实例化模型对象并生成聊天补全结果：

```python
from langchain_modelscope import ModelScopeEndpoint

llm = ModelScopeEndpoint(
    model="Qwen/Qwen2.5-Coder-32B-Instruct",
    temperature=0,
    max_tokens=1024,
    timeout=60,
)
```

## 调用

```python
input_text = "Write a quick sort algorithm in python"

completion = llm.invoke(input_text)
completion
```

```text
'当然可以！快速排序是一种流行且高效的排序算法，采用分治法对元素进行排序。以下是 Python 中快速排序算法的简单实现：\n\n\`\`\`python\ndef quick_sort(arr):\n    # 基本情况：如果数组为空或只有一个元素，则已排序\n    if len(arr) <= 1:\n        return arr\n    else:\n        # 从数组中选择一个基准元素\n        pivot = arr[len(arr) // 2]\n        \n        # 将数组划分为三个部分：\n        # - 小于基准的元素\n        # - 等于基准的元素\n        # - 大于基准的元素\n        less_than_pivot = [x for x in arr if x < pivot]\n        equal_to_pivot = [x for x in arr if x == pivot]\n        greater_than_pivot = [x for x in arr if x > pivot]\n        \n        # 递归地对 less_than_pivot 和 greater_than_pivot 子数组应用 quick_sort\n        return quick_sort(less_than_pivot) + equal_to_pivot + quick_sort(greater_than_pivot)\n\n# 示例用法：\narr = [3, 6, 8, 10, 1, 2, 1]\nsorted_arr = quick_sort(arr)\nprint("排序后的数组：", sorted_arr)\n\`\`\`\n\n### 说明：\n1. **基本情况**：如果数组包含一个或零个元素，则已经排序，因此直接返回。\n2. **基准选择**：我们选择数组的中间元素作为基准。这是一种简单的策略，但还有其他选择基准的策略。\n3. **分区**：我们将数组划分为三个列表：\n   - `less_than_pivot`：小于基准的元素。\n   - `equal_to_pivot`：等于基准的元素。\n   - `greater_than_pivot`：大于基准的元素。\n4. **递归排序**：我们递归地对 `less_than_pivot` 和 `greater_than_pivot` 列表进行排序，并将它们与 `equal_to_pivot` 列表连接起来，以获得最终排序后的数组。\n\n此实现直观易懂，但由于使用了额外的列表，在空间复杂度方面可能不是最高效的。对于原地版本的快速排序，您可以修改算法以在其自身的内存空间中排序数组。'
```
