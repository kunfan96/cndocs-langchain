>[WolframAlpha](https://en.wikipedia.org/wiki/WolframAlpha) 是由 `Wolfram Research` 开发的答案引擎。它通过从外部数据源计算答案来回答事实性查询。

本页介绍如何在 LangChain 中使用 `Wolfram Alpha API`。

## 安装与设置
- 通过以下方式安装依赖：
<CodeGroup>
```bash pip
pip install wolframalpha
```

```bash uv
uv add wolframalpha
```
</CodeGroup>
- 前往 Wolfram Alpha 并在[此处](https://developer.wolframalpha.com/)注册开发者账号
- 创建一个应用并获取你的 `APP ID`
- 将 APP ID 设置为环境变量 `WOLFRAM_ALPHA_APPID`


## 封装

### 工具类

有一个封装此 API 的 WolframAlphaAPIWrapper 工具类。导入此工具类：

```python
from langchain_community.utilities.wolfram_alpha import WolframAlphaAPIWrapper
```

有关此封装的更详细说明，请参阅[此笔记本](/oss/python/integrations/tools/wolfram_alpha)。

### 工具

你也可以将此封装作为工具（用于 Agent）轻松加载。可以这样实现：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["wolfram-alpha"])
```

有关工具的更多信息，请参阅[此页面](/oss/python/integrations/tools)。
