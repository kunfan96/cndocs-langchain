>[ROAM](https://roamresearch.com/) 是一款用于网状思维的笔记工具，旨在构建个人知识库。

本文档介绍如何从 Roam 数据库加载文档，灵感来源于[此示例仓库](https://github.com/JimmyLv/roam-qa)。

## 🧑 导入自有数据集的操作步骤

从 Roam Research 导出数据集。您可以点击右上角的三个点，然后点击 `Export` 完成导出。

导出时，请确保选择 `Markdown & CSV` 格式。

这将在您的下载文件夹中生成一个 `.zip` 文件。将该 `.zip` 文件移动到本仓库目录中。

运行以下命令解压 zip 文件（根据需要将 `Export...` 替换为您的实际文件名）：

```shell
unzip Roam-Export-1675782732639.zip -d Roam_DB
```

```python
from langchain_community.document_loaders import RoamLoader
```

```python
loader = RoamLoader("Roam_DB")
```

```python
docs = loader.load()
```
