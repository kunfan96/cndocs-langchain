>[Datadog](https://www.datadoghq.com/) 是一个面向云规模应用程序的监控与分析平台。

## 安装与设置

<CodeGroup>
```bash pip
pip install datadog_api_client
```

```bash uv
uv add datadog_api_client
```
</CodeGroup>

我们必须使用 Datadog API 密钥和 APP 密钥初始化加载器，并需要设置查询以提取所需的日志。

## 文档加载器

请参阅[使用示例](/oss/python/integrations/document_loaders/datadog_logs)。
