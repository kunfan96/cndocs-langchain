>[Yahoo（维基百科）](https://en.wikipedia.org/wiki/Yahoo) 是一家美国网络服务提供商。
>
> 它提供门户网站、搜索引擎 Yahoo Search 以及相关服务，包括 `My Yahoo`、`Yahoo Mail`、`Yahoo News`、`Yahoo Finance`、`Yahoo Sports` 及其广告平台 `Yahoo Native`。


## 工具

### Yahoo Finance 新闻

我们需要安装一个 Python 包：

<CodeGroup>
```bash pip
pip install yfinance
```

```bash uv
uv add yfinance
```
</CodeGroup>

查看[使用示例](/oss/python/integrations/tools/yahoo_finance_news)。


```python
from langchain_community.tools import YahooFinanceNewsTool
```
