>[Argilla](https://argilla.io/) 是一个面向 LLM 的开源数据策管平台。
> 使用 `Argilla`，每个人都可以通过人工和机器反馈更快速地策管数据，构建强大的语言模型。
> `Argilla` 为 MLOps 周期中的每个步骤提供支持，从数据标注到模型监控。

## 安装与设置

获取您的 [API 密钥](https://platform.openai.com/account/api-keys)。

安装 Python 包：

<CodeGroup>
```bash pip
pip install argilla
```

```bash uv
uv add argilla
```
</CodeGroup>

## 回调


```python
from langchain.callbacks import ArgillaCallbackHandler
```

查看[示例](/oss/python/integrations/callbacks/argilla)。
