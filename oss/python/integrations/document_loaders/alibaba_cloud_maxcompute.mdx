>[阿里云 MaxCompute](https://www.alibabacloud.com/product/maxcompute)（前身为 ODPS）是一个通用的、全托管的、多租户数据处理平台，用于大规模数据仓库。MaxCompute 支持多种数据导入方案和分布式计算模型，使用户能够高效地查询海量数据集、降低生产成本并确保数据安全。

`MaxComputeLoader` 允许您执行 MaxCompute SQL 查询，并将结果按行加载为文档，每行对应一个文档。

## 基本用法

要实例化加载器，您需要提供一个待执行的 SQL 查询、MaxCompute 的 endpoint 和项目名称，以及您的 access ID 和密钥。access ID 和密钥可以通过 `access_id` 和 `secret_access_key` 参数直接传入，也可以设置为环境变量 `MAX_COMPUTE_ACCESS_ID` 和 `MAX_COMPUTE_SECRET_ACCESS_KEY`。

```python
from langchain_community.document_loaders import MaxComputeLoader
```

```python
base_query = """
SELECT *
FROM (
    SELECT 1 AS id, 'content1' AS content, 'meta_info1' AS meta_info
    UNION ALL
    SELECT 2 AS id, 'content2' AS content, 'meta_info2' AS meta_info
    UNION ALL
    SELECT 3 AS id, 'content3' AS content, 'meta_info3' AS meta_info
) mydata;
"""
```

```python
endpoint = "<ENDPOINT>"
project = "<PROJECT>"
ACCESS_ID = "<ACCESS ID>"
SECRET_ACCESS_KEY = "<SECRET ACCESS KEY>"
```

```python
loader = MaxComputeLoader.from_params(
    base_query,
    endpoint,
    project,
    access_id=ACCESS_ID,
    secret_access_key=SECRET_ACCESS_KEY,
)
data = loader.load()
```

## 指定内容列与元数据列

您可以使用 `page_content_columns` 和 `metadata_columns` 参数来配置哪些列作为文档内容加载，哪些列作为元数据加载。

```python
loader = MaxComputeLoader.from_params(
    base_query,
    endpoint,
    project,
    page_content_columns=["content"],  # Specify Document page content
    metadata_columns=["id", "meta_info"],  # Specify Document metadata
    access_id=ACCESS_ID,
    secret_access_key=SECRET_ACCESS_KEY,
)
data = loader.load()
```

```python
print(data[0].page_content)
```

```text
content: content1
```

```python
print(data[0].metadata)
```

```python
{'id': 1, 'meta_info': 'meta_info1'}
```
