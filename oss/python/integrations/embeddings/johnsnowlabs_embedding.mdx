>[John Snow Labs](https://nlp.johnsnowlabs.com/) NLP 与 LLM 生态系统包含用于大规模先进人工智能、负责任的人工智能、无代码人工智能的软件库，并提供对医疗、法律、金融等领域超过 20,000 个模型的访问权限。
>
>模型通过 [nlp.load](https://nlp.johnsnowlabs.com/docs/en/jsl/load_api) 加载，底层会自动启动 Spark 会话，使用 >[nlp.start()](https://nlp.johnsnowlabs.com/docs/en/jsl/start-a-sparksession)。
>查看所有 24,000 多个模型，请参阅 [John Snow Labs 模型中心](https://nlp.johnsnowlabs.com/models)

## 示例

```python
from langchain_community.embeddings.johnsnowlabs import JohnSnowLabsEmbeddings
```

初始化 JohnSnowLabs 嵌入模型与 Spark 会话

```python
embedder = JohnSnowLabsEmbeddings("en.embed_sentence.biobert.clinical_base_cased")
```

定义一些示例文本。这些可以是您希望分析的任何文档——例如新闻文章、社交媒体帖子或产品评论。

```python
texts = ["Cancer is caused by smoking", "Antibiotics aren't painkiller"]
```

生成并打印文本的嵌入向量。JohnSnowLabsEmbeddings 类会为每个文档生成一个嵌入向量，该向量是文档内容的数值表示。这些嵌入向量可用于各种自然语言处理任务，例如文档相似度比较或文本分类。

```python
embeddings = embedder.embed_documents(texts)
for i, embedding in enumerate(embeddings):
    print(f"Embedding for document {i + 1}: {embedding}")
```

生成并打印单条文本的嵌入向量。您还可以为单条文本（例如搜索查询）生成嵌入向量。这在信息检索等任务中非常有用，您可以借此查找与给定查询相似的文档。

```python
query = "Cancer is caused by smoking"
query_embedding = embedder.embed_query(query)
print(f"Embedding for query: {query_embedding}")
```
