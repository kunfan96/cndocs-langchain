>[Breebs](https://www.breebs.com/) 是一个开放的协作知识平台。
>任何人都可以基于存储在 Google Drive 文件夹中的 PDF 创建一个 `Breeb`（知识胶囊）。
>`Breeb` 可被任何 LLM/聊天机器人使用，以提升其专业能力、减少幻觉并提供信息来源访问。
>在底层，`Breebs` 实现了多种`检索增强生成（RAG）`模型，
> 在每次迭代中无缝提供有用的上下文。

## 检索器

```python
from langchain_classic.retrievers import BreebsRetriever
```
