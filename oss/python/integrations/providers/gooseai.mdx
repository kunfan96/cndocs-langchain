>[GooseAI](https://goose.ai) 使部署 NLP 服务变得更加简单和普及。
> `GooseAI` 是一项通过 API 交付的全托管推理服务。
> 在与其它知名 API 功能对等的基础上，`GooseAI` 提供了一种即插即用的解决方案，
> 仅需修改代码中的 2 行，即可实现开源语言模型的部署，并享有业界最优的成本效益。

## 安装与设置

- 使用 `pip install openai` 安装 Python SDK
- 从 [GooseAI 官网](https://goose.ai/) 获取您的 GooseAI API 密钥。
- 设置环境变量（`GOOSEAI_API_KEY`）。

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```


## 大语言模型 (LLMs)

查看 [使用示例](/oss/python/integrations/llms/gooseai)。

```python
from langchain_community.llms import GooseAI
```
