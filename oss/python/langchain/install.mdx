---
title: 安装 LangChain
sidebarTitle: 安装
---

要安装 LangChain 包：

<CodeGroup>
    ```bash pip
    pip install -U langchain
    # 需要 Python 3.10+
    ```

    ```bash uv
    uv add langchain
    # 需要 Python 3.10+
    ```

</CodeGroup>

LangChain 提供了对数百个 LLM 和数千个其他集成的支持。这些集成存在于独立的提供程序包中。

<CodeGroup>
    ```bash pip
    # 安装 OpenAI 集成
    pip install -U langchain-openai

    # 安装 Anthropic 集成
    pip install -U langchain-anthropic
    ```
    ```bash uv
    # 安装 OpenAI 集成
    uv add langchain-openai

    # 安装 Anthropic 集成
    uv add langchain-anthropic
    ```

</CodeGroup>

<Tip>
  查看 [集成标签](/oss/python/integrations/providers/overview)
  以获取可用集成的完整列表。
</Tip>

现在您已安装 LangChain，可以通过 [快速入门指南](/oss/python/langchain/quickstart) 开始使用。
