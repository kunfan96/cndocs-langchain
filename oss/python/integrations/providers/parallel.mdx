## 安装与配置

`Parallel` 集成位于其独立的[合作伙伴包](https://pypi.org/project/langchain-parallel/)中。您可以使用以下命令进行安装：

<CodeGroup>
    ```bash pip
    pip install -qU langchain-parallel
    ```
    ```bash uv
    uv add langchain-parallel
    ```
</CodeGroup>

要使用该包，您还需将 `PARALLEL_API_KEY` 环境变量设置为您的 Parallel API 密钥。您可以通过[在 Parallel 注册](https://platform.parallel.ai)来获取 API 密钥。
