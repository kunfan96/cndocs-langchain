>[Airbyte](https://github.com/airbytehq/airbyte) 是一个面向 ELT 管道的数据集成平台，支持将数据从 API、
> 数据库和文件传输至数据仓库和数据湖。它拥有最丰富的面向数据仓库和数据库的 ELT 连接器目录。

## 安装与设置

<CodeGroup>
```bash pip
pip install -U langchain-airbyte
```

```bash uv
uv add langchain-airbyte
```
</CodeGroup>

<Note>
**目前，`langchain-airbyte` 库尚不支持 Pydantic v2。**

请降级至 Pydantic v1 以使用此包。

此外，此包目前还需要 Python 3.10 或更高版本。

</Note>

该集成包不需要设置任何全局环境变量，但某些集成（例如 `source-github`）可能需要传入凭据。
