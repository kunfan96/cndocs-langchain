Oracle AI 数据库专为 AI 工作负载构建，您可以按**语义（含义）**查询数据，而不仅仅是关键字。它在同一系统中结合了**对非结构化内容的语义搜索**和**对业务数据的关系过滤**——因此您可以构建 AI 检索工作流（如 RAG），无需引入单独的向量数据库，也不会造成多平台数据碎片化。

### 为什么重要

- **AI 与业务数据的统一系统：** 向量和运营数据存储在一起。
- **减少碎片化：** 比单独的向量存储移动部件更少。
- **数据库级保证：** 对同一 AI 工作负载应用安全性、事务、扩展性和可用性。

## 先决条件

安装 `langchain-oracledb`。`python-oracledb` 驱动程序将作为依赖项自动安装。

<CodeGroup>
```bash pip
pip install -qU langchain-oracledb
```
```bash uv
uv add langchain-oracledb
```
</CodeGroup>

## 文档加载器

请查看 [使用示例](/oss/python/integrations/document_loaders/oracleai)。

```python
from langchain_oracledb.document_loaders.oracleai import OracleDocLoader
```
