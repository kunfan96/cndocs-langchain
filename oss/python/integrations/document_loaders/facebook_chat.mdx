>[Messenger](https://en.wikipedia.org/wiki/Messenger_(software)) 是由 `Meta Platforms` 开发的一款美国专有即时通讯应用和平台。该应用最初于 2008 年以 `Facebook Chat` 的名称推出，公司于 2010 年对其消息服务进行了全面改版。

本笔记本介绍如何将 [Facebook Chats](https://www.facebook.com/business/help/1646890868956360) 中的数据加载为可导入 LangChain 的格式。

```python
# pip install pandas
```

```python
from langchain_community.document_loaders import FacebookChatLoader
```

```python
loader = FacebookChatLoader("example_data/facebook_chat.json")
```

```python
loader.load()
```
