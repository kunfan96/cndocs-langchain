<Note>
    尽管我们会偶尔分享集成信息，但我们更注重提供有意义的见解和最佳实践内容。我们的主要社交渠道是[Twitter](https://x.com/LangChain) 和 [LinkedIn](https://www.linkedin.com/company/langchain/)，在那里我们展示最好的示例。
</Note>

### 我们乐于推广的内容

<AccordionGroup>
    <Accordion title="教育性内容" icon="school">
        展示教育性内容的博客、YouTube 视频和其他媒体。请注意，我们更倾向于那些不以“这里是如何使用 XYZ 集成”为框架的内容，而是以“这里是如何实现 ABC”的形式呈现，因为这对我们来说更具教育意义和帮助。
    </Accordion>

    <Accordion title="端到端应用程序" icon="cube">
        端到端的应用程序是开发者构建时很好的资源。我们更倾向于展示那些更为复杂/主动性质的应用，并且使用 [LangGraph](https://github.com/langchain-ai/langgraph) 作为编排框架。我们特别兴奋于涉及以下内容的任何项目：

        - 长期记忆系统
        - 人类在环交互模式
        - 多代理架构
    </Accordion>

    <Accordion title="研究" icon="flask">
        我们热爱展示新颖的研究！无论是基于 LangChain 的研究还是与之集成的研究。
    </Accordion>
</AccordionGroup>
