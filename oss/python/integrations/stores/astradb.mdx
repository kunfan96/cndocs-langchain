> AI 就绪数据库，基于 `Apache Cassandra®` 构建，并通过易用的 JSON API 便捷提供服务。

## 配置

要创建 `AstraDBByteStore` 字节存储，你需要[创建一个 DataStax 账户](https://www.datastax.com/products/datastax-astra)。

### 凭据

注册后，设置以下凭据：

```python
from getpass import getpass

ASTRA_DB_API_ENDPOINT = getpass("ASTRA_DB_API_ENDPOINT = ")
ASTRA_DB_APPLICATION_TOKEN = getpass("ASTRA_DB_APPLICATION_TOKEN = ")
```

### 安装

LangChain AstraDB 集成位于 `langchain-astradb` 包中：

```python
pip install -qU langchain-astradb
```

## 实例化

现在可以实例化字节存储：

```python
from langchain_astradb import AstraDBByteStore

kv_store = AstraDBByteStore(
    api_endpoint=ASTRA_DB_API_ENDPOINT,
    token=ASTRA_DB_APPLICATION_TOKEN,
    collection_name="my_store",
)
```

## 使用方法

可以使用 `mset` 方法在键下设置数据：

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[b'value1', b'value2']
```

可以使用 `mdelete` 方法删除数据：

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[None, None]
```

你可以在任何使用其他 ByteStore 的地方使用 `AstraDBByteStore`。
