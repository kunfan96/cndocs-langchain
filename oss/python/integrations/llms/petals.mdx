`Petals` 在家庭设备中以 BitTorrent 模式运行 1000 亿参数以上的语言模型。

本笔记本将介绍如何将 LangChain 与 [Petals](https://github.com/bigscience-workshop/petals) 配合使用。

## 安装 petals

使用 Petals API 需要安装 `petals` 包。请使用 `pip3 install petals` 安装 `petals`。

Apple Silicon（M1/M2）用户请按照此指南 [https://github.com/bigscience-workshop/petals/issues/147#issuecomment-1365379642](https://github.com/bigscience-workshop/petals/issues/147#issuecomment-1365379642) 安装 petals

```python
!pip3 install petals
```

## 导入模块

```python
import os

from langchain_classic.chains import LLMChain
from langchain_community.llms import Petals
from langchain_core.prompts import PromptTemplate
```

## 设置环境变量 API Key

请确保已从 Hugging Face 获取 [你的 API Key](https://huggingface.co/docs/api-inference/quicktour#get-your-api-token)。

```python
from getpass import getpass

HUGGINGFACE_API_KEY = getpass()
```


```python
os.environ["HUGGINGFACE_API_KEY"] = HUGGINGFACE_API_KEY
```

## 创建 petals 实例

你可以指定不同的参数，例如模型名称、最大生成令牌数、温度等。

```python
# 下载大文件可能需要几分钟时间！

llm = Petals(model_name="bigscience/bloom-petals")
```

```text

```

## 创建 Prompt 模板

我们将为问答任务创建一个 prompt 模板。

```python
template = """Question: {question}

Answer: Let's think step by step."""

prompt = PromptTemplate.from_template(template)
```

## 初始化 LLMChain

```python
llm_chain = LLMChain(prompt=prompt, llm=llm)
```

## 运行 LLMChain

提供一个问题并运行 LLMChain。

```python
question = "What NFL team won the Super Bowl in the year Justin Beiber was born?"

llm_chain.run(question)
```
