[Dria](https://dria.co/) 检索器允许代理在全面的知识中心执行基于文本的搜索。

## 设置

要使用 Dria 检索器，首先安装 Dria JS 客户端：

```bash npm
npm install dria
```
你需要向检索器提供两样东西：

- **API 密钥**：你可以在创建账户时在[个人资料页面](https://dria.co/)获取。
- **合约 ID**：在查看知识时页面顶部或其 URL 中可访问。
  例如，比特币白皮书上传到 Dria 的地址是 https://dria.co/knowledge/2KxNbEb040GKQ1DSDNDsA-Fsj_BlQIEAlzBNuiapBR0，因此其合约 ID 是 `2KxNbEb040GKQ1DSDNDsA-Fsj_BlQIEAlzBNuiapBR0`。
    合约 ID 可以在实例化时省略，稍后通过 `dria.contractId = "your-contract"` 设置。

Dria 检索器还公开了底层的 [Dria 客户端](https://npmjs.com/package/dria)，请参考 [Dria 文档](https://github.com/firstbatchxyz/dria-js-client?tab=readme-ov-file#usage) 以了解更多关于客户端的信息。

## 用法

<Tip>
请参阅[此部分获取安装 LangChain 包的一般说明](/oss/javascript/langchain/install)。
</Tip>

```bash npm
npm install dria @langchain/community @langchain/core
```

```typescript
import { DriaRetriever } from "@langchain/community/retrievers/dria";

// 上传到 Dria 的 TypeScript Handbook v4.9 的合约
// https://dria.co/knowledge/-B64DjhUtCwBdXSpsRytlRQCu-bie-vSTvTIT8Ap3g0
const contractId = "-B64DjhUtCwBdXSpsRytlRQCu-bie-vSTvTIT8Ap3g0";

const retriever = new DriaRetriever({
  contractId, // 要连接的知识
  apiKey: "DRIA_API_KEY", // 如果未提供，将检查环境变量 `DRIA_API_KEY`
  topK: 15, // 可选：默认值为 10
});

const docs = await retriever.invoke("什么是联合类型？");
console.log(docs);
```
