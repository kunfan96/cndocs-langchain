>[EverNote](https://evernote.com/) 旨在归档和创建笔记，其中可以嵌入照片、音频和保存的网页内容。笔记存储在虚拟的“笔记本”中，并可以被标记、注释、编辑、搜索和导出。

此笔记本展示了如何从磁盘加载 `Evernote` [导出](https://help.evernote.com/hc/en-us/articles/209005557-Export-notes-and-notebooks-as-ENEX-or-HTML) 文件 (.enex)。

导出中的每个笔记都将创建一个文档。

```python
# lxml 和 html2text 是解析 EverNote 笔记所必需的
pip install -qU  lxml
pip install -qU  html2text
```

```python
from langchain_community.document_loaders import EverNoteLoader

# 默认情况下，所有笔记会合并为单个 Document
loader = EverNoteLoader("example_data/testing.enex")
loader.load()
```

```python
[Document(page_content='testing this\n\nwhat happens?\n\nto the world?**Jan - March 2022**', metadata={'source': 'example_data/testing.enex'})]
```

```python
# 为每个笔记返回一个 Document 可能更有用
loader = EverNoteLoader("example_data/testing.enex", load_single_document=False)
loader.load()
```
