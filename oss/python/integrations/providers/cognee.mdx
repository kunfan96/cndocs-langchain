Cognee 实现了可扩展、模块化的 ECL（提取、认知化、加载）流水线，使您能够互联并检索历史对话、文档及音频转录文本，同时降低幻觉率、开发工作量与成本。

Cognee 融合图数据库与向量数据库，以揭示数据中隐藏的关系和新模式。您可以自动建模、加载和检索代表业务领域的实体和对象，分析它们之间的关系，从而发现仅靠向量存储或图存储单独无法提供的见解。

您可以在 Google Colab <a href="https://colab.research.google.com/drive/1g-Qnx6l_ecHZi0IOw23rg0qC4TYvEvWZ?usp=sharing">Notebook</a> 中尝试使用，或查看 <a href="https://docs.cognee.ai">文档</a>。

如有任何问题，请加入 cognee <a href="https://discord.gg/NQPKmU5CCg">Discord</a> 社区。

您是否看过 cognee 的 <a href="https://github.com/topoteretes/cognee-starter">入门仓库</a>？快去看看吧！


## 安装与配置

<CodeGroup>
```bash pip
pip install langchain-cognee
```

```bash uv
uv add langchain-cognee
```
</CodeGroup>
