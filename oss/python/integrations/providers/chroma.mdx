>[Chroma](https://docs.trychroma.com/getting-started) 是一个用于构建基于嵌入（Embedding）的 AI 应用程序的数据库。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-chroma
```

```bash uv
uv add langchain-chroma
```
</CodeGroup>


## 向量存储

Chroma 向量数据库提供了一个封装器，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

```python
from langchain_chroma import Chroma
```

如需更详细地了解 Chroma 封装器的使用指南，请参阅[此笔记本](/oss/python/integrations/vectorstores/chroma)
