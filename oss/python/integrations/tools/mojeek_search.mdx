以下内容将介绍如何使用 Mojeek Search 获取结果。请访问 [Mojeek 官网](https://www.mojeek.com/services/search/web-search-api/) 获取 API 密钥。

```python
from langchain_community.tools import MojeekSearch
```

```python
api_key = "KEY"  # obtained from Mojeek Website
```

```python
search = MojeekSearch.config(api_key=api_key, search_kwargs={"t": 10})
```

在 `search_kwargs` 中，您可以添加 [Mojeek 文档](https://www.mojeek.com/support/api/search/request_parameters.html) 中列出的任何搜索参数。

```python
search.run("mojeek")
```
