---
title: "Layerup 安全集成"
description: "使用 LangChain Python 与 Layerup Security LLM 进行集成。"
---

[Layerup Security](https://uselayerup.com) 集成允许你保护对任何 LangChain LLM、LLM 链或 LLM 代理的调用。该 LLM 对象会包装现有的任意 LLM 对象，从而在你的用户和你的 LLM 之间提供一个安全层。

尽管 Layerup Security 对象被设计为 LLM，但它本身并不是一个 LLM，它仅是对 LLM 进行包装，使其能够适配底层 LLM 的相同功能。

## 设置
首先，你需要从 Layerup [网站](https://uselayerup.com) 获取一个 Layerup Security 账户。

接下来，通过 [控制面板](https://dashboard.uselayerup.com) 创建一个项目，并复制你的 API 密钥。我们建议将 API 密钥保存在项目的运行环境中。

安装 Layerup Security SDK：
<CodeGroup>
```bash pip
pip install LayerupSecurity
```

```bash uv
uv add LayerupSecurity
```
</CodeGroup>

并安装 LangChain Community：
<CodeGroup>
```bash pip
pip install langchain-community
```

```bash uv
uv add langchain-community
```
</CodeGroup>

现在，你就可以开始使用 Layerup Security 来保护你的 LLM 调用了！

```python
from langchain_community.llms.layerup_security import LayerupSecurity
from langchain_openai import OpenAI

# 创建你常用的 LLM 实例
openai = OpenAI(
    model_name="gpt-3.5-turbo",
    openai_api_key="OPENAI_API_KEY",
)

# 配置 Layerup Security
layerup_security = LayerupSecurity(
    # 指定 Layerup Security 将要包装的 LLM
    llm=openai,

    # Layerup API 密钥，来自 Layerup 控制面板
    layerup_api_key="LAYERUP_API_KEY",

    # 自定义基础 URL（如果是自托管部署）
    layerup_api_base_url="https://api.uselayerup.com/v1",

    # 在调用 LLM 之前，需在提示词上执行的护栏规则列表
    prompt_guardrails=[],

    # 在 LLM 响应上执行的护栏规则列表
    response_guardrails=["layerup.hallucination"],

    # 是否在将提示词发送至 LLM 前对其中的 PII（个人身份信息）及敏感数据进行掩码处理
    mask=False,

    # 用于滥用行为追踪、客户追踪和范围追踪的元数据。
    metadata={"customer": "example@uselayerup.com"},

    # 提示词护栏违规事件的处理程序
    handle_prompt_guardrail_violation=(
        lambda violation: {
            "role": "assistant",
            "content": (
                "There was sensitive data! I cannot respond. "
                "Here's a dynamic canned response. Current date: {}"
            ).format(datetime.now())
        }
        if violation["offending_guardrail"] == "layerup.sensitive_data"
        else None
    ),

    # 响应护栏违规事件的处理程序
    handle_response_guardrail_violation=(
        lambda violation: {
            "role": "assistant",
            "content": (
                "Custom canned response with dynamic data! "
                "The violation rule was {}."
            ).format(violation["offending_guardrail"])
        }
    ),
)

response = layerup_security.invoke(
    "Summarize this message: my name is Bob Dylan. My SSN is 123-45-6789."
)
```
