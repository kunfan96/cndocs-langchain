本页介绍如何在 LangChain 中使用 Hazy Research 生态系统。
内容分为两部分：安装与设置，以及针对特定 Hazy Research 包装器的参考文档。

## 安装与设置
- 要使用 `manifest`，请通过 `pip install manifest-ml` 进行安装

## 包装器

### LLM

存在一个基于 Hazy Research 的 `manifest` 库构建的 LLM 包装器。
`manifest` 是一个 Python 库，其本身封装了多个模型提供商，并提供了缓存、历史记录等附加功能。

要使用此包装器：
```python
from langchain_community.llms.manifest import ManifestWrapper
```
