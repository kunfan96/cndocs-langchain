[Nia](https://trynia.ai) 是一项搜索与索引 API，能够持续提供来自文档、研究论文、数据集、代码库等内容的上下文信息——确保智能体从不依赖过时数据。具备可扩展性、成本降低 5 倍且可靠。

## 安装与配置

<CodeGroup>
```bash pip
pip install langchain-nia
```

```bash uv
uv add langchain-nia
```
</CodeGroup>

设置你的 API 密钥：

```bash
export NIA_API_KEY="nk_..."
```

## 工具

`langchain-nia` 包提供了 20 个工具，并整合为一个可配置的工具包。

有关完整详情、使用示例及所有可用工具的完整列表，请参阅 [Nia Toolkit 集成指南](/oss/python/integrations/tools/nia)。

```python
from langchain_nia import NiaToolkit

toolkit = NiaToolkit()
tools = toolkit.get_tools()  # 20 个工具
```
