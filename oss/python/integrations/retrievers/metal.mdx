>[Metal](https://github.com/getmetal/metal-python) 是一个用于机器学习嵌入的托管服务。

本 notebook 展示如何使用 [Metal](https://docs.getmetal.io/introduction) 的检索器。

首先，你需要注册 Metal 并获取 API 密钥。可以在[这里](https://docs.getmetal.io/misc-create-app)完成注册。

```python
pip install -qU  metal_sdk
```

```python
from metal_sdk.metal import Metal

API_KEY = ""
CLIENT_ID = ""
INDEX_ID = ""

metal = Metal(API_KEY, CLIENT_ID, INDEX_ID)
```

## 导入文档

如果你尚未设置索引，只需执行此步骤

```python
metal.index({"text": "foo1"})
metal.index({"text": "foo"})
```

```text
{'data': {'id': '642739aa7559b026b4430e42',
  'text': 'foo',
  'createdAt': '2023-03-31T19:51:06.748Z'}}
```

## 查询

现在索引已设置好，我们可以配置检索器并开始查询。

```python
from langchain_community.retrievers import MetalRetriever
```

```python
retriever = MetalRetriever(metal, params={"limit": 2})
```

```python
retriever.invoke("foo1")
```
