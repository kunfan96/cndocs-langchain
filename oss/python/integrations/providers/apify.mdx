>[Apify](https://apify.com) 是一个用于网页抓取和数据提取的云平台，
>它提供了一个包含上千个名为 *Actors* 的现成应用的[生态系统](https://apify.com/store)，适用于各种抓取、爬取和提取用例。

## 概述

Apify 提供了对数千种预构建工具（Actors）的访问权限，用于网页抓取、数据提取和自动化。
该平台负责基础设施管理，使您能够专注于数据提取逻辑。

## 何时使用 Apify

- **可访问数千种预构建的 Actors**，适用于各种平台（社交媒体、电子商务、搜索引擎等）
- **自定义网页抓取和自动化工作流**，超越简单的搜索功能
- **灵活的 Actor 生态系统** – 运行 Apify Store 中的任何 Actor

此集成允许您在 `Apify` 平台上运行 Actors，并将其结果加载到 LangChain 中，从而将来自网络的文档和数据填充到您的向量索引中，例如从带有文档、博客或知识库的网站生成答案。

## 安装与设置

- 使用以下命令安装 LangChain Apify Python 包：

<CodeGroup>
```bash pip
pip install langchain-apify
```

```bash uv
uv add langchain-apify
```
</CodeGroup>

- 获取您的 [Apify API 令牌](https://console.apify.com/account/integrations)，并将其设置为环境变量（`APIFY_TOKEN`），或在构造函数中将其作为 `apify_api_token` 传入。

## 工具

您可以使用 `ApifyActorsTool` 在智能体中使用 Apify Actors。

```python
from langchain_apify import ApifyActorsTool
```

请参阅[此笔记本](/oss/python/integrations/tools/apify_actors)以获取示例用法，以及[Apify LangGraph 智能体 Actor 模板](https://apify.com/templates/python-langgraph)中包含 LangGraph 的工具调用智能体的完整示例。

有关如何使用此工具的更多信息，请访问[Apify 集成文档](https://docs.apify.com/platform/integrations/langgraph)。

## 包装器

您可以使用 `ApifyWrapper` 在 Apify 平台上运行 Actors。

```python
from langchain_apify import ApifyWrapper
```

有关如何使用此包装器的更多信息，请参阅[Apify 集成文档](https://docs.apify.com/platform/integrations/langchain)。

## 应用场景

- **网页抓取**：从网站、社交媒体、电子商务站点提取数据
- **搜索引擎结果**：抓取 Google、Bing 及其他搜索引擎的结果
- **数据收集**：收集结构化数据以用于分析和机器学习流水线
- **内容聚合**：从多个来源收集内容以用于 RAG 应用

## 文档加载器

您还可以使用我们的 `ApifyDatasetLoader` 从 Apify 数据集中获取数据。

```python
from langchain_apify import ApifyDatasetLoader
```

有关此加载器的更详细教程，请参阅[此笔记本](/oss/python/integrations/document_loaders/apify_dataset)。

## 定价

Apify 采用按使用量付费或按事件付费的定价模式，并提供免费套餐。具体价格因 Actor 而异：

- 部分 Actors 免费（您仅需支付平台计算单元费用）
- 其他则按结果或事件收费
- **按事件付费（PPE）定价**：许多 Actor 支持[PPE 定价](https://docs.apify.com/platform/actors/publishing/monetize/pay-per-event)，当您在智能体部署中希望获得可预测的、基于使用量的成本时非常有用
- 详情请参阅[Apify 定价](https://apify.com/pricing)

此集成的源代码可在[LangChain Apify 仓库](https://github.com/apify/langchain-apify)中找到。

## MCP 服务器

不确定该使用哪个 Actor 或它需要哪些参数？
Apify 提供了一个[MCP（模型上下文协议）服务器](https://mcp.apify.com)，帮助您发现可用的 Actors、探索其输入模式并了解参数要求。

通过 HTTP 连接到 Apify MCP 服务器时，请在请求头中包含您的 Apify 令牌：

```text
Authorization: Bearer <APIFY_TOKEN>
```
