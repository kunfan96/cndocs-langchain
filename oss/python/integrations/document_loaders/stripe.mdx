>[Stripe](https://stripe.com/en-ca) 是一家爱尔兰裔美国金融服务和软件即服务（SaaS）公司，为电子商务网站和移动应用程序提供支付处理软件及应用程序编程接口。

本笔记本介绍如何将 `Stripe REST API` 中的数据加载为 LangChain 可用的格式，并提供了向量化使用的示例。

```python
from langchain.indexes import VectorstoreIndexCreator
from langchain_community.document_loaders import StripeLoader
```

Stripe API 需要一个访问令牌，可在 Stripe 控制台中找到。

该文档加载器还需要一个 `resource` 选项，用于指定要加载的数据类型。

以下资源可用：

`balance_transations` [文档](https://stripe.com/docs/api/balance_transactions/list)

`charges` [文档](https://stripe.com/docs/api/charges/list)

`customers` [文档](https://stripe.com/docs/api/customers/list)

`events` [文档](https://stripe.com/docs/api/events/list)

`refunds` [文档](https://stripe.com/docs/api/refunds/list)

`disputes` [文档](https://stripe.com/docs/api/disputes/list)

```python
stripe_loader = StripeLoader("charges")
```

```python
# 从加载器创建向量存储检索器
# 详情请参阅 https://python.langchain.com/en/latest/modules/data_connection/getting_started.html

index = VectorstoreIndexCreator().from_loaders([stripe_loader])
stripe_doc_retriever = index.vectorstore.as_retriever()
```
