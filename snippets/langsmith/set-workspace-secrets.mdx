在 [LangSmith UI](https://smith.langchain.com) 中，确保您的 API 密钥已设置为[工作区密钥](/langsmith/set-up-hierarchy#configure-workspace-settings)。

1. 导航到 <Icon icon="settings" /> **设置**，然后转到 **密钥** 选项卡。
1. 选择 **添加密钥**，输入密钥环境变量（例如，`OPENAI_API_KEY` 或 `ANTHROPIC_API_KEY`）和您的 API 密钥作为**值**。
1. 选择 **保存密钥**。

<Note> 在 LangSmith UI 中添加工作区密钥时，请确保密钥名称与您的模型提供商预期的环境变量名称匹配。</Note>
