>[DeepEval](https://confident-ai.com) 包用于 LLM 单元测试。
> 使用 Confident，每个人都可以通过更快的迭代构建健壮的语言模型
> 使用单元测试和集成测试。我们为迭代中的每个步骤提供支持
> 从合成数据创建到测试。

在本指南中，我们将演示如何测试和衡量 LLM 性能。我们将展示如何使用我们的 Callback 来衡量性能，以及如何定义您自己的指标并将它们记录到我们的仪表板中。

DeepEval 还提供：

- 如何生成合成数据
- 如何衡量性能
- 一个用于随时间监控和审查结果的仪表板

## 安装和设置

```python
pip install -qU  langchain langchain-openai langchain-community deepeval langchain-chroma
```

### 获取 API 凭证

要获取 DeepEval API 凭证，请遵循以下步骤：

1. 前往 [app.confident-ai.com](https://app.confident-ai.com)
2. 点击 "组织"
3. 复制 API Key。

当您登录时，您还将被要求设置 `implementation` 名称。实现名称用于描述实现的类型。（想想您想要如何称呼您的项目。我们建议使其具有描述性。）

```python
!deepeval login
```

### 设置 DeepEval

默认情况下，您可以使用 `DeepEvalCallbackHandler` 来设置您想要跟踪的指标。然而，目前它对指标的支持有限（很快将添加更多）。它当前支持：

- [答案相关性](https://docs.confident-ai.com/docs/measuring_llm_performance/answer_relevancy)
- [偏见](https://docs.confident-ai.com/docs/measuring_llm_performance/debias)
- [毒性](https://docs.confident-ai.com/docs/measuring_llm_performance/non_toxic)

```python
from deepeval.metrics.answer_relevancy import AnswerRelevancy

# 这里我们要确保答案最小程度相关
answer_relevancy_metric = AnswerRelevancy(minimum_score=0.5)
```

## 开始使用

要使用 `DeepEvalCallbackHandler`，我们需要 `implementation_name`。

```python
from langchain_community.callbacks.confident_callback import DeepEvalCallbackHandler

deepeval_callback = DeepEvalCallbackHandler(
    implementation_name="langchainQuickstart", metrics=[answer_relevancy_metric]
)
```

### 场景 1：输入到 LLM

然后您可以将其与 OpenAI 一起输入到您的 LLM 中。

```python
from langchain_openai import OpenAI

llm = OpenAI(
    temperature=0,
    callbacks=[deepeval_callback],
    verbose=True,
    openai_api_key="<YOUR_API_KEY>",
)
output = llm.generate(
    [
        "What is the best evaluation tool out there? (no bias at all)",
    ]
)
```

### 场景 2：在没有 callback 的链中跟踪 LLM

要在没有 callback 的情况下跟踪链中的 LLM，您可以在末尾接入它。

我们可以首先定义一个简单的链，如下所示。

```python
import requests
from langchain_classic.chains import RetrievalQA
from langchain_chroma import Chroma
from langchain_community.document_loaders import TextLoader
from langchain_openai import OpenAI, OpenAIEmbeddings
from langchain_text_splitters import CharacterTextSplitter

text_file_url = "https://raw.githubusercontent.com/hwchase17/chat-your-data/master/state_of_the_union.txt"

openai_api_key = "sk-XXX"

with open("state_of_the_union.txt", "w") as f:
    response = requests.get(text_file_url)
    f.write(response.text)

loader = TextLoader("state_of_the_union.txt")
documents = loader.load()
text_splitter = CharacterTextSplitter(chunk_size=1000, chunk_overlap=0)
texts = text_splitter.split_documents(documents)

embeddings = OpenAIEmbeddings(openai_api_key=openai_api_key)
docsearch = Chroma.from_documents(texts, embeddings)

qa = RetrievalQA.from_chain_type(
    llm=OpenAI(openai_api_key=openai_api_key),
    chain_type="stuff",
    retriever=docsearch.as_retriever(),
)

# 提供一个新的问答管道
query = "Who is the president?"
result = qa.run(query)
```

定义链之后，您可以手动检查答案相似性。

```python
answer_relevancy_metric.measure(result, query)
answer_relevancy_metric.is_successful()
```
