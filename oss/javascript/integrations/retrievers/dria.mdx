[Dria](https://dria.co/) 检索器允许代理在综合知识中心执行基于文本的搜索。

## 设置

要使用 Dria 检索器，首先安装 Dria JS 客户端：

```bash npm
npm install dria
```
你需要向检索器提供两件事：

- **API 密钥**：创建帐户后，你可以在[个人资料页面](https://dria.co/)获取你的密钥。
- **合约 ID**：在查看知识时可在页面顶部或其 URL 中访问。
  例如，比特币白皮书已上传到 Dria，网址为 https://dria.co/knowledge/2KxNbEb040GKQ1DSDNDsA-Fsj_BlQIEAlzBNuiapBR0，因此其合约 ID 为 `2KxNbEb040GKQ1DSDNDsA-Fsj_BlQIEAlzBNuiapBR0`。
    在实例化期间可以省略合约 ID，稍后可以通过 `dria.contractId = "your-contract"` 进行设置

Dria 检索器也公开了底层的 [Dria 客户端](https://npmjs.com/package/dria)，请参阅 [Dria 文档](https://github.com/firstbatchxyz/dria-js-client?tab=readme-ov-file#usage)以了解有关客户端的更多信息。

## 用法

<Tip>
请参阅 [此部分以获取有关安装 LangChain 包的一般说明](/oss/javascript/langchain/install)。
</Tip>

```bash npm
npm install dria @langchain/community @langchain/core
```

```typescript
import { DriaRetriever } from "@langchain/community/retrievers/dria";

// 上传到 Dria 的 TypeScript Handbook v4.9 的合约
// https://dria.co/knowledge/-B64DjhUtCwBdXSpsRytlRQCu-bie-vSTvTIT8Ap3g0
const contractId = "-B64DjhUtCwBdXSpsRytlRQCu-bie-vSTvTIT8Ap3g0";

const retriever = new DriaRetriever({
  contractId, // 要连接的知识
  apiKey: "DRIA_API_KEY", // 如果未提供，将检查 env 中的 `DRIA_API_KEY`
  topK: 15, // 可选：默认值为 10
});

const docs = await retriever.invoke("What is a union type?");
console.log(docs);
```
