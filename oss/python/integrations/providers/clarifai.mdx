>[Clarifai](https://clarifai.com) 是成立于 2013 年的首批深度学习平台之一。Clarifai 提供涵盖图像、视频、文本和音频数据的数据探索、数据标注、模型训练、评估与推理的完整 AI 生命周期 AI 平台。在 LangChain 生态系统中，据我们所知，Clarifai 是唯一在一个生产级平台上同时支持 LLMs、Embeddings 和 Vector Store 的提供商，使其成为将您的 LangChain 实现投入生产环境的绝佳选择。
>
> `Clarifai` 为众多不同的用例提供了数千种 AI 模型。您可以[在此处浏览](https://clarifai.com/explore)以找到最适合您用例的模型。这些模型包括由其他提供商（如 OpenAI、Anthropic、Cohere、AI21 等）创建的模型，以及来自开源领域的最先进技术（如 Falcon、InstructorXL 等），助您将顶尖的 AI 能力融入您的产品中。您可以按创建者的 `user_id` 以及我们称之为“应用程序”（Applications）的项目（由其 `app_id` 标识）来查找这些模型。除了 `model_id` 和可选的 `version_id` 之外，您还需要这些 ID，因此一旦找到最适合您用例的最佳模型，请务必记录下所有这些 ID！
>
>此外请注意，鉴于有许多用于图像、视频、文本和音频理解的模型，您可以构建一些有趣的 AI Agent，利用多样化的 AI 模型作为专家来理解这些数据类型。


## 安装与配置
- 安装 Python SDK：
<CodeGroup>
```bash pip
pip install clarifai
```

```bash uv
uv add clarifai
```
</CodeGroup>
注册 [Clarifai 账号](https://clarifai.com/signup)，然后从您的[安全设置](https://clarifai.com/settings/security)获取个人访问令牌以访问 Clarifai API，并将其设置为环境变量（`CLARIFAI_PAT`）。


## LLMs

要在 Clarifai 平台上查看可用的 LLMs 列表，您可以选择[文本到文本模型类型](https://clarifai.com/explore/models?filterData=%5B%7B%22field%22%3A%22model_type_id%22%2C%22value%22%3A%5B%22text-to-text%22%5D%7D%5D&page=1&perPage=24)。

```python
from langchain_community.llms import Clarifai
llm = Clarifai(pat=CLARIFAI_PAT, user_id=USER_ID, app_id=APP_ID, model_id=MODEL_ID)
```

更多详情，请参阅 Clarifai LLM Wrapper 文档中的[详细指南](/oss/python/integrations/llms/clarifai)。


## Embedding models

要在 Clarifai 平台上查看可用的 Embedding 模型列表，您可以选择[文本到 Embedding 模型类型](https://clarifai.com/explore/models?page=1&perPage=24&filterData=%5B%7B%22field%22%3A%22model_type_id%22%2C%22value%22%3A%5B%22text-embedder%22%5D%7D%5D)。

LangChain 中提供了 Clarifai Embedding 模型，您可以通过以下方式访问：
```python
from langchain_community.embeddings import ClarifaiEmbeddings
embeddings = ClarifaiEmbeddings(pat=CLARIFAI_PAT, user_id=USER_ID, app_id=APP_ID, model_id=MODEL_ID)
```

请参阅[使用示例](/oss/python/integrations/document_loaders/couchbase)。


## Vectorstore

Clarifai 的 Vector DB 于 2016 年推出，并经过优化以支持实时搜索查询。借助 Clarifai 平台的 Workflows，您的数据会自动由 Embedding 模型建立索引，并可选择性地由其他模型辅助索引，从而在数据库中建立索引以供搜索。您不仅可以基于向量查询数据库，还可以根据元数据匹配、其他 AI 预测的概念进行过滤，甚至执行地理坐标搜索。只需创建一个应用程序，为您的数据类型选择合适的基础 Workflow，然后上传数据即可（可通过 API 上传，[此处有文档说明](https://docs.clarifai.com/api-guide/data/create-get-update-delete)，或通过 clarifai.com 上的 UI 上传）。

您也可以直接从 LangChain 添加数据，系统将为您自动完成索引过程。您会发现这与其他的 Vector Store 略有不同，在其他 Vector Store 中，您需要在构造函数中提供 Embedding 模型，并由 LangChain 协调从文本获取 Embedding 并将其写入索引的过程。这不仅更加便捷，而且使用 Clarifai 的分布式云在后台完成所有索引操作也具备更高的可扩展性。

```python
from langchain_community.vectorstores import Clarifai
clarifai_vector_db = Clarifai.from_texts(user_id=USER_ID, app_id=APP_ID, texts=texts, pat=CLARIFAI_PAT, number_of_docs=NUMBER_OF_DOCS, metadatas = metadatas)
```
更多详情，请参阅 Clarifai Vector Store 文档中的[详细指南](/oss/python/integrations/vectorstores/clarifai)。
