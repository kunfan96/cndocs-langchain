[JSONFormer](https://github.com/1rgs/jsonformer) 是一个库，用于包装本地 Hugging Face pipeline 模型，以对 JSON Schema 的子集进行结构化解码。

它的工作原理是填充结构 token，然后从模型中采样内容 token。

**警告 - 此模块仍处于实验阶段**

```python
pip install -qU  jsonformer > /dev/null
```

### Hugging Face 基线

首先，让我们通过检查未进行结构化解码的模型输出来建立定性基线。

```python
import logging

logging.basicConfig(level=logging.ERROR)
```

```python
import json
import os

import requests
from langchain.tools import tool

HF_TOKEN = os.environ.get("HUGGINGFACE_API_KEY")


@tool
def ask_star_coder(query: str, temperature: float = 1.0, max_new_tokens: float = 250):
    """查询 BigCode StarCoder 模型以获取编程问题相关答案。"""
    url = "https://api-inference.huggingface.co/models/bigcode/starcoder"
    headers = {
        "Authorization": f"Bearer {HF_TOKEN}",
        "content-type": "application/json",
    }
    payload = {
        "inputs": f"{query}\n\nAnswer:",
        "temperature": temperature,
        "max_new_tokens": int(max_new_tokens),
    }
    response = requests.post(url, headers=headers, data=json.dumps(payload))
    response.raise_for_status()
    return json.loads(response.content.decode("utf-8"))
```

```python
prompt = """你必须使用 JSON 格式响应，包含单个 action 和单个 action_input。
你可以使用 'ask_star_coder' 寻求编程问题的帮助。

{arg_schema}

示例
----
人类："那么关于 GIL 到底是怎么回事？"
AI 助手：{{
  "action": "ask_star_coder",
```

```python
from langchain_huggingface import HuggingFacePipeline
```

## JSONFormer LLM 封装器

让我们再试一次，这次向模型提供 Action input 的 JSON Schema。

```python
decoder_schema = {
    "title": "Decoding Schema",
    "type": "object",
    "properties": {
        "action": {"type": "string", "default": ask_star_coder.name},
        "action_input": {
            "type": "object",
            "properties": ask_star_coder.args,
        },
    },
}
```

```python
from langchain_experimental.llms import JsonFormer

json_former = JsonFormer(json_schema=decoder_schema, pipeline=hf_model)
```

```python
results = json_former.predict(prompt, stop=["Observation:", "Human:"])
print(results)
```

```json
{"action": "ask_star_coder", "action_input": {"query": "What's the difference between an iterator and an iter", "temperature": 0.0, "max_new_tokens": 50.0}}
```
