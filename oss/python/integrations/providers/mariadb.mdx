本页介绍如何在 LangChain 中使用 [MariaDB](https://github.com/mariadb/) 生态系统。
内容分两个部分：安装与设置，以及对特定 PGVector 封装器的说明。

## 安装
- 安装 c/c 连接器

Debian、Ubuntu 系统：
```bash
sudo apt install libmariadb3 libmariadb-dev
```

CentOS、RHEL、Rocky Linux 系统：
```bash
sudo yum install MariaDB-shared MariaDB-devel
```

- 使用 `pip install mariadb` 安装 Python 连接器包


## 设置
1. 第一步是安装 MariaDB 11.7.1 或更高版本。

    Docker 镜像是最简单的入门方式。

## 封装器

### 向量存储

MariaDB 向量数据库有一个封装器，可将其作为向量存储使用，支持语义搜索或示例选择。

导入该向量存储：
```python
from langchain_mariadb import MariaDBStore
```
