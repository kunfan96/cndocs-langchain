本页介绍如何在 LangChain 中使用 [SerpApi 网络搜索 API](https://serpapi.com/)。
内容分为两部分：安装与设置，以及对特定 SerpAPI 封装器的参考说明。

## 安装与设置
- 使用 `pip install google-search-results` 安装依赖
- 从[此处](https://serpapi.com/manage-api-key)获取 SerpApi API 密钥，并将其设置为环境变量（`SERPAPI_API_KEY`）

## 封装器

### 工具类

存在一个封装此 API 的 SerpAPI 工具类。导入方式：

```python
from langchain_community.utilities import SerpAPIWrapper
```

有关此封装器的更详细演练，请参阅[此 notebook](/oss/python/integrations/tools/serpapi)。

### 工具

您也可以轻松地将此封装器加载为工具（与代理一起使用）。
操作方式：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["serpapi"])
```

有关更多信息，请参阅[此页面](/oss/python/integrations/tools)
