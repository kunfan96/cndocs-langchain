Baichuan Inc.（[www.baichuan-ai.com/](https://www.baichuan-ai.com/)）是一家处于 AGI 时代的中国初创公司，致力于解决人类的基本需求：效率、健康与幸福。

```python
## 安装使用此集成所需的 langchain 包
pip install -qU langchain-community
```

## 前置条件

访问 Baichuan LLM API 需要 API 密钥。请访问 [platform.baichuan-ai.com/](https://platform.baichuan-ai.com/) 获取您的 API 密钥。

## 使用 Baichuan LLM

```python
import os

os.environ["BAICHUAN_API_KEY"] = "YOUR_API_KEY"
```

```python
from langchain_community.llms import BaichuanLLM

# 加载模型
llm = BaichuanLLM()

res = llm.invoke("What's your name?")
print(res)
```

```python
res = llm.generate(prompts=["你好！"])
res
```

```python
for res in llm.stream("Who won the second world war?"):
    print(res)
```

```python
import asyncio


async def run_aio_stream():
    async for res in llm.astream("Write a poem about the sun."):
        print(res)


asyncio.run(run_aio_stream())
```
