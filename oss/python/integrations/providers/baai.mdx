>[北京智源人工智能研究院（BAAI）（维基百科）](https://en.wikipedia.org/wiki/Beijing_Academy_of_Artificial_Intelligence),
> 又称 `智源研究院`，是一家中国非营利性的
> 人工智能（AI）研究实验室。`BAAI` 开展 AI 研究
> 致力于促进学术界与产业界的合作，
> 以及培养顶尖人才，并专注于对 AI 技术基础
> 的长期研究。作为协作枢纽，BAAI 的创始
> 成员包括领先的 AI 公司、大学和研究机构。


## 嵌入模型

### HuggingFaceBgeEmbeddings

>[HuggingFace 上的 BGE 模型](https://huggingface.co/BAAI/bge-large-en-v1.5)
> 是 [最佳开源嵌入模型之一](https://huggingface.co/spaces/mteb/leaderboard)。

查看 [使用示例](/oss/python/integrations/embeddings/bge_huggingface)。

```python
from langchain_community.embeddings import HuggingFaceBgeEmbeddings
```

### IpexLLMBgeEmbeddings

>[IPEX-LLM](https://github.com/intel-analytics/ipex-llm) 是一个 PyTorch
> 库，用于在 Intel CPU 和 GPU 上运行 LLM（例如，配备集成显卡的本地 PC，
> 或 Arc、Flex 和 Max 等独立显卡），可实现极低的延迟。

查看 [在 Intel CPU 上运行模型的使用示例](/oss/python/integrations/embeddings/ipex_llm)。
查看 [在 Intel GPU 上运行模型的使用示例](/oss/python/integrations/embeddings/ipex_llm_gpu)。

```python
from langchain_community.embeddings import IpexLLMBgeEmbeddings
```

### QuantizedBgeEmbeddings

查看 [使用示例](/oss/python/integrations/embeddings/itrex)。

```python
from langchain_community.embeddings import QuantizedBgeEmbeddings
```
