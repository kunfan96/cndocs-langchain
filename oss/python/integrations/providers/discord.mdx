>[Discord](https://discord.com/) 是一款 VoIP 和即时通讯社交平台。用户能够进行通信，
> 方式包括语音通话、视频通话、文本消息、媒体和文件，这些通信可在私人聊天中进行，也可作为被称为
> “服务器”的社区的一部分。服务器是一组持久的聊天室和语音频道的集合，可通过邀请链接访问。

## 安装与设置

<CodeGroup>
```bash pip
pip install pandas
```

```bash uv
uv add pandas
```
</CodeGroup>

请按照以下步骤下载您的 `Discord` 数据：

1. 进入您的 **用户设置**
2. 然后进入 **隐私与安全**
3. 前往 **请求我的所有数据** 并点击 **请求数据** 按钮

接收您的数据可能需要 30 天时间。您将在注册到 Discord 的邮箱地址收到一封电子邮件。
该邮件将包含一个下载按钮，通过它您可以下载个人的 Discord 数据。
