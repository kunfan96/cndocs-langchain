[GPT4All](https://gpt4all.io/index.html) 是一个免费使用、本地运行、注重隐私的聊天机器人。无需 GPU 或互联网。它包含流行模型及其自有模型，如 GPT4All Falcon、Wizard 等。

本笔记本介绍如何使用 [GPT4All 嵌入](https://docs.gpt4all.io/gpt4all_python_embedding.html#gpt4all.gpt4all.Embed4All) 与 LangChain。

## 安装 GPT4All 的 Python 绑定

```python
pip install -qU  gpt4all > /dev/null
```

```python
from langchain_community.embeddings import GPT4AllEmbeddings
```

```python
gpt4all_embd = GPT4AllEmbeddings()
```

```text
Model downloaded at:  /Users/rlm/.cache/gpt4all/ggml-all-MiniLM-L6-v2-f16.bin
```

```python
text = "This is a test document."
```

## 嵌入文本数据

```python
query_result = gpt4all_embd.embed_query(text)
```

使用 embed_documents 可以嵌入多段文本。您还可以将这些嵌入与 [Nomic 的 Atlas](https://docs.nomic.ai/index.html) 映射，以查看数据的可视化表示。

```python
doc_result = gpt4all_embd.embed_documents([text])
```
