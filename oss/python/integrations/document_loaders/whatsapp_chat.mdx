>[WhatsApp](https://www.whatsapp.com/)（也称为 `WhatsApp Messenger`）是一款免费、跨平台、集中式的即时消息（IM）和互联网语音协议（VoIP）服务。它允许用户发送文本和语音消息、进行语音和视频通话，以及分享图片、文档、用户位置和其他内容。

本笔记本介绍如何将 `WhatsApp Chats` 中的数据加载为可被 LangChain 接收的格式。

```python
from langchain_community.document_loaders import WhatsAppChatLoader
```

```python
loader = WhatsAppChatLoader("example_data/whatsapp_chat.txt")
```

```python
loader.load()
```
