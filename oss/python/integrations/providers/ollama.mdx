本页面涵盖所有 LangChain 与 [Ollama](https://ollama.com/) 的集成。

Ollama 允许您在本地运行开源模型（如 [`gpt-oss`](https://ollama.com/library/gpt-oss)）。

有关支持的模型和变体的完整列表，请参见 [Ollama 模型库](https://ollama.ai/library)。
