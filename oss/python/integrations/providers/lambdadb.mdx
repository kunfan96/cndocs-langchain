title: LambdaDB
---

>[LambdaDB](https://lambdadb.ai/) 是一款面向 RAG 和 Agent 的无服务器 AI 数据库。在灵活的文档模型上统一全文检索、多向量和混合搜索。即时处理无限持久内存和海量并发——成本仅为传统方案的十分之一。

## 安装与设置

安装 Python SDK：

<CodeGroup>
    ```bash pip
    pip install langchain-lambdadb
    ```

    ```bash uv
    uv add langchain-lambdadb
    ```
</CodeGroup>

您还需要一个 LambdaDB 账户，包含：
- 项目 URL
- 项目 API 密钥

在 [lambdadb.ai](https://lambdadb.ai) 获取您的凭据。

## 向量存储

`LambdaDB` 集合有一个封装器，可将其作为向量存储用于语义搜索、RAG 和其他 AI 应用。

导入该向量存储：
```python
from langchain_lambdadb import LambdaDBVectorStore
```

### 快速开始

```python
from langchain_lambdadb import LambdaDBVectorStore
from langchain_openai import OpenAIEmbeddings
from lambdadb import LambdaDB
import os

# Initialize client
client = LambdaDB(
    server_url=os.environ["LAMBDADB_SERVER_URL"],
    project_api_key=os.environ["LAMBDADB_API_KEY"]
)

# Create vector store with existing collection
vector_store = LambdaDBVectorStore(
    client=client,
    collection_name="my_collection",  # Must exist beforehand
    embedding=OpenAIEmbeddings()
)
```

更详细的 LambdaDB 封装器演练，请参阅[完整文档](/oss/python/integrations/vectorstores/lambdadb)。
