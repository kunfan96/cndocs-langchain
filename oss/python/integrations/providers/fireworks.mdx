>[Fireworks AI](https://fireworks.ai) 是一个生成式 AI 推理平台，以业界领先的速度和
> 生产就绪性运行和自定义模型。

## 安装与设置

- 安装 Fireworks 集成包。

  ```bash
  pip install langchain-fireworks
  ```

- 在 [fireworks.ai](https://fireworks.ai) 注册以获取 Fireworks API 密钥。
- 通过设置 FIREWORKS_API_KEY 环境变量进行认证。

### 认证

使用 Fireworks API 密钥进行认证有两种方式：

1.  设置 `FIREWORKS_API_KEY` 环境变量。

    ```python
    os.environ["FIREWORKS_API_KEY"] = "<KEY>"
    ```

2.  在 Fireworks LLM 模块中设置 `api_key` 字段。

    ```python
    llm = Fireworks(api_key="<KEY>")
    ```
