title: 实现 LangChain 集成
sidebarTitle: 实现
---

集成包是用户可以安装在项目中使用的 Python 包。它们实现了一个或多个符合 LangChain 接口标准的组件。

LangChain 组件是 [`langchain-core`](https://github.com/langchain-ai/langchain/tree/master/libs/core) 中基类的子类。示例包括[聊天模型](/oss/python/integrations/chat)、[工具](/oss/python/integrations/tools)、[检索器](/oss/python/integrations/retrievers)等。

您的集成包通常会实现至少一个组件的子类。展开以下选项卡查看每个组件的详细信息。

<Tabs>
    <Tab title="聊天模型">
        聊天模型是 [`BaseChatModel`](https://reference.langchain.com/python/langchain-core/language_models/chat_models/BaseChatModel) 类的子类。它们实现了生成聊天补全、处理消息格式化以及管理模型参数的方法。

        <Warning>
            聊天模型集成指南目前正在完善中。与此同时，请阅读[聊天模型概念指南](/oss/python/langchain/models)，了解 LangChain 聊天模型的工作原理。
        </Warning>
    </Tab>
    <Tab title="工具">
        工具有两种主要使用方式：

        1. 定义"输入模式"或"参数模式"，与文本请求一起传递给聊天模型的工具调用功能，使聊天模型能够生成"工具调用"，即调用该工具所需的参数。
        2. 接收上述生成的"工具调用"，执行某些操作并返回可作为 ToolMessage 传回给聊天模型的响应。

        工具类必须继承自 [`BaseTool`](https://reference.langchain.com/python/langchain-core/tools/base/BaseTool) 基类。该接口有 3 个属性和 2 个方法需要在子类中实现。

        <Warning>
            工具集成指南目前正在完善中。与此同时，请阅读[工具概念指南](/oss/python/langchain/tools)，了解 LangChain 工具的工作原理。
        </Warning>
    </Tab>
    <Tab title="检索器">
        检索器用于根据查询从 API、数据库或其他来源检索文档。检索器类必须继承自 BaseRetriever 基类。

        <Warning>
            检索器集成指南目前正在完善中。与此同时，请阅读[检索器概念指南](/oss/python/integrations/retrievers)，了解 LangChain 检索器的工作原理。
        </Warning>
    </Tab>
    <Tab title="向量存储">
        所有向量存储都必须继承自 [`VectorStore`](https://reference.langchain.com/python/langchain-core/vectorstores/base/VectorStore) 基类。该接口包含用于在向量存储中写入、删除和搜索文档的方法。

        请参阅[向量存储集成指南](/oss/python/integrations/vectorstores)，了解如何实现向量存储集成。

        <Warning>
            向量存储集成指南目前正在完善中。与此同时，请阅读[向量存储概念指南](/oss/python/integrations/vectorstores)，了解 LangChain 向量存储的工作原理。
        </Warning>
    </Tab>
    <Tab title="嵌入模型">
        嵌入模型是 [`Embeddings`](https://reference.langchain.com/python/langchain-core/embeddings/embeddings/Embeddings) 类的子类。

        <Warning>
            嵌入模型集成指南目前正在完善中。与此同时，请阅读[嵌入模型概念指南](/oss/python/integrations/text_embedding)，了解 LangChain 嵌入模型的工作原理。
        </Warning>
    </Tab>
    <Tab title="沙盒">
        沙盒集成使 [Deep Agents](/oss/python/deepagents/overview) 能够在隔离环境中运行代码。

        **参考实现：** 请参阅 [Daytona 合作伙伴集成](https://github.com/langchain-ai/deepagents/tree/main/libs/partners/daytona)，了解结构和模式。
    </Tab>
</Tabs>
