> [Kùzu](https://kuzudb.com/) 是一款可嵌入、可扩展、极速的图数据库。
> 采用 MIT 许可证，您可以在[这里](https://github.com/kuzudb/kuzu)查看其源代码。

> Kùzu 的主要特点：
>- 性能与可扩展性：为图实现了现代先进的连接算法。
>- 易用性：无需服务器（嵌入式架构），极易安装和上手。
>- 互操作性：可方便地扫描和复制来自外部列式格式、CSV、JSON 和关系数据库的数据。
>- 结构化属性图模型：实现属性图模型，并添加了结构化支持。
>- Cypher 支持：允许使用声明式查询语言 Cypher 方便地查询图。

> 访问 Kùzu 的[文档](https://docs.kuzudb.com/)开始使用。


## 安装与设置

按如下方式安装 Python SDK：

<CodeGroup>
```bash pip
pip install -U langchain-kuzu
```

```bash uv
uv add langchain-kuzu
```
</CodeGroup>
