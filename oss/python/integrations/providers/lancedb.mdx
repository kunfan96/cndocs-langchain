本页介绍如何在 LangChain 中使用 [LanceDB](https://github.com/lancedb/lancedb)。
内容分两个部分：安装与设置，以及对特定 LanceDB 封装器的说明。

## 安装与设置

- 使用 `pip install lancedb` 安装 Python SDK

## 封装器

### 向量存储

LanceDB 数据库有一个封装器，可将其作为向量存储使用，支持语义搜索或示例选择。

导入该向量存储：

```python
from langchain_community.vectorstores import LanceDB
```

更详细的 LanceDB 封装器演练，请参阅[此 notebook](/oss/python/integrations/vectorstores/lancedb)
