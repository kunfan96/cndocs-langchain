## 安装与设置

GreenNode 集成可以通过 pip 安装：

```python
pip install -qU langchain-greennode
```

### API 密钥

要使用 GreenNode 无服务器 AI，你需要一个 API 密钥，可以从 [GreenNode 无服务器 AI](https://aiplatform.console.greennode.ai/api-keys) 获取。API 密钥可以作为初始化参数 `api_key` 传递，或设置为环境变量 `GREENNODE_API_KEY`。

```python
import getpass
import os

if not os.getenv("GREENNODE_API_KEY"):
    os.environ["GREENNODE_API_KEY"] = getpass.getpass("Enter your GreenNode API key: ")
```

## 聊天模型

```python
from langchain_greennode import ChatGreenNode

chat = ChatGreenNode(
    model="deepseek-ai/DeepSeek-R1-Distill-Qwen-32B",  # 从可用模型中选择
    temperature=0.6,
    top_p=0.95,
)
```

查看 GreenNode [聊天模型](https://python.langchain.com/docs/integrations/chat/greennode/) 的使用方法

## 嵌入模型

```python
from langchain_greennode import GreenNodeEmbeddings

# 初始化嵌入
embeddings = GreenNodeEmbeddings(
    model="BAAI/bge-m3"  # 从可用模型中选择
)
```

查看 GreenNode [嵌入模型](https://python.langchain.com/docs/integrations/text_embedding/greennode) 的使用方法

## 重排序

```python
from langchain_greennode import GreenNodeRerank

# 初始化重排序器
rerank = GreenNodeRerank(
    model="BAAI/bge-reranker-v2-m3",  # 从可用模型中选择
    top_n=-1,
)
```

查看 GreenNode [重排序模型](https://python.langchain.com/docs/integrations/retrievers/greennode-reranker) 的使用方法
