[ScraperAPI](https://www.scraperapi.com/) 通过其网页抓取 API 支持从任何公开网站采集数据，无需担心代理、浏览器或 CAPTCHA 处理问题。[langchain-scraperapi](https://github.com/scraperapi/langchain-scraperapi) 对该服务进行了封装，使 AI 代理能够轻松浏览网页并从中抓取数据。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-scraperapi
```

```bash uv
uv add langchain-scraperapi
```
</CodeGroup>

从 [ScraperAPI](https://www.scraperapi.com/) 获取 API 密钥并将其设置为环境变量：

```python
import os

os.environ["SCRAPERAPI_API_KEY"] = "your-api-key"
```

## 工具

该包提供 3 种工具，分别用于抓取任意网站、获取 Google 搜索结构化结果以及获取 Amazon 搜索结构化结果。

查看每种工具的详细文档：
- [ScraperAPITool](/oss/python/integrations/tools/scraperapi) - 浏览和抓取任意网站
- ScraperAPIGoogleSearchTool - 获取 Google 搜索 SERP 结构化数据
- ScraperAPIAmazonSearchTool - 获取 Amazon 产品搜索结构化数据

更详细的演练请参阅[官方仓库](https://github.com/scraperapi/langchain-scraperapi)。
