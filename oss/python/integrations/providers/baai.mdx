>[北京人工智能研究院（BAAI）（维基百科）](https://en.wikipedia.org/wiki/Beijing_Academy_of_Artificial_Intelligence)，
> 又称`智源研究院`，是一家中国非营利人工智能研究机构。`BAAI` 开展 AI 研究，
> 致力于促进学术界和产业界之间的合作，培养顶尖人才，并专注于 AI 技术基础的长期研究。
> 作为协作枢纽，BAAI 的创始成员包括国内领先的 AI 公司、高校和研究机构。


## 嵌入模型

### HuggingFaceBgeEmbeddings

>[HuggingFace 上的 BGE 模型](https://huggingface.co/BAAI/bge-large-en-v1.5)
> 是[最优秀的开源嵌入模型之一](https://huggingface.co/spaces/mteb/leaderboard)。

查看[使用示例](/oss/python/integrations/text_embedding/bge_huggingface)。

```python
from langchain_community.embeddings import HuggingFaceBgeEmbeddings
```

### IpexLLMBgeEmbeddings

>[IPEX-LLM](https://github.com/intel-analytics/ipex-llm) 是一个 PyTorch 库，
> 可在英特尔 CPU 和 GPU（例如配备集成显卡的本地 PC、Arc、Flex 和 Max 等独立显卡）上以极低延迟运行 LLM。

查看[在 Intel CPU 上运行模型的使用示例](/oss/python/integrations/text_embedding/ipex_llm)。
查看[在 Intel GPU 上运行模型的使用示例](/oss/python/integrations/text_embedding/ipex_llm_gpu)。

```python
from langchain_community.embeddings import IpexLLMBgeEmbeddings
```

### QuantizedBgeEmbeddings

查看[使用示例](/oss/python/integrations/text_embedding/itrex)。

```python
from langchain_community.embeddings import QuantizedBgeEmbeddings
```
