---
title: 安装 LangGraph
sidebarTitle: 安装
---



要安装基础 LangGraph 包：



<CodeGroup>
```bash npm
npm install @langchain/langgraph @langchain/core
```

```bash pnpm
pnpm add @langchain/langgraph @langchain/core
```

```bash yarn
yarn add @langchain/langgraph @langchain/core
```

```bash bun
bun add @langchain/langgraph @langchain/core
```
</CodeGroup>


要使用 LangGraph，您通常需要访问 LLM 并定义工具。
您可以根据自己的需要以任何方式实现。

一种方法（我们将在文档中使用）是使用 [LangChain](/oss/javascript/langchain/overview)。

安装 LangChain：



<CodeGroup>
```bash npm
npm install langchain
```

```bash pnpm
pnpm add langchain
```

```bash yarn
yarn add langchain
```

```bash bun
bun add langchain
```
</CodeGroup>


要使用特定的 LLM 提供程序包，您需要单独安装它们。

请参考 [集成](/oss/javascript/integrations/providers/overview) 页面获取特定提供程序的安装说明。
