[RunPod](https://www.runpod.io/) 提供 GPU 云基础设施，包括针对部署和扩展 AI 模型优化的 Serverless 端点。

本指南介绍如何使用 `langchain-runpod` 集成包将 LangChain 应用程序连接到 [RunPod Serverless](https://www.runpod.io/serverless-gpu) 上托管的模型。

该集成提供了标准语言模型 (LLM) 和聊天模型的接口。

## 安装

安装专用的合作伙伴包：

```python
pip install -qU langchain-runpod
```

## 设置

### 1. 在 RunPod 上部署端点

- 导航到您的 [RunPod Serverless 控制台](https://www.runpod.io/console/serverless/user/endpoints)。
- 创建“新端点”，选择适合您模型和预期输入/输出格式的 GPU 和模板（例如 vLLM、TGI、text-generation-webui）（请参阅组件指南或包 [README](https://github.com/runpod/langchain-runpod)）。
- 配置设置并部署。
- **关键：部署后复制端点 ID**。

### 2. 设置 API 凭据

集成需要您的 RunPod API 密钥和端点 ID。将它们设置为环境变量以确保安全访问：

```python
import getpass
import os

os.environ["RUNPOD_API_KEY"] = getpass.getpass("Enter your RunPod API Key: ")
os.environ["RUNPOD_ENDPOINT_ID"] = input("Enter your RunPod Endpoint ID: ")
```

（可选）如果为 LLM 和聊天模型使用不同的端点，您可能需要设置 `RUNPOD_CHAT_ENDPOINT_ID` 或在初始化期间直接传递 ID。

## 组件

该包提供两个主要组件：

### 1. LLM

用于与标准文本补全模型交互。

请参阅 [RunPod LLM 集成指南](/oss/python/integrations/llms/runpod) 了解详细用法。

```python
from langchain_runpod import RunPod

# 示例初始化（使用环境变量）
llm = RunPod(model_kwargs={"max_new_tokens": 100})  # 在此处添加生成参数

# 示例调用
try:
    response = llm.invoke("Write a short poem about the cloud.")
    print(response)
except Exception as e:
    print(
        f"Error invoking LLM: {e}. Ensure endpoint ID and API key are correct and endpoint is active."
    )
```

### 2. 聊天模型

用于与对话模型交互。

请参阅 [RunPod 聊天模型集成指南](/oss/python/integrations/chat/runpod) 了解详细用法和功能支持。

```python
from langchain.messages import HumanMessage
from langchain_runpod import ChatRunPod

# 示例初始化（使用环境变量）
chat = ChatRunPod(model_kwargs={"temperature": 0.8})  # 在此处添加生成参数

# 示例调用
try:
    response = chat.invoke(
        [HumanMessage(content="Explain RunPod Serverless in one sentence.")]
    )
    print(response.content)
except Exception as e:
    print(
        f"Error invoking Chat Model: {e}. Ensure endpoint ID and API key are correct and endpoint is active."
    )
```
