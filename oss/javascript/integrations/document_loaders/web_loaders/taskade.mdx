[Taskade](https://www.taskade.com) 是用于 AI 驱动写作、项目管理和任务自动化的终极工具。Taskade 旨在成为您的第二大脑，简化项目执行并增强团队从始至终的协作。

## 概述

通过 [Taskade](https://www.taskade.com)，您可以构建、训练和部署自己的 AI 代理团队，以自动化任务并简化工作流程。Taskade 提供了构思、协作和执行工具的无缝融合——从结构化列表到现代表格和思维导图，所有这些都可以自定义以适应您独特的工作流程和需求。

```typescript
import { TaskadeProjectLoader } from "@langchain/community/document_loaders/web/taskade";

const loader = new TaskadeProjectLoader({
  personalAccessToken: "TASKADE_PERSONAL_ACCESS_TOKEN", // 或从 process.env.TASKADE_PERSONAL_ACCESS_TOKEN 加载
  projectId: "projectId",
});
const docs = await loader.load();

console.log({ docs });
```

您可以通过在浏览器中打开项目并从 URL 中提取来找到您的 Taskade 项目 ID：

```
https://www.taskade.com/d/<您的项目 ID 在此>
```
