[Egnyte](https://www.egnyte.com) 是一个基于云的内容协作与文件共享平台，使组织能够安全地存储、共享和管理其文件。Egnyte 将云端内容管理的能力与高级安全性、合规性及 AI 驱动搜索功能相结合。

本包为 Egnyte 的混合搜索 API 提供了 LangChain 集成，该 API 结合了语义搜索和关键词搜索，帮助您从 Egnyte 实例中查找和检索文档。

## 安装与配置

<CodeGroup>
```bash pip
pip install -U egnyte-langchain-connector
```

```bash uv
uv add egnyte-langchain-connector
```
</CodeGroup>

# egnyte-langchain-connector

本包包含与 Egnyte 的 LangChain 集成。有关 Egnyte 的更多信息，请参阅 [Egnyte 开发者文档](https://developers.egnyte.com/)。

## 前提条件

为了与 Egnyte 集成，您需要：

* Egnyte 账户 —— 如果您不是当前的 Egnyte 客户，或希望在生产环境之外的 Egnyte 实例中进行测试，可以使用[免费开发者账户](https://developers.egnyte.com/member/register)。
* Egnyte 应用 —— 此应用在[开发者控制台](https://developers.egnyte.com/)中配置，且必须启用相应的权限范围。
* 该应用必须由管理员启用。

## 身份验证

`egnyte-langchain-connector` 包使用带有 Egnyte 用户令牌的 Bearer Token 认证方式。

### 生成用户令牌

要生成用于身份验证的 Egnyte 用户令牌：

1. **注册开发者账户**
   - 访问 [https://developers.egnyte.com/member/register](https://developers.egnyte.com/member/register)
   - 创建您的免费开发者账户

2. **生成用户令牌**
   - 按照 [公共 API 认证指南](https://developers.egnyte.com/docs/read/Public_API_Authentication) 使用您的 API 密钥生成用户令牌
   - **重要提示**：在生成令牌时使用 `Egnyte.ai` 权限范围，以确保正确访问 AI 驱动的搜索功能

### 使用令牌

您可以直接将令牌传递给检索器：

```python
import getpass

egnyte_user_token = getpass.getpass("输入您的 Egnyte 用户令牌：")
domain = "company.egnyte.com"  # 您的 Egnyte 域名（不含 https://）
```

或者使用环境变量：

```python
import os
from dotenv import load_dotenv

load_dotenv()

egnyte_user_token = os.getenv("EGNYTE_USER_TOKEN")
domain = os.getenv("EGNYTE_DOMAIN")
```

from langchain_egnyte import EgnyteRetriever

retriever = EgnyteRetriever(domain="company.egnyte.com")

## 工具类

### EgnyteSearchOptions

配置高级搜索参数：

```python
from langchain_egnyte import EgnyteSearchOptions

search_options = EgnyteSearchOptions(
    limit=50,
    folderPath="/policies",
    excludeFolderPaths=["/temp", "/archive"],
    createdAfter=1640995200000,  # 毫秒级 Unix 时间戳
    createdBefore=1672531200000
)
```
