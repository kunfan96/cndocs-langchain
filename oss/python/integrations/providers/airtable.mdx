>[Airtable](https://en.wikipedia.org/wiki/Airtable) 是一项云协作服务。
`Airtable` 是一种电子表格与数据库的混合体，具备数据库的功能，但应用于电子表格中。
> Airtable 表中的字段类似于电子表格中的单元格，但具有诸如“复选框”、
> “电话号码”和“下拉列表”等类型，并且可以引用图像等文件附件。

> 用户可以创建数据库、设置列类型、添加记录、将表相互链接、进行协作、排序记录
> 并将视图发布到外部网站。

## 安装与配置

```bash
pip install pyairtable
```

* 获取您的 [API 密钥](https://support.airtable.com/docs/creating-and-using-api-keys-and-access-tokens)。
* 获取您的 [Base ID](https://airtable.com/developers/web/api/introduction)。
* 从表 URL 中获取 [表 ID](https://www.highviewapps.com/kb/where-can-i-find-the-airtable-base-id-and-table-id/#:~:text=Both%20the%20Airtable%20Base%20ID,URL%20that%20begins%20with%20tbl)。
