> [Embedchain](https://github.com/embedchain/embedchain) 是一个用于构建数据管道的 RAG 框架。它负责加载、索引、检索和同步所有数据。
>
> 它可作为[开源软件包](https://github.com/embedchain/embedchain)提供，也可作为[托管平台解决方案](https://app.embedchain.ai/)使用。


## 安装与配置

使用 pip 安装该软件包：

<CodeGroup>
```bash pip
pip install embedchain
```

```bash uv
uv add embedchain
```
</CodeGroup>
