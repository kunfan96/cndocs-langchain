## 设置

要访问 Parallel 模型，您需要安装 `langchain-parallel` 集成包并获取 [Parallel](https://platform.parallel.ai) API 密钥。

### 安装

<CodeGroup>
    ```bash pip
    pip install -U langchain-parallel
    ```
    ```bash uv
    uv add langchain-parallel
    ```
</CodeGroup>

### 凭证

前往 [Parallel](https://platform.parallel.ai) 注册并生成 API 密钥。完成后，在您的环境中设置 `PARALLEL_API_KEY` 环境变量：

```python
import getpass
import os

if not os.environ.get("PARALLEL_API_KEY"):
    os.environ["PARALLEL_API_KEY"] = getpass.getpass("输入您的 Parallel API 密钥： ")
```

## 实例化

现在我们可以实例化我们的模型对象并生成响应。默认模型是 `"speed"`，它提供快速响应：

```python
from langchain_parallel import ChatParallelWeb

llm = ChatParallelWeb(
    model="speed",
    # temperature=0.7,
    # max_tokens=None,
    # timeout=None,
    # max_retries=2,
    # api_key="...",  # 如果您更喜欢直接传递 API 密钥
    # base_url="https://api.parallel.ai",
    # 其他参数...
)
```

请参阅 [`ChatParallelWeb`](https://reference.langchain.com/python/langchain-parallel/chat_models/ChatParallelWeb) API 参考以获取完整的可用模型参数集。

<Note>
    **OpenAI 兼容性**

    Parallel 支持许多 OpenAI 兼容参数，以便于迁移（例如，`response_format`、`tools`、`top_p`），但大多数参数会被 Parallel API 忽略。有关更多详细信息，请参阅 [OpenAI 兼容性](#openai-compatibility) 部分。
</Note>

messages = [

## 链式调用

我们可以将模型与提示模板链式调用，如下所示：

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate(
    [
        (
            "system",
            "您是一个有用的研究助手，可以访问实时网络信息。 "
            "提供关于 {topic} 的全面答案，并使用当前数据。",
        ),
        ("human", "{question}"),
    ]
)

chain = prompt | llm
chain.invoke(
    {
        "topic": "人工智能",
        "question": "2025 年最重大的人工智能突破是什么？",
    }
)
```

```text
AIMessage(content="根据提供的搜索结果，以下是重大人工智能突破和趋势的摘要...", additional_kwargs={}, response_metadata={'model': 'speed', 'finish_reason': 'stop', 'created': 1764043419}, id='run--9c521362-6724-4299-9e65-0565ec13d997-0')
```

## 流式传输

ChatParallelWeb 支持流式响应以实现实时输出：

```python
for chunk in llm.stream(messages):
    print(chunk.content, end="")
```

---

## 异步

您也可以使用异步操作：

```python
# 异步调用
ai_msg = await llm.ainvoke(messages)

# 异步流式传输
async for chunk in llm.astream(messages):
    print(chunk.content, end="")
```

## 错误处理

该集成提供了针对常见场景的增强错误处理：

```python
from langchain_parallel import ChatParallelWeb

try:
    llm = ChatParallelWeb(api_key="invalid-key")
    response = llm.invoke([("human", "Hello")])
except ValueError as e:
    if "Authentication failed" in str(e):
        print("提供了无效的 API 密钥")
    elif "Rate limit exceeded" in str(e):
        print("API 速率限制已超出，请稍后再试")
```

## OpenAI 兼容性

<Info>
    **OpenAI 兼容 API**

    ChatParallelWeb 与许多 [OpenAI Chat Completions API](https://platform.openai.com/docs/api-reference/chat) 参数完全兼容，使迁移变得无缝。但是，大多数高级参数（如 `response_format`、`tools`、`top_p`）会被接受但被 Parallel API 忽略。
</Info>

```python
llm = ChatParallelWeb(
    model="speed",
    # 这些参数会被接受但被 Parallel 忽略
    response_format={"type": "json_object"},
    tools=[{"type": "function", "function": {"name": "example"}}],
    tool_choice="auto",
    top_p=1.0,
    frequency_penalty=0.0,
    presence_penalty=0.0,
    logit_bias={},
    seed=42,
    user="user-123"
)
```

## 消息处理

该集成自动处理消息格式，并合并相同类型的连续消息以满足 API 要求：

```python
from langchain.messages import HumanMessage, SystemMessage

# 这些连续的系统消息将自动合并
messages = [
    SystemMessage("您是一个有用的助手。"),
    SystemMessage("始终保持礼貌和简洁。"),
    HumanMessage("今天天气怎么样？")
]

# 在 API 调用前自动合并为单个系统消息
response = llm.invoke(messages)
```
