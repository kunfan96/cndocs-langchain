`NomicEmbeddings` 类使用 Nomic AI API 为给定文本生成嵌入。

## 设置

要使用 Nomic API，您需要[注册 Nomic 账户并创建 API 密钥](https://atlas.nomic.ai/)。

您首先需要安装 [`@langchain/nomic`](https://www.npmjs.com/package/@langchain/nomic) 包：

<Tip>
请参阅[此部分获取安装 LangChain 包的一般说明](/oss/javascript/langchain/install)。
</Tip>

```bash npm
npm install @langchain/nomic @langchain/core
```

## 用法

```typescript
import { NomicEmbeddings } from "@langchain/nomic";

/* 嵌入查询 */
const nomicEmbeddings = new NomicEmbeddings();
const res = await nomicEmbeddings.embedQuery("Hello world");
console.log(res);
/* 嵌入文档 */
const documentRes = await nomicEmbeddings.embedDocuments([
  "Hello world",
  "Bye bye",
]);
console.log(documentRes);
```
