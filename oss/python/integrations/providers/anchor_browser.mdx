[Anchor](https://anchorbrowser.io?utm=langchain) 是一个用于 AI 智能体浏览器自动化的平台，它解决了为缺乏 API 或 API 覆盖范围有限的 Web 应用程序自动化工作流的挑战。它简化了基于浏览器的自动化的创建、部署和管理，将复杂的 Web 交互转化为简单的 API 端点。

`langchain-anchorbrowser` 提供 3 个主要工具：
- `AnchorContentTool` - 用于以 Markdown 或 HTML 格式提取网页内容。
- `AnchorScreenshotTool` - 用于网页截图。
- `AnchorWebTaskTools` - 用于执行网页任务。

## 快速入门

### 安装

安装该包：

<CodeGroup>
```bash pip
pip install langchain-anchorbrowser
```

```bash uv
uv add langchain-anchorbrowser
```
</CodeGroup>

### 用法

导入并使用您所需的工具。有关 Anchor Browser 可用工具的完整列表，请参阅 [Anchor Browser 工具页面](/oss/python/integrations/tools/anchor_browser) 中的 **工具功能** 表格。

```python
from langchain_anchorbrowser import AnchorContentTool

# 获取 https://www.anchorbrowser.io 的 Markdown 内容
AnchorContentTool().invoke(
    {"url": "https://www.anchorbrowser.io", "format": "markdown"}
)
```
