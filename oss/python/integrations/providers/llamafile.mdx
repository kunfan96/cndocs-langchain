>[llamafile](https://github.com/Mozilla-Ocho/llamafile) 让您可以用单个文件分发和运行 LLM。

>`llamafile` 通过将 [llama.cpp](https://github.com/ggerganov/llama.cpp) 与
> [Cosmopolitan Libc](https://github.com/jart/cosmopolitan) 组合成一个框架，
> 将 LLM 的所有复杂性压缩成一个单文件可执行程序（称为"llamafile"），
> 使开源 LLM 对开发者和最终用户都更易访问。
> 该程序无需安装即可在大多数计算机上本地运行。


## 安装与设置

查看[安装说明](https://github.com/Mozilla-Ocho/llamafile?tab=readme-ov-file#quickstart)。

## LLM

查看[使用示例](/oss/python/integrations/llms/llamafile)。

```python
from langchain_community.llms.llamafile import Llamafile
```

## 嵌入模型


```python
from langchain_community.embeddings import LlamafileEmbeddings
```
