>[Quip](https://quip.com) 是一款面向移动端和 Web 端的协作生产力软件套件，允许群组共同创建和编辑文档及电子表格，通常用于商业目的。

这是一个用于加载 `Quip` 文档的加载器。

请参阅[此处](https://quip.com/dev/automation/documentation/current#section/Authentication/Get-Access-to-Quip's-APIs)了解如何获取个人访问令牌。

指定 `folder_ids` 列表和/或 `thread_ids` 列表，可将对应文档加载为 Document 对象。若两者均已指定，加载器将根据 `folder_ids` 获取该文件夹下所有 `thread_ids`，并与传入的 `thread_ids` 合并，返回两个集合的并集。

* 如何获取 folder_id？
  进入 Quip 文件夹，右键单击文件夹并复制链接，从链接中提取末尾部分作为 folder_id。提示：`https://example.quip.com/<folder_id>`
* 如何获取 thread_id？
  thread_id 即文档 ID。进入 Quip 文档，右键单击文档并复制链接，从链接中提取末尾部分作为 thread_id。提示：`https://exmaple.quip.com/<thread_id>`

您也可以将 `include_all_folders` 设置为 `True` 来获取 group_folder_ids。
还可以指定布尔值 `include_attachments` 以包含附件，默认为 False；若设为 True，则所有附件将被下载，QuipLoader 会提取附件中的文本并添加到 Document 对象中。当前支持的附件类型包括：`PDF`、`PNG`、`JPEG/JPG`、`SVG`、`Word` 和 `Excel`。另外，您可以指定布尔值 `include_comments` 以在文档中包含评论，默认为 False；若设为 True，则文档中所有评论将被获取，QuipLoader 会将其添加到 Document 对象中。

在使用 QuipLoader 之前，请确保已安装最新版本的 quip-api 包：

```python
pip install -qU  quip-api
```

## 示例

### 个人访问令牌

```python
from langchain_community.document_loaders.quip import QuipLoader

loader = QuipLoader(
    api_url="https://platform.quip.com", access_token="change_me", request_timeout=60
)
documents = loader.load(
    folder_ids={"123", "456"},
    thread_ids={"abc", "efg"},
    include_attachments=False,
    include_comments=False,
)
```
