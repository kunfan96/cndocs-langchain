>[Konko](https://www.konko.ai/) API 是一个完全托管的 Web API，旨在帮助应用程序开发人员：

1. **选择**适用于其应用的最佳开源或专有 LLM
2. **快速构建**应用程序，通过与领先的应用程序框架集成和完全托管的 API
3. **微调**较小的开源LLM以实现行业领先的性能，成本仅为传统方法的一小部分
4. **部署满足安全、隐私、吞吐量和延迟 SLA 的生产级 API**，无需基础设施设置或管理，使用 Konko AI 的符合 SOC 2 标准的多云基础设施

本示例介绍了如何使用 LangChain 与 `Konko` ChatCompletion 模型进行交互。[Konko 文档](https://docs.konko.ai/docs/list-of-models#konko-hosted-models-for-chatcompletion) 中提供了相关信息。

要运行此笔记本，您需要 Konko API 密钥。请登录我们的 Web 应用程序以 [创建 API 密钥](https://platform.konko.ai/settings/api-keys)，以便访问模型。

```python
from langchain_community.chat_models import ChatKonko
from langchain.messages import HumanMessage, SystemMessage
```

#### 设置环境变量

1. 您可以设置以下环境变量：
   1. KONKO_API_KEY（必填）
   2. OPENAI_API_KEY（可选）
2. 在当前 shell 会话中，使用 export 命令：

```shell
export KONKO_API_KEY={您的_KONKO_API_KEY_here}
export OPENAI_API_KEY={您的_OPENAI_API_KEY_here} # 可选
```

## 调用模型

在 [Konko 概览页面](https://docs.konko.ai/docs/list-of-models) 上查找一个模型。

另一种方法是通过此 [端点](https://docs.konko.ai/reference/get-models) 查找 Konko 实例上运行的模型列表。

从这里，我们可以初始化我们的模型：

```python
chat = ChatKonko(max_tokens=400, model="meta-llama/llama-2-13b-chat")
```

```python
messages = [
    SystemMessage(content="You are a helpful assistant."),
    HumanMessage(content="Explain Big Bang Theory briefly"),
]
chat(messages)
```
