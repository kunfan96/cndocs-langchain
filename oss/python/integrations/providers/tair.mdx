>[阿里云 Tair](https://www.alibabacloud.com/help/en/tair/latest/what-is-tair) 是阿里云开发的云原生内存数据库服务。它提供丰富的数据模型和企业级功能，
> 支持您的实时在线场景，同时与开源 `Redis` 保持完全兼容。
> `Tair` 还引入了基于新型非易失性内存（NVM）存储介质的持久化内存优化实例。

## 安装与设置

安装 Tair Python SDK：

<CodeGroup>
```bash pip
pip install tair
```

```bash uv
uv add tair
```
</CodeGroup>

## 向量存储

```python
from langchain_community.vectorstores import Tair
```

查看[使用示例](/oss/python/integrations/vectorstores/tair)。
