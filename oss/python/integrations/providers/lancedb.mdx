本页介绍如何在 LangChain 中使用 [LanceDB](https://github.com/lancedb/lancedb)。
本文档分为两部分：安装与设置，以及针对特定 LanceDB 包装器的参考文档。

## 安装与设置

- 使用 `pip install lancedb` 安装 Python SDK

## 包装器

### VectorStore

提供了针对 LanceDB 数据库的包装器，使其可作为 VectorStore 使用，适用于语义搜索或示例选择等场景。

要导入此 VectorStore：

```python
from langchain_community.vectorstores import LanceDB
```

有关 LanceDB 包装器的更详细逐步指南，请参阅 [此 Notebook](/oss/python/integrations/vectorstores/lancedb)
