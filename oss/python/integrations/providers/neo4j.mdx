>- Neo4j 是一款专注于图数据库技术的`开源数据库管理系统`。
>- Neo4j 允许您用节点和边来表示和存储数据，非常适合处理关联数据和关系。
>- Neo4j 提供 `Cypher 查询语言`，使与图数据的交互和查询变得简单。
>- 借助 Neo4j，您可以实现高性能的`图遍历和查询`，适合生产级系统。

>访问 [Neo4j 官网](https://neo4j.com/)开始使用。

## 安装与设置

- 使用 `pip install neo4j langchain-neo4j` 安装 Python SDK


## 向量存储

Neo4j 向量索引用作向量存储，支持语义搜索或示例选择。

```python
from langchain_neo4j import Neo4jVector
```

查看[使用示例](/oss/python/integrations/vectorstores/neo4jvector)

## GraphCypherQAChain

Neo4j 图数据库有一个封装器，允许您根据用户输入生成 Cypher 语句，并使用它们从数据库中检索相关信息。

```python
from langchain_neo4j import GraphCypherQAChain, Neo4jGraph
```

查看[使用示例](/oss/python/integrations/graphs/neo4j_cypher)

## 从文本构建知识图谱

文本数据通常包含丰富的关系和洞察，可用于各种分析、推荐引擎或知识管理应用。
Diffbot 的 NLP API 允许从非结构化文本数据中提取实体、关系和语义含义。
将 Diffbot 的 NLP API 与图数据库 Neo4j 结合，可以基于从文本中提取的信息创建强大的动态图结构。
这些图结构完全可查询，并可集成到各种应用中。

```python
from langchain_neo4j import Neo4jGraph
from langchain_experimental.graph_transformers.diffbot import DiffbotGraphTransformer
```

查看[使用示例](/oss/python/integrations/graphs/diffbot)

## 检查点保存器

Neo4j 实现的 LangGraph 检查点保存器，用于持久化具有分支时间旅行支持的 GitHub Agent 内存。检查点保存器在每个超级步骤持久化图状态，支持会话内存、人在环路工作流、时间旅行和容错。
适用于 LangGraph 图和 LangChain Agent：

```python
from langchain_neo4j import Neo4jSaver

with Neo4jSaver.from_conn_string(
    uri="bolt://localhost:7687",
    user="neo4j",
    password="password"
) as checkpointer:
    checkpointer.setup()  # Create indexes (run once)
    # Create agent with checkpointer
    agent = create_agent(
        model="anthropic:claude-sonnet-4-6",
        tools=[get_weather],
        system_prompt="You are a helpful assistant",
        checkpointer=checkpointer
    )
```
