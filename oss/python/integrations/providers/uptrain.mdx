>[UpTrain](https://uptrain.ai/) 是一个开源统一平台，用于评估和改进生成式 AI 应用。它提供 20 多项预配置评估（涵盖语言、代码、嵌入使用场景）的评分，对失败案例进行根因分析，并提供解决方案建议。

## 安装与设置

<CodeGroup>
```bash pip
pip install uptrain
```

```bash uv
uv add uptrain
```
</CodeGroup>

## 回调

```python
from langchain_community.callbacks.uptrain_callback import UpTrainCallbackHandler
```

查看[示例](/oss/python/integrations/callbacks/uptrain)。
