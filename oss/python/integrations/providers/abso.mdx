[Abso](https://abso.ai/#router) 是一个开源 LLM 代理，根据提示复杂度自动在快速和慢速模型之间路由请求。它使用各种启发式方法选择合适的模型。它速度非常快且延迟低。

## 安装和设置

```bash
pip install langchain-abso
```
