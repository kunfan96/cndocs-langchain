>[Trubrics](https://trubrics.com) 是一个 LLM 用户分析平台，让您能够收集、分析和管理 AI 模型上的用户提示和反馈。
>
>查看 [Trubrics 仓库](https://github.com/trubrics/trubrics-sdk) 了解有关 `Trubrics` 的更多信息。

## 安装与设置

我们需要安装 `trubrics` Python 包：

<CodeGroup>
```bash pip
pip install trubrics
```

```bash uv
uv add trubrics
```
</CodeGroup>


## 回调

查看[使用示例](/oss/python/integrations/callbacks/trubrics)。

```python
from langchain.callbacks import TrubricsCallbackHandler
```
