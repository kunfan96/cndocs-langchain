> [Bagel](https://www.bagel.net/)（`AI 的开放向量数据库`）就像 AI 数据的 GitHub。
它是一个协作平台，用户可以在此创建、分享和管理向量数据集。它可以支持独立开发者的私人项目、
企业的内部协作以及数据 DAO 的公开贡献。

## 安装与设置

<CodeGroup>
```bash pip
pip install bagelML
```

```bash uv
uv add bagelML
```
</CodeGroup>


## 向量存储

查看[使用示例](/oss/python/integrations/vectorstores/bagel)。

```python
from langchain_community.vectorstores import Bagel
```
