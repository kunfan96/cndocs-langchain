> `FalkorDB` 是什么？
> - FalkorDB 是一个专注于图数据库技术的`开源数据库管理系统`。
> - FalkorDB 允许你将数据表示并存储在节点和边中，使其成为处理关联数据和关系的理想选择。
> - FalkorDB 支持带有专有扩展的 OpenCypher 查询语言，便于你与图数据进行交互和查询。
> - 借助 FalkorDB，你可以实现高性能的`图遍历与查询`，适用于生产级系统。
> 访问[其官方网站](https://docs.falkordb.com/)即可开始使用 FalkorDB。

## 安装与配置

- 使用 `pip install falkordb langchain-falkordb` 安装 Python SDK。

## VectorStore

FalkorDB 向量索引用作向量存储（vectorstore），
无论是用于语义搜索还是示例选择。

```python
from langchain_community.vectorstores.falkordb_vector import FalkorDBVector
```

或

```python
from langchain_falkordb.vectorstore import FalkorDBVector
```

请参阅[使用示例](/oss/python/integrations/vectorstores/falkordbvector.ipynb)。
