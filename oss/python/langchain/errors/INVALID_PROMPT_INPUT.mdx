当[提示词模板](https://github.com/langchain-ai/langchain/blob/v0.3/docs/docs/concepts/prompt_templates.mdx)接收到缺失或无效的输入变量时，会发生此错误。




## 故障排查

要解决此错误，你可以：

1. 检查提示词模板的正确性。使用 f-string 格式时，确保正确转义花括号：
   * 在 f-string 中使用 `{{` 表示单个花括号
   * 在 f-string 中使用 `{{{{` 表示双花括号
2. 使用 `MessagesPlaceholder` 组件时，确认你传入的是消息数组或类消息对象。如果使用简写元组，请将变量名用花括号括起来，例如 `["placeholder", "{messages}"]`
3. 通过 [LangSmith](/langsmith/home) 或日志记录来检查提示词模板的实际输入，以验证它们是否符合预期
4. 如果从 LangChain [Prompt Hub](https://smith.langchain.com/prompts) 获取提示词，请使用示例输入单独测试该提示词，以确保其按预期运行
