> 来自 [Unstructured.IO](https://www.unstructured.io/) 的 `unstructured` 包从原始源文档（如 PDF 和 Word 文档）中提取干净的文本。
本页介绍如何在 LangChain 中使用 [`unstructured`](https://github.com/Unstructured-IO/unstructured) 生态系统。

## 安装和设置

如果您使用的是本地运行的加载器，请按照以下步骤获取 `unstructured` 及其依赖项。

- 为了获得最小的安装占用空间并利用开源 `unstructured` 包中不可用的功能，请使用 `pip install unstructured-client` 安装 Python SDK，并配合 `pip install langchain-unstructured` 以使用 `UnstructuredLoader` 并远程针对 Unstructured API 进行分区。此加载器位于 LangChain 合作伙伴仓库中，而不是 `langchain-community` 仓库中，您需要一个 `api_key`。您可以在 [Unstructured API 密钥页面](https://unstructured.io/api-key/) 生成一个免费密钥。
    - Unstructured SDK 的文档可以在这里找到：
      https://docs.unstructured.io/api-reference/api-services/sdk

- 要在本地运行所有内容，请使用 `pip install unstructured` 安装开源 Python 包，并配合 `pip install langchain-community`，并使用与上述相同的 `UnstructuredLoader`。
    - 您可以使用额外功能安装特定于文档的依赖项，例如 `pip install "unstructured[docx]"`。在[完整安装文档](https://docs.unstructured.io/open-source/installation/full-installation)中了解更多关于额外功能的信息。
    - 要安装所有文档类型的依赖项，请使用 `pip install "unstructured[all-docs]"`。
- 如果您的系统上尚未安装以下系统依赖项，请安装它们，例如在 Mac 上使用 `brew install`。根据您解析的文档类型，您可能不需要全部这些依赖项。
    - `libmagic-dev`（文件类型检测）
    - `poppler-utils`（图像和 PDF）
    - `tesseract-ocr`（图像和 PDF）
    - `qpdf`（PDF）
    - `libreoffice`（MS Office 文档）
    - `pandoc`（EPUB）
- 本地运行时，Unstructured 还建议使用 Docker [按照本指南](https://docs.unstructured.io/open-source/installation/docker-installation) 确保所有系统依赖项正确安装。

Unstructured API 需要 API 密钥才能发出请求。
您可以[请求一个 API 密钥](https://unstructured.io/api-key-hosted) 并立即开始使用！
查看 [Unstructured API README](https://github.com/Unstructured-IO/unstructured-api) 以开始进行 API 调用。
我们很乐意听取您的反馈，请在我们的[社区 Slack](https://join.slack.com/t/unstructuredw-kbe4326/shared_invite/zt-1x7cgo0pg-PTptXWylzPQF9xZolzCnwQ) 中告诉我们进展如何。
并关注质量和性能的改进！
如果您想自托管 Unstructured API 或在本地运行，请查看 [Docker 自托管说明](https://github.com/Unstructured-IO/unstructured-api#dizzy-instructions-for-using-the-docker-image)。

## 数据加载器

`Unstructured` 的主要用途是在数据加载器中。

### UnstructuredLoader

查看[使用示例](/oss/python/integrations/document_loaders/unstructured_file)，了解如何使用此加载器进行本地分区和远程使用无服务器 Unstructured API。

```python
from langchain_unstructured import UnstructuredLoader
```

### UnstructuredCHMLoader

`CHM` 表示 `Microsoft 编译的 HTML 帮助`。

```python
from langchain_community.document_loaders import UnstructuredCHMLoader
```

### UnstructuredCSVLoader

`逗号分隔值` (`CSV`) 文件是一种使用逗号分隔值的分隔文本文件。文件的每一行都是一个数据记录。
每个记录由一个或多个字段组成，字段之间用逗号分隔。

查看[使用示例](/oss/python/integrations/document_loaders/csv#unstructuredcsvloader)。

```python
from langchain_community.document_loaders import UnstructuredCSVLoader
```

### UnstructuredEmailLoader

查看[使用示例](/oss/python/integrations/document_loaders/email)。

```python
from langchain_community.document_loaders import UnstructuredEmailLoader
```

### UnstructuredEPubLoader

[EPUB](https://en.wikipedia.org/wiki/EPUB) 是一种使用“.epub”文件扩展名的`电子书文件格式`。该术语是电子出版物的缩写，有时也写作 `ePub`。`EPUB` 受许多电子阅读器支持，并且适用于大多数智能手机、平板电脑和计算机的兼容软件。

查看[使用示例](/oss/python/integrations/document_loaders/epub)。

```python
from langchain_community.document_loaders import UnstructuredEPubLoader
```

### UnstructuredODTLoader

`办公应用程序开放文档格式 (ODF)`，也称为 `OpenDocument`，是一种用于文字处理文档、电子表格、演示文稿和图形的开放文件格式，使用 ZIP 压缩的 XML 文件。它的开发旨在为办公应用程序提供开放的、基于 XML 的文件格式规范。

查看[使用示例](/oss/python/integrations/document_loaders/odt)。

