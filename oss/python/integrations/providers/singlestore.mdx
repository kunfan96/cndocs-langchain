[SingleStore](https://singlestore.com/) 是一款高性能分布式 SQL 数据库，专为在[云端](https://www.singlestore.com/cloud/)和本地环境中均表现出色而设计。它提供多样化的功能集、无缝的部署选项和卓越的性能。

此集成提供以下组件以充分利用 SingleStore 的能力：

- **`SingleStoreLoader`**：直接从 SingleStore 数据库表加载文档。
- **`SingleStoreSemanticCache`**：将 SingleStore 用作语义缓存，高效存储和检索嵌入。
- **`SingleStoreChatMessageHistory`**：在 SingleStore 中存储和检索聊天消息历史记录。
- **`SingleStoreVectorStore`**：存储文档嵌入并执行快速向量和全文搜索。

这些组件实现了高效的文档存储、嵌入管理以及高级搜索功能，结合全文搜索和基于向量的搜索，实现快速准确的查询。

```python
from langchain_singlestore import (
    SingleStoreChatMessageHistory,
    SingleStoreLoader,
    SingleStoreSemanticCache,
    SingleStoreVectorStore,
)
```
