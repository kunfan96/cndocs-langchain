本笔记本介绍如何从您只想复制粘贴的内容加载文档对象。在这种情况下，您甚至不需要使用 DocumentLoader，而是可以直接构造 Document。

```python
from langchain_core.documents import Document
```

```python
text = "..... 将您复制粘贴的文本放在这里......"
```

```python
doc = Document(page_content=text)
```

## 元数据

如果您想添加关于获取此文本片段来源的元数据，可以使用 metadata 键轻松实现。

```python
metadata = {"source": "internet", "date": "Friday"}
```

```python
doc = Document(page_content=text, metadata=metadata)
```
