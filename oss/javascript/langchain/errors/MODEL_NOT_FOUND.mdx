---
title: MODEL_NOT_FOUND
---

<Note>
    目前仅在 `langchainjs` (JavaScript/TypeScript) 中使用。
</Note>

您指定的模型名称未被您的提供者认可。

## 故障排除

要解决此错误：

1. **验证模型标识符**: 仔细检查您传入的模型字符串。确保拼写和格式正确
2. **检查代理/包装器配置**: 如果您正在使用带有模型包装器的代理或其他替代主机，请确认允许的模型名称未受到限制或更改

该错误通常源于模型名称字符串本身的拼写错误，或者代码与提供者 API 之间的代理服务或模型包装器施加的限制。
