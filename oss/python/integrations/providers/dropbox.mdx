>[Dropbox](https://en.wikipedia.org/wiki/Dropbox) 是一种文件托管服务，将传统文件、云内容和网络快捷方式汇集到一处。


## 安装与设置

请参阅详细的[安装指南](/oss/python/integrations/document_loaders/dropbox#prerequisites)。

<CodeGroup>
```bash pip
pip install -U dropbox
```

```bash uv
uv add dropbox
```
</CodeGroup>

## 文档加载器

查看[使用示例](/oss/python/integrations/document_loaders/dropbox)。

```python
from langchain_community.document_loaders import DropboxLoader
```
