>[SAP SE（维基百科）](https://www.sap.com/about/company.html) 是一家德国跨国软件公司。它开发用于管理业务运营和客户关系的企业软件。该公司是全球领先的`企业资源规划（ERP）`软件供应商。

## 安装与设置

我们需要安装 `langchain-hana` Python 包。

<CodeGroup>
```bash pip
pip install langchain-hana
```

```bash uv
uv add langchain-hana
```
</CodeGroup>

## 向量存储

>[SAP HANA Cloud 向量引擎](https://help.sap.com/docs/hana-cloud-database/sap-hana-cloud-sap-hana-database-vector-engine-guide/sap-hana-cloud-sap-hana-database-vector-engine-guide) 是
> 完全集成到 `SAP HANA Cloud` 数据库中的向量存储。

查看[使用示例](/oss/python/integrations/vectorstores/sap_hanavector)。

```python
from langchain_hana import HanaDB
```

## 自查询检索器
>[SAP HANA Cloud 向量引擎](https://help.sap.com/docs/hana-cloud-database/sap-hana-cloud-sap-hana-database-vector-engine-guide/sap-hana-cloud-sap-hana-database-vector-engine-guide)
> 还通过 `HanaTranslator` 类提供了自查询检索器实现。

查看[使用示例](/oss/python/integrations/retrievers/self_query/hanavector_self_query)。

```python
from langchain_hana import HanaTranslator
```

## 图
>[SAP HANA Cloud 知识图谱引擎](https://help.sap.com/docs/hana-cloud-database/sap-hana-cloud-sap-hana-database-knowledge-graph-guide/sap-hana-cloud-sap-hana-database-knowledge-graph-engine-guide)
> 通过 `HanaRdfGraph` 类支持利用知识图谱。

查看[使用示例](/oss/python/integrations/graphs/sap_hana_rdf_graph)。

```python
from langchain_hana import HanaRdfGraph
```

## 链
还提供了 `SparqlQAChain`，可与 `HanaRdfGraph` 一起用于 SPARQL-QA 任务。

查看[使用示例](/oss/python/integrations/chains/sap_hana_sparql_qa_chain)。

```python
from langchain_hana import HanaSparqlQAChain
```
