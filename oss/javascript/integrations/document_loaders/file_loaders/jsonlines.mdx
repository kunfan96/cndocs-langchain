---
title: "Jsonlines 文件 - 集成"
description: "使用 LangChain JavaScript 与 Jsonlines 文件 - 文档加载器进行集成。"
---

本示例介绍如何从 JSONLines 或 JSONL 文件加载数据。第二个参数是一个 JSONPointer，用于指定从文件中每个 JSON 对象提取的属性。文件中的每个 JSON 对象都将创建一个文档。

示例 JSONLines 文件：

```json
{"html": "This is a sentence."}
{"html": "This is another sentence."}
```

示例代码：

```typescript
import { JSONLinesLoader } from "@langchain/classic/document_loaders/fs/json";

const loader = new JSONLinesLoader(
  "src/document_loaders/example_data/example.jsonl",
  "/html"
);

const docs = await loader.load();
/*
[
  Document {
    "metadata": {
      "blobType": "application/jsonl+json",
      "line": 1,
      "source": "blob",
    },
    "pageContent": "This is a sentence.",
  },
  Document {
    "metadata": {
      "blobType": "application/jsonl+json",
      "line": 2,
      "source": "blob",
    },
    "pageContent": "This is another sentence.",
  },
]
*/
```
