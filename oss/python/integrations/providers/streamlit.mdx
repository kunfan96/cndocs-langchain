>[Streamlit](https://streamlit.io/) 是一种更快地构建和共享数据应用的方式。
>`Streamlit` 在几分钟内将数据脚本转换为可共享的 Web 应用。全部使用纯 Python。无需前端经验。
>更多示例请参见 [streamlit.io/generative-ai](https://streamlit.io/generative-ai)。

## 安装与设置

我们需要安装 `streamlit` Python 包：

<CodeGroup>
```bash pip
pip install streamlit
```

```bash uv
uv add streamlit
```
</CodeGroup>

## 回调

请参阅[使用示例](/oss/python/integrations/callbacks/streamlit)。

```python
from langchain_community.callbacks import StreamlitCallbackHandler
```
