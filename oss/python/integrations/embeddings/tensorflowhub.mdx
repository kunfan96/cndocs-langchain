>[TensorFlow Hub](https://www.tensorflow.org/hub) 是一个预训练机器学习模型的仓库，已准备好进行微调并可在任何地方部署。只需几行代码即可复用 `BERT` 和 `Faster R-CNN` 等预训练模型。
>
>
接下来加载 TensorflowHub Embedding 类。

```python
from langchain_community.embeddings import TensorflowHubEmbeddings
```

```python
embeddings = TensorflowHubEmbeddings()
```
