[AskNews](https://asknews.app/) 通过每日处理并索引超过 30 万篇文章，为语言模型提供最新的全球或历史新闻，
通过低延迟端点提供针对提示词优化的响应，
并确保其新闻报道的透明度与多样性。

## 安装与设置

首先，您需要安装 `asknews` Python 包。

<CodeGroup>
```bash pip
pip install asknews
```

```bash uv
uv add asknews
```
</CodeGroup>

您还需要配置我们的 AskNews API 凭据，这些凭据可在
[AskNews 控制台](https://my.asknews.app/) 生成。


## 检索器

请参阅[使用示例](/oss/python/integrations/retrievers/asknews)。

```python
from langchain_community.retrievers import AskNewsRetriever
```

## 工具

请参阅[使用示例](/oss/python/integrations/tools/asknews)。

```python
from langchain_community.tools.asknews import AskNewsSearch
```
