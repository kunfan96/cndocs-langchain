`TitanML` 通过我们的训练、压缩和推理优化平台，帮助企业构建、部署更优、更小、成本更低且速度更快的 NLP 模型。

我们的推理服务器 [Titan Takeoff](https://docs.titanml.co/docs/intro) 支持通过单条命令在本地硬件上部署大语言模型（LLM）。它支持大多数生成式模型架构，例如 Falcon、Llama 2、GPT2、T5 等。如果您在使用特定模型时遇到问题，请通过 [hello@titanml.co](mailto:hello@titanml.co) 联系我们。

## 示例用法

以下是一些帮助快速上手使用 Titan Takeoff Server 的实用示例。运行这些命令前，请确保 Takeoff Server 已在后台启动。更多信息请参阅 [启动 Takeoff 的文档页面](https://docs.titanml.co/docs/Docs/launching/)。

```python
import time

# 注意：导入 TitanTakeoffPro 而非 TitanTakeoff 同样有效，两者底层使用相同的对象
from langchain_community.llms import TitanTakeoff
from langchain_core.callbacks import CallbackManager, StreamingStdOutCallbackHandler
from langchain_core.prompts import PromptTemplate
```

### 示例 1

基本用法，假设 Takeoff 正在您的机器上使用默认端口运行（即 localhost:3000）。

```python
llm = TitanTakeoff()
output = llm.invoke("What is the weather in London in August?")
print(output)
```

### 示例 2

指定端口及其他生成参数

```python
llm = TitanTakeoff(port=3000)
# 完整的参数列表可查阅 https://docs.titanml.co/docs/next/apis/Takeoff%20inference_REST_API/generate#request
output = llm.invoke(
    "What is the largest rainforest in the world?",
    consumer_group="primary",
    min_new_tokens=128,
    max_new_tokens=512,
    no_repeat_ngram_size=2,
    sampling_topk=1,
    sampling_topp=1.0,
    sampling_temperature=1.0,
    repetition_penalty=1.0,
    regex_string="",
    json_schema=None,
)
print(output)
```

### 示例 3

使用 generate 处理多个输入

```python
llm = TitanTakeoff()
rich_output = llm.generate(["What is Deep Learning?", "What is Machine Learning?"])
print(rich_output.generations)
```

### 示例 4

流式输出

```python
llm = TitanTakeoff(
    streaming=True, callback_manager=CallbackManager([StreamingStdOutCallbackHandler()])
)
prompt = "What is the capital of France?"
output = llm.invoke(prompt)
print(output)
```

### 示例 5

使用 LCEL

```python
llm = TitanTakeoff()
prompt = PromptTemplate.from_template("Tell me about {topic}")
chain = prompt | llm
output = chain.invoke({"topic": "the universe"})
print(output)
```

### 示例 6

使用 TitanTakeoff Python 包装器启动读取器（Readers）。如果在首次启动 Takeoff 时尚未创建任何读取器，或者您希望添加其他读取器，可以在初始化 TitanTakeoff 对象时完成此操作。只需将想要启动的模型配置列表作为 `models` 参数传入即可。

```python
# Llama 模型的模型配置，您可在此指定以下参数：
#   model_name (str): 要使用的模型名称
#   device (str): 用于推理的设备，cuda 或 cpu
#   consumer_group (str): 将读取器分配到的消费者组
#   tensor_parallel (Optional[int]): 模型希望拆分运行的 GPU 数量
#   max_seq_length (int): 用于推理的最大序列长度，默认为 512
#   max_batch_size (int): 请求连续批处理的最大批次大小
llama_model = {
    "model_name": "TheBloke/Llama-2-7b-Chat-AWQ",
    "device": "cuda",
    "consumer_group": "llama",
}
llm = TitanTakeoff(models=[llama_model])

# 模型需要时间启动，所需时间取决于模型大小和网络连接速度
time.sleep(60)

prompt = "What is the capital of France?"
output = llm.invoke(prompt, consumer_group="llama")
print(output)
```
