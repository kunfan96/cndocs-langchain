title: "MindsDB 集成"
description: "使用 LangChain Python 与 MindsDB 进行集成。"
---

MindsDB 是一个基于企业数据定制 AI 的平台。借助 MindsDB 及其近 200 个针对 [数据源](https://docs.mindsdb.com/integrations/data-overview) 和 [AI/ML 框架](https://docs.mindsdb.com/integrations/ai-overview) 的集成，任何开发者都可以利用其企业数据，更快、更安全地为其目的定制 AI。

通过 MindsDB，您可以将任意数据源连接到任意 AI/ML 模型，以实现并自动化 AI 驱动的应用程序。利用来自数据库、向量存储或应用程序的数据，实时部署、提供服务并微调模型。所有这些操作均可使用开发者已熟悉的通用工具完成。

MindsDB 与 LangChain 集成，使用户能够：


- 在 MindsDB 中部署通过 LangChain 提供的模型，使其可被众多数据源访问。
- 在 MindsDB 中使用实时和动态数据对通过 LangChain 提供的模型进行微调。
- 结合 LangChain 和 MindsDB 自动化 AI 工作流。

请参阅[我们的文档](https://docs.mindsdb.com/integrations/ai-engines/langchain)，了解更多关于 MindsDB 与 LangChain 集成的信息并查看示例。
