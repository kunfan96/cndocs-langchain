所有与 `Naver` 相关的功能，包括 HyperCLOVA X 模型，尤其是那些可通过 `Naver Cloud` [CLOVA Studio](https://clovastudio.ncloud.com/) 访问的模型。

> [Naver](https://navercorp.com/) 是一家拥有尖端技术和多元化业务组合的全球科技公司，业务涵盖搜索、电商、金融科技、内容、云服务和人工智能。

> [Naver Cloud](https://www.navercloudcorp.com/lang/en/) 是 Naver 的云计算部门，作为领先的云服务提供商，通过其 [Naver Cloud Platform (NCP)](https://www.ncloud.com/) 为企业提供全面的云服务套件。

有关更详细的说明，请参阅 [NCP 用户指南](https://guide.ncloud-docs.com/docs/clovastudio-overview)（亦提供韩语版本）。

## 安装与配置

- 通过[申请获取](https://api.ncloud-docs.com/docs/ai-naver-clovastudio-summary#API%ED%82%A4) CLOVA Studio API Key，并将其设置为环境变量（`CLOVASTUDIO_API_KEY`）。


Naver 集成分布在两个软件包中：

- `langchain-naver`：专为 Naver 设计的集成软件包。
- `langchain-naver-community`：由社区维护的软件包，并非由 Naver 或 LangChain 官方维护。

<CodeGroup>
```bash pip
pip install -U langchain-naver
# pip install -U langchain-naver-community // 安装以使用 Naver Search 工具。
```

```bash uv
uv add langchain-naver
# uv add langchain-naver-community // 安装以使用 Naver Search 工具。
```
</CodeGroup>

> **（注意）** 通过 `langchain-community`（一个[第三方集成](https://python.langchain.com/docs/concepts/architecture/#langchain-community)集合）进行的 Naver 集成已过时。
> - **请改用 `langchain-naver`，新功能应仅通过此软件包实现**。
> - 如果您使用的是 `langchain-community`（过时版本）并获得了旧版 API Key（不以 `nv-*` 前缀开头），您应将其设置为 `NCP_CLOVASTUDIO_API_KEY`，并且可能需要通过[创建您的应用](https://guide.ncloud-docs.com/docs/en/clovastudio-playground01#create-test-app)获取额外的 API Gateway API Key，并将其设置为 `NCP_APIGW_API_KEY`。

## 工具

### Naver 搜索

Naver Search 集成允许您的 LangChain 应用程序从 Naver 搜索引擎中检索信息。这对于韩语查询以及获取关于韩国主题的最新资讯特别有用。

要使用 Naver Search 工具，您需要：

1. 登录 [Naver 开发者门户](https://developers.naver.com/main/)
2. 创建新应用并启用搜索 API
3. 从“应用列表”部分获取您的 **NAVER_CLIENT_ID** 和 **NAVER_CLIENT_SECRET**
4. 将这些变量设置为应用程序中的环境变量

```python
from langchain_naver_community.tool import NaverSearchResults
from langchain_naver_community.utils import NaverSearchAPIWrapper

# 设置搜索包装器
search = NaverSearchAPIWrapper()

# 创建工具
tool = NaverSearchResults(api_wrapper=search)
```

有关更多详细信息，请参阅[使用示例](/oss/python/integrations/tools/naver_search)。

### 专用搜索工具

该软件包还为不同类型的內容提供了专用搜索工具：

```python
from langchain_naver_community.tool import NaverNewsSearch  # 用于新闻文章
from langchain_naver_community.tool import NaverBlogSearch  # 用于博客文章
from langchain_naver_community.tool import NaverImageSearch  # 用于图片
```

这些工具均可在 LangChain 智能体中使用，以提供更精准的搜索能力。
