本笔记本介绍如何使用 [Twilio](https://www.twilio.com) API 包装器通过 SMS 或 [Twilio 消息渠道](https://www.twilio.com/docs/messaging/channels) 发送消息。

Twilio 消息渠道便于与第三方消息应用集成，并允许您通过 WhatsApp Business Platform（GA）、Facebook Messenger（Public Beta）和 Google Business Messages（Private Beta）发送消息。

## 设置

要使用此工具，您需要安装 Python Twilio 包 `twilio`。

```python
pip install -qU  twilio
```

您还需要设置 Twilio 帐户并获取您的凭据。您需要您的帐户字符串标识符 (SID) 和身份验证令牌。您还需要一个用于发送消息的号码。

您可以将这些作为命名参数 `account_sid`、`auth_token`、`from_number` 传递给 TwilioAPIWrapper，或者您可以设置环境变量 `TWILIO_ACCOUNT_SID`、`TWILIO_AUTH_TOKEN`、`TWILIO_FROM_NUMBER`。

## 发送 SMS

```python
from langchain_community.utilities.twilio import TwilioAPIWrapper
```

```python
twilio = TwilioAPIWrapper(
    #     account_sid="foo",
    #     auth_token="bar",
    #     from_number="baz,"
)
```

```python
twilio.run("hello world", "+16162904619")
```

## 发送 WhatsApp 消息

您需要将您的 WhatsApp Business 帐户与 Twilio 链接。您还需要确保用于发送消息的号码在 Twilio 上配置为 WhatsApp 启用的发件人，并在 WhatsApp 上注册。

```python
from langchain_community.utilities.twilio import TwilioAPIWrapper
```

```python
twilio = TwilioAPIWrapper(
    #     account_sid="foo",
    #     auth_token="bar",
    #     from_number="whatsapp: baz,"
)
```

```python
twilio.run("hello world", "whatsapp: +16162904619")
```
