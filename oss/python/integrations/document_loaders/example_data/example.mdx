# 示例 Markdown 文档

## 简介

欢迎阅读此示例 Markdown 文档。Markdown 是一种用于格式化文本的轻量级标记语言。它广泛用于文档、自述文件等。

## 功能

### 标题

Markdown 支持多级标题：

- **标题 1**：`# 标题 1`
- **标题 2**：`## 标题 2`
- **标题 3**：`### 标题 3`

### 列表

#### 无序列表

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

#### 有序列表

1. 第一项
2. 第二项
3. 第三项
