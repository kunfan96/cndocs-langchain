>[Spreedly](https://docs.spreedly.com/) 是一项允许您安全存储信用卡并将其用于向任意数量的支付网关和第三方 API 进行交易的服务。它通过同时提供信用卡代码化/保管服务以及网关和接收器集成服务来实现这一点。通过 Spreedly 代码化的支付方式存储在 `Spreedly` 中，使您可以独立存储一张卡，然后根据业务需求将该卡传递给不同的终端。
