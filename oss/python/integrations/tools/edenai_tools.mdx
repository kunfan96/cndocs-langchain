本 Jupyter Notebook 演示了如何将 Eden AI 工具与 Agent 配合使用。

Eden AI 正在通过整合顶级 AI 服务提供商来革新 AI 领域，让用户能够解锁无限可能，充分挖掘人工智能的真正潜力。借助这个一站式、全面且无障碍的平台，用户可以极速将 AI 功能部署到生产环境，通过单一 API 轻松访问完整的 AI 能力。（官网：[edenai.co/](https://edenai.co/)）

通过将 Eden AI 工具添加到提供给 Agent 的工具列表中，您可以赋予 Agent 执行多种任务的能力，例如：

- 语音转文字
- 文字转语音
- 文本不良内容检测
- 图像不良内容检测
- 目标检测
- OCR 发票解析
- OCR 身份证解析

在本示例中，我们将演示如何使用 Eden AI 工具创建一个能够执行上述部分任务的 Agent。

---------------------------------------------------------------------------
访问 EDENAI 的 API 需要 API 密钥，

您可以通过在 [app.edenai.run/user/register](https://app.edenai.run/user/register) 注册账户，然后前往 [app.edenai.run/admin/account/settings](https://app.edenai.run/admin/account/settings) 获取密钥。

获取密钥后，您可以将其设置为环境变量 `EDENAI_API_KEY`，或在初始化 EdenAI 工具时通过 `edenai_api_key` 命名参数直接传入，例如 `EdenAiTextModerationTool(edenai_api_key="...")`

```python
pip install -qU langchain-community
```

```python
from langchain_community.tools.edenai import (
    EdenAiExplicitImageTool,
    EdenAiObjectDetectionTool,
    EdenAiParsingIDTool,
    EdenAiParsingInvoiceTool,
    EdenAiSpeechToTextTool,
    EdenAiTextModerationTool,
    EdenAiTextToSpeechTool,
)
```

```python
from langchain.agents import create_agent
from langchain_community.llms import EdenAI

llm = EdenAI(
    feature="text", provider="openai", params={"temperature": 0.2, "max_tokens": 250}
)

tools = [
    EdenAiTextModerationTool(providers=["openai"], language="en"),
    EdenAiObjectDetectionTool(providers=["google", "api4ai"]),
    EdenAiTextToSpeechTool(providers=["amazon"], language="en", voice="MALE"),
    EdenAiExplicitImageTool(providers=["amazon", "google"]),
    EdenAiSpeechToTextTool(providers=["amazon"]),
    EdenAiParsingIDTool(providers=["amazon", "klippa"], language="en"),
    EdenAiParsingInvoiceTool(providers=["amazon", "google"], language="en"),
]
agent_chain = create_agent(
    model=llm,
    tools=tools,
    verbose=True,
)
```
