> [VDMS](https://github.com/IntelLabs/vdms/blob/master/README.md) 是一种用于高效访问大型“视觉”数据的存储解决方案，旨在通过搜索存储为图的视觉元数据相关的视觉数据，并启用对视觉数据的机器友好增强以实现更快访问，从而达到云规模。

## 安装与设置

### 安装客户端

<CodeGroup>
```bash pip
pip install langchain-vdms
```

```bash uv
uv add langchain-vdms
```
</CodeGroup>

### 安装数据库

有两种方式开始使用 VDMS：

1. 通过 Docker 在本地机器上安装 VDMS
    ```bash
        docker run -d -p 55555:55555 intellabs/vdms:latest
    ```

2. 直接在本地机器上安装 VDMS。请参阅
[安装说明](https://github.com/IntelLabs/vdms/blob/master/INSTALL.md)。

## VectorStore

要导入此向量存储：

```python
from langchain_vdms import VDMS
from langchain_vdms.vectorstores import VDMS
```
要导入 VDMS 客户端连接器：

```python
from langchain_vdms.vectorstores import VDMS_Client
```

有关 VDMS 包装器的更详细演练，请参阅 [VDMS](/oss/python/integrations/vectorstores/vdms)。
