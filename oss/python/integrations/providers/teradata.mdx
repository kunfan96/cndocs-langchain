本页面介绍如何在 LangChain 中使用 Teradata 向量存储。
它分为两部分：安装和设置，以及特定 Teradata 包装器的参考。

## 安装
- 使用 `pip install langchain-teradata` 安装 Python 包

## 设置
第一步是创建与 Teradata Vantage 系统的连接。

    您需要 Teradata 凭据，包括主机名、用户名、密码，以及云部署的可选 API 令牌。有关详细设置说明，请参阅 [Teradata 向量存储用户指南](https://docs.teradata.com/r/Enterprise_IntelliFlex_VMware/Teradata-Vector-Store-User-Guide/Introduction-to-the-Enterprise-Vector-Store-User-Guide)
和 [Vantage Cloud Lake 入门](https://docs.teradata.com/r/Lake-Getting-Started-with-VantageCloud-Lake/)

## 包装器

### VectorStore

存在一个围绕 Teradata 向量数据库的包装器，允许您将其用作向量存储，无论是用于相似性搜索还是 RAG 管道。

要导入此向量存储：
```python
from langchain_teradata import TeradataVectorStore
```

### 用法

有关 Teradata VectorStore 的更详细演练，请参阅 [langchain-teradata 用户指南](https://docs.teradata.com/r/Enterprise_IntelliFlex_VMware/Teradata-Package-for-LangChain-User-Guide) 和 [langchain-teradata 工作流](https://github.com/Teradata/langchain-teradata)
