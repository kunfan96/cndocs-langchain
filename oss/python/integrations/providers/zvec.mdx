> [Zvec](https://zvec.org/en/) 是一个开源的、进程内的向量数据库——轻量级、闪电般快速，并旨在直接嵌入到应用程序中。它基于 Proxima（阿里巴巴经过实战检验的向量搜索引擎）构建，以最小的设置提供生产级、低延迟、可扩展的相似性搜索。

本文档演示了如何在 LangChain 生态系统中利用 Zvec。具体而言，它展示了如何安装 Zvec，以及如何将其用作 LangChain 中的 VectorStore 插件。

## 安装与设置

安装 Python SDK：

<CodeGroup>
```bash pip
pip install zvec
```

```bash uv
uv add zvec
```
</CodeGroup>

## 嵌入模型

```python
from langchain_community.embeddings import DashScopeEmbeddings
```

请参阅[使用示例](/oss/python/integrations/vectorstores/zvec)。

## 向量存储

Zvec Collection 被包装为一个熟悉的 VectorStore，以便在 LangChain 中原生使用，这使其能够轻松用于各种场景，例如语义搜索或示例选择。

您可以通过以下方式导入向量存储：

```python
from langchain_community.vectorstores import Zvec
```

有关 Zvec 包装器的详细演练，请参阅[此笔记本](/oss/python/integrations/vectorstores/zvec)。
