>[Golden](https://golden.com) 提供了一组基于 Golden 知识图谱的用于查询和增强的自然语言 API。例如，`Products from OpenAI`、`Generative ai companies with series a funding` 和 `rappers who invest` 等查询可用于检索相关实体的结构化数据。
>
>`golden-query` LangChain 工具是对 [Golden Query API](https://docs.golden.com/reference/query-api) 的封装，可实现对这些结果的编程访问。
>有关更多信息，请参阅 [Golden Query API 文档](https://docs.golden.com/reference/query-api)。

## 安装与设置
- 前往 [Golden API 文档](https://docs.golden.com/) 了解 Golden API 概览。
- 从 [Golden API 设置](https://golden.com/settings/api) 页面获取您的 API 密钥。
- 将您的 API 密钥保存至 `GOLDEN_API_KEY` 环境变量中

## 封装器

### 实用工具

存在一个名为 `GoldenQueryAPIWrapper` 的实用工具，用于封装该 API。要导入此实用工具：

```python
from langchain_community.utilities.golden_query import GoldenQueryAPIWrapper
```

如需更详细的逐步指南，请参阅[此笔记本](/oss/python/integrations/tools/golden_query)。

### 工具

您还可以轻松将此封装器加载为 Tool（以便与 Agent 配合使用）。
您可以通过以下方式实现：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["golden-query"])
```

有关工具的更多信息，请参阅[工具](/oss/python/integrations/tools)。
