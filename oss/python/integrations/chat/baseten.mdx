## 配置

要访问 Baseten 模型，您需要创建一个 Baseten 账户、获取 API 密钥并安装 `langchain-baseten` 整合包。

前往 [Baseten 网站](https://app.baseten.co) 创建账户并生成 API 密钥。完成这些步骤后，请设置 `BASETEN_API_KEY` 环境变量：

### 资格凭证

```python
Set API key icon="key"
import getpass
import os

if "BASETEN_API_KEY" not in os.environ:
    os.environ["BASETEN_API_KEY"] = getpass.getpass("Enter your Baseten API key: ")
```

要启用自动 <Tooltip tip="记录模型执行的每一步以进行调试和改进">跟踪</Tooltip>，请设置您的 [LangSmith](/langsmith/home) API 密钥：

```python
Enable tracing icon="flask"
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### 安装

LangChain Baseten 整合位于 `langchain-baseten` 包中：

<CodeGroup>
    ```python
    pip install -U langchain-baseten
    ```
    ```python
    uv add langchain-baseten
    ```
</CodeGroup>

## 实例化

Baseten 提供了两种访问聊天模型的方法：

1. **模型 API**：用于访问最新、最受欢迎的开源模型。
2. **专用 URL**：使用特定模型部署和专用资源。

这两种方法都支持自动端点规范化。

```python
Initialize with model slug icon="robot"
from langchain_baseten import ChatBaseten

# 选项 1: 使用模型 API 和模型标识符
model = ChatBaseten(
    model="moonshotai/Kimi-K2-Instruct-0905",  # 从可用的模型标识符中选择：https://docs.baseten.co/development/model-apis/overview#supported-models
    api_key="your-api-key",  # 或设置 BASETEN_API_KEY 环境变量
)
```

```python
Initialize with model URL icon="link"
from langchain_baseten import ChatBaseten

# 选项 2: 使用专用部署和模型 URL
model = ChatBaseten(
    model_url="https://model-<id>.api.baseten.co/environments/production/predict",
    api_key="your-api-key",  # 或设置 BASETEN_API_KEY 环境变量
)
```

## 调用

```python
Basic invocation icon="player-play"
# 使用聊天模型
response = model.invoke("Hello, how are you?")
print(response.content)
```

```text
Hello! I'm doing well, thank you for asking! How about you?
```

您也可以使用消息对象进行更复杂的对话：

<CodeGroup>
    ```python
    Dictionary format icon="book"
    messages = [
        {"role": "system", "content": "You are a poetry expert"},
        {"role": "user", "content": "Write a haiku about spring"},
    ]
    response = model.invoke(messages)
    print(response)
    ```
</CodeGroup>
