[Hyperbrowser](https://hyperbrowser.ai) 是一个用于运行浏览器智能体和扩展无头浏览器的平台。它可以让你大规模启动和管理浏览器会话，并为任何网页抓取需求提供易于使用的解决方案，例如抓取单个页面或爬取整个网站。

主要功能：

- 即时可扩展性 - 在几秒钟内启动数百个浏览器会话，无需担心基础设施问题
- 简单集成 - 与 Puppeteer 和 Playwright 等流行工具无缝配合
- 强大的 API - 用于抓取/爬取任意网站的易用 API，以及更多功能
- 绕过反爬虫措施 - 内置隐身模式、广告拦截、自动验证码解决和轮换代理

本指南提供了 Hyperbrowser 工具入门的快速概览。

有关 Hyperbrowser 的更多信息，请访问 [Hyperbrowser 网站](https://hyperbrowser.ai)，如果你想查看文档，可以访问 [Hyperbrowser 文档](https://docs.hyperbrowser.ai)。

## 浏览器智能体

Hyperbrowser 提供强大的浏览器智能体工具，使 AI 模型能够以编程方式与网络浏览器交互。这些浏览器智能体可以导航网站、填写表单、点击按钮、提取数据，并执行复杂的网页自动化任务。

浏览器智能体尤其适用于：

- 从复杂网站进行网页抓取和数据提取
- 自动化重复性网页任务
- 与需要身份验证的 Web 应用程序交互
- 跨多个网站进行研究
- 测试 Web 应用程序

Hyperbrowser 提供三种类型的浏览器智能体工具：

- **Browser Use Tool**：通用浏览器自动化工具
- **OpenAI CUA Tool**：与 OpenAI 计算机使用智能体的集成
- **Claude Computer Use Tool**：与 Anthropic Claude 计算机使用功能的集成

## 设置

要访问 Hyperbrowser 工具，你需要安装 `langchain-hyperbrowser` 集成包，并创建一个 Hyperbrowser 账户并获取 API 密钥。

### 凭据

前往 [Hyperbrowser](https://app.hyperbrowser.ai/) 注册并生成 API 密钥。完成后设置 HYPERBROWSER_API_KEY 环境变量：

```bash
export HYPERBROWSER_API_KEY=<your-api-key>
```

### 安装

安装 **langchain-hyperbrowser**。

```python
pip install -qU langchain-hyperbrowser
```

## 实例化

### Browser Use 工具

`HyperbrowserBrowserUseTool` 是一个使用浏览器智能体（特别是 Browser-Use 智能体）执行网页自动化任务的工具。

```python
from langchain_hyperbrowser import HyperbrowserBrowserUseTool
tool = HyperbrowserBrowserUseTool()
```

### OpenAI CUA 工具

`HyperbrowserOpenAICUATool` 是一个通过 Hyperbrowser 利用 OpenAI 计算机使用智能体（CUA）能力的专用工具。

```python
from langchain_hyperbrowser import HyperbrowserOpenAICUATool
tool = HyperbrowserOpenAICUATool()
```

### Claude Computer Use 工具

`HyperbrowserClaudeComputerUseTool` 是一个通过 Hyperbrowser 利用 Claude 计算机使用能力的专用工具。

```python
from langchain_hyperbrowser import HyperbrowserClaudeComputerUseTool
tool = HyperbrowserClaudeComputerUseTool()
```

### 使用自定义会话选项

所有工具均支持自定义会话选项：

```python
result = tool.run(
    {
        "task": "Go to npmjs.com, and tell me when react package was last updated.",
        "session_options": {
            "session_options": {"use_proxy": True, "accept_cookies": True}
        },
    }
)
print(result)
```

```python
{'data': 'I have found that the react package was last published 11 hours ago. This is the most recently updated package I could find.', 'error': None}
```

### 异步用法

所有工具均支持异步用法：

```python
async def browse_website():
    tool = HyperbrowserBrowserUseTool()
    result = await tool.arun(
        {
            "task": "Go to npmjs.com, click the first visible package, and tell me when it was updated"
        }
    )
    return result


result = await browse_website()
```

## 在智能体中使用

以下是如何在智能体中使用任意 Hyperbrowser 工具：

```python
from langchain.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_hyperbrowser import browser_use_tool
from langchain_openai import ChatOpenAI
from langchain.agents import create_agent


model = ChatOpenAI(temperature=0)

# You can use any of the three tools here
browser_use_tool = HyperbrowserBrowserUseTool()
agent = create_agent(model, [browser_use_tool])

user_input = "Go to npmjs.com, and tell me when react package was last updated."
for step in agent.stream(
    {"messages": user_input},
    stream_mode="values",
):
    step["messages"][-1].pretty_print()
```

## 配置选项

Claude Computer Use、OpenAI CUA 和 Browser Use 具有以下可用参数：

- `task`：使用智能体执行的任务
- `max_steps`：智能体完成任务可以执行的最大交互步骤数
- `session_options`：浏览器会话配置

更多详情，请参阅各自的 API 参考：

- [OpenAI CUA API 参考](https://docs.hyperbrowser.ai/reference/api-reference/agents/openai-cua)
