---
title: "HuggingFace 推理集成"
description: "使用 LangChain JavaScript 与 HuggingFace 推理嵌入模型进行集成。"
---

此嵌入集成使用 HuggingFace 推理 API 为给定文本生成嵌入，默认使用 `BAAI/bge-base-en-v1.5` 模型。您可以向构造函数传递不同的模型名称以使用其他模型。

## 设置

首先需要安装 [`@langchain/community`](https://www.npmjs.com/package/@langchain/community) 包和所需的对等依赖项：

<Tip>
请参阅[此部分获取安装 LangChain 包的一般说明](/oss/javascript/langchain/install)。
</Tip>

```bash npm
npm install @langchain/community @langchain/core @huggingface/inference@4
```
## 用法

```typescript
import { HuggingFaceInferenceEmbeddings } from "@langchain/community/embeddings/hf";

const embeddings = new HuggingFaceInferenceEmbeddings({
  apiKey: "YOUR-API-KEY", // 默认为 process.env.HUGGINGFACEHUB_API_KEY
  model: "MODEL-NAME", // 如果未提供，默认为 `BAAI/bge-base-en-v1.5`
  provider: "MODEL-PROVIDER", // 如果未提供，将回退到 Hugging Face 推理 API 内的自动选择机制
});
```

> **注意：**
> 如果您不提供 `model`，将记录警告并使用默认模型 `BAAI/bge-base-en-v1.5`。
> 如果您不提供 `provider`，Hugging Face 将默认使用 `auto` 选择，这将根据您在 https://hf.co/settings/inference-providers 的设置选择模型的第一个可用提供者。

> **提示：**
> `hf-inference` 是由 Hugging Face 直接托管的模型的提供者名称。
