>[SemaDB](https://semafind.com/) 是一款简便的向量相似性搜索引擎。它提供低成本的云托管版本，帮助您轻松构建 AI 应用程序。

使用 SemaDB Cloud（我们的托管版本），简便意味着无需计算 Pod 大小、无需定义模式、无需分区设置、无需调优参数、无需搜索算法调优、无需复杂安装、无需复杂 API。它集成了 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb)，提供透明的计费、自动分片以及交互式 API 演练场。

## 安装

无需任何安装，直接在 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb) 上开始使用 SemaDB Cloud。

## 向量存储

SemaDB 集合有一个基础封装器，允许您将其用作向量存储。

```python
from langchain_community.vectorstores import SemaDB
```

您可以在[此 notebook](/oss/python/integrations/vectorstores/semadb) 中查看如何使用该封装器的教程。
