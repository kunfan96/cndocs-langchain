>[DocArrayHnswSearch](https://docs.docarray.org/user_guide/storing/index_hnswlib/) 是 [Docarray](https://github.com/docarray/docarray) 提供的一种轻量级文档索引实现，完全在本地运行，最适合中小型数据集。它将向量存储在磁盘上的 [hnswlib](https://github.com/nmslib/hnswlib) 中，并将所有其他数据存储在 [SQLite](https://www.sqlite.org/index.html) 中。

使用此集成需要通过 `pip install -qU langchain-community` 安装 `langchain-community`。

本 notebook 展示如何使用与 `DocArrayHnswSearch` 相关的功能。

## 使用 DocArrayHnswSearch

```python
from langchain_community.document_loaders import TextLoader
from langchain_community.vectorstores import DocArrayHnswSearch
from langchain_openai import OpenAIEmbeddings
from langchain_text_splitters import CharacterTextSplitter
```

```python
documents = TextLoader("../../how_to/state_of_the_union.txt").load()
text_splitter = CharacterTextSplitter(chunk_size=1000, chunk_overlap=0)
docs = text_splitter.split_documents(documents)

embeddings = OpenAIEmbeddings()

db = DocArrayHnswSearch.from_documents(
    docs, embeddings, work_dir="hnswlib_store/", n_dim=1536
)
```

### 相似度搜索

```python
query = "What did the president say about Ketanji Brown Jackson"
docs = db.similarity_search(query)
```

```python
print(docs[0].page_content)
```
