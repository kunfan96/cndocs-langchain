>[Metal](https://github.com/getmetal/metal-python) 是一个用于 ML Embeddings 的托管服务。

本笔记本展示了如何使用 [Metal 的](https://docs.getmetal.io/introduction) 检索器。

首先，您需要注册 Metal 并获取 API 密钥。您可以在 [Metal 创建应用指南](https://docs.getmetal.io/misc-create-app) 中完成此操作。

```python
pip install -qU  metal_sdk
```

```python
from metal_sdk.metal import Metal

API_KEY = ""
CLIENT_ID = ""
INDEX_ID = ""

metal = Metal(API_KEY, CLIENT_ID, INDEX_ID)
```

## 摄入文档

仅当您尚未设置索引时才需要执行此操作

```python
metal.index({"text": "foo1"})
metal.index({"text": "foo"})
```

```text
{'data': {'id': '642739aa7559b026b4430e42',
  'text': 'foo',
  'createdAt': '2023-03-31T19:51:06.748Z'}}
```

## 查询

现在我们的索引已设置好，我们可以设置检索器并开始查询。

```python
from langchain_community.retrievers import MetalRetriever
```

```python
retriever = MetalRetriever(metal, params={"limit": 2})
```

```python
retriever.invoke("foo1")
```
