本示例介绍如何使用 LangChain 与 SambaNovaCloud 模型进行交互。

## 环境配置

### 凭据

要访问 SambaNovaCloud 模型，您需要创建一个 [SambaNovaCloud 账户](https://cloud.sambanova.ai/)，获取 API 密钥，并将其设置为 `SAMBANOVA_API_KEY` 环境变量：

```python
import getpass
import os

if "SAMBANOVA_API_KEY" not in os.environ:
    os.environ["SAMBANOVA_API_KEY"] = getpass.getpass()
```

### 安装

该集成位于 `langchain-community` 包中。我们还需要安装 [sseclient-py](https://pypi.org/project/sseclient-py/) 包，这是运行流式预测所必需的。

```python
pip install --quiet -U langchain-community sseclient-py
```

## 实例化

```python
from langchain_community.llms.sambanova import SambaNovaCloud

llm = SambaNovaCloud(
    model="Meta-Llama-3.3-70B-Instruct",
    max_tokens_to_generate=1000,
    temperature=0.01,
    # top_k = 50,
    # top_p = 1.0
)
```

## 调用

现在我们可以实例化模型对象并生成补全结果：

```python
input_text = "为什么我应该使用开源模型？"

completion = llm.invoke(input_text)
completion
```

## 链式调用

我们可以将补全模型与提示模板进行链式组合，如下所示：

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template("如何用 {output_language} 表达 {input}：\n")

chain = prompt | llm
chain.invoke(
    {
        "output_language": "德语",
        "input": "我爱编程。",
    }
)
```

```text
'"我爱编程"的德语翻译为：\n\n"Ich liebe das Programmieren."\n\n以下是该句子的结构解析：\n\n* "Ich" 意为 "我"\n* "liebe" 是动词 "爱" 的第一人称单数形式（我爱）\n* "das" 是 "Programmieren"（编程）的定冠词\n* "Programmieren" 是动词 "编程" 的不定式形式，但在此语境中作为名词使用，指代编程活动。\n\n因此，"Ich liebe das Programmieren" 是德语中表达您对编程热爱的常用方式。'
```
