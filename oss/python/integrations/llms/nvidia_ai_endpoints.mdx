## 概述

`langchain-nvidia-ai-endpoints` 软件包包含用于构建基于 NVIDIA NIM 推理微服务的模型的 LangChain 集成。这些模型由 NVIDIA 优化，可在 NVIDIA 加速基础设施上提供最佳性能，并以 NIM 的形式部署，这是一种易于使用的预构建容器，只需一条命令即可在 NVIDIA 加速基础设施上的任何位置部署。

您可以在 [NVIDIA API 目录](https://build.nvidia.com/) 上测试由 NVIDIA 托管的 NIM 部署。测试完成后，可使用 NVIDIA AI Enterprise 许可证从 NVIDIA API 目录导出 NIM，并在本地或云端运行，使企业能够拥有并完全控制其知识产权和 AI 应用。

NIM 按模型打包为容器镜像，并通过 NVIDIA NGC 目录以 NGC 容器镜像的形式分发。其核心在于，NIM 为运行 AI 模型推理提供了简单、一致且熟悉的 API。

本示例介绍如何使用 LangChain 通过 `NVIDIA` 类与受支持的 NVIDIA 模型进行交互。

有关通过此 API 访问 LLM 模型的更多信息，请查阅 [NVIDIA](https://python.langchain.com/docs/integrations/llms/nvidia_ai_endpoints/) 文档。

## 环境配置

**开始之前：**

1. 在 [NVIDIA](https://build.nvidia.com/) 上创建一个免费账户，该平台托管了 NVIDIA AI Foundation 模型。

2. 点击您选择的模型。

3. 在 `Input`（输入）下选择 `Python` 选项卡，然后点击 `Get API Key`（获取 API 密钥）。接着点击 `Generate Key`（生成密钥）。

4. 复制生成的密钥并将其保存为 `NVIDIA_API_KEY`。此后，您应该可以访问相关端点。

### 凭据

```python
import getpass
import os

if not os.getenv("NVIDIA_API_KEY"):
    # 注意：API 密钥应以 "nvapi-" 开头
    os.environ["NVIDIA_API_KEY"] = getpass.getpass("请输入您的 NVIDIA API 密钥：")
```

### 安装

LangChain NVIDIA AI Endpoints 集成位于 `langchain-nvidia-ai-endpoints` 软件包中：

```python
pip install -qU langchain-nvidia-ai-endpoints
```

## 实例化

完整功能请参阅 [LLM](/oss/python/langchain/models)。

```python
from langchain_nvidia_ai_endpoints import NVIDIA
```

```python
llm = NVIDIA().bind(max_tokens=256)
llm
```

## 调用

```python
prompt = "# Function that does quicksort written in Rust without comments:"
```

```python
print(llm.invoke(prompt))
```

## 流式传输、批处理与异步

这些模型原生支持流式传输，并且与所有 LangChain LLM 一样，它们提供了用于处理并发请求的批处理方法，以及用于 invoke、stream 和 batch 的异步方法。以下是一些示例。

```python
for chunk in llm.stream(prompt):
    print(chunk, end="", flush=True)
```

```python
llm.batch([prompt])
```

```python
await llm.ainvoke(prompt)
```

```python
async for chunk in llm.astream(prompt):
    print(chunk, end="", flush=True)
```

```python
await llm.abatch([prompt])
```

```python
async for chunk in llm.astream_log(prompt):
    print(chunk)
```

```python
response = llm.invoke(
    "X_train, y_train, X_test, y_test = train_test_split(X, y, test_size=0.1) #训练逻辑回归模型，在测试集上预测标签并计算准确率得分"
)
print(response)
```
