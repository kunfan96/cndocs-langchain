> [any-llm](https://github.com/mozilla-ai/any-llm) 是一个统一的接口，用于调用 OpenAI、Anthropic、Google、本地模型（通过 Ollama/LocalAI）以及[更多](https://mozilla-ai.github.io/any-llm/providers/)。只需更改一个字符串即可在提供商之间切换。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-anyllm
```

```bash uv
uv add langchain-anyllm
```
</CodeGroup>

您需要为所选的提供商提供相应的 API 密钥。API 密钥可以通过 `api_key` 参数传递，或设置为环境变量（例如 `OPENAI_API_KEY`、`ANTHROPIC_API_KEY`）。有关特定提供商的要求，请参阅 [any-llm 文档](https://mozilla-ai.github.io/any-llm/providers/)。

## 聊天模型

```python
from langchain_anyllm import ChatAnyLLM
```
