Cognee 实现了可扩展的模块化 ECL（提取、认知化、加载）管道，使您能够互联和检索过去的对话、文档和音频转录，同时减少幻觉、开发工作量和成本。

Cognee 融合了图数据库和向量数据库，以发现数据中隐藏的关系和新模式。您可以自动建模、加载和检索代表业务领域的实体和对象，并分析其关系，揭示单独使用向量存储或图存储都无法提供的洞察。

您是否看过 cognee 的 <a href="https://github.com/topoteretes/cognee-starter">入门仓库</a>？快来看看吧！


## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-cognee
```

```bash uv
uv add langchain-cognee
```
</CodeGroup>
