---
title: GRAPH_RECURSION_LIMIT
---

你的 LangGraph [`StateGraph`](https://langchain-ai.github.io/langgraph/reference/graphs/#langgraph.graph.state.StateGraph) 在达到停止条件之前已达到最大步数。
这通常是由于无限循环引起的，例如以下代码：

```python
class State(TypedDict):
    some_key: str

builder = StateGraph(State)
builder.add_node("a", ...)
builder.add_node("b", ...)
builder.add_edge("a", "b")
builder.add_edge("b", "a")
...

graph = builder.compile()
```





然而，复杂的图可能会自然地达到默认限制。

## 故障排除

* 如果你不希望图经过多次迭代，你可能存在循环。请检查你的逻辑是否存在无限循环。

* 如果你有复杂的图，可以在调用图时将更高的 `recursion_limit` 值传递到 `config` 对象中：

```python
graph.invoke({...}, {"recursion_limit": 100})
```
