>[Doctran](https://github.com/psychic-api/doctran) 是一个 Python 包。它利用 LLM 和开源
> NLP 库将原始文本转换为干净、结构化且信息密集的文档，这些文档针对向量空间检索进行了优化。你可以将 `Doctran` 视为一个黑盒，其中
> 杂乱的字符串进入，整洁、干净且带标签的字符串输出。


## 安装与配置

<CodeGroup>
```bash pip
pip install doctran
```

```bash uv
uv add doctran
```
</CodeGroup>
