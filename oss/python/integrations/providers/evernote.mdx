>[EverNote](https://evernote.com/) 旨在用于归档和创建笔记，其中可嵌入照片、音频和保存的网页内容。笔记存储在虚拟“笔记本”中，支持标记、批注、编辑、搜索和导出。

## 安装与配置

首先，请安装 `lxml` 和 `html2text` Python 包。

<CodeGroup>
```bash pip
pip install lxml
pip install html2text
```

```bash uv
uv add lxml
uv add html2text
```
</CodeGroup>

## 文档加载器

请参阅[使用示例](/oss/python/integrations/document_loaders/evernote)。

```python
from langchain_community.document_loaders import EverNoteLoader
```
