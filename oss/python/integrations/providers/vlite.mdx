本页介绍如何在 LangChain 中使用 [vlite](https://github.com/sdan/vlite)。vlite 是一个简单快速的向量数据库，用于存储和检索嵌入向量。

## 安装与设置

安装 vlite，运行以下命令：

<CodeGroup>
```bash pip
pip install vlite
```

```bash uv
uv add vlite
```
</CodeGroup>

要支持 PDF OCR，安装 `vlite[ocr]` 扩展：

<CodeGroup>
```bash pip
pip install "vlite[ocr]"
```

```bash uv
uv add "vlite[ocr]"
```
</CodeGroup>

## 向量存储

vlite 对其向量数据库提供了封装，允许你将其用作向量存储，用于语义搜索和示例选择。

导入 vlite 向量存储：

```python
from langchain_community.vectorstores import vlite
```

### 使用方法

有关 vlite 封装的更详细说明，请参阅[此笔记本](/oss/python/integrations/vectorstores/vlite)。
