## 概述

`UpstashRedisStore` 是 `ByteStore` 的一个实现，将所有内容存储在你的 [Upstash](https://upstash.com/) 托管 Redis 实例中。

如需使用基础 `RedisStore`，请参阅[本指南](/oss/python/integrations/stores/redis/)。

## 配置

首先需要[注册 Upstash 账户](https://upstash.com/docs/redis/overall/getstarted)，然后创建一个 Redis 数据库进行连接。

### 凭据

创建数据库后，获取数据库 URL（不要忘记 `https://`！）和 token：

```python
from getpass import getpass

URL = getpass("Enter your Upstash URL")
TOKEN = getpass("Enter your Upstash REST token")
```

### 安装

LangChain Upstash 集成位于 `langchain-community` 包中。还需要安装 `upstash-redis` 包作为对等依赖：

```python
pip install -qU langchain-community upstash-redis
```

## 实例化

现在可以实例化字节存储：

```python
from langchain_community.storage import UpstashRedisByteStore
from upstash_redis import Redis

redis_client = Redis(url=URL, token=TOKEN)
kv_store = UpstashRedisByteStore(client=redis_client, ttl=None, namespace="test-ns")
```

## 使用方法

可以使用 `mset` 方法在键下设置数据：

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[b'value1', b'value2']
```

可以使用 `mdelete` 方法删除数据：

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[None, None]
```
