OpenLLM 允许开发者通过**一条命令**运行任何**开源 LLM**，并将其作为**兼容 OpenAI 的 API** 端点提供服务。

- 🔬 专为快速部署和生产环境使用而构建
- 🚂 支持 llama3、qwen2、gemma 等模型，以及大量**量化版本** [完整列表](https://github.com/bentoml/openllm-models)
- ⛓️ 兼容 OpenAI 的 API
- 💬 内置类 ChatGPT 的用户界面
- 🔥 借助最先进的推理后端加速 LLM 解码过程
- 🌥️ 随时准备进行企业级云部署（支持 Kubernetes、Docker 和 BentoCloud）

## 安装与配置

通过 PyPI 安装 OpenLLM 包：

<CodeGroup>
```bash pip
pip install openllm
```

```bash uv
uv add openllm
```
</CodeGroup>

## LLM

OpenLLM 支持广泛的开源 LLM，同时也支持托管用户自行微调的 LLM。使用 `openllm model` 命令可查看所有已为 OpenLLM 预优化的可用模型。

## Wrappers

提供了一个 OpenLLM 封装器，支持与正在运行的 OpenLLM 服务器进行交互：

```python
from langchain_community.llms import OpenLLM
```

### Wrapper for OpenLLM server

该封装器支持与 OpenLLM 的兼容 OpenAI 的端点进行交互。

要运行模型，请执行以下操作：

```bash
openllm hello
```

封装器用法：

```python
from langchain_community.llms import OpenLLM

llm = OpenLLM(base_url="http://localhost:3000/v1", api_key="na")

llm("What is the difference between a duck and a goose? And why there are so many Goose in Canada?")
```
