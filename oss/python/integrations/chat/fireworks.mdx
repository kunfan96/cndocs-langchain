## 配置

要访问 Fireworks 模型，您需要创建一个 Fireworks 账户、获取 API 密钥并安装 `langchain-fireworks` 集成包。

### 凭据

前往 [fireworks.ai](https://app.fireworks.ai/login) 注册 Fireworks 并生成 API 密钥。完成此操作后，设置 FIREWORKS_API_KEY 环境变量：

```python
import getpass
import os

if "FIREWORKS_API_KEY" not in os.environ:
    os.environ["FIREWORKS_API_KEY"] = getpass.getpass("请输入您的 Fireworks API 密钥: ")
```

要启用对模型调用的自动跟踪，请设置 [LangSmith](/langsmith/home) API 密钥：

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("请输入您的 LangSmith API 密钥: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### 安装

LangChain Fireworks 集成位于 `langchain-fireworks` 包中：

```bash
pip install -qU langchain-fireworks
```

## 实例化

现在我们可以实例化我们的模型对象并生成聊天完成结果：

```python
from langchain_fireworks import ChatFireworks

llm = ChatFireworks(
    model="accounts/fireworks/models/kimi-k2-instruct-0905",
    temperature=0,
    max_tokens=None,
    timeout=None,
    max_retries=2,
    # 其他参数...
)
```

## 调用

```python
messages = [
    (
        "system",
        "您是一个将英语翻译成法语的有帮助的助手。请翻译用户句子。",
    ),
    ("human", "我爱编程。"),
]
ai_msg = llm.invoke(messages)
ai_msg
```
