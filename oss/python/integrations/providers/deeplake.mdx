[DeepLake](https://www.deeplake.ai/) 是一款专为人工智能和深度学习应用优化的数据库。


## 安装与配置

<CodeGroup>
```bash pip
pip install langchain-deeplake
```

```bash uv
uv add langchain-deeplake
```
</CodeGroup>

## 向量存储

有关可用向量存储的更多详细信息，请参阅 [Activeloop DeepLake 向量存储集成](/oss/python/integrations/vectorstores/activeloop_deeplake)。
