**LangGraph v1 是面向代理运行时的稳定性发布。** 它保留了核心图 API 与执行模型，同时在类型安全、文档与开发体验上做了改进。

LangGraph 旨在与 [LangChain v1](/oss/javascript/releases/langchain-v1) 无缝配合，后者的 `createAgent` 就是构建在 LangGraph 之上。你可以用 LangChain 快速上手，必要时再使用 LangGraph 进行精细编排。
