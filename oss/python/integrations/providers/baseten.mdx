>[Baseten](https://baseten.co) 提供了部署和服务机器学习模型所需的全部基础设施，具备高性能、可靠性和可扩展性。

>作为模型推理平台，`Baseten` 在 LangChain 生态系统中是一个 `Provider`。
`Baseten` 集成目前实现了`聊天模型`和`嵌入`组件。

>`Baseten` 允许您通过指定模型 [slug](https://docs.baseten.co/development/model-apis/overview#supported-models) 访问 Kimi K2 或 GPT OSS 等开源模型的 API，
也支持通过指定 `model_url` 在专用 GPU 上运行专有或微调模型。

## 安装与设置

在 LangChain 中使用 Baseten 模型需要两样东西：

* 一个 [Baseten 账户](https://baseten.co)
* 一个 [API 密钥](https://docs.baseten.co/observability/api-keys)

将您的 API 密钥导出为名为 `BASETEN_API_KEY` 的环境变量。

```sh
export BASETEN_API_KEY="paste_your_api_key_here"
```

## 聊天模型（模型 API 和专用部署）

查看[使用示例](/oss/python/integrations/chat/baseten)。

```python
from langchain_baseten import ChatBaseten
```

## 嵌入（仅限专用部署）

查看[使用示例](/oss/python/integrations/text_embedding/baseten)。

```python
from langchain_baseten import BasetenEmbeddings
```
