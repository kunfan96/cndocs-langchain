>[Fauna](https://fauna.com/) 是一种文档数据库。

查询 `Fauna` 文档

```python
pip install -qU  fauna
```

## 数据查询示例

```python
from langchain_community.document_loaders.fauna import FaunaLoader

secret = "<enter-valid-fauna-secret>"
query = "Item.all()"  # Fauna 查询语句，假设集合名称为 "Item"
field = "text"  # 包含页面内容的字段，假设字段名为 "text"

loader = FaunaLoader(query, field, secret)
docs = loader.lazy_load()

for value in docs:
    print(value)
```

### 分页查询

如果还有更多数据，您将获得一个 `after` 值。您可以通过在查询中传入 `after` 字符串来获取游标之后的值。

更多信息请参阅[此链接](https://fqlx-beta--fauna-docs.netlify.app/fqlx/beta/reference/schema_entities/set/static-paginate)

```python
query = """
Item.paginate("hs+DzoPOg ... aY1hOohozrV7A")
Item.all()
"""
loader = FaunaLoader(query, field, secret)
```
