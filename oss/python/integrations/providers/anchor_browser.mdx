[Anchor](https://anchorbrowser.io?utm=langchain) 是面向 AI Agent 的浏览器自动化平台，专为解决那些缺乏 API 或 API 覆盖有限的 Web 应用程序自动化工作流问题。它简化了基于浏览器的自动化的创建、部署和管理，将复杂的 Web 交互转化为简单的 API 端点。

`langchain-anchorbrowser` 提供三种主要工具：
- `AnchorContentTool` - 以 Markdown 或 HTML 格式提取网页内容。
- `AnchorScreenshotTool` - 截取网页截图。
- `AnchorWebTaskTools` - 执行 Web 任务。

## 快速入门

### 安装

安装包：

<CodeGroup>
```bash pip
pip install langchain-anchorbrowser
```

```bash uv
uv add langchain-anchorbrowser
```
</CodeGroup>

### 使用

导入并使用所需工具。完整的 Anchor Browser 可用工具列表，请参见 [Anchor Browser 工具页面](/oss/python/integrations/tools/anchor_browser)中的**工具功能**表格。

```python
from langchain_anchorbrowser import AnchorContentTool

# 获取 https://www.anchorbrowser.io 的 Markdown 内容
AnchorContentTool().invoke(
    {"url": "https://www.anchorbrowser.io", "format": "markdown"}
)
```
