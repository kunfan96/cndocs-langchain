>[StarRocks](https://www.starrocks.io/) 是一款高性能分析数据库。
`StarRocks` 是新一代亚秒级 MPP 数据库，支持多维分析、实时分析和即席查询等全场景分析。

>`StarRocks` 通常被归类为 OLAP，并在 [ClickBench — 分析型 DBMS 基准测试](https://benchmark.clickhouse.com/) 中表现出色。由于其具备超快速的向量化执行引擎，它也可以用作快速向量数据库。

## 安装与设置


<CodeGroup>
```bash pip
pip install pymysql
```

```bash uv
uv add pymysql
```
</CodeGroup>
