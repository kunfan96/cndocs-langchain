[AgentQL](https://www.agentql.com/) 可通过 [AgentQL 查询语言](https://docs.agentql.com/agentql-query)或自然语言提示，从任意网页中进行网页交互和结构化数据提取。AgentQL 支持多种语言和网页，且不会因时间推移或页面变化而失效。

## 安装与设置

安装集成包：

<CodeGroup>
```bash pip
pip install langchain-agentql
```

```bash uv
uv add langchain-agentql
```
</CodeGroup>

## API 密钥

从我们的 [开发者门户](https://dev.agentql.com/) 获取 API 密钥，并将其添加到环境变量中：
```
export AGENTQL_API_KEY="your-api-key-here"
```

## 文档加载器
AgentQL 的文档加载器可通过 AgentQL 查询从任意网页中提取结构化数据。

```python
from langchain_agentql.document_loaders import AgentQLLoader
```
查看我们的[文档加载器文档及使用示例](/oss/python/integrations/document_loaders/agentql)。

## 工具与工具包
AgentQL 工具可通过 AgentQL 查询或自然语言提示，在任意网页上进行交互和结构化数据提取。

```python
from langchain_agentql.tools import ExtractWebDataTool, ExtractWebDataBrowserTool, GetWebElementBrowserTool
from langchain_agentql import AgentQLBrowserToolkit
```
查看我们的[工具文档及使用示例](/oss/python/integrations/tools/agentql)。
