>[Confluence](https://www.atlassian.com/software/confluence) 是一个用于保存和组织所有项目相关材料的 Wiki 协作平台。`Confluence` 是一个主要处理内容管理活动的知识库。


## 安装与设置

<CodeGroup>
```bash pip
pip install atlassian-python-api
```

```bash uv
uv add atlassian-python-api
```
</CodeGroup>

我们需要配置 `username/api_key` 或 `OAuth2 登录`。
请参阅[说明文档](https://support.atlassian.com/atlassian-account/docs/manage-api-tokens-for-your-atlassian-account/)。
