---

基于字符的分割是最简单的文本分割方法。它使用指定的字符序列（默认为 `"\n\n"`）来分割文本，块长度以字符数衡量。

**关键点**：
1. **文本如何分割**：通过给定的字符分隔符。
2. **块大小如何衡量**：通过字符计数。

您可以选择以下方法：
- `.split_text` — 返回纯字符串块。
- `.create_documents` — 返回 LangChain [Document](https://reference.langchain.com/python/langchain-core/documents/base/Document) 对象，当需要为下游任务保留元数据时非常有用。

```python
from langchain_text_splitters import CharacterTextSplitter

# 加载示例文档
with open("state_of_the_union.txt") as f:
    state_of_the_union = f.read()

text_splitter = CharacterTextSplitter(
    separator="\n\n",
    chunk_size=1000,
    chunk_overlap=200,
    length_function=len,
    is_separator_regex=False,
)
texts = text_splitter.create_documents([state_of_the_union])
print(texts[0])
```

使用 `.create_documents` 将每个文档关联的元数据传播到输出块：

```python
metadatas = [{"document": 1}, {"document": 2}]
documents = text_splitter.create_documents(
    [state_of_the_union, state_of_the_union], metadatas=metadatas
)
print(documents[0])
```
