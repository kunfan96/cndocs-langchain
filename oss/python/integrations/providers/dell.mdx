戴尔是一家全球性科技公司，提供各类硬件、软件及服务，其中包括人工智能解决方案。其人工智能产品组合包括专为人工智能工作负载打造的基础设施，其中包括针对人工智能数据管理优化的戴尔 PowerScale 存储系统。

## PowerScale

戴尔 [PowerScale](https://www.dell.com/en-us/shop/powerscale-family/sf/powerscale) 是一款企业级横向扩展存储系统，搭载行业领先的 OneFS 文件系统，支持本地部署或云端部署。

### 安装与配置

<CodeGroup>
```bash pip
pip install powerscale-rag-connector
```

```bash uv
uv add powerscale-rag-connector
```
</CodeGroup>
