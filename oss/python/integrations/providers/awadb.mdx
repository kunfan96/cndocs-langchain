>[AwaDB](https://github.com/awa-ai/awadb) 是一款 AI 原生数据库，用于搜索和存储 LLM 应用所使用的嵌入向量。

## 安装与设置

```bash
pip install awadb
```

## 向量存储

```python
from langchain_community.vectorstores import AwaDB
```

请参阅[使用示例](/oss/python/integrations/vectorstores/awadb)。

## 嵌入模型

```python
from langchain_community.embeddings import AwaEmbeddings
```

请参阅[使用示例](/oss/python/integrations/embeddings/awadb)。
