> [DashVector](https://help.aliyun.com/document_detail/2510225.html) 是一项全托管的向量数据库服务，支持高维稠密向量和稀疏向量、实时写入与过滤搜索。它具备自动扩展能力，可适应不同的应用需求。

本文档演示了如何在 LangChain 生态系统中利用 DashVector。具体而言，它展示了如何安装 DashVector，以及如何将其作为 LangChain 中的 VectorStore 插件使用。内容分为两部分：安装与设置，以及针对特定 DashVector 包装器的参考说明。

## 安装与设置


安装 Python SDK：

<CodeGroup>
```bash pip
pip install dashvector
```

```bash uv
uv add dashvector
```
</CodeGroup>

您需要拥有一个 API Key。以下是[安装指南](https://help.aliyun.com/document_detail/2510223.html)。


## 嵌入模型

```python
from langchain_community.embeddings import DashScopeEmbeddings
```

请参阅[使用示例](/oss/python/integrations/vectorstores/dashvector)。


## 向量存储

DashVector Collection 被封装为熟悉的 VectorStore，以便在 LangChain 中原生使用，
使其能够轻松应用于各种场景，例如语义搜索或示例选择。

您可以通过以下方式导入该向量存储：
```python
from langchain_community.vectorstores import DashVector
```

如需详细了解 DashVector 包装器，请参阅[此笔记本](/oss/python/integrations/vectorstores/dashvector)
