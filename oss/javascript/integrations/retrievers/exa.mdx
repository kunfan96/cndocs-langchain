## 概述

[Exa](https://exa.ai/) 是一个搜索引擎，可根据输入查询从网络检索相关内容。

本指南将帮助您开始使用 `ExaRetriever` [检索](/oss/javascript/langchain/retrieval)。有关所有 `ExaRetriever` 功能和配置的详细文档，请前往 [API 参考](https://reference.langchain.com/javascript/langchain-exa/ExaRetriever)。

### 集成详情

| 检索器 | 来源 | 包 |
| :--- | :--- | :---: |
[ExaRetriever](https://reference.langchain.com/javascript/langchain-exa/ExaRetriever) | 网络上的信息。 | [`@langchain/exa`](https://www.npmjs.com/package/@langchain/exa) |

## 设置

您需要将 API 密钥设置为环境变量。

`Exa` 类在搜索您的 API 密钥时默认使用 `EXASEARCH_API_KEY`。

```typescript
process.env.EXASEARCH_API_KEY="<您的 API 密钥>";
```

如果您想从单个查询获取自动跟踪，也可以通过取消注释以下内容来设置您的 [LangSmith](/langsmith/home) API 密钥：

```typescript
// process.env.LANGSMITH_API_KEY = "<您的 API 密钥>";
// process.env.LANGSMITH_TRACING = "true";
```

### 安装

此检索器位于 `@langchain/exa` 包中：

<CodeGroup>
```bash npm
npm install @langchain/exa @langchain/core
```
```bash yarn
yarn add @langchain/exa @langchain/core
```
```bash pnpm
pnpm add @langchain/exa @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化我们的检索器：

```typescript
import { ExaRetriever } from "@langchain/exa";
import Exa from "exa-js";

const retriever = new ExaRetriever({
  // @lc-ts-ignore
  client: new Exa(
    process.env.EXASEARCH_API_KEY // 默认 API 密钥
  ),
  searchArgs: {
    numResults: 2,
  }
});
```

## 用法

```typescript
const query = "演讲者在 2022 年国情咨文中关于布雷耶大法官说了什么？";

await retriever.invoke(query);
```
