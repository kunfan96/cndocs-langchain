Oracle AI Database 专为 AI 工作负载而构建，支持按**含义**（语义）而非仅关键词查询数据。它将**对非结构化内容的语义搜索**与**对业务数据的关系过滤**整合在同一系统中——因此，您可以构建 AI 检索工作流（如 RAG），而无需引入独立的向量数据库，也避免了数据在多个平台间的碎片化。

### 为什么重要

- **AI 与业务数据一体化：** 向量与操作数据共存。
- **减少碎片化：** 相比独立的向量存储，组件更少。
- **数据库级保障：** 将安全性、事务、扩展性和可用性应用于同一 AI 工作负载。

## 前提条件

安装 `langchain-oracledb`。`python-oracledb` 驱动将作为依赖项自动安装。

<CodeGroup>
```bash pip
pip install -qU langchain-oracledb
```
```bash uv
uv add langchain-oracledb
```
</CodeGroup>
