>[ElevenLabs](https://elevenlabs.io/about) 是一家语音 AI 研究和部署公司，
> 其使命是让内容以任何语言和声音都能被普遍访问。
>
>`ElevenLabs` 创造了最逼真、最多样化且最具情境感知能力的
> AI 音频，能够用 29 种语言的数百种新旧声音生成语音。

## 安装与设置

首先，你需要注册一个 ElevenLabs 账号。你可以按照
[此处的说明](https://docs.elevenlabs.io/welcome/introduction) 进行操作。

安装 Python 包：

<CodeGroup>
```bash pip
pip install elevenlabs
```

```bash uv
uv add elevenlabs
```
</CodeGroup>

## 工具

查看[使用示例](/oss/python/integrations/tools/eleven_labs_tts)。

```python
from langchain_community.tools import ElevenLabsText2SpeechTool
```
