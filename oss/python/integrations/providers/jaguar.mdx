本页介绍如何在 LangChain 中使用 Jaguar 向量数据库。
内容分三个部分：简介、安装与设置，以及 Jaguar API。


## 简介

Jaguar 向量数据库具有以下特点：

1. 分布式向量数据库
2. JaguarDB 的"ZeroMove"特性支持即时水平扩展
3. 多模态：嵌入、文本、图像、视频、PDF、音频、时序数据和地理空间数据
4. 全主节点架构：支持并行读写
5. 异常检测能力
6. RAG 支持：将 LLM 与专有数据和实时数据结合
7. 共享元数据：跨多个向量索引共享元数据
8. 距离度量：欧氏距离、余弦相似度、内积、曼哈顿距离、切比雪夫距离、汉明距离、Jaccard 距离、闵可夫斯基距离

[Jaguar 可扩展向量数据库概览](http://www.jaguardb.com)

您可以在 Docker 容器中运行 JaguarDB，也可以下载软件并在云端或本地运行。

## 安装与设置

- 在一台或多台主机上安装 JaguarDB
- 在一台主机上安装 Jaguar HTTP 网关服务器
- 安装 JaguarDB HTTP 客户端包

详细步骤请参阅 [Jaguar 文档](http://www.jaguardb.com/support.html)

客户端程序中的环境变量：

        export OPENAI_API_KEY="......"
        export JAGUAR_API_KEY="......"


## Jaguar API

配合 LangChain，可通过 Python 导入 Jaguar 客户端类：

```python
from langchain_community.vectorstores.jaguar import Jaguar
```

Jaguar 类支持的 API 函数：

- `add_texts`
- `add_documents`
- `from_texts`
- `from_documents`
- `similarity_search`
- `is_anomalous`
- `create`
- `delete`
- `clear`
- `drop`
- `login`
- `logout`


更多 Jaguar API 详情，请参阅[此 notebook](/oss/python/integrations/vectorstores/jaguar)
