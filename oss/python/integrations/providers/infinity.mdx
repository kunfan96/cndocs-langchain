>[Infinity](https://github.com/michaelfeil/infinity) 支持创建文本 Embedding。

## 文本 Embedding 模型

存在一个 Infinity Embedding 模型，您可以通过以下方式访问它：
```python
from langchain_community.embeddings import InfinityEmbeddings
```
有关此内容的更详细教程，请参阅[此笔记本](/oss/python/integrations/embeddings/infinity)
