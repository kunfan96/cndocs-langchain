>[PipelineAI](https://pipeline.ai) 允许您在云端大规模运行您的 ML 模型。它还提供了对[多个 LLM 模型](https://pipeline.ai)的 API 访问权限。

本笔记本将介绍如何在 [PipelineAI](https://docs.pipeline.ai/docs) 中使用 LangChain。

## PipelineAI 示例

[此示例展示了 PipelineAI 如何与 LangChain 集成](https://docs.pipeline.ai/docs/langchain)，由 PipelineAI 创建。

## 设置

使用 `PipelineAI` API（即 `Pipeline Cloud`）需要安装 `pipeline-ai` 库。请使用 `pip install pipeline-ai` 安装 `pipeline-ai`。

```python
# 安装该包
pip install -qU  pipeline-ai
```

## 示例

### 导入模块

```python
import os

from langchain_community.llms import PipelineAI
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import PromptTemplate
```

### 设置环境变量 API Key

请确保从 PipelineAI 获取您的 API Key。查看[云快速入门指南](https://docs.pipeline.ai/docs/cloud-quickstart)。您将获得为期 30 天的免费试用，包含 10 小时的无服务器 GPU 计算资源，用于测试不同的模型。

```python
os.environ["PIPELINE_API_KEY"] = "YOUR_API_KEY_HERE"
```

## 创建 PipelineAI 实例

在实例化 PipelineAI 时，您需要指定要使用的 pipeline 的 ID 或标签，例如 `pipeline_key = "public/gpt-j:base"`。随后，您可以选择传递其他特定于 pipeline 的关键字参数：

```python
llm = PipelineAI(pipeline_key="YOUR_PIPELINE_KEY", pipeline_kwargs={...})
```

### 创建 Prompt 模板

我们将为问答（Question and Answer）创建一个 prompt 模板。

```python
template = """Question: {question}

Answer: Let's think step by step."""

prompt = PromptTemplate.from_template(template)
```

### 初始化 LLMChain

```python
llm_chain = prompt | llm | StrOutputParser()
```

### 运行 LLMChain

提供一个问题并运行 LLMChain。

```python
question = "What NFL team won the Super Bowl in the year Justin Beiber was born?"

llm_chain.invoke(question)
```
