>[Discord](https://discord.com/) 是一个 VoIP 和即时通讯社交平台。用户可以通过私聊或名为"服务器"的社区进行语音通话、视频通话、文字聊天、媒体和文件分享。服务器是一系列可以通过邀请链接访问的持久聊天室和语音频道的集合。

## 安装与设置

<CodeGroup>
```bash pip
pip install pandas
```

```bash uv
uv add pandas
```
</CodeGroup>

按照以下步骤下载您的 `Discord` 数据：

1. 前往**用户设置**
2. 然后前往**隐私与安全**
3. 前往**请求我的所有数据**，点击**请求数据**按钮

您可能需要 30 天才能收到您的数据。您将在注册 Discord 时使用的地址收到一封电子邮件，其中包含一个下载链接，您可以通过该链接下载个人 Discord 数据。
