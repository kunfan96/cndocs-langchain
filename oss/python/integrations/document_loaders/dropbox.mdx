[Dropbox](https://en.wikipedia.org/wiki/Dropbox) 是一项文件托管服务，将传统文件、云端内容和网络快捷方式汇聚在一处。

本 notebook 介绍如何从 *Dropbox* 加载文档。除了文本和 PDF 等常见文件外，它还支持 *Dropbox Paper* 文件。

## 前提条件

1. 创建一个 Dropbox 应用。
2. 为应用授予以下作用域权限：`files.metadata.read` 和 `files.content.read`。
3. 生成访问令牌：[www.dropbox.com/developers/apps/create](https://www.dropbox.com/developers/apps/create)。
4. `pip install dropbox`（处理 PDF 文件类型需要 `pip install "unstructured[pdf]"`）。

## 使用说明

`DropboxLoader`` 需要您创建一个 Dropbox 应用并生成访问令牌。可通过 [www.dropbox.com/developers/apps/create](https://www.dropbox.com/developers/apps/create) 完成。您还需要安装 Dropbox Python SDK（pip install dropbox）。

DropboxLoader 可以从 Dropbox 文件路径列表或单个 Dropbox 文件夹路径加载数据。两种路径都应相对于与访问令牌关联的 Dropbox 账户的根目录。

```python
pip install dropbox
```

```python
from langchain_community.document_loaders import DropboxLoader
```

```python
# Generate access token: https://www.dropbox.com/developers/apps/create.
dropbox_access_token = "<DROPBOX_ACCESS_TOKEN>"
# Dropbox root folder
dropbox_folder_path = ""
```

```python
loader = DropboxLoader(
    dropbox_access_token=dropbox_access_token,
    dropbox_folder_path=dropbox_folder_path,
    recursive=False,
)
```

```python
documents = loader.load()
```

```text
File /JHSfLKn0.jpeg could not be decoded as text. Skipping.
File /A REPORT ON WILES’ CAMBRIDGE LECTURES.pdf could not be decoded as text. Skipping.
```

```python
for document in documents:
    print(document)
```
