> [any-llm](https://github.com/mozilla-ai/any-llm) 是一个统一接口，支持调用 OpenAI、Anthropic、Google、本地模型（通过 Ollama/LocalAI）以及[更多](https://mozilla-ai.github.io/any-llm/providers/)提供商。只需更改一个字符串即可切换提供商。

## 安装与设置

<CodeGroup>
```bash pip
pip install langchain-anyllm
```

```bash uv
uv add langchain-anyllm
```
</CodeGroup>

您需要为所选提供商准备相应的 API 密钥。API 密钥可通过 `api_key` 参数传入，也可以设置为环境变量（例如 `OPENAI_API_KEY`、`ANTHROPIC_API_KEY`）。各提供商的具体要求请参阅 [any-llm 文档](https://mozilla-ai.github.io/any-llm/providers/)。

## 聊天模型

```python
from langchain_anyllm import ChatAnyLLM
```
