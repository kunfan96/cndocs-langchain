>[Apple Inc.（维基百科）](https://en.wikipedia.org/wiki/Apple_Inc.) 是一家美国跨国公司和科技公司。
>
> [iMessage（维基百科）](https://en.wikipedia.org/wiki/IMessage) 是由 Apple Inc. 开发并于 2011 年推出的即时通讯服务。
> `iMessage` 仅在 Apple 平台上运行。

## 安装与设置

查看[设置说明](/oss/python/integrations/chat_loaders/imessage)。

## 聊天加载器

从 `iMessage` 的 `chat.db` SQLite 文件中加载聊天会话。

查看[使用示例](/oss/python/integrations/chat_loaders/imessage)。

```python
from langchain_community.chat_loaders.imessage import IMessageChatLoader
```
