>[Airbyte](https://github.com/airbytehq/airbyte) 是一个用于构建从 API、数据库和文件到数据仓库及数据湖的 ELT 流水线的数据集成平台。它拥有最大规模的 ELT 连接器目录，涵盖各类数据仓库和数据库。

## 安装与设置

<CodeGroup>
```bash pip
pip install -U langchain-airbyte
```

```bash uv
uv add langchain-airbyte
```
</CodeGroup>

<Note>
**目前，`langchain-airbyte` 库不支持 Pydantic v2。**

请降级到 Pydantic v1 以使用此包。

此包目前还需要 Python 3.10 及以上版本。

</Note>

集成包不需要设置任何全局环境变量，但某些集成（例如 `source-github`）可能需要传入凭据。
