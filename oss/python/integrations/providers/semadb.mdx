>[SemaDB](https://semafind.com/) 是一个无繁琐的向量相似性搜索引擎。它提供了一个低成本的云托管版本，帮助您轻松构建 AI 应用程序。

使用 SemaDB Cloud（我们的托管版本），无繁琐意味着无需计算 Pod 大小、无需模式定义、无需分区设置、无需参数调优、无需搜索算法调优、无需复杂安装、无需复杂 API。它与 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb) 集成，提供透明计费、自动分片和交互式 API 演练场。

## 安装

无需安装，直接通过 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb) 开始使用 SemaDB Cloud。

## 向量存储

有一个围绕 `SemaDB` 集合的基本包装器，允许您将其用作向量存储。

```python
from langchain_community.vectorstores import SemaDB
```

您可以按照[此笔记本](/oss/python/integrations/vectorstores/semadb)中的教程了解如何使用该包装器。
