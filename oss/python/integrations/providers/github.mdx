>[GitHub](https://github.com/) 是一个开发者平台，允许开发者创建、
> 存储、管理和共享代码。它使用 `Git` 软件，提供 Git 的分布式版本控制以及访问控制、缺陷跟踪、
> 软件功能请求、任务管理、持续集成，并为每个项目提供 Wiki。


## 安装与配置

要访问 GitHub API，您需要一个 [个人访问令牌](https://github.com/settings/tokens)。


## 文档加载器

GitHub 目前提供两种文档加载器。

请参阅 [使用示例](/oss/python/integrations/document_loaders/github)。

```python
from langchain_community.document_loaders import GitHubIssuesLoader, GithubFileLoader
```

## 工具/工具包

### GitHubToolkit
`GitHub` 工具包包含一系列工具，使 LLM 代理能够与 GitHub 仓库进行交互。

该工具包是对 `PyGitHub` 库的封装。

```python
from langchain_community.agent_toolkits.github.toolkit import GitHubToolkit
```

更多详情请参阅 [示例笔记本](/oss/python/integrations/tools/github)。

### GitHubAction

用于与 GitHub API 交互的工具。

```python
from langchain_community.tools.github.tool import GitHubAction
```
