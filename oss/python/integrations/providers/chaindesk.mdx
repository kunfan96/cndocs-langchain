>[Chaindesk](https://chaindesk.ai) 是一个[开源](https://github.com/gmpetrov/databerry)文档检索平台，旨在帮助您将个人数据与大语言模型连接起来。


## 安装与设置

我们需要注册 Chaindesk，创建数据存储库，添加一些数据，并获取您的数据存储库 API 端点 URL。
我们需要 [API 密钥](https://docs.chaindesk.ai/api-reference/authentication)。

## 检索器

请参阅[使用示例](/oss/python/integrations/retrievers/chaindesk)。
