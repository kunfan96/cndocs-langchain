[Spider](https://spider.cloud/) 是 [最快](https://github.com/spider-rs/spider/blob/main/benches/BENCHMARKS.md) 且最经济的爬虫和抓取工具，可返回适用于 LLM 的数据。

## 设置

```python
pip install spider-client
```

## 用法

要使用 spider，您需要从 [spider.cloud](https://spider.cloud/) 获取 API 密钥。

```python
from langchain_community.document_loaders import SpiderLoader

loader = SpiderLoader(
    api_key="YOUR_API_KEY",
    url="https://spider.cloud",
    mode="scrape",  # 如果未提供 API 密钥，它将在环境中查找 SPIDER_API_KEY
)

data = loader.load()
print(data)
```

## 模式

- `scrape`：默认模式，抓取单个 URL
- `crawl`：爬取提供的域名 URL 的所有子页面

## 爬虫选项

`params` 参数是一个字典，可以传递给加载器。请参阅 [Spider 文档](https://spider.cloud/docs/api) 查看所有可用参数。
