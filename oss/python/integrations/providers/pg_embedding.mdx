> [pg_embedding](https://github.com/neondatabase/pg_embedding) 是一个用于
> 使用 `Postgres` 和 `Hierarchical Navigable Small Worlds`
> 算法进行向量相似性搜索的近似最近邻搜索开源包。

## 安装与设置

我们需要安装几个 Python 包。

<CodeGroup>
```bash pip
pip install psycopg2-binary
```

```bash uv
uv add psycopg2-binary
```
</CodeGroup>

## 向量存储

参见 [使用示例](/oss/python/integrations/vectorstores/pgembedding)。

```python
from langchain_community.vectorstores import PGEmbedding
```
