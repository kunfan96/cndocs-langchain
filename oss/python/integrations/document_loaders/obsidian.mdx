>[Obsidian](https://obsidian.md/) 是一个强大且可扩展的知识库，它基于您本地的纯文本文件夹工作。

本笔记本介绍如何从 `Obsidian` 数据库加载文档。

由于 `Obsidian` 仅作为 Markdown 文件夹存储在磁盘上，加载器只需获取此目录的路径。

`Obsidian` 文件有时也包含[元数据](https://help.obsidian.md/Editing+and+formatting/Metadata)，这是文件顶部的 YAML 块。这些值将被添加到文档的元数据中。（也可以向 `ObsidianLoader` 传递 `collect_metadata=False` 参数来禁用此行为。）

```python
from langchain_community.document_loaders import ObsidianLoader
```

```python
loader = ObsidianLoader("<path-to-obsidian>")
```

```python
docs = loader.load()
```
