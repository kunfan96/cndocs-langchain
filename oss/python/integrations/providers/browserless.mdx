> [Browserless](https://www.browserless.io/docs/start) 是一项允许您
> 在云端运行无头 Chrome 实例的服务。这是在无需担心管理自身基础设施的情况下，大规模运行基于浏览器的
> 自动化任务的绝佳方式。

## 安装与设置

请从 [Browserless 定价页面](https://www.browserless.io/pricing/) 获取 API 密钥。
