>[UpTrain](https://uptrain.ai/) 是一个开源统一平台，用于评估和
>改进生成式 AI 应用程序。它为 20 多个预配置的评估提供评分
>（涵盖语言、代码、嵌入用例），对失败案例进行根本原因分析，
>并提供关于如何解决这些问题的见解。

## 安装和设置

<CodeGroup>
```bash pip
pip install uptrain
```

```bash uv
uv add uptrain
```
</CodeGroup>

## 回调

```python
from langchain_community.callbacks.uptrain_callback import UpTrainCallbackHandler
```

请参阅[示例](/oss/python/integrations/callbacks/uptrain)。
