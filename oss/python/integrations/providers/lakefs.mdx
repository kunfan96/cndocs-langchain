>[lakeFS](https://docs.lakefs.io/) 为数据湖提供可扩展的版本控制，
> 并使用类似 Git 的语义来创建和访问这些版本。

## 安装与设置

获取 `ENDPOINT`、`LAKEFS_ACCESS_KEY` 和 `LAKEFS_SECRET_KEY`。
您可以在[这里](https://docs.lakefs.io/quickstart/launch.html)找到安装说明。


## 文档加载器

查看[使用示例](/oss/python/integrations/document_loaders/lakefs)。

