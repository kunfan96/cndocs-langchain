[ClickSend](https://www.clicksend.com/) 是一个基于云的通信平台，使开发者能够通过统一的 API 发送短信、电子邮件、语音、传真和邮政消息。

## 概述

### 集成详情

| 类 | 包 | 可序列化 | [Python 支持](/oss/javascript/integrations/tools/clicksend) | 版本 |
|:---|:---|:---:|:---:|:---:|
| `ClickSendTools` | [@clicksend/langchain-clicksend-mcp](https://www.npmjs.com/package/@clicksend/langchain-clicksend-mcp) | ❌ | ❌ | ![npm version](https://img.shields.io/npm/v/@clicksend/langchain-clicksend-mcp?style=flat-square&label=%20) |

### 工具特性

- **短信消息**：向全球移动号码发送文本消息
- **电子邮件**：发送事务性和营销电子邮件
- **语音**：发送自动语音消息和呼叫
- **MCP 集成**：通过模型上下文协议连接，实现无缝代理集成

## 设置

集成位于 `@clicksend/langchain-clicksend-mcp` 包中。

```bash
npm install @clicksend/langchain-clicksend-mcp
```

### 凭据

您需要一个具有 API 凭据的 ClickSend 账户。在 [clicksend.com](https://www.clicksend.com/) 注册以获取您的用户名和 API 密钥。

```typescript 设置凭据
import dotenv from 'dotenv';
dotenv.config();
```

## 实例化

通过连接到 ClickSend MCP 服务器来初始化 ClickSend 工具：

```typescript 初始化 ClickSend
import { ClickSendTools } from "@clicksend/langchain-clicksend-mcp";

const tools = await ClickSendTools.fromHTTP({
  url: "https://mcp.clicksend.com",
  username: process.env.CLICKSEND_USERNAME,
  apiKey: process.env.CLICKSEND_API_KEY,
});

console.log(`Loaded ${tools.length} ClickSend tools`);
```

## 调用

在您的 LangChain 代理中使用这些工具来发送通信：

```typescript 与代理一起使用
import { ChatOpenAI } from "@langchain/openai";
import { createAgent } from "langchain";

// 使用工具创建代理
const agent = await createAgent({
  model: new ChatOpenAI({ 
    model: "gpt-4o-mini",
    temperature: 0 
  }),
  tools,
});

// 使用代理
const result = await agent.invoke({
  messages: [
    { role: "user", content: "向 +1234567890 发送一条短信，内容为 'Hello from LangChain!'" },
  ],
});

console.log(result);
```
