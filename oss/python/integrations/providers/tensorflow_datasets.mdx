>[TensorFlow Datasets](https://www.tensorflow.org/datasets) 是一个可直接使用的数据集集合，
> 适用于 TensorFlow 或其他 Python ML 框架（如 Jax）。所有数据集均以
> [tf.data.Datasets](https://www.tensorflow.org/api_docs/python/tf/data/Dataset) 形式暴露，
> 支持易用、高性能的输入管道。请参阅
> [指南](https://www.tensorflow.org/datasets/overview) 和
> [数据集列表](https://www.tensorflow.org/datasets/catalog/overview#all_datasets) 以开始使用。



## 安装与设置

您需要安装 `tensorflow` 和 `tensorflow-datasets` Python 包。

<CodeGroup>
```bash pip
pip install tensorflow
```

```bash uv
uv add tensorflow
```
</CodeGroup>

<CodeGroup>
```bash pip
pip install tensorflow-dataset
```

```bash uv
uv add tensorflow-dataset
```
</CodeGroup>
