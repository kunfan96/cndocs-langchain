## 概述

`RedisStore` 是 `ByteStore` 的一个实现，它将所有内容存储在您的 Redis 实例中。

## 设置

要创建 Redis 字节存储，您需要设置一个 Redis 实例。您可以在本地设置，也可以通过提供商设置 - 请参阅我们的 [Redis 指南](/oss/python/integrations/providers/redis) 了解选项概述。

### 安装

LangChain `RedisStore` 集成位于 `langchain-community` 包中：

```python
pip install -qU langchain-community redis
```

## 实例化

现在我们可以实例化我们的字节存储：

```python
from langchain_community.storage import RedisStore

kv_store = RedisStore(redis_url="redis://localhost:6379")
```

## 用法

您可以使用 `mset` 方法在键下设置数据，如下所示：

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[b'value1', b'value2']
```

并且您可以使用 `mdelete` 方法删除数据：

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```text
[None, None]
```
