>[Milvus](https://milvus.io/docs/overview.md) 是一款存储、索引和管理深度神经网络及其他机器学习（ML）模型生成的海量嵌入向量的数据库。


## 安装与设置

安装 Python SDK：

<CodeGroup>
```bash pip
pip install langchain-milvus
```

```bash uv
uv add langchain-milvus
```
</CodeGroup>

## 向量存储

查看[使用示例](/oss/python/integrations/vectorstores/milvus)。

导入该向量存储：
```python
from langchain_milvus import Milvus
```
